Talk Details:-
Injection vulnerabilities have been listed as the top item in the OWASP Top
10 since 2010 and this is not by mistake.  Injection vulnerabilities are
fairly easy to overlook and many times just as easy to exploit.  More times
than not, what seems like a simple injection may be the key to a very
successful devastating attack vector.  Join us in a hands-on workshop
getting to know some of the more common types of injections and how you can
find and exploit these vulnerabilities using the Mutillidae vulnerability
training web app. Exploit vulnerabilities like SQL Injection. Use tools
like SQL map and Havij. Session will be good for Infosec beginners and
Intermediates.
