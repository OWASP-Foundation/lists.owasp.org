Hi All,
we will have an hands on event on OWASP nettacker tool. Please find the
details below and kindly fill the form to confirm attendance.

Date:- June 20th, 6 pm to 8 pm.
Venue:-   New Relic,  188 Spear Street in San Francisco
Registration Form :- https://goo.gl/forms/qfBw6yLXDdctI5FQ2

Description:
This session is aimed at introducing OWASP Nettacker project at the people
who work as security engineers, penetration testers and sysadmins, this
project is created to automate information gathering, vulnerability
scanning and eventually generating a report for networks, including
services, bugs, vulnerabilities, misconfigurations, and other information.
Despite this framework is a new starter, we will explain how to use and
develop the project by a live demonstrating. after this session you will be
able to develop new modules of your own idea and monitor the networks in
the way you want, in addition, we will show you how to use API and WebUI to
get the best use of it.

Session steps,

1- Introductions
2- Authors
3- Why OWASP Nettacker?
4- Features
5- Usage
6- API & WebUI
7- How to Contribute?
7.1- Roadmap
7.2- Media
7.3- Language Library
7.4- Payloads
7.5- Modules
7.5.1- Modules Structure - Framework
7.5.2- Add a New Module
8- Questions

Requirements,
1- Python scripting knowledge
2- Laptop
3- Installed Python & PyPI


About the speaker
Vahid Behzadan is a freelance security consultant and PhD candidate at
Kansas State University. His professional portfolio spans over 15 years of
experience in numerous offensive and defensive security projects focused on
web applications, networks, RF communications, unmanned aerial vehicles,
distributed systems, counter-terrorism, and artificial intelligence. His
research interests include safety and security of intelligent systems,
security of complex adaptive systems, game theory, multi-agent systems, and
the economics of threat intelligence. He is currently affiliated with the
ISCAAS lab and the AI Safety Research Initiative at K-State, as well as the
Offsec Research group.
