Hi All,I am happy to announce our first OWASP Bay area hands-on meet up - Hacker Thursdays. Looking forward for your participation.
Topic - Introduction to Application Security and OWASP Top10 with Webgoat
Have you ever wanted to understand how white hat hackers identify vulnerabilities within web applications? Learn web application penetration basics using web goat and burp proxy.  Specifically learn how to set-up a local vulnerable web app using OWASP’s WebGoat and test for specific vulnerabilities like Cross-Site Scripting (XSS), SQL Injection (SQLi) and Business Logic Flaws. 
TrainerTy Sbano, Director at Lending Club
Hardware/Software requirementsLaptop with Docker Installed, nothing major for requirement beyond having a modern computing device.
