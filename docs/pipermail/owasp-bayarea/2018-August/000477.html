> We are glad to announce an OWASP hands-on session on "Active Directory

> presentation.  Please fill the below form to confirm registration.
>
> Date:- August 15th,  6 pm to 8 pm.
> Venue:- Spirent , 2708 Orchard Pkwy #20, San Jose, CA 95134
> Form:- https://goo.gl/forms/IsPyN1KFRcsvpaGs2
>
> Description:
> This will be a completely hands on session on getting started with
> reconnaissance of Windows environments. We will commence with basics of
> active directory, setting up a Domain Environment, and performing
> reconnaissance/situational awareness.
>
> At a bare minimum, the following topics will be covered:
> 1. Basics of Active Directory
> 2. Setting up a simple Windows domain environment
> 3. Unauthenticated reconnaissance
> 4. Authenticated reconnaissance as a normal/unprivileged user
> 5. Using ADRecon tool(https://github.com/sense-of-security/ADRecon)
>
>
> What to bring if you would like to try things
> 1. Get your laptop
> 2. Two virtual machines, Windows Server 2016 and Windows 10. Evaluation
> copies are available at: https://www.microsoft.com/en-us/evalcenter/
