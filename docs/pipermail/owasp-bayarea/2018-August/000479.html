Hi All,
Happy to announce an OWASP Hacker Thursdays event on topic "DevSecOps the
Kubernetes Way". We will utilize 5.30 to 6 pm for food and networking,
session will start from 6 pm. Drinks and pizza will be served at the
meet.Please fill below form to confirm attendance.


Date:- August 30th, 2018. 5.30 to 8 pm
Venue:- Segment, 100 California St, Suite 700, San Francisco, CA,
Form:- https://goo.gl/forms/YSPaArGGtZlT6rcr1

Talk Description:
The Cloud as we know it is changing. Containers have taken the center stage
as the preferred method of developing and deploying software into
production. As security practitioners, we must adapt to the latest
technologies or be left in the dust.
This workshop is a mix of lecture and labs and focuses on the ins and outs
of building a Kubernetes infrastructure capable of taking containers from a
developer’s laptop to production, in a secure manner.
The hands-on portion of the workshop will rely on Kubernetes for the
deployment and orchestration of Docker containers. Each student will build
a sandbox Kubernetes cluster from scratch on their local machines using
Goole Kubernetes Engine. We will cover the following topics throughout the
workshop:

    Fast and Furious Introduction to Kubernetes
    Common Kubernetes Security Issues and Remediations
    Kubernetes Authentication and Authorization
    Logging in Kubernetes
    Building an Automated Security Testing Pipeline in Kubernetes

Plenty of time will be left for Q&A and open discussion.

Student Requirements
Familiarity with at least one public cloud provider is recommend. Students
should also have basic Docker knowledge and experience launching and
managing basic cloud instances. Basic command line and scripting skills are
highly recommended. No Kubernetes experience is required.

Laptop Requirements
Any laptop with a modern web browser to access Google Cloud Platform. GCP
credentials will be provided.

About the Speaker
Jimmy Mesta is a veteran application security leader that has spent time on
both the offense and defense side of the industry. His core focus has been
in application and cloud security with an emphasis on secure architecture,
automated testing, developer training and modern defensive techniques.
He is the chapter leader of OWASP Santa Barbara and co-organizes the AppSec
California and LocoMoco security conferences. He is a frequent speaker and
trainer at conferences such as AppSec USA and BSides.

Again, please fill below form to confirm attendance.

Form:- https://goo.gl/forms/YSPaArGGtZlT6rcr1
