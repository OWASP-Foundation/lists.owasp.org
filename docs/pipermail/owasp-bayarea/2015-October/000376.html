If you want to warn on the use of an old SSL protocol without having to 
actually support it, there are ways to pull that off. You can serve 
icons from different subdomains, each configured to support only a 
single protocol.  From the main site (running only “secure” protocols), 
javascript can try to load a 1x1 image from each address.  Based on the 
errors from each request, it’s possible to detect & warn about a 
particular protocol without serving anything of importance on it.
