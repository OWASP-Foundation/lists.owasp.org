6:30 a 7:15 - Integración de Biométricos con Aplicaciones Web - Alejandro

Azteca

7:15 a 8:15 - SQL Injection Avanzado - Víctor Chapela - Sm4rt Security

8:15 a 9:00 - Intercambio de ideas y discusión acerca de temas de interés
