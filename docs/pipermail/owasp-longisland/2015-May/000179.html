OWASP LI Specialists!

We are excited to announce Gabriel Gumbs will be presenting Part 2 of the
OWASP Application Security Program Quick Start Guide during our June 23,
2015 chapter meeting. This guide is intended to be a short, straightforward
introductory guide to start or improve an Application Security Program. The
Quick Start Guide is a true OWASP project and it can be found at
https://www.owasp.org/index.php/OWASP_Application_Security_Program_Quick_Start_Guide_Project
.

Please join us at 6:30 PM for networking opportunities prior to the
presentation.

Food and drink provided by Secure Decisions.

Details:

When:
Tuesday, June 23rd, 2015 @ 6:30 PM

Where:
Huntington LaunchPad / (2nd Floor)
315 Main St
Huntington, NY 11743

Full meeting details:
http://www.meetup.com/OWASP-Long-Island-Meetup/events/221033379/


We look forward to seeing everyone!
