OWASP LI Specialists,

Let's get our hands dirty and give back to the OWASP Foundation. Come
out next month for the OWASP Code Review Guide Hands-On Workshop led
by Kenneth F. Belva.

Through his personal experience at OWASP AppSecUSA 2015, Ken will
explain how he became involved in the OWASP Code Review project and
why you should too. He will talk about his conference experiences with
the Code Review working group and help to guide the participants to a
fun and successful experience contributing to the guide. Be prepared
to learn and to teach others in a grassroots manner that defines the
OWASP experience.

Non-contributing participants are welcome to learn too. To contribute,
laptops are required otherwise laptops are highly recommended but
optional to learn. If you are interested in secure code at a high
level or are involved in the technical aspects of it this session is
for you.


When:
Wednesday, March 2nd, 2016 @ 6:30 PM

Where:
Swirbul Library | ** Room 100 **
1 South Avenue
Garden City, NY 11530

Space is limited for this event! Please only RSVP if you truly intend
to participate!
