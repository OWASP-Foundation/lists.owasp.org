OWASP LI Specialists,

We are excited to announce Mike Richter, senior Solutions Architect
from Microsoft will be presenting "Application security in the
Microsoft cloud" during our Wednesday, November 4, 2015 chapter
meeting.

Mike will provide an interactive forum with lots of live demos showing
you how Azure’s built-in security layers and Application Services
Environments offer protection, how to configure the Application
Settings and Web Application Firewalls for optimal security, and how
to configure OAuth and MFA with Azure Active Directory. Mike will also
talk about the Azure Marketplace, and how to make your application
available to the Azure community.

When:
Wednesday, November 4, 2015 @ 6:30 PM

Where:
LaunchPad LI (Huntington) | 2nd Floor
315 Main Street
Huntington, NY 11743

Full meeting details:
http://www.meetup.com/OWASP-Long-Island-Meetup/events/225204441/

We look forward to seeing everyone!
