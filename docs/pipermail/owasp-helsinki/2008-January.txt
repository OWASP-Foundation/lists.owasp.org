Position Description
 
Corporate Security is looking for experienced Senior Security
Specialist, Information Security to independently provide security
support for demanding internal customers globally in a networked working
environment.

Main responsibilities:  
* Creates understanding of customers' information security needs   
* Provides expertice and consultative support to customers on their
information security issues, improvement needs and security enhancement
projects (e.g. scoping projects, selecting controls, supporting controls
implementation, developing good practices)  
* Keeps business owner informed about ongoing security activities   
* Manages global and local information security development projects  
* Conducts global or local information security risk analysis and
technical security audits   
* Provides incident response services  
* Develops tools to support information security delivery  
* Develops global information security policies, quidelines and
awareness material  
* Develops and delivers information security trainings for selected
target groups as required  
* Shares competences and knowledge with security specialists Network  

Operates in co-operation with security management, area security teams
and internal stakeholders e.g. Business Infrastructure, Communications,
Legal and Risk Management. 

 
Requirements
 
Competencies:
Excellent knowledge of information security technology, deep
understanding of best practises in information security and risk
management. Excellent project management, planning, documentation,
communication, organizing, collaboration, teamworking, training and
negotiation skills. Fluent English. Ability to create new things,
influence others and manage several tasks in parallel.   

Education and Experience:
M.Sc./B.Sc. in computer security, information technology,
telecommunications, engineering or equivalent. Typically 6-9 years
business experience in information security. Key contributor, recognised
specialist with proven track record. This position requires traveling.
