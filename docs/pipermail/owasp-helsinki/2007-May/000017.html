Moi!

Tiistaina 5.6. järjestämme seuraavan tapaamisen Smilehousen tiloissa Lauttasaaressa. Päivän fokus ei ihan keskity SOA-juttuihin (puhujia oli yllättävän vaikea löytää...), mutta tiedossa on Gunnar Petersonin SOA/WS/XML security -aihe sekä hänen päivityksensä projektista "OWASP XML Security Gateway Evaluation Criteria" - jota Gunnar itse vetää. Lisäksi luvassa on tekninen esitys OWASPin työkalujen käytöstä softatestauksessa (Nokian tuoteturvallisuudessa). Kolmas esitys on vielä auki koska esiintyjä ei ole vahvistanut osallistumistaan (juttu kyllä valmiina!).

Gunnar pitää mainitusta aiheesta koko päivän koulutuksen Tietoturva ry:n kautta ja keskustelu aiheesta jatkuu illalla OWASPin puitteissa.

Eli jos kiinnostaa niin laita meiliä - tila on rajattu ja about 30 mahtuu mukaan.

Btw, vapaamuotoinen keskustelu jatkuu "paikallisessa" virallisen osuuden jälkeen...
