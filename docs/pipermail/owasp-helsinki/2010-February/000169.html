Moi,

Summary: Larry Suto has published an interesting comparison of  
different web application scanners by running all scanners against  
each others' test sites.

http://ha.ckers.org/blog/20100203/accuracy-and-time-costs-of-web-application-security-scanner-report/

Larry Suto on julkaissut mielenkiintoisen vertailun eri  
sovellusskannerien kesken.

Vertailu tehtiin ajamalla jokaisella skannit kaikkien valmistajien  
omia testisaitteja vastaan.

Tuloksena NTOspider ja Appscan pärjäsivät parhaiten. Mielenkiintoista  
oli, että osa skannereista ei pärjännyt hyvin edes omaa testisaittiaan  
vastaan. Hintaerot skannereissa ovat myös suuria, mutta tätä  
vertailussa ei huomioitu.

Tulokset voivat olla hyvin erilaisia riippuen kohdesovelluksen  
kirjautumis- ja istunnonhallintamenettelyistä. Jos skanneria miettii  
esimerkiksi oman organisaation sovelluskehityksen tueksi, kannattaa  
ensin tarkastaa ja vertailla, miten hyvin skanneri toimii käytettyjen  
teknologioiden kanssa.
