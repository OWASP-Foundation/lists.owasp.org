Hi everybody,

OWASP chapter board had a meeting a while ago where we planned the upcoming
OWASP chapter activities. Plan is to have at least 4 events again this
year. You all have received an invitation to the next meeting which is held
on 24th of April. Here's the link to registration once more:
http://www.eventbrite.com/event/6240751255. As I mentioned in the previous
meeting, there might be even some new OWASP merchandise for participants :).

Here are some themes for upcoming events that have been on the "drafting
table" among others:
- web applications and privacy
- waf 2.0
- new web application threats
- security related issues with SSL. Convergence.
- information security in gaming industry.

Is there something else that you would like to see in the upcoming OWASP
chapter events (topic/training/demos/presenter)? Let me know.

Also if you are interested in participating more actively in planning OWASP
chapter activities, let me know. There's always room for active volunteers
in OWASP especially if you have a developer background.

Best regards,

Petteri
