Hello,

Haaga-Helia University Of Applied Sciences is organizing a cyber security
seminar in cooperation with OWASP Helsinki and FISA. Main theme of the
seminar will be secure software development and fulfillment of security and
privacy requirements during the systems development life cycle.

The seminar is open and free for everyone.

Venue: HAAGA-HELIA UAS, Pasila Campus, Ratapihantie 13, Helsinki
Date: Tuesday, 30 September 2014, 12.15. p.m. - 17.00 p.m.

Agenda:
12.15 Opening of the seminar
12.45 External security and privacy requirements for system development
13.30 Implementing security and privacy as part of enterprise and
IT-architectures
14.15 Coffee break
14.45 Implementing security and privacy as part of agile software
development processes + Cyber risk exploitation demo
15.30 Security and privacy concerns during software development
16.15 Current threats against information systems and detecting and
managing data breaches

Please see the link below for detailed programme and registration:
http://www.haaga-helia.fi/fi/cyber-security-seminar

With best regards,
Pyry
