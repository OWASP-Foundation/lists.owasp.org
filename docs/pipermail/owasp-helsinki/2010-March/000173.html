English-language summary:

Googler Michal Zalewski has written a new open source web security  
scanner which is available on Google Code.

Google on julkaissut Michal Zalewskin (mies Browser Security Handbook - 
kirjan takana) laatiman webbisovellusskannerin nimeltään skipfish.

Maailmalta kuuluneiden ensikokemusten mukaan työkalu on  
poikkeuksellisen nopea, mutta raskaahko generoimiensa HTTP-pyyntöjen  
määrän takia.
Työkalu on tällä hetkellä versionumerossa 1.19 Beta.

http://code.google.com/p/skipfish/
