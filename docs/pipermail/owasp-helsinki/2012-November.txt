Hi everybody,

Next OWASP chapter meeting will be held on December 4th. In this meeting
Erlend Oftedal appears as guest speaker. He's a developer at Bekk
Consulting AS specialized in application security; java script security
among others.

Please register at Eventbrite: http://www.eventbrite.com/event/4856878053

Location: Nokia House, Keilalahdentie 4, Espoo.

Agenda:
17:00 Socializing time (details unconfirmed)
18:00 Opening words /Petteri Arola, OWASP and Oleg Podsechin, HelsinkiJS
18:10 Word from our sponsor /Nokia
18:20 Securing JavaScript based web apps /Erlend Oftedal
Single page web applications move much of the application logic to the
client side. We now also see applications using JavaScript on the server
side. How do we handle such applications from a security perspective? What
problems are introduced and how do we handle them?
19:05 RESTful Security /Erlend Oftedal
Many applications rely on web services and ws-security for integration. But
for more lightweight services with simpler protocols, REST is quickly
gaining popularity. How do we secure REST services? What problems do we
need to be aware of?

20:00 Official program ends

Please register by Friday November 30th.

Welcome!
