The OWASP Chapter germany is currently soliciting papers for the 4. German
OWASP Day (formerly known as AppSec Germany) which takes place November
17th, 2011 in München

The German OWASP Day is primarily oriented towards a German speaking
audience but presentations in English are also welcome.

We are interested in all topics related to Web Application Security (in no
particular
order):

 * Technical presentations related to operations
 * Frameworks and best practices for secure development
 * Cloud Security
 * Privacy in Web Applications
 * Secure Development Lifecycle
 * Metrics for application security
 * Security awareness programs for developers, testers, architects and
project owners
 * Security management for applications in the corporate environment
 * Application security for Outsourcing and Offshoring projekts
 * Field reports from corporations regarding the institution of Web
Application Security
  processes, internal and external auditing etc.
 * OWASP in your workplace, university, etc.

For more details, please visit
       https://www.owasp.org/index.php/German_OWASP_Day_2011

How to make a submission:
 * Submit the filled form at
       https://www.easychair.org/conferences/?conf=owaspger11
Submit Proposals to:
       https://www.easychair.org/conferences/?conf=owaspger11

Submission deadline is September 9th, 2011

Conference Website:
       https://www.owasp.org/index.php/German_OWASP_Day_2011

OWASP Website:
http://www.owasp.org <http://www.owasp.org/>

Please forward to all interested practitioners and colleagues.
