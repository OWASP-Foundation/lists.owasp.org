Dear , 

I would like to invite you to the next OWASP Belgium Chapter meeting
on March 4th in Leuven!  

WHEN 

Tuesday, March 4th, 2008 (18pm-21pm) 

WHERE 

Katholieke Universiteit Leuven [1] sponsors the venue. 

Location: Department of Computer Science (auditorium 00.225)
          Celestijnenlaan 200 A, 3001 Heverlee 

Map: Detailed map [2] General map [3] 

PROGRAM 

The agenda looks as follows: 
18h00 - 18h30: Welcome, Refreshments and drinks
18h30 - 18h45: OWASP Update (by Sebastien Deleersnyder, OWASP BeLux)
18h45 - 19h00: CAcert.org (by Ken Van Wyk, KRvW Associates) 
19h00 - 20h00: Development life cycle issues (by Ken Van Wyk, KRvW Associates)
20h00 - 20h15: break 
20h15 - 21h00: Improvements for secure software development
               processes (by Bart De Win, DistriNet, KULeuven)

More information about the talks and the speakers can be found
online [4].  

REGISTRATION  

Please send a mail to belgium at owasp.org [5] if you plan to attend,
so we can size the venue appropriately and keep you updated on
last-minute changes.  

WHO 

Anyone interested in Web Application Security (management, security
professionals, developers, students, etc). OWASP Belgium chapter
membership is free. All meetings are free. There are never vendor
pitches or sales presentations at OWASP meetings. Check our chapter
page http://www.owasp.org/index.php/Belgium [6] on meeting details,
sign up to the chapter mailing list and introduce yourself. 

Please forward to anyone you feel would have an interest in these
events.  

Thank you, 

Lieven Desmet
