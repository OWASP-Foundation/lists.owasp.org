Dear  ,

February/March seems a pretty busy period for web application
security, so we quickly inform you about some interesting events:

	*OWASP @ FOSDEM
	*ISSA Cyber Crime Survey
	*ISSA Linux Security Event
	*OWASP Belgium Chapter meeting
	*Call For Papers/Topics OWASP AppSec Conference Europe (Belgium)

OWASP @ FOSDEM

Rogan Dawes comes to Brussels to talk about WebScarab-NG this
week-end (Sunday februari 24) ! 

Rogan will talk briefly about the Open Web Application Security
Project (OWASP) [1], then introduce, explain and demonstrate
OWASP's WebScarab-NG testing tool [2]. More information on
http://www.fosdem.org/2008/schedule/events/oswap_webscarab_ng [3]

ISSA CYBER CRIME SURVEY
OWASP supports the following Cyber Crime Survey organised by ISSA
Belgium: 

* English version [4]: https://www.surveymonkey.com/s.aspx?sm=TogOWc05vIo3NOtNJyOEzg_3d_3d
*Version francaise [5]:
https://www.surveymonkey.com/s.aspx?sm=xDns5IKHewedpCTWfNCRsQ_3d_3d
*Nederlandse versie [6]:
https://www.surveymonkey.com/s.aspx?sm=I86ijdGYdbdMmeWp5GzwVA_3d_3d

Contribute to this survey and make it valuable for the community and
yourself! 

The preliminary results of this survey will be presented on
19-03-2008 (11h30 - 12h30) at INFOSECURITY.BE (where OWASP will also
be presenting).

ISSA LINUX SECURITY EVENT

The ISSA Brussels-European Chapter also organises a Linux Security
Event: "A practical guide to Linux security." [7] 

This evening event will be held on Monday February 25th, 2008 

Agenda: 

*1800h: Welcome with drinks and sandwiches
*1830h: "A practical guide to Linux security (part 1) by Mr Jan Guldentops, CEO / Open Source and Security consultant at Better Access (BA)	
*1930h: short break
*1945h: "A practical guide to Linux security (part 2)"
*2045h: closing drink

This event will be held in room "Isabella" of the Provinciehuis
Vlaams-Brabant, Provincieplein 1, 3010 Leuven More information on
http://www.issa-be.org/ [8]
 
OWASP BELGIUM CHAPTER MEETING

The upcoming OWASP Belgium Chapter Meeting on 4 Mar 2008 in Leuven
includes the following topics:

*CAcert.org (by Ken Van Wyk, KRvW Associates)
*Development life cycle issues (by Ken Van Wyk, KRvW Associates)	
*Improvements for secure software development processes (by Bart De Win, DistriNet, K.U.Leuven)

More information can be found at
http://www.owasp.org/index.php/Belgium#Chapter_Meeting_.284-Mar-2008.29_in_Leuven
[9]

CALL FOR PAPERS/TOPICS OWASP APPSEC CONFERENCE EUROPE (BELGIUM)

The Call For Papers for the refereed papers track, focussing on
academic research, are due to March 1st, 2008. More information about
the refereed paper track can be found at
http://www.cs.kuleuven.be/~lieven/AppSec2008/ [10]

The Call For Topics for the Business & Technical Sessions are also
due to March 1st, 2008. More information about this Call For topics
can be found at
http://www.owasp.org/index.php/OWASP_AppSec_Europe_2008_-_Belgium/CFTP
[11]

The OWASP AppSec Conference Europe 2008 [12] will be organized in
Belgium on May 20th-23rd, 2008.
