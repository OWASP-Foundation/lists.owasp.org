Dag List,

Bij de OWASP meeting vorige week heb ik in het kort verteld over het
Open Review Project.

Het doel is Open Source projecten te reviewen op security zwakheden.
Dit doen we door middel van sterke automatische tooling en door middel
van handmatige review (ook ondersteund door collaboration tooling).

Het project is in opstart fase. We zijn bezig de tooling helemaal af
te maken (is grotendeels gebeurd) en het eerste project te kiezen
(contacten zijn gelegd).

Er is inmiddels een aantal mensen aangehaakt, maar zoals gezegd
tijdens de meeting: ik heb nog meer enthousiastelingen nodig die
kunnen en willen reviewen. De talen waar we ons nu op focussen zijn
PHP en Java, maar wanneer je andere talen spreekt ben ik ook
geïnteresseerd.

Laat van je horen, ook al heb je weinig tijd! Het is een kans om een
bijdrage aan OWASP te leveren, kennis op te doen, mensen te leren
kennen en de wereld een beetje veiliger te maken.

De project pagina is:
http://www.owasp.org/index.php/Category:OWASP_Open_Review_Project
Je vindt daar ook de mailing list van het project waarop je je kunt abonneren.

Met vriendelijke groet,
Mario de Boer
