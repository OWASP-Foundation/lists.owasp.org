Abstract:<br>
<br>
By now pretty much everyone has heard of "big data", right? If not,<br>
here's Wikipedia's short summary: "big data is a collection of data sets<br>
so large and complex that it becomes difficult to process using on-hand<br>
database management tools or traditional data processing applications."<br>
Did you ever wonder how big data affects security? Or why the "bad guys"<br>
are still winning? The industry invests large amounts of money and<br>
energy in protecting assets via the use of firewalls, WAF, IDS/IPS, AV,<br>
etc., but is it really working? Come and join Lars Ewe, VP of<br>
Engineering at Click Security, a leader in the big data real-time<br>
security analytics (RtSA) space, to learn more about how big data RtSA<br>
can help you catch the "bad guys" by better leveraging your existing<br>
security infrastructure investments.<br>
