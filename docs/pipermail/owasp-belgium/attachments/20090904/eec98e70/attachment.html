<br>
<strong>WHAT</strong><br>
<br>
The Open Web Application Security Project (<a class="moz-txt-link-abbreviated" href="http://www.owasp.org">www.owasp.org</a>) Belgium
Chapter<br>
organizes their next Chapter meeting. OWASP's all-volunteer participants<br>
produce free, professional quality, open-source documentation, tools,
and<br>
standards on application security. An example of this is the famous
OWASP<br>
top ten of most critical web application security flaws. The OWASP<br>
community facilitates conferences, local chapters, articles, and message<br>
forums. Participation in OWASP is free and open to all, as are all the<br>
materials we produce.<br>
