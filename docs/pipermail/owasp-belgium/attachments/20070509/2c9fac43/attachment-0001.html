<P><FONT face=Arial size=2>Subject: [SC-L] WASC Announcement: Distributed Open 
Proxy Honeypot</FONT></P>
<P><FONT face=Arial size=2>ProjectData Released</FONT></P>
<P><FONT face=Arial size=2>The Web Application Security Consortium (WASC) is 
pleased to announce the</FONT></P>
<P><FONT face=Arial size=2>inital release of data collected by the Distributed 
Open Proxy</FONT></P>
<P><FONT face=Arial size=2>Honeypot Project. This first release of information 
is for data gathered</FONT></P>
<P><FONT face=Arial size=2>from January - April, 2007. During this timeframe, we 
had 7 internationally</FONT></P>
<P><FONT face=Arial size=2>placed honeypot sensors deployed and sending their 
data back to our central</FONT></P>
<P><FONT face=Arial size=2>logging host.</FONT></P>
<P><FONT face=Arial size=2>What did we see? Here are some brief highlights 
-</FONT></P>
<P><FONT face=Arial size=2>- SQL Injection Attacks</FONT></P>
<P><FONT face=Arial size=2>- Brute Force Attacks</FONT></P>
<P><FONT face=Arial size=2>- OS Command Injection</FONT></P>
<P><FONT face=Arial size=2>- Web Defacement Attempts</FONT></P>
<P><FONT face=Arial size=2>- Google-Abuses (Google-Hacking and Proxying for 
BannerAd/Click Fraud)</FONT></P>
<P><FONT face=Arial size=2>- Information Leakage</FONT></P>
<P><FONT face=Arial size=2>We have created a PDF document here -</FONT></P>
<P><A 
href="http://www.webappsec.org/projects/honeypots/Threat_Report_05072007.pdf"><U><FONT 
color=#0000ff><FONT face=Arial 
size=2>http://www.webappsec.org/projects/honeypots/Threat_Report_05072007.pdf</FONT></U></FONT></A></P>
<P><FONT face=Arial size=2>. The attacks are mapped to the WASC Threat 
Classification categories.</FONT></P>
<P><FONT face=Arial size=2>There are some high-level statistics shown, however 
they are very crude as</FONT></P>
<P><FONT face=Arial size=2>this was not the focus of this phase of the project. 
We understand that the</FONT></P>
<P><FONT face=Arial size=2>data presented is a bit raw, however we wanted to 
release this information</FONT></P>
<P><FONT face=Arial size=2>so that the public may have a chance to review it and 
provide feedback. Our</FONT></P>
<P><FONT face=Arial size=2>initial goal was to identify the types of current 
attacks that are using</FONT></P>
<P><FONT face=Arial size=2>open proxy servers. In our future deployments, we 
will attempt to refine</FONT></P>
<P><FONT face=Arial size=2>the data analysis processes to extract out trend data 
and high level</FONT></P>
<P><FONT face=Arial size=2>concepts. In the near future, we will be updating 
both the VMware honeypot</FONT></P>
<P><FONT face=Arial size=2>sensors themselves and will also use a newer version 
of the centralize</FONT></P>
<P><FONT face=Arial size=2>logging host (ModSecurity Console).</FONT></P>
<P><FONT face=Arial size=2>We are also planning to release more frequent 
information in the form of</FONT></P>
<P><FONT face=Arial size=2>diary entries on the project webpage as new 
attacks/trends are identified.</FONT></P>
<P><FONT face=Arial size=2>While the initial deployment was a success, we still 
need participants who</FONT></P>
<P><FONT face=Arial size=2>are willing to participate by deploying our VMware 
honeypot sensor on their</FONT></P>
<P><FONT face=Arial size=2>network. If you are interested in participating, 
please send an email to</FONT></P>
<P><FONT face=Arial size=2>Ryan Barnett at - RCBarnett_@_gmail.com.</FONT></P>
<P><FONT face=Arial size=2>URL:</FONT></P>

color=#0000ff><FONT face=Arial 
size=2>http://www.webappsec.org/projects/honeypots/</FONT></U></FONT></A></P>
