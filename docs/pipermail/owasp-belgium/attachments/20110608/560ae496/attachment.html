<p>WHAT</p>
<p>The Open Web Application Security Project (<a href="http://www.owasp.org">www.owasp.org</a>) Belgium Chapter organizes their next Chapter meeting. OWASP&#39;s all-volunteer participants produce free, professional quality, open-source documentation, tools, and standards on application security. An example of this is the famous OWASP top ten of most critical web application security flaws. The OWASP community facilitates conferences, local chapters, articles, and message forums. Participation in OWASP is free and open to all, as are all the materials we produce.</p>
