Job description:
Electrabel provides energy solutions for industrial
enterprises. It 
offers smaller businesses and residential customers a
quality 
locally based offer. Its diversified generating
equipment is 
primarily made up of gas power stations and extremely
reliable 
nuclear facilities. Electrabel engages in trading
activities on all

distribution 
system operators. The company's matrix organisation

Mission:
As Security Architect your responsabilities will be:
Assist the security manager in setting up an
information security 
improvement program.
Provide security support to projects (provide advice,
evaluate 
architectures and designs, security test support).
Setting up information security policies for the IT
community in 
specific technical and organizational domains.
Play an active role in project and change management
with respect to 
implementing and improving information security.
Set-up and lead the ITS-security forums.
Follow-up the implementation of information security
best 
practices.. Assist the security manager in setting up
"continuous 
improvement" mechanisms.
Provide guidance and assistance in the development and
maintenance 
of the necessary tools.
Performing information security audits.
Proactive alerting on security problems.
Responding to, escalating and following-up of security
breakages.
Provide adequate reporting on information security
activities to the 
security manager. 

Desired profile 
In depth (hands-on) technical know-how in at least 3
of the 
following domains, and a good to basic knowledge of
other domains:
Secure web architecture
Network security
Application security
System security (Windows, AIX, Linux, VMS, OS380)
OO software development (Java)
Vulnerability and threat analysis
Experience in security testing of end-to-end
applications.
Pen-testing experience and know-how is definitely an
asset
A strong team player, able to work in hierarchical and
matrix-based 
teams
Trilingual (Dutch, French, English)
Good communication and report writing skills 
Analytical and accurate
