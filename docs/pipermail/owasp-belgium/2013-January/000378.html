Dear Security Professional, Computer Geek or Developer with Security
Affinity,

SANS is coming to Brussels from 18 to 23 February 2012 and will be
providing the following courses in Belgium:
- Security Essentials Bootcamp Style - SEC 401 by Dave Shackleford (US)
- Hacker Techniques, Exploits & Incident Handling - SEC 504 by Steve
Armstrong (UK)
- Web Application Penetration Testing & Ethical Hacking - SEC 542 by Pieter
Danhieux (BE)

nViso is happy to provide you with a 10% discount code (SANSBE13NVISO) that
can be used by all OWASP members and security professionals in Belgium.
More info on this event can be found at this URL:
http://www.sans.org/event/belgium-2013

Additionally, SANS and nViso are organising a SANS at night that is accessible
by anyone without charge.
- Steve Armstrong will be talking about "How to manage an incident and tips
for finding hackers"
- Pieter Danhieux will be talking about "Patching your employees brain"
More info on this SANS at Night event can be found at this URL:
http://www.sans.org/event/belgium-2013/bonus-sessions/

We hope to see you there.

Regards,

Erik Van Buggenhout
