Tom,

most of those proxy tools use the same techniques more or less like you
note. I would however would like to have an overview of any special
features of those tools. I work with the BURP suite for example which has
the nice feature of doing automated and customizable attacks on parameters.
I don't think Paros and others feature this kind of option, but they will
probably have other nice features. It would be interesting to get an
overview of the tools and any special functions.

topic. Everyone how to find bad input filtering techniques in websites, but
is it possible to circumvent certain filters?
