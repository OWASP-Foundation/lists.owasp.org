Part 1 : "Information Security Intelligence"

"Knowledge is power". It's the motto all "Intelligence Agencies" around the
world live by. "Intelligence" could be described as "the process of
gathering, evaluating, correlating and interpreting of information, and
disseminating it to the decision makers." As a result, it reduces
uncertainty in the decisionmaking process. Which is something every "boss"
appreciates. To some degree, everybody in every organisation does this, but
we could do a lot better, and this presentation by Maarten Van Horenbeeck
from Cybertrust will show you the way! 

 

Part 2 : "Using the Windows UserAssist registry keys for forensic analysis"

An incredible amount of data is being stored into the Microsoft Windows
Registry database. A whole lot of it is being documented in official
Microsoft documents, but not everything... But if one has a truely
inquisitive mindset like Didier Stevens from Contraste Europe has, one is
bound to find some very interesting needles is this haystack!

SPEAKERS' BIO'S AND ABSTRACTS OF THE PRESENTATIONS :

Maarten Van Horenbeeck, Security Consultant at Cybertrust

bio : 
As a security consultant, Maarten Van Horenbeeck is active within
Cybertrust, providing services that secure critical data, protect identities
and help customers demonstrate ongoing compliance. He has an academic
background in information security and intelligence and has experience in

Prior to his current role, he was a senior security analyst and subsequently

Maarten is also an incident handler with the SANS Internet Storm Center. 

 

abstract : "Information Security Intelligence

The threat landscape has seen some drastic changes over the last few years.
The fading perimeter, format rendering vulnerabilities and targeted attacks
have all changed the way we deal with technical controls and how we choose 
to mitigate risk. Organizations are having a hard time dealing with these
new threats, and are increasingly looking to formal security intelligence
processes for assistance. Maarten will introduce the intelligence principles

that underlie the way we deal with security information, apply them to a
number of recent security incidents, and show how they can be and are being
used by organizations and security vendors to more effectively deal with 
threats to information security.

Mr Didier Stevens, Security Consultant at Contraste Europe 

Didier Stevens (CISSP, MCSD .NET, MCSE/Security) is an IT Security
Consultant currently working at a large Belgian financial corporation. He is
employed by Contraste Europe NV, an IT Consulting Services company. (note
from Tomas : "His blog and his IT Security Research and Open Source tools
are renown around the world, but he is Mr Modesty in person!") 

abstract : 

Windows XP (and later Windows versions) keeps a list of all programs
executed by a user. This list is kept in the registry (UserAssist keys). For
each executed program, a running counter and a timestamp is maintained. 
The data in the UserAssist registry keys is used by Windows to display the
Most Frequently Used Programs list on the start menu, but it can also be
used to analyse which programs a user has executed.
Exploiting these keys for forensic analysis is a little known technique,
because there is no publicly available official documentation from Microsoft
about these registry keys. This presentation will detail Didier's findings
and the use of the Open Source tool he wrote specifically to reveal this
info in a userfriendly way.
