All,

The next OWASP London Chapter Meeting will take place on Wednesday 24th 
October 2018 at J.P. Morgan offices in Canary Wharf

Location: J.P. Morgan, 25 Bank Street, Canary Wharf, London, E14 5JP

Nearest Tubes: Canary Wharf (5-minute walk - take the exit signposted 
DLR), Heron Quays DLR (2-minute walk)

Doors Open at 6:00pm for registration, food, drinks and networking. The 
talks start at 6:30pm (we start on time)

Please aim to arrive at 6:00pm to get a chance to register, grab some 
food & drink and network with other participants.

The full details of the talks and speakers are now available on the 
OWASP London Wiki Page here:

https://www.owasp.org/index.php/London#Wednesday.2C_24th_October_2018_.28Canary_Wharf.29

You can register to attend this event here:

https://www.eventbrite.co.uk/e/owasp-london-chapter-meeting-wednesday-24th-october-2018-630pm-tickets-51430094798?aff=ml 


Please note that space at this event is limited, so please only book 
tickets if you are able to attend.  We recommend business/smart casual 
dress code for this venue.

Looking forward to seeing you there!

Regards,

Sam Stepanyan, Sherif Mansour and Dr. Greg Fragkos
OWASP London Chapter Leaders
