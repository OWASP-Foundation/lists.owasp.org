> I have a project that might interest people, here is the oulines:
>
> Secure communication Framework
>
>
>  The goal is to offer a secure layer of communication using a well managed
> PKI, to tiers applications, users or systems. This layer would be on the top
> of any operating system, and just under the application layer.
>
> This would be a multi-platform software ( Windows, Mac OS, Linux, embeded
> OS)
>
> The general system would be highly distributed, without real central point,
> apart from some specialized components, like the certification authority.
>
>
>  The main parts would be :
>
>    -
>
>    Automated certificate generation, distribution and revocation (as far
>    as it is applicable).
>    -
>
>    Integrated authentication via certificate.
>    -
>
>    Messaging protocol offered for tier applications, this would include a
>    complete routing system and a naming system.
>    -
>
>    Possibility for any application to use the certificate or to use a
>    secure channel from the framework.
>    -
>
>    multi-platform: Windows, Mac OS, Linux, embeded OS, ... (Mainly due to
>    the use of Qt framework: http://trolltech.com/products/qt/ )
>
>
>  Potentials applications:
>
>    -
>
>    Secure Instant messaging
>    -
>
>    Secure VOIP
>    -
>
>    Secure administration of complex network, from user stations to
>    servers, including switches, firewall, routers.
>    -
>
>    User mobility management (laptop, phone, PDA)
>
> Let me know if that sounds good, and if this funding is also open to non-UK
> people (french as a matter of fact)
