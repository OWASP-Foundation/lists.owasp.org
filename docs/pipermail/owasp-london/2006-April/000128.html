I totally agree, and that is my objective with the proposed Owasp-London
Web App Firewall session. To get the vendors to do a technical
presentation to technical people.

The current plan is to get the vendors of these products to show how
they are able to protect against the vulnerabilities that currently
exist in SiteGenerator.

I don't want to hear about the 'wonderful' features of their product, I
want to know how they deliver real value on real issues (i.e.
vulnerabilities) that exist on real applications

Dinis
