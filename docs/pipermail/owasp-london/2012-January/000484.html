The question is not Dull or Smart, it is about making IT creative.

There is probably a need for a short section on personal IT security,
probably the equivalent for safe sex in biology.   Trying to teach kids
about industrial strength security  would fall in to the Dull catagory.   

Presenting security as a "who-dunnit" story might make it more attractive
and their spark imagination.  Relate it to what they see in the media -
Lizbeth Salander or Spooks.

We have to give security the same "cool" as writing video games 

So as we start to think about who engage to support these initiatives,
remember Marsh McLuhan, the medium is the message.

Richard
