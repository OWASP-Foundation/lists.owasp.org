</font><b>Title: Advanced Asp.Net Exploits and Countermeasures</b><br>
<p><font face="Times New Roman, Times, serif"><b>Overview:</b><br>
<br>
In this 2 day course you will push Asp.Net to the limit and will be
shown how Asp .NET applications and environments can be exploited by<br>
skilled attackers. Advanced exploitation techniques will be presented
together with low-level technical analysis of the .Net Framework. You<br>
will also learn advanced defense techniques such as: Building an Asp
.NET Security Protection layer (also called a Web Application Firewall)<br>
and Real time patching of vulnerabilities in the target application,
the .Net Framework or the CLR."<br>
<br>
<b>Structure:</b><br>
<br>
The Course is made of 4 modules (2 per day, one in the morning and one
in the afternoon)<br>
<br>
<b> Module 1: Security principles and .NET Framework Architecture</b><br>
<br>
<b> Module 2: Guerrilla Threat Modeling and Exploiting Asp.Net
Applications</b><br>
</font></p>
<ul>
  <li><font face="Times New Roman, Times, serif">Using quick-and-dirty
threat models to discover vulnerabilities in the target application</font></li>
  <li><font face="Times New Roman, Times, serif">Exploiting
vulnerabilities in Asp.Net applications: Data Validation,
Authorization, Authentication, SessionState, XSS, Cookies, AJAX, Web
Services, Remoting, etc.. (using basic and advanced techniques)</font></li>
  <li><font face="Times New Roman, Times, serif">Exploiting Buffer
Overflows and Windows vulnerabilities via Asp.Net Applications</font></li>
</ul>
<p><font face="Times New Roman, Times, serif"><br>
<b> Module 3: Exploiting Full Trust and Partial Trust Asp.Net
Environments</b><br>
</font></p>
<ul>
  <li><font face="Times New Roman, Times, serif">Practical
demonstrations of the power of Full Trust Asp.Net: Rooting the CLR
(e.g. patching the .Net Framework and CLR), Reflection,</font></li>
  <li><font face="Times New Roman, Times, serif">IIS Metabase,
Shellcode injection, Launching internal attacks to compromise the
server and the data center</font></li>
  <li><font face="Times New Roman, Times, serif">Full Trust
non-verification and Type Safety attacks (via MSIL manipulation)</font></li>
  <li><font face="Times New Roman, Times, serif">Exploiting Insecure
Partial Trust Asp.Net Environments</font><br>
  </li>
</ul>
<p><font face="Times New Roman, Times, serif"><b> Module 4: Advanced
Asp.Net Countermeasures</b><br>
</font></p>
<ul>
  <li><font face="Times New Roman, Times, serif">Applying real-time
security patches in the target application, .Net Framework and CLR <br>
    </font></li>
  <li><font face="Times New Roman, Times, serif">Solutions to create
secure Data Validation and Authorization architectures</font></li>
  <li><font face="Times New Roman, Times, serif">Creating secure
Asp.Net hosting environments</font></li>
  <li><font face="Times New Roman, Times, serif">Building an Asp.Net
Security Protection layer (also called web Application Firewall);</font></li>
  <li><font face="Times New Roman, Times, serif">Using Mono</font><br>
  </li>
</ul>
<p><font face="Times New Roman, Times, serif">You will walk away from
this class with a much better understanding of some of the weaknesses
of .NET applications, particularly the internals of the .NET framework.
You will also get the chance to put your skills to the test against a
target application over the course of the class.<br>
