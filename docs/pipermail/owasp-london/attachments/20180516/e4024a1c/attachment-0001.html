      AppSecEU is the premier application security conference for
      European software developers and security experts. Application
      Security Europe 2018 provides thought leadership, amazing talks,
      informative sessions and great social experiences.<br>
      <br>
      The main AppSecEU conference this year spans two days from 5th to
      6th of July 2018, offering four full tracks of talks: for
      pentesters and ethical hackers, developers and security engineers,
      DevOps/DevSecOps practices and GRC/Risk level talks for managers
      and CISOs.<br>
      <br>
      July 2nd-July 4th will be dedicated to Training sessions and
      workshops delivered by the industry experts for hands-on learning
      in top applications security areas.<br>
