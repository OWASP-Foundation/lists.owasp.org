'Older versions'??
Its called "SOAP Scanner" and its still available for the latest versions
of ZAP.

For the record, while importing WSDL / WADL or any other API definition
will help you get started, I think the best option is proxying real traffic
through your security tool.
You can do this manually or using functional regression tests. For
automation you'll want to do the latter, and if you dont have any then you
might like to write some ;)

And if anyone has any suggestions as to how we could improve ZAP for
scanning APIs then please let me know.
