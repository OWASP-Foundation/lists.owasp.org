Hi, if you are around this evening please join us for the 4th GDPR Working
Session.

Tonight we going to focus on *"GDPR Patterns based on Threat Models"*

What are GDPR patterns? They are reusable mappings of data journeys across
specific threat modeling scenarios. This continues to work we started at
the last GDPR Working Session.

You can read more details here: https://www.linkedin.
com/pulse/join-us-tomorrow-gdpr-patterns-working-session-based-threat-cruz/

You can get a free ticket (for both onsite and remote participation) from
https://www.eventbrite.com/e/gdpr-patterns-owasp-working-session-tickets-
42421937156 (we use this to send detailed instructions on how to get into
the building, and the remote participation Google Hangouts link)

We have started to add all the materials to this GitHub repo
https://github.com/OWASP/gdpr-patterns

See you tonight
