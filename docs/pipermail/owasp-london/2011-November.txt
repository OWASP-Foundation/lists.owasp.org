Hi folks,
cc Leeds and London folks,

Booking is now open for the OWASP Manchester meeting on Novemeber 16th!
We've got another 2 great talks lined up:

   - Policy is the best honesty: Dr Daniel Dresner, Head of Information
   Assurance Practice, NCC

   - Non-alphanumeric code in JavaScript and PHP: Gareth Heyes, independent
   security researcher

See https://www.owasp.org/index.php/Manchester for full details, and
booking is again via Eventbrite <http://www.eventbrite.com/event/2460447264>
.

Please note that this time the doors open at 18:45, talks start at 19:00
prompt - please make sure you are there on time.

This time we have 50 spaces available - but book soon if you want to attend
- last time we sold out and there were 10 people on the waiting list!

Many thanks to KPMG for supporting OWASP by hosting this event.

Cheers,

Simon Bennetts
