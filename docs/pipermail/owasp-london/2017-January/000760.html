The next OWASP London Chapter meeting will take place on Thursday, 26th

This event is kindly sponsored and hosted by J.P. Morgan.

Location: 6th Floor, JP Morgan, 60 Victoria Embankment, London, EC4Y 0JP

Nearest Tube: Blackfriars (3 minute walk)

Doors Open at 6pm, the talks will start at 6:30pm (we start on time).


The details of the talks and speakers are now available on the OWASP 
London Wiki Page here:

https://www.owasp.org/index.php/London

You can register to attend this event here:

https://www.eventbrite.co.uk/e/owasp-london-chapter-meeting-thursday-26th-january-2017-630pm-tickets-31043174972

Please note that space is limited, so please only book tickets if you 
are able to attend.

Ticket booking will close the evening before the event so we get names 
for the building security.
