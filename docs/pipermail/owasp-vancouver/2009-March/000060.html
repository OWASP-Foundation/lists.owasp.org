Hello,
 
We have scheduled a presentation of our 4-day "Secure Web 
Application Development and Testing" course in VANCOUVER BC 
for April 28 through May 1 2009 (Tuesday to Friday). More 
information is available online at 
http://www.wavefrontcg.com/IntroWebAppSec_2009a.pdf 

Course: Secure Web Application Development and Testing
Dates: April 28 - May 1 2009 (4 days)
Location: Vancouver (downtown)
Instructor: Rui Pereira,CISSP,CISA,CPTS/CPTE

A brief summary follows...

"This 4-day course aims to provide web application 
developers with an understanding of application security 
issues and attack vectors (XSS, SQL Injection, Broken Access 
Control, Forced Browsing, etc.), and the skills necessary to 
code defensively against web attacks. We will show how 
hackers can abuse a web application, and what developers 
can do to prevent this. In addition, we show how developers 
and testers can test their own applications in order to 
determine if they are susceptible to web application 
attacks. The material would also be of interest for 
software testers (QA); system, network and database 
administrators; software, network and system architects; 
Information Security personnel; and Compliance Officers 
(Privacy, SOX, PCI, etc.)"

If you know of anyone else who may benefit from this training, 
please pass this information on to them. The more the merrier 
we always say (although the course is capped at 14 students).
