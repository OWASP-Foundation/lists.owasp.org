Hi OWASP Members,

I wanted to raise awareness of a new initiative to create a forum for Q&A on
Application Security.

There is a service called Stack Exchange which provides online communities
hosting for StackOverflow Q&A style forums; any community can launch a
proposal and some of the great people at OWASP have created one for
Application Security.  The site has moved out of the definition stage and
into commitment phase; if you think this sort of service would be useful, go
to the site below and indicate wether or not you would use the service.  If
you are not familiar with StackOverflow, then I strongly recommend checking
it out!

We need more people to commit to using the site, and I know that some
extremely talented, top name people in the AppSec field have already
contributed and committed to participation, so this is an opportunity to get
world class talent to focus on the issues you are interested in for free :D
