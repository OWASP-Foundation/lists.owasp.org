Hi OWASP Vancouver Chapter Members,

I have lined up our speaker for September 23rd meeting, Dana Epp from
Scorpion Software will lead a collaborative session using the Microsoft
Threat Analysis and Modelling Tool.

This is an excellent opportunity to get an interactive tutorial on how to
use this free tool from Microsoft, and get some valuable insight from a
talented security professional into the threats and risks associated with
