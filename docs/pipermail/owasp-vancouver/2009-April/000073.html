     The Building Security in Maturity Model (BSIMM) and Your Software
Security Program

Developed by Fortify and Cigital, the Building Security In Maturity Model
(BSIMM) is meant as a guide for building and evolving a software security
initiative. Based on a study of nine large enterprise software security
programs (including those of Adobe, Depository Trust Clearing Corporation
[DTCC], EMC, Google, Microsoft, QUALCOMM, and Wells Fargo) it identifies key
software security roles, 110 activities in twelve practices, as well as the
ten core activities that all programs studied carry out. Citigal's Sammy
Migues will join analysts Ramon Krikken (SRMS) and Kirk Knoernschild (APS)
to discuss the model, and how it fits security and development practices in
other enterprises.

Presented by: Ramon Krikken
