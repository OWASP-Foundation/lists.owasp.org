*Web Input Vector Extractor Teaser
<http://code.google.com/p/wivet/>*(WIVET) is a benchmarking project
that aims to statistically analyze web
link extractors. In general, web application vulnerability scanners fall
into this category. WIVET provides a good sum of input vectors to any
extractor and presents the results...

http://code.google.com/p/wivet/ (project page)
http://code.google.com/p/wivet/wiki/CurrentResults (results I get so far,
you can send me any other results you have)
http://www.webguvenligi.org/wivet/ (for online use)
http://www.webguvenligi.org/software/wivet/wivet_v3.rar (for offline use)

WIVET is an OWASP/Turkey project, a local project :)
