The Security Spending Benchmarks Project seeks to produce guidance and an
industry accepted benchmark for justifying overall Web application security
spending. We want to quantify how many dollars and human resources should be
allocated towards the software development life-cycle, security training,
security software/tools, independent third-party reviews, Web application
firewalls, etc

https://www.owasp.org/index.php/Category:OWASP_Security_Spending_Benchmarks

This is your call to get involved and take a look.
