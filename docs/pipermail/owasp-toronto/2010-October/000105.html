SecTor - Security Education Conference Toronto 2010
October 25 - 27, 2010, Metro Toronto Convention Centre

SecTor 2010 Opens Sponsor Expo October 27th from 1:30 to 4:00PM

For the first time in SecTor’s history, the larger security community is invited to attend the Sponsor Expo on Wednesday, October 27th from 1:30 to 4:00 pm.   In addition to meeting the world’s top security companies, Expo Attendees are encouraged to visit the SecTor Enterprise Demo Lab, featuring the latest security products and solutions available.  As well, Expo Attendees can visit the Hardware Hacking Village and Lock Pick Village.   Learn a little about physical security and try your hand at picking a lock.   It’s an opportunity you don’t want to miss.

For free admission to the SecTor Sponsor Expo, register before October 27th with the discount code OWASPexpo
Registration after October 26th or without a discount code is $50 + HST.

From security considerations with cloud computing and the latest web application attacks, testing and protection measures, to hands on lock-picking, and enterprise demo lab, it's the one must attend conference of the year.
