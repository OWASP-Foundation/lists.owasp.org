6:00 - 6:10 : Registration and introductions
6:15 - 7:00 : Comparing Web App Scanners - Case Study

Comparing Web App Scanners - Case Study

Description: Andre and his TSS team undertook an evaluation for the
acquisition process for an enterprise-class web security scanning product.
The presentation will describe some of the objective (and subjective)
selection criteria, the observations and opinions of the evaluation team,
impressions of the selected tool after several real-world engagements, and
the pros and cons of web security scanners in the areas of client
confidentiality, of coverage, and of extensibility. 
Presenter : Andre Rochefort
About The Presenter:  Andre, is the Web Application Security Team Lead for
TELUS Security Solutions (Former - Assurent Secure Techbologies).  Andre who
is responsible for National Practice Leadership for web app security within
TSS and is a key developer on the NSS-certified Assurelogic Web Application
Firewall product, and a co-author of our Secure Development Methodology.

Web Application Firewalls - Panel Discussion
Andre and a couple of other security professionals will be part of a panel
where we will discuss with the attendees the advantages and disadvantages of
Web Application Firewalls.
