I know this is an extremely low traffic list, but I thought I'd ask a 
question.

As a coauthor of the original ASVS, it included our opinion of what
automation should be able to do. At Level 1 we have:

1A: Dynamic Analysis
1B: Static Analysis

Are any of you aware of any work to compare an actual tool against
ASVS to see what it can do relative to what we thought they should be
able to do (in
2009 anyway)? I know some of the tools, like Watcher, actually
implement some ASVS specific checks. I was hoping someone actually did
a study looking at one or more major commercial tools in this space.
