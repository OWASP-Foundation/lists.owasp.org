This year we are happy to announce OWASP Poland Day, the largest OWASP
conference for application and software security in Poland, aimed at
developers, testers, architects, product designers, and managers - in
short, anyone involved with securing software lifecycle! OWASP Poland Day
will take place this year on October 2nd, 2017 in Krakow. URL with more
information about the event: *https://www.owasp.org/index.php/OWASPPolandDay
<https://www.owasp.org/index.php/OWASPPolandDay>*

We are starting our CFP and would like to invite you to participate in the
event as a speaker on one of the following topics:
- web security
- devops security
- cloud security
- mobile security
- application security
- software development
- quality assurance
- software maintenance or operations

Use the link: https://goo.gl/forms/KiXlU4ImdCuCowVt2 to apply, and do
zobaczenia w Krakowie!
