<span style="font-size:12pt"><span style="font-family:arial,helvetica,sans-serif"><strong>This year OWASP has been selected as an official Google Summer of Code ("GSoC") mentoring organization!</strong><br>
<br>
Open source software is changing the world and creating the future.  Want to help shaping it?  We're looking for students to join us in making 2014 the best Summer of Code yet!<br>
<br>
All students currently enrolled in an accredited institution are welcome to participate in the Google Summer of Code 2014 program along with the OWASP Foundation.<br>
<br>
<strong>What is GSoC?</strong><br>
<br>
The Google Summer of Code program ("GSoC") is designed to encourage student participation in open source developent.  Throught GSoC, accepted student applicants will be paired with OWASP mentors from participating organizations.<br>

<br>
<strong>Benefits to students include:</strong></span></span>
<ul>
        <li><span style="font-size:12pt"><span style="font-family:arial,helvetica,sans-serif">Gaining exposure to real-world software development scenarios,</span></span></li>
        <li><span style="font-size:12pt"><span style="font-family:arial,helvetica,sans-serif">An opportunity for employment in areas related to their academic pursuits, and</span></span></li>
        <li><span style="font-size:12pt"><span style="font-family:arial,helvetica,sans-serif">Google will be offering successful student contributors a $5,000 USD stipend, enabling them to focus on their coding projects for three months.</span></span></li>

</ul>
<span style="font-size:12pt"><span style="font-family:arial,helvetica,sans-serif">This program is done completely online.  Students and mentors from more than 100 countries have participated in past years.<br>
<br>
For a detailed timeline and FAQ about the GSoC program <a href="http://cts.vresp.com/c/?TheOWASPFoundation/d5fe3dca41/ffcb799a6d/d7581f38c4" target="_blank">CLICK HERE</a><br>
<br>
GSoC + OWASP:  Call for Student Proposals<br>
<br>
Are you a student and want to code for an OWASP project?<br>
<br>
<strong>Here are the steps and some tips on getting started:</strong><br>
<br>
1.  Think of a good idea.  <a href="http://cts.vresp.com/c/?TheOWASPFoundation/d5fe3dca41/ffcb799a6d/223581de08" target="_blank">CLICK HERE</a> for reference.<br>
2.  Do some research yourself based on the idea.<br>
3.  Write up a proposal draft.<br>
4.  Post it to the <a href="http://cts.vresp.com/c/?TheOWASPFoundation/d5fe3dca41/ffcb799a6d/7288c99ab4" target="_blank">OWASP GSoC group</a> for initial discussions with OWASP mentors.<br>
5.  Based on feedback, write up a full proposal - <a href="http://cts.vresp.com/c/?TheOWASPFoundation/d5fe3dca41/ffcb799a6d/a17c726b48" target="_blank">SEE TEMPLATE</a><br>
6.  Submit your proposal to Google Melange from March 10th-March 21st, 2014<br>
<br>
Students wishing to participate in GSoC must realize this is a formal commitment to produce code for the selected OWASP Project during three months.  You will also take some resources from OWASP project leaders, who will dedicate a portion of their time to mentor you.  Therefore, we'd like to have candidates who are committed to helping the OWASP mission.  You don't have to be a proven developer - in fact, this whole program is meant to facilitate joining OWASP and other Open Source communities.  However, experience in coding and applications are welcome.  <br>

<br>
<a href="http://cts.vresp.com/c/?TheOWASPFoundation/d5fe3dca41/ffcb799a6d/eddc4d3008" target="_blank">CLICK HERE</a> for full details and requirements for participating.<br>
<br>
We wish you the best of luck,<br>
OWASP GSoC team</span></span><br>
