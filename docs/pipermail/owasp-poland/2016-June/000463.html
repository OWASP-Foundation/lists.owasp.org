W ramach ciekawostek, właśnie udostępniono OWASP ASVS 3.0.1:
https://www.owasp.org/index.php/Category:OWASP_Application_Security_Verification_Standard_Project#tab=Downloads

Dodano przeszło 20 nowych sprawdzeń, w tym nie takie całkiem
kosmetyczne, np.:

"Verify that anti-automation is in place to prevent breached
credential testing, brute forcing, and account lockout
attacks."

" Verify that forms containing credentials are not filled in by the
application. Pre-filling by the application implies that credentials are
stored in plaintext or a reversible format, which is explicitly prohibited"

" Verify that all authentication decisions can be logged, without
storing sensitive session identifiers or passwords. This should include
requests with relevant metadata needed for security investigations."

Z ciekawostek, chyba zapomnieli zmienić rok - teraz w pdf-ie jest data
publikacji July 2015
