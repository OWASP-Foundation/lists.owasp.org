Cześć!
Dostałam od znajomego bardzo ciekawą ofertę z pogranicza security. Może
ktoś będzie zainteresowany:


*Full-stack Senior Security Automation Engineer - 6 months project
collaboration*

 We are looking for a full-stack senior software engineer that can help us
building security automation tools.

We offer a well-paid six-months collaboration, combined with an additional
bonus in case of project success.

Our client is a global Software-as-a-Service platform which allows coding
in Javascript and Java. To help them catching web security vulnerabilities
as early as possible, we are working with them to design and develop a web
scanner that combines custom code with the power of Burp Suite -
https://portswigger.net/burp. The scanner will be packaged as a standalone
service that can be installed on Ubuntu-compatible Linux hosts. We will use
static analyis to derive knowledge from configuration files and code. Then,
we will instrument Burp Scanner with the acquired knowledge in order to
leverage the power of the tool. The overall system will require a custom
web UI to configure and manage the scanner instance as well as to provide
reporting.


*Responsibilities*
 ● Full-stack development of the scanner
 ● Work with co-founder on solving technical challenges
 ● Partner with the customer to ensure customer's objectives are achieved

* Requirements *
● Experience developing multi-tiered software applications or products. We
will use Node.js and Java, and require proficiency on those languages
● Ability to work with standard dev tools and techniques (IDE, git, ...)
● You're passionate about building great software and can have fun while
doing it
● Interested in web security, with basic understanding of common software
vulnerabilities (e.g. XSS, SQL Injection, ...)
● You’re self-driven and can focus on a project to make it happen
● Eager to learn, adapt and perfect your work
