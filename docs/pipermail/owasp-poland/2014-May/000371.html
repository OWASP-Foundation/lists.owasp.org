Kolejne spotkanie OWASP Poland odbędzie się w Krakowie, w poniedziałek
2014-05-25.
Jeśli wybieracie się również na CONFidence to mam dla Was dobrą wiadomość.
Nasze spotkanie będzie połączone z "before party" CONFidence. Jeśli nie -
to nie martwcie się - wstęp jest wolny.
Miejsce: Barka Augusta (niedaleko skrzyżowania Podgórskiej i Gazowej)
50.047431,19.948313 http://www.restauracjaaugusta.pl/

Tym razem zaprosiliśmy specjalistów, którzy mają coś do powiedzenia w
temacie security + continous delivery.

Agenda spotkania:
19.00-19.15 OWASP News
19.15-19.45 Security in continuous delivery environment, Jakub Nawalaniec,
Base CRM
19.45-20.00 short break
20.00-20.40 BDD Security - Automated security testing based on BDD,
Krystian Piwowarczyk, Rule Financial

Ponieważ spodziewamy się że na spotkanie przyjdą również uczestnicy
