Leaders,



I am glad to announce we’ve just set a new project up, the OWASP
Security Tools for Developers, lead by Mark Curphey, which already has
contributors as John Wilander and

Psiinon. Please join me to congratulate them!

https://www.owasp.org/index.php/OWASP_Security_Tools_for_Developers_Project

http://www.curphey.com/author/admin/

https://www.owasp.org/index.php/User:John.wilander

https://www.owasp.org/index.php/User:Psiinon



While the project is open to all, the project leader is particularly
looking for developers who will actively contribute code. Mark is
especially interested in any developers that have experience in
customizing Jenkins, extending Git, unit testing frameworks or
customizing management tools like ScrumDo. He is also interested in
any developers focused in extending behaviour driven development
testing frameworks like JBehave.
You can add yourself to the mailing list
https://lists.owasp.org/mailman/admin/owasp-std . As always, your
suggestions and contributions would be greatly appreciated.
Thanks,
- Paulo
