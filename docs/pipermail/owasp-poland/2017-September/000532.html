For the first time in Poland OWASP organises the AppSec Day - unique
possibility to update application security knowledge and to network with
other security experts and developers.

The agenda looks very interesting:

- Martin Knobloch - Keynote - Your software, your responsibility!
- Taras Ivaschenko - Security in developer’s life: knowledge is a power
- Zacharia Rachid - OWASP ZAP - security testing automation for developers
-  Networking Lunch ;)
- Luca Carettoni - A Study of Electron Security
- Glenn Ten Gate - OWASP SKF Making the web secure by design
- Michał Kurek - Embeding security into SDLC in the light of GDPR
- Wojciech Dworakowski - OWASP Top10 2017

Register now to take advantage of the special early bird prices!

More information on event web page: http://owaspday.pl
See you there!
