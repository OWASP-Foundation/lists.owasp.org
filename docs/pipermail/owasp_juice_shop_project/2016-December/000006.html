Hello all, 
  
a new version of OWASP Juice Shop has just been released, adding the
following: 
  
--------------------- 
  
Challenges 
- added Premium Challenge behind a Bitcoin paywall (that can obviously be
bypassed) 
- adopted Continue Code challenge to new save/restore feature 
  
Features 
- added QR code modal dialog for Bitcoin donation option (on Your Basket
screen) 
- added Your Hacking Session as a convenient way to save and restore hacking
progress (on the Score Board) 
  - replaces Continue Code input form 
  
Tests 
- verify that XSS/CSRF combo attack can be used for one of the challenges 
  
--------------------- 
  
Get v2.20.0 by simply updating your Docker image, running `npm install` on
your from-source checkout or by downloading the latest pre-packaged
distributions here:
https://github.com/bkimminich/juice-shop/releases/tag/v2.20.0!

  
Cheers, 
Bjoern
