Hello juice shoppers,

the next release (v2.16) of Juice Shop will switch to node.js v6 as the default. This is time wise coupled with v6 going officially into LTS state. Juice Shop will still work and be CI tested with v4. We dropped official v5 support and testing, because that version is abandoned by node.js development team.

TL;DR: For users, things should work as they did before! Resume your challenge hunt! 🏹
