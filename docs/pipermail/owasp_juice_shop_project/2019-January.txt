Dear all,

we just released v6.0.0 of the Juice Shop CTF Extension which adds support
for the new data format of CTFd 2.x and some fixes! See below for the full
release notes. Get the latest version via
https://npmjs.com/package/juice-shop-ctf-cli with

npm i -g juice-shop-ctf-cli

Our detailed setup guide for CTFs with the Juice Shop was updated as well:

https://bkimminich.gitbooks.io/pwning-owasp-juice-shop/content/part1/ctf.html

Enjoy! Cheers,
Bj?rn

v6.0.0 <https://github.com/bkimminich/juice-shop-ctf/releases/tag/v6.0.0>

This release contains breaking changes ([image: zap]) which will make
existing config files for the Juice Shop CTF Extension invalid!

CTF Frameworks

   - #57 <https://github.com/bkimminich/juice-shop-ctf/issues/57>: Added
   support for CTFd 2.x in interactive mode and configuration file
   - Renamed "CTFd" framework selection into "CTFd 1.x" ([image: zap])
   - Set "CTFd 2.x" as the default framework selection ([image: zap])

Bugfixes

   - Added missing category *Difficulty 6* to FBCTF data exports
   - insertHintUrls is now truly optional in config files with FBCTF as
   ctfFramework
