Dear infinitely valued customers, 
  
we would like to let you know that a new version of our wonderful eCommerce
application has been released today. The following enhancements were made
for your pleasure and our profit:

  
Changes 
----------- 
* QR code modal dialogs for BitCoin and Dash donations 
* Challenges now have a category field (e.g. for Jeopardy-like CTFs) 
* Solved challenges now show a flag-code for CTFs 
* Fixed feedback carousel problems 
* Documentation updates 
* Dependency updates 
  
Translations 
---------------- 
* Czech 
* Russian 
* Indonesian 
  
You can find the latest release here for download
https://github.com/bkimminich/juice-shop/releases/tag/v2.21.1  or you simply
update your local installation/Docker image. Enjoy! 

  
Cheers, 
Bjoern
