# v2.18.0
[https://github.com/bkimminich/juice-shop/releases/tag/v2.18.0]

## Functional changes
* significantly increase difficulty of Bender's password change challenge
* change continue code generation (pre 2.18 continue codes will not work any more!)

## Non-functional changes
* Removed SauceLabs integration
  * no more cross browser tests
  * no more hanging builds due to sauceconnect tunnel problems
  * run e2e tests on Travis directly and for all node versions
  * CI cycle time down from 20min to 5-6min
  * no more need to limit parallel builds
  * e2e tests now impact build success
