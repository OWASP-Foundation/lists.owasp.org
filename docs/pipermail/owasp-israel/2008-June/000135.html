שלום חברים. 

היום יצא העיתון המכוונן של MSDN ישראל - MSDN Pulse (מהדורה שלישית).  העיתון מומלץ לכל מפתח .net  בעיתון ניתן ללמוד על טכניקות פיתוח חדשות, עדכונים, דוגמאות , מאמרים ועוד... 

במהדורת החודש אחד מהמאמרים הינו מאמר שאני כתבתי (מאמר ראשון מתוך סדרה של 3 מאמרים בנושא ... ) בנושא Code Access Security

להלן לינק למאמר: http://www.microsoft.com/israel/msdn/Article/CAS-Part1.mspx <http://www.microsoft.com/israel/msdn/Article/CAS-Part1.mspx>   

 

אשמח לקבל תגובות. 

 

להלן לינק לעיתון החודש: http://www.microsoft.com/israel/msdn/pulse/2008/06/default.htm <http://www.microsoft.com/israel/msdn/pulse/2008/06/default.htm>  

 

בהמשך אני מתכוון להעביר הרצאה בנושא (CAS) אצלנו בקבוצה - .net security user group  http://www.microsoft.com/israel/communities/usergroups/securedev.mspx <http://www.microsoft.com/israel/communities/usergroups/securedev.mspx>  . 

אשמח לדעת מי מחברי הקבוצה מתעניין בנושא ורוצה לשמוע עליו באחד מהמפגשים הבאים. 

כולם מוזמנים לקרוא וללמוד משהו :-) 

 

המשך שבוע טוב.
