Hi all,

Amdocs is looking for information security analysts to fill a new role:


Role Definition:
• As an Information Security Analyst, candidate has to focus on identifying
and assessing vulnerabilities in software systems, Networks and mobile
based application.
• The major focus will be on Application Penetration testing followed by
Network Penetration Testing and Mobile Security assessments.
• The work involves Test Case Creation, Penetration Testing, Source code
reviews, Report Creation & presentation to stakeholders along with
operation and construction of tools to assist in these tasks.
• To activily contribute to the Vulnerability management efforts of the
organization via developer query resolution on vulnerabilities and defect
tracking to closure.
• Well versed with OWASP – Top Ten and WASC Threat Classifications
• Expertise in Vulnerability Assessment and Penetration Testing of Web
Applications
• Business‐Logic based application testing
• Penetration testing of Mobile applications and websites.
• Exploitation of the issues found and presenting the impact occurred
• Source Code Reviews - Well versed in Java Secure Code Review
• Expertise in Automated Scanning using CheckMarx  and Fortify
• Well versed with OWASP Code Review concepts & identifiers



3 reasons why you should apply for this role!
1. Application Security Testing (Web based, Thick client, web services,
Mobile)
2. Network Security Testing (Network, OS, Databases etc)
3. Static Code Analysis/ Secure Code Review



Critical Experiences and skills:
Must have minimum 3 years of experience in application and network
penetration testing, secure code reviews etc.
1. Understanding of the nature and sources of security vulnerabilities, how
to identify and exploit them
2. Strong expertise in security technologies and significant experiences in
information technology focusing on security related vulnerabilities
3. Good to have programming experience in Java, shell scripting, Perl, or
Python
4. Sound Knowledge of TCP/IP protocol Stack, HTTP protocol, encoding
standards, encryption technologies and development frameworks.
5. Familiar with popular tools:.
a. Application Proxy: Burp suite, Paros, OWASP ZAP, WireShark
b. Vulnerability Scanners: IBM AppScan, HP WebInspect, Nessus, NTO Spider
c. Exploit Toolkits: Metasploit, Exploit DB etc



Soft Skills:
1. Ability to work directly in a customer facing environment
2. Interact with management and Customer Delivery groups to co-ordinate
effort estimates and technical discussions.
3. Excellent English communication skills (verbal and writing)
4. Have an ownership mentality to resolve issues on own or with the help of
others



Please Apply using the the following url:
https://career4.successfactors.com/sfcareer/jobreqcareer?jobId=31949&company=amdocs


Thanks.
