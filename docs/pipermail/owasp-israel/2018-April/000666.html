Hey All
I would like to invite you to our DevSecOps meetup, hosted by Soluto.
I’ll introduce Glue, a tool developed by OWASP that ease the integration of security tools into the CI - including live demo. The tool has built in capabilities for filtering (for example, handling false positives) and reporting (for example to Jira or TeamCity).
Danny from Snyk will share details and insights from their research into open source security and demonstrate how it is possible to automatically identify and track publicly known vulnerabilities from the CVE database, source code hosted on GitHub, and various other interesting sources.

The talks will be recorded, and we’re working on live streaming from our Twitter account - @SolutoEng.

Looking forward to see you all!

Omer
