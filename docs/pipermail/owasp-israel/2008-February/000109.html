שלום לכולם,

מבקש להזכיר לכם שמחר יום שלישי( 05.02.08 ) מתקיים מפגש קהילת אבטחת המידע למפתחי .net במיקרוסופט רעננה(הגעה משעה 17:30 , תחילת הרצאה 18:00 ) .

המפגש יועבר ע"י ארז מטולה, מומחה אבטחת מידע באפליקציות ומנהל תחום האפליקציות בחברת 2bsecure.

המפגש בנושא דרכים ושיטות לאתר מפגעי Cross Site Scripting (XSS) באפליקציות.

 

מפגעי XSS ממוקמים במקום הראשון ברשימת OWASP TOP 10, ונחשבים למפגעים הבעייתים ביותר באתרי WEB אשר יכולים להוביל לבעיות פישינג, defacement, גניבת זהויות וכו'.

המצגת תארך כשעתיים וחצי והינה הרחבה למצגת שהועברה לראשונה ב OWASP Conference האחרון.

 

כמו כן מחר אנחנו חוגגים שנתיים פעילות לקבוצה, ולכן הזמנו עוגה כדי שנוכל לחגוג את האירוע כמו שצריך, במילים אחרות אפשר להגיע רעבים ... :-) 

 

כולם מוזמנים להגיע, ההשתתפות הינה בחינם.

להלן לינק להרשמה:
