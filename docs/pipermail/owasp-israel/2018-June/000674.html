Application Security Engineer (Entry Level)

Technical Skills:

-          Hands on experience developing web applications

-          Understanding of application technologies and its components

-          Expertise with REST APIs

-          OSCP/GWAPT Certification - Advantage

Soft skills:

-          Proficiency in English for written and verbal communication

-          Ability to multitask and be able to juggle different tasks with ease

-          Analytical Skills
