שלום לכולם,

 

להזכירכם, פגישת OWASP ישראל הקרובה תהיה מחר, ביום שלישי ה-22/6/2010 בשעה
17:00. 

הפגישה תיערך במשרדי IBM Watchfire, ספיר 1, הרצליה. (הוראות הגעה מפורטות
מצורפות למטה).

 

מי שעוד לא שלח לי אישור הגעה – אנא שלחו לי אישור. זה עוזר לנו להתארגן יותר
טוב למפגש!

 

אג'נדה:

SDL for Agile

מרצה: אבי דגלן

בהרצאה זו יסקור אבי את הקשיים העומדים בפני ישום אבטחת מידע בעת פיתוח
במתודולוגיית Agile, וכיצד ניתן לשלב תהליכי פיתוח מאובטחים בעולם ה-Agile.

 

Voice Biometrics

מרצה: אלמוג עלי-רז, Per Say

בהרצה זו יסקור אלמוג את טכנולוגיית הזיהוי הביומטרית מבוססת קול וכיצד ניתן
לשלבה באפליקציות ווב ומובייל.

 

SQL Injection Anywhere

מרצה: גיל כהן, Hacktics

בהרצאה זו יסקור גיל טכניקה חדשה לביצוע SQL Injection המאפשר ניצול של SQL
Injection בכל מצב, גם במצבים בהם טכניקות אחרות נכשלות, לצורך שליפת מידע.
