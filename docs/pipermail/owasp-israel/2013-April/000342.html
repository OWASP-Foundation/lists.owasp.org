Hi all,
Trustwave SpiderLabs have several open positions in Israel which you may find very compelling:

Security Researcher<https://hire.jobvite.com/j/?cj=olvrWfwX> and Security Research Intern<https://hire.jobvite.com/j/?cj=ooSSWfwO> - Key team members of the web server security research team whose focus is tracking new trends in the web application security field. These positions will conduct vulnerability research on web applications and other server-side web attacks. For more details: https://hire.jobvite.com/j/?cj=olvrWfwX and https://hire.jobvite.com/j/?cj=ooSSWfwO

Security Researcher<https://hire.jobvite.com/j/?cj=ot5kXfwz> - A key team member of the web client security research team whose focus is tracking vulnerabilities and exploits for browsers and other client-side software. The researcher will evaluate new emerging threats, exploitation methodologies and techniques and develop detection logic. For more details: https://hire.jobvite.com/j/?cj=ot5kXfwz

Interested candidates, who qualify the job requirements, are welcome to submit your resumes.
