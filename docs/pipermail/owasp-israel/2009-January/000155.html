אנחנו מעוניינים לגייס לחברת 2Bsecure עובד נוסף לצוות אבטחת אפליקציות (מחלקת ייעוץ), עבור תחום Web Application Firewall   .
להלן הדרישות לתפקיד:

יכולות טכנולוגיות כולל:

 1.  ידע באבטחת מידע בכלל ואבטחת מידע באפליקציות בפרט,  שנה ניסיון מינימום.
 2.  היכרות לעומק עם מגוון סוגי המתקפות האפליקטיביות, עלפי OWASP.
 3.  ידע כללי במערכות אבטחת מידע – FW,WAF, IPS וכו' - יתרון.
 4.  ידע \ ניסיון (יתרון) בפיתוח עבור מערכות אינטרנטיות.  ברמת הבנה, ידע.
    *   הכרת שפת JAVA\JAVA SCRIPT,HTML,.NET לסוגיו. Visual Basic. PHP,שפות פיתוח נוספות - יתרון.
 5.  הכרת פרוטוקולי -  HTTP, SOAP, TCP, וכו'
יכולות אישיות כולל:

 1.  יכולות הדרכה- יתרון.
 2.  יכולת כתיבת מסמכים בשפה עברית \ אנגלית.
