We (AppSec Reseach Group of Checkmarx) keep growing and looking for an
AppSec Researcher who is passionate about security but also about

or CV.

Some more details:

AppSec Researcher and E-Learning content creator DESCRIPTION

In addition to our Application Security testing solutions, our SaaS
platform is built specifically to train, refresh and validate Software
Developers and DevOps personnel on Application Security and Secure Coding
principles; helping business to develop and maintain more secure software
and applications.

We are looking for a talented AppSec researcher and content writer for
application security courses for beginners, intermediate, and advanced
learners.

The AppSec Researcher is part of the Checkmarx AppSec Research Group,
working with other researchers, developers and Application Security content
writers.


Responsibilities

   - Research security aspects of coding languages and technologies.
   - Come up with a full course or specific lesson ideas in different
   programming languages and technologies.
   - Work closely with technical writers - Explain and write vulnerability
   descriptions.
   - Develop basic vulnerable/safe applications for education purposes.
   - Review application security lessons content.
   - Perform code review and analysis of security results.
   - Support Checkmarx developers with the development of Checkmarx
   solutions by providing application security guidance.

Requirements

   - Ability to plan and perform independent research.
   - Good general information security knowledge – Hands-on experience is
   necessary.
   - Adept in application security – Deep understanding of security
   concepts, vulnerabilities, mitigations and coding practices.
   - Security work experience in a professional environment (security
   research, web application penetration testing / secure development /
   security code review).
   - Development experience both with compiled and script languages.
   - Ability to learn new programming languages and programming
   technologies independently.
   - Familiarity with CWE, AppSec compliance and application security
   standards like OWASP’s.
   - English language – High working level.
   - Strong communication skills.
   - An advantage - Experience in training or teaching.
