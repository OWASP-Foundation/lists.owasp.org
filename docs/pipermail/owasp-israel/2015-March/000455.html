Hi,

See below - this is a tremendous opportunity for anyone who ever wanted to do web security research at huge scale!!!



Overview:The Senior Security Researcher will be a key team member of Akamai’s Threat Research team, and will focus on tracking new trends in the web application security field, and conduct research on new trends in web application layer attacks, analyze and evaluate new threats and develop defensive protections - all based on Akamai’s unparalleled security big data system, that stores over 4 Petabytes of data, from Akamai’s visibility into as much as 30% of all Internet traffic. The Senior Security Researcher brings experience in web attack trends and research and uses it to extend the security research and the detection capabilities of the Akamai’s Web Application Firewall (WAF) product, Akamai’s Web Client Reputation, as well as other Akamai cloud security offerings.

Responsibilities:

- Track new trends in the web application security field by researching Akamai’s security big data
- Conduct attack trend & web-based exploit research using security big data and open source intelligence
- Analyze and evaluate new web application security threats, and develop protection mechanisms for Akamai’s WAF, Client-Reputation and other security offerings
- Author research whitepapers, news articles and blog posts on relevant topics
- Engage in community web application security efforts such as OWASP and WASC

Qualifications:

Basic Qualifications
- BA in CS or similar field, or equivalent experience
- Experience in a security research position-4+ years
- Hands-on experience in researching big data & security intelligence
- Proven understanding of databases and SQL
- Proven strong understanding of Internet protocols including HTTP, SSL/TLS, TCP/IP, DNS
- Programming skills: scripting languages (Python, etc.), web-related languages (PHP, ASP.NET<http://ASP.NET>, JavaEE, JavaScript)
- Creativity and innovation
- Self-motivated, independent and able to understand complex systems
- Bilingual Hebrew/English, verbal and written

Desired Qualifications

- Experience in the web application security field is an advantage
- Experience in writing vulnerability and attack detection signatures is an advantage
- Experience with Hadoop & HIVE query language
