Time of the 3rd meeting of OWASP IL! The Open Web Application Security
Project (OWASP) is an all-volunteer group that produces free,
professional-quality, open-source documentation, tools, and standards. The
OWASP community facilitates conferences, local chapters, articles, papers,
and message forums. It is most known for the OWASP guide and the OWASP top
10, which are among the most important references regarding applications
security. 

 

The 32nd meeting of the Israeli OWASP chapter will be held on March 29th,
17:15, at Breach Security <http://www.breach.com/>  offices, 11th Bareket
St. Herzliya. Participation is free and open to all, but please inform us
(mail me at ofers at breach.com) that you are coming so room is limited.

 

Feel free to distribute this e-mail to others in your organization or
outside that you think may be interested in the meeting. You can also
register to the OWASP Israel mailing list at
http://lists.sourceforge.net/lists/listinfo/owasp-israel in order to receive
updates regarding chapter's meetings. For further details please contact me.

 

The meeting agenda will be: 


17:15

Gathering, Socializing and Pizzas

התכנסות, מפגש ופיצות


17:30

The Web Application Firewall Evaluation Criteria
Ofer Shezaf, CTO, Breach Security

 

The Web Application Security Consortium had released standard criteria for
evaluating web application firewalls that Ofer has co-wrote. The new
document provides an opportunity to review the web application firewalls
market.

 

The presentation will discuss the new standard, its significance as well as
recent developments in web application firewalls that are evident in the
standard. 

תקן הערכת חומות אש ליישומי Web

עפר שיזף, מנהל טכנולוגיות ראשי, חברת בריץ' סקיוריטי

 

הארגון Web Application Security Consortium פרסם תקן המשמש לצורך הערכה
והשוואה של Web Application Firewalls שעפר הוא בין כותביו. תקן זה הוא הזדמנות
מצוינת לסקור את שוק ה- Web Application Firewalls.

 

ההרצאה תציג את התקן החדש, תדון במשמעויותיו ותסקור התפתחויות אחרונות במוצרי
Web Application Firewall המתבטאות בתקן.


18:00

Google Hacking 
Eyal Gruner, BugSec

 

Al Alqueda guide states that 80% of the useful information can be gathered
through open sources. Hackers have also found that out and Google has become
heaven for locating vulnerabilities in web sites.

 

Eyal will explain how advanced Google usage helps hackers and will
demonstrate how to locate web application vulnerabilities using Google.

גוגל הקינג

אייל גרונר, חברת בגסק

 

מדריך של אלקאידה מציין כי 80% מהמידע השימושי ניתן למצוא במקורות גלויים. גם
הקרים גילו זאת וגוגל הפך לגן עדן לאיתור פגמים באתרי ווב.

 

אייל יסביר כיצד חיפוש מתקדם בגוגל עוזר להקרים וידגים איך למצוא פגמים באתרי
ווב באמצעות גוגל.


19:00

Preventing Spoofing, Phishing and Spamming by Secure Usability and
Cryptography
Prof. Amir Herzberg, dept. of computer science, Bar-Ilan University, Israel

 

Spoofing, phishing and spamming are of the worst security problems in the
Internet. Amir will present vulnerabilities in the current email and web
systems, causing the proliferation of such attacks. 

 

Amir will then discuss some recent proposals made by him as well as others
to improve security against these threats. Some solutions involve secure
usability, some use (simple) cryptographic protocols, while others involve
both areas.

מניעת התחזות, פישינג ודואר זבל באמצעות "שימושיות מאובטחת" וצפינה

פרופ. אמיר הרצברג, מדעי המחשב, אונ' בר-אילן

 

מניעת התחזות, פישינג ודואר זבל הם מבעיות האבטחה החמורות באינטרנט. אמיר יציג
את הפגמים במערכות דואר אלקטרוני וווב המאפשרים את ההתרבות של התקפות אלו.

 

אמיר ימשיך ויציג הצעות חדשניות שהוצאו על ידו ועל ידי אחרים לשיפור האבטחה
כנגד התקפות אלו. כמה מהפתרונות דורשים "שימושיות מאובטחת", אחרים צפינה ואחרים
משלבים בין השיטות.
