*About the Group*

Security is a key element of LivePerson's operations and DNA, as thousands
of customers trust our platform and expect us to deliver the most secure
services to allow storage and processing of their sensitive
data. LivePerson’s Security team has one of the most interesting and
challenging roles: our main objective is to provide a world class security
framework that enables the business and supports its growth. Our team is
deeply involved in all major processes within the company, to identify
potential security risks and outline the best solutions to help protect our
customers and assets.



*Job Requirements*

We are looking for a white-hat hacker to work with R&D, QA, Product and
System teams on building Security into our applications. As an application
security specialist you will be leading all Secure Software Development
Lifecycle processes at LivePerson and implementation efforts of security
design, coding & testing methods within the company.

In addition, manage penetration testing and vulnerability assessments
processes (internal and external facing).



*Experience/Skills required *

   - True hacker by mind & by heart, curious tech geek
   - At least 3 years in the security industry, mainly in Application
   Security / PT roles:

- Vast knowledge in web application ( server side and client side)

- Experience with penetration testing processes

- Experience with vulnerability assessment

- Good knowledge with JAVA and JavaScript - advantage

- Working with Static code analysis - advantage

- Knowledge of OWASP frameworks – advantage

- Security Incident Response (systems and processes)  - advantage



Perfect English (both speaking and writing)

Strong leadership and process management skills

Familiar with Cloud infrastructure and technologies (AWS, OpenStack)

Active in application security communities, blogs, etc.



*Other Requirements*

   - Ability to work long hours (US timezone in addition to the local
   Israel timezone)
   - Perfect English (both speaking and writing)
   - Ability to travel occasionally.
