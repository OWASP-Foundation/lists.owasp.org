Amdocs is looking for Application Security analyst/Penetration tester to join our Application Security Analyst group.
The position is as follow:
Brief Description
APDO testing team is looking for a Security Analyst to join a growing team offering a professional and challenging position, assigned to ensure the quality and readiness of the Amdocs products such as Billing, CRM, OMS and AMSS from Application Security perspective.
Security Analysts should have good technical skills and in dept knowledge in SQL and Web applications. They should be autodidact, good problem solvers, thinkers and analyzers with attention to details. They should understand and respond to new technology in the security and hacking world.
Detailed Description
Performing application level penetration tests (Hacking), security reviews, and consulting Amdocs products developers in order to secure their applications.                                                                                                                                              Design of security test rules and test scenarios based on Specs and Communication with Developers.
Analyze security faults as they arise in design or execution phase.                                                                                                                                                                          Identify and propose work-around for critical defects.
 Behavioral Competencies

Education / Experience
