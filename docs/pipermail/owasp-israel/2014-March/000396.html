*1) Application Security Specialist*

Job Description:
Security penetration testing group offers an exciting and challenging
position for a better security quality of Amdocs products.

Main roles and responsibilities:
Application level penetration tests.
Vulnerabilities detection and ability to provide recommendations for
mitigation.
Consulting and guidance throughout product Software Development Life Cycle
(SDLC).
Gain broad knowledge of Amdocs Product's integrations; understand suite
architecture, technology, and its business.
Good communication and presentation skills.

Job Requirements:
 At least one year experience in conducting pen-tests for web applications.
Experience in conducting pen-tests for mobile applications.
Deep up-to-date knowledge in threats, vulnerabilities, security breaches
and useful tools
Such as scanners, proxies etc.
Knowledge in programming (SQL, Java, Python, C++, DotNet) & protocols
(TCP/IP, HTTP, SSL, DNS)
Fluent in UNIX, troubleshooting and scripting.
Writing skills for reports & policy documentation.
Self-learner.
English written and verbal skills.
B.Sc. graduated in Computer Science or relevant qualification in
information security - advantage.
Experience in code review - advantage.
Other relevant skills (Reverse engineering, Threat-Modeling, Cryptography)
- advantage.
