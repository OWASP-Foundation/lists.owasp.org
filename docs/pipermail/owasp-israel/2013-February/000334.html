Amdocs Israel are recruiting experienced application security specialist.

Security penetration testing group offers an exciting and challenging position, performing Application level penetration tests, Vulnerabilities detection and recommendations for mitigation, Consulting and guidance throughout product Software Development Life Cycle (SDLC).

Requirements:

*         At least one year experience in conducting pen-tests for web applications.
