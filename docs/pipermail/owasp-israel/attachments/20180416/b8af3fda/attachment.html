<p class="MsoNormal"><span style="font-size:11.0pt">Hey All<o:p></o:p></span></p>
<p class="MsoNormal"><span style="font-size:11.0pt">I would like to invite you to our DevSecOps meetup, hosted by Soluto.
<o:p></o:p></span></p>
<p class="MsoNormal"><span style="font-size:11.0pt">I’ll introduce Glue, a tool developed by OWASP that ease the integration of security tools into the CI - including live demo. The tool has built in capabilities for filtering (for example, handling false positives)
 and reporting (for example to Jira or TeamCity). <o:p></o:p></span></p>
<p class="MsoNormal"><span style="font-size:11.0pt">Danny from Snyk will share details and insights from their research into open source security and demonstrate how it is possible to automatically identify and track publicly known vulnerabilities from the
 CVE database, source code hosted on GitHub, and various other interesting sources.<o:p></o:p></span></p>
