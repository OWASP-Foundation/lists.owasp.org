Hi Everyone,

 

The next OWASP Israel meeting would be on Thursday, March 26th, 16:00-19:00
at Tel-Aviv university, hosted by the Checkpoint Institute for Information
Security Studies. This meeting replaces the one formerly scheduled for March
17th at F5 as we found out that we need much more room than they have.

 

We are looking for speakers for the meeting. Let me know if you want to
talk.

 

We are especially interested in customer experience, so if you work in a
security department in an organization and would like to talk about your
application security experience, let me know. It does not need to be an hour
long presentation, but can also be a short 15 minutes talk or a panel with
some others that I will arrange.
