שלום לכולם.

אני שמח להזמין אתכם להשתתף במפגש קבוצת .net security user group הקרוב שיתקיים בתאריך – ה -  1.4.08 (וזאת לא בדיחה :-) ). 

נושא המפגש: איסוף מידע – הצעד ה-1 של התוקף.(אירוע מספר 1032372543)

איסוף מידע הינו השלב ה-1 והבסיסי ביותר של התוקף הזדוני לפני תקיפת אפליקציה. במפגש זה נתמקד בטכניקות בהם משתמש התוקף בכדי לאסוף את המידע המקדים לתקיפה. כמו כן נראה שימוש בכלים אוטומטים הניתנים להורדה בחינם.

טל ארגוני (יועץ אבטחת אפליקציות מחברת 2bsecure ) ידגים את השימוש בכלים וכיצד ניתן באמצעות הטכניקות המוצגות לאסוף את המידע הרלוונטי ביותר לתקיפה ממוקדת.

כמו כן, נדון בדרכים לצמצום האינפורמציה הנחשפת בכדי להקשות על תוקפים אופציונאליים.

- לקראת סיום המפגש נפתח פאנל לשאלות בנושא security כלליות, כולם מוזמנים להשתתף.
