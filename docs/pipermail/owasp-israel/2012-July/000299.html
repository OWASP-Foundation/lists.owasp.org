Hello,

Akamai is looking for talented researchers in the field of web application security.

The application security researcher will be a part of Akamai’s cloud security team whose focus will be to monitor and analyze new trends in the web application security field, and to conduct security research on web and mobile application technologies, analyze and evaluate new threats and develop defensive protections.

Here's a link to Akamai's security offerings web page: http://www.akamai.com/security

Responsibilities

• Track new trends in the web application security field
• Conduct vulnerability research on web and mobile application technologies
• Analyze and evaluate new web application security threats, and develop protection mechanisms for Akamai’s security offerings
• Engage in public web application security efforts such as OWASP and WASC

Qualifications

• BA in CS or similar field (or equivalent experience)
• 2+ years experience in a security research position
• Proven strong understanding of Internet protocols including HTTP, SSL, TCP/IP
• Proven experience in building secure internet and web systems and applications
• Programming skills: web-related languages is a must (e.g. HTML, JavaScript, PHP, ASP.NET<http://ASP.NET>, Java EE, RnR), scripting languages (Perl, Python, etc.)
• Creativity and innovation
• Self-motivated, independent and able to understand complex systems
• Bilingual Hebrew/English, verbal and written
• Experience in the web application security field (advantage)
• Experience writing exploit, vulnerability and attack detection signatures (advantage)
• Experience with web application firewalls (WAF) concepts and technologies (advantage)


Thank you,
