Subject: [OWASP Israel] New paper - Detecting remote file inclusion attacks
Message-ID: <8A1B4B015C52A54886DF15E9A97C7FB9F5B14D6F@MONET.utopiasystems.net>

Hi,

One of the big challenges standing in front of security experts is to detect attacks that cannot easily be detected using signatures; remote file inclusion (RFI) is a good example of such an attack.
In this paper I will try to present the challenges of accurately detecting RFI attacks. In order to achieve detection of unknown variants of the RFI attack I will try to define a generic solution to the problem instead of specific solution for known vulnerabilities. The solution is a generic rule set that will enable protecting applications from RFI attack.

http://www.breach.com/resources/whitepapers/downloads/WP_Detecting_Remote_File.pdf
