Looking for a Rockstar Security Researcher (Malware Focused)

Keywords: Malware analysis, C&C communication, Large-scale Data classification, DNS, HTTP(S), Botnets, Sink-holing.

Hey There,

We're slowly filling up the open researcher positions  at Akamai's Threat Research team. Don't miss the opportunity to join a team of high caliber world-renowned security researchers!. If you are interested in researching massive data sets, solving hard problems, and have a deep passion for security, Akamai is the place for you.

This role was created to help identify and classify potential threats and malicious activity using Akamai's extensive Cloud Security Intelligence big data platform

Responsibilities:

  *   Research latest threats on a global scale
  *   Work with team members and developers on creating innovative solutions through automation, research, and development
  *   Research and implement innovative security classification techniques
  *   Construct and prototype PoC classification models
  *   Build new algorithms for large-scale classification
  *   Contribute to security communities (papers, blogs, presentations)
  *   Identify malware, hostile websites, network communications, and vulnerabilities
  *   Analyze and reverse engineer malware

Required Qualification:

  *   Strong understanding of the DNS and HTTP(S), how they function, and how they are used across the Internet
  *   Strong understanding of malware communications, botnet architectures, exploit kit delivery techniques
  *   4 years minimum of real-world security experience
  *   Strong knowledge of TCP/IP stack including TCP, UDP, and IP protocols and network traffic analysis
  *   BA in CS or similar field or equivalent years in security trenches
  *   Experience with data modeling, graph theory, or machine learning recommended
  *   Experience at analyzing and mining very large data sets by using existing utilities and writing new tools if needed
  *   Programming experience. Python, C/C++ or Java
  *   Experience with at least one data storage/database system (preferably experience with relational and non-relational databases)
  *   Working knowledge of incident response practices

Link to Jobs @ Akamai: http://jobs.akamai.com/job/Herzliyya-Principal-Security-Researcher-Job/279706500/
