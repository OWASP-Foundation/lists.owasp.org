Hi everyone,
as you know it's planned to transfer the OWASP Switzerland Local Chapter
into an association so we can handle finances, contracts etc. more easily.

Attached you'll find a first draft of the bylaws (in German). If there's
any feedback or question, please let me know :)

If you want to become a founding member, you should make sure to have a
valid OWASP Membership as that's the requirement to become a member of
OWASP Switzerland. (https://www.owasp.org/index.php/Membership)

A concrete date for the founding will be announced as soon as possible.

Best regards and have a nice weekend,
