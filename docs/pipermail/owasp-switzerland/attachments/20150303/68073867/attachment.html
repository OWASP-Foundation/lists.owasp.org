<div>As partner of the Security Interest Group Switzerland, Swiss Cyber Storm is organizing a special tech track at the SIGs Technology Summit Information on the 23rd of June 2015 in Bern, Switzerland. The SIGS Technology Summit is an emerging platform to learn about new technologies, methods, oportunities, challenges and threats in the information security world that help decision makers and IT security professionals shape their roadmap.</div>
<div>The SIGS Technology Summit offers 5 parallel session where the Swiss Cyber Storm track is one of them.</div>
<div>Swiss Cyber Storm is a non-profit organization that organizes the Swiss Cyber Storm Security Challenge and Swiss Cyber Storm conference, an international IT security conference taking place on the 21st of October in the KKL Lucerne.</div>
<div> </div>
<div>Scope</div>
<div> </div>
<div>In particular, we are looking for topics in the following domains:</div>
<div> </div>
<div>    Operating system and application security</div>
<div>    Wired and wireless network security</div>
<div>    Mobile communication security</div>
<div>    Forensics and anti-forensics</div>
<div>    Digital privacy and anonymous communication</div>
<div>    Advanced Exploitation (Bypassing all the security measures)</div>
<div>    Malware collection and analysis</div>
<div>    Botnet analysis</div>
<div>    Covert communication channels</div>
<div>    Electronic voting</div>
<div>    Industrial espionage</div>
<div>    Industrial control system security / SCADA</div>
<div>    Security metrics and visualization</div>
<div>    Intrusion detection and prevention</div>
<div>    Cloud computing security</div>
<div>    Cryptography and security protocols</div>
<div>    Biometric system security</div>
<div>    Quantitative and model based IT risk management</div>

<div>Submissions from academic and scientific institutions are welcome.</div>
