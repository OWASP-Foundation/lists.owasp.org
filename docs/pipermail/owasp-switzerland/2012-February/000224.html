Hi everyone,
I hereby like to invite you to the next OWASP Switzerland Meeting.

This time we decided to not strictly focus on web related security
issues, as OWASP covers all application security related areas.

* What:
"Analysis of the RSA Security Breach" by Gianni Gnesa

Several posts have been written and several talks have been given
about the RSA Security Breach, but none of these posts or talks have
ever explained in detail the attack. This talk will analyze each part
of the attack, from the malicious excel file to the 0day used to
install a backdoor into an RSA employee's PC. Finally, I will give
some recommendations on how to protect your IT infrastructure and
minimize the impact of 0days.

Gianni Gnesa is a security researcher who works for Ptrace Security, a
Swiss-based company dedicate to providing information security
solutions.

If someone else can offer an additional short talk 5-30min, please let
me (Sven) know.
