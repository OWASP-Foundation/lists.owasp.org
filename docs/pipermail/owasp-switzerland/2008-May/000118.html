Dear Colleague,

next Tuesday, Urs P. Kuederli from Microsoft Switzerland will give a
talk in the ZISC Information Security Colloquium. The talk is open to
the public and will be given in English.

- ---------------------
Topic: "Software Security Engineering and Development Today
Security Development Lifecycle (SDL) - A Process to Develop More Secure
Software"

Speaker: Urs P. Kuederli, Microsoft Switzerland

Place: ETH Zurich, HG F 5 (main building, floor F, room 5)

Date: Tuesday, May 27, 2008

Time: 17:15 - 18:15 h
- ---------------------


Abstract:

In the last couple of years, the threats have changed and current
software development methods fail to produce secure software. The short
introduction to Security Development Lifecycle (SDL) is about the need
for such a framework and about processes, examples and learnings. SDL
demonstrably changed the way Microsoft is building new software.
