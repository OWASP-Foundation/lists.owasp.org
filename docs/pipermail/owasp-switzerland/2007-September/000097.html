Am Samstag und Sonntag 29./30. September 2007 findet in der Aula der FH
Rapperswil das erste öffentliche Wargame der Schweiz statt.

Als Wargame wird eine IT Security Laboraufgabe bezeichnet, bei der die
Teilnehmer versuchen eine gestellte Hacking-Aufgabe zu lösen.
Beispielsweise ist das Ziel, 'root' Rechte in einem Linux System oder
mit Shatter die Admin Privilegien unter Windows zu erlangen.

Der Anlass richtet sich an Informatiker, Studenten von der ETH,UNI,FH
oder IT Security Interessierte, die Freude am Lösen von Puzzles haben.
Die Wargames sind letztlich Hacking Challenges, die Sie im Labor
entschlüsseln werden. Das LERNEN steht im Vordergrund!

Der Anlass hat den Charakter einer LAN Party - ist kostenlos und findet
über 2 Tage in der Aula der FH Rapperswil statt.

Mehr Infos unter

http://www.hackergame.ch/

Grüsse

Ivan Buetler
