    Yes, these encoders are not meant to protect HTML markup. The HTML
    tags will be encoded and will be displayed as text in the browser ON
    PURPOSE so they do not execute. This is how you stop XSS.<br>
    <br>
    If you need to accept and render HTML from user sources, use this
    project instead.<br>
