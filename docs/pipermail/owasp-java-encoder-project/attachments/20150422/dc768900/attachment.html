    > What canonicalization functions are you thinking of?<br>
    <br>
    Detection of encoded input.<br>
    <br>
    Like if a user took an attack and then URL encoded it and then HTML
    Entity encoded it in an attempt to circumvent filters.<br>
    <br>
    By noticing that input is already decoded - you might want to reject
    it as a form of intrusion detection for certain fields.<br>
    <br>
    So like...<br>
    <br>
    if (rawContent !=
    htmlDecode(urlDecode(javascriptDecode(cssDecode(rawContent)))) {<br>
       //we might want to reject this<br>
    }<br>
    <br>
    //kindda like this<br>
    canonicalize(rawInput) {<br>
      return
    htmlDecode(urlDecode(javascriptDecode(cssDecode(rawContent))));<br>
    }<br>
