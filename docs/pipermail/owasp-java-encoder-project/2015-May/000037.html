In your example you didn't encode data properly.

You need to change this:

        <% String pass = data;     
            if(pass!=null){
                out.println(pass);
            }
        %>

To this or similar:

        <% String pass = data;     
            if(pass!=null){
                out.println(Encode.forHtml(pass));
            }
        %>
