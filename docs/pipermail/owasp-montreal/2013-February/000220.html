OWASP Montréal - 26 février 2013 - Secure Coding pour Java

OWASP Montréal présente mardi le 26 février prochain une conférence
intitulé "Secure Coding pour Java". Sébastien Gioria, consultant sénior
en Sécurité des applications, Chapter Leader de l'OWASP France et membre
du OWASP Global Education Committee introduira la pratique de "Secure
Coding" en Java qui permet de développer une application java sécurisée
via différents matériels OWASP.

La présentation aura lieu le 26 février 2013 à 18h00.
