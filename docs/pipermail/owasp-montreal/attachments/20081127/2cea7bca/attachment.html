Some XSRF are very easy to find. Most &quot;logout&quot; feature are vulnerable to xsrf. A good way to find XSRF is to try to reproduce the usual requests with netcat. Once you get a &quot;feel&quot; how the application works and what parameters are needed, you can test it easily. Simply create a local html file and embed your request in img tag. Then, load your page and voila.<br>
