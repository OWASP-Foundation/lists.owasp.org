We would like to invite you to the first OWASP Montreal Chapter, Web
Application Security meeting on February 24th 2009 in Montreal!

This meeting is sponsored by Microsoft, CGI and Centrale Taxes.

* WHEN:
Tuesday, February 24th 2009, 18h00

* WHERE:
Location: 111 Duke 7th floor, Montreal, QC, H3C 2M1 (room 734.1)

* PROGRAM:
- Welcome, lunch and refreshments
- OWASP Goal and Top Ten (by Benoit Guerette, Centrale Taxes)
- Security Development Lifecycle for IT (by Rob Labbe, Microsoft)
- Montreal Chapter, what do you expect from us

* REGISTRATION:
Please send a mail to owaspmontreal at gmail.com if you plan to
attend, we can keep you updated on last-minute changes.

*** Registration is mandatory for security reasons in this building. ***

* WHO:

Anyone interested in Web Application Security (management, security
professionals, developers, students, etc). OWASP Montreal chapter
membership is free. All meetings are free. There are never vendor
pitches or sales presentations at OWASP meetings. Please forward to
anyone you feel would have an interest in these events.
