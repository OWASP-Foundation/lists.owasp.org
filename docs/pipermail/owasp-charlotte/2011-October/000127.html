I am a member of the Enterprise Developers Guild (
http://www.developersguild.org) here in Charlotte and I can see if we can
get on their agenda for next year.  It's a .NET developers group that has
meetings on every 4th Tuesday of the month as well as various Code Camps
that they run on the weekends.  They don't normally have application
security topics so it would be a good venue for introducing our group and
what OWASP can offer (e.g. - ESAPI).

With that said, I would like to introduce myself to the group.  My name is
Tim Leedy and have been a developer for around a decade and have been a
security professional since 2001.  I am excited about working with this
group and look forward to getting back into Application Security as I have
been away from the field for a few years.
