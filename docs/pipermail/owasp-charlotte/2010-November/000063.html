It's actually not unusual at all (from what I've seen) to move from network security to application security.  Application security as its own discipline is new enough (again, from what I've seen) that there aren't a lot of people looking for jobs in it that have been trained in the ways of appsec - so usually you see network security folks learn the application side, or you see application developers learn the security side.  I think the former is more common than the latter - again, just my experience.

But welcome aboard!  You're certainly not alone with the experience you bring.

Will
