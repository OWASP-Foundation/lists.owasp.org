-- (EVENTS @ YVERDON) // IT SECURITY DAYS
Trois jours durant, respectivement en Février, Mars et Avril prochain,
la Haute Ecole d'Ingénierie et de Gestion du canton de Vaud (HEIG-VD,
Yverdon) organisera les IT Security Days, un évènement exceptionnel
durant lequel des experts reconnus en sécurité de l'information
viendront diffuser leurs connaissances et partager leur expérience sur
différents domaines tels que le ethical hacking, la sécurité dans les
applications web et la guerre électronique.

Agenda:
- Jour 1: Ethical Hacking (9 février)
- Jour 2: Web applications (in-)security (16 mars)
- Jour 3: Security in the cyberspace (7 avril)

Pour cette occasion, la HEIG-VD a invité un représentant de la section
OWASP Suisse, qui participera à la journée dédiée à la sécurité des
applications web et présentera une activité de sécurité fréquemment
sous-estimée dans les projets de développement web: la modélisation de
menaces (threat modeling).

Informations et inscriptions:
http://securitydays.heig-vd.ch

(Attention: cette manifestation n'est pas organisée par OWASP. Merci
de contacter directement les organisateurs des IT Security Days à
travers le site web officiel pour toute information supplémentaire.)



-- (EVENTS @ YVERDON) // IT SECURITY DAYS
During three separate days from February to April 2011, the School of
Business and Engineering Vaud (HEIG-VD, Yverdon) will host the IT
Security Days, an exceptional event were recognized information
security experts will disseminate high-quality knowledge to
participating individuals and companies.

Agenda:
- Day 1: Ethical Hacking (February 9th)
- Day 2: Web applications (in-)security (March 16th)
- Day 3: Security in the cyberspace (April 7th)

As part of the Web Applications (in-)Security day, the HEIG-VD invited
an OWASP Switzerland representative to talk about one of among the
most under-estimated software security activities: threat modeling.

Information and registration:
http://securitydays.heig-vd.ch

(Warning: this event is not organized by OWASP. Please address all
inquiries directly to the IT Security Days organizers through the
official website.)
