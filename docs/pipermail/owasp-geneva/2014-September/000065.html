La très attendue quatrième édition du Testing Guide de l'OWASP est
terminée et désormais disponible au téléchargement.

---Téléchargement
https://www.owasp.org/images/1/19/OTGv4.pdf (document complet)
https://www.owasp.org/index.php/OWASP_Testing_Guide_v4_Table_of_Contents

---Nouveautés
1. En plus de la mise à jour des contrôles, le guide propose une
numérotation harmonisée avec les deux guides OWASP annexes: Developers
Guide et Code Review Guide.

Cette harmonisation permet de rapidement associer une bonne pratique de
développement (Developpers Guide) avec un test (Testing Guide) ou un
point de revue de code (Code Review Guide).


2. Quatre groupes de contrôles viennent s'ajouter à l'ancienne version
du guide:
- mécanismes de gestion de l'identité (Identity Management Testing)
- traitement d'erreurs et exceptions (Error Handling)
- usage de la cryptographie (Cryptography)
- code exécuté côté client (Client Side Testing)

contribuer à une traduction du document, voire la piloter, votre
participation est plus que bienvenue.

Contactez Andrew pour vous annoncer (andrew.muller at owasp.org), ou
Thomas/Antonio pour les plus timides.
