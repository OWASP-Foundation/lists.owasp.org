<p>Bien entendu, les listes de diffusion OWASP restent un <u>service gratuit</u>.<o:p></o:p></p>
<p>En cas de doute, le team se tiendra à votre disposition pour répondre à vos questions.<o:p></o:p></p>

<a href="https://lists.owasp.org/pipermail/owasp-leaders/2019-February/019695.html">
post</a> on the Leaders List  with some advice on setting to help out with this including
<a href="https://lists.owasp.org/pipermail/owasp-leaders/attachments/20190222/1bbda485/attachment-0004.png">
this screenshot</a>.  I'd recommend the advice on that <a href="http://lists.owasp.org/pipermail/owasp-leaders/2019-February/019694.html">
thread</a>.<o:p></o:p></li><li class="MsoNormal" style="mso-margin-top-alt:auto;mso-margin-bottom-alt:auto;mso-list:l0 level1 lfo3">
