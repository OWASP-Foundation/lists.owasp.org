Abonnées, Abonnés,

Vous le savez peut-être déjà, la 5ème édition de l'AppSec Forum aura lieu
les 4, 5 et 6 novembre prochains.

La journée du 4 sera dédiée à des formations d'une journée, sur les thèmes
suivants :
- Creating a strong two factors authentication device for less than CHF
100.-
- Secure Coding for Java
- Droid Application Hack & Smash
- XSS & PHP: A Happily Married Couple
- Understanding malwares

Deux journées de conférences bien remplies nous attendent les 5 et 6
novembre.

En sa qualité de partenaire de l'événement, votre section locale de l'OWASP
se réjouit de vous offrir un code de réduction de 10% sur le montant de
votre inscription, valable pour toutes les formations
En outre, les membres OWASP bénéficient d'un rabais supplémentaire, pour un
rabais total de 15% (pour les formations également).

Pour bénéficier du rabais abonné, il suffit de rentrer le code de réduction
pr0fit1z lors de votre commande.
Pour le rabais membre, contactez-nous par mail (geneva at owasp.ch), nous vous
transmettons le code spécifique.

Plus d'informations, un programme détaillé et les formulaires d'inscription
sont disponibles sur http://2014.appsec-forum.ch ...


Pour OWASP Geneva,
