Αγαπητοί φίλοι,

Η Ελληνική Ομάδα Εργασίας του Open Web Application Security Project
(http://www.owasp.gr) σας προσκαλεί στην παρουσίαση με θέμα "Detecting
Web Application Vulnerabilities Using Open Source Means -  Εύρεση Κενών
Ασφαλείας Web Εφαρμογών με τη χρήση ΕΛ/ΛΑΚ", στα πλαίσια του 3ου
Συνεδρίου Ελεύθερου Λογισμικού / Λογισμικού Ανοιχτού Κώδικα
(http://conf.ellak.gr),

την Τρίτη 27 Μαΐου 2008 και ώρα 11:15 στο Εθνικό Μετσόβιο Πολυτεχνείο,
στην Πολυτεχνειούπολη Ζωγράφου (Αίθουσα Τελετών Πρυτανείας).

Το αναλυτικό πρόγραμμα του συνεδρίου, που διοργανώνεται από το
e-business forum και την κοινότητα ΕΛ/ΛΑΚ, μπορείτε να το βρείτε στο
site: http://conf.ellak.gr

Η παρουσία σας θα μας τιμήσει ιδιαίτερα.
