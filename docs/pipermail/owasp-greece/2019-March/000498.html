Λυπούμαστε που εισβάλλουμε στη λίστα αλληλογραφίας σας, αλλά πρέπει να
ενημερώσω τα μέλη του Owasp ότι η υπηρεσία OWASP Foundation κλείνει όλες
τις λίστες αλληλογραφίας που εκτελούνται από τον Mailman στη λίστα
lists.owasp.org στις 22 Μαρτίου 2019.

Ακολουθεί ένας σύνδεσμος με ένα κλικ για να συμμετάσχετε στη νέα Ομάδα
Google που αντικαθιστά αυτή τη λίστα:
https://groups.google.com/a/owasp.org/forum/#!forum/greece-chapter/join

Ο νέος Όμιλος είναι επίσης διαθέσιμος στο διαδίκτυο στη διεύθυνση:
https://groups.google.com/a/owasp.org/forum/#!forum/greece-chapter

Λεπτομερείς πληροφορίες σχετικά με τον τρόπο συμμετοχής στο Συγκρότημα
βρίσκονται στη διεύθυνση
https://drive.google.com/open?id=1TBzgvB8Tb0aAZnEy2qYzLnrzJuRzK_T91Em09DVf5YA

ΣΗΜΕΙΩΣΗ: Δεν απαιτείται να έχετε λογαριασμό Google, διεύθυνση ηλεκτρονικού
ταχυδρομείου @ owasp.org ή να δώσετε τον αριθμό τηλεφώνου σας για να
συμμετάσχετε και να συμμετάσχετε στη νέα ομάδα Κεφάλαιο Ελλάδας - απλά μια
διεύθυνση ηλεκτρονικού ταχυδρομείου.

Τέλος, όλες οι λεπτομέρειες σχετικά με το σχέδιο απόσυρσης του Mailman,
συνδέσμους και συχνές ερωτήσεις είναι στη διεύθυνση:
https://www.owasp.org/index.php/Staff-Projects/Mailman-EOL

Ευχαριστώ για τις παρελθούσες και μελλοντικές συνεισφορές σας στην
κοινότητά μας!
