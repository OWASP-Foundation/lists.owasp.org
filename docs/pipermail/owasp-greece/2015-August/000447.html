Zητείται νέα/νέος απόφοιτος πανεπιστημιακού τμήματος πληροφορικής με ενδιαφέρον για την ασφάλεια υπολογιστών για άμεση πρόσληψη απο συμβουλευτική εταιρεία του εξωτερικού σε θέματα ασφάλειας πληροφοριακών συστημάτων.

Απαραίτητα Προσόντα
==
+ Πτυχίο (ΑΕΙ ή ΤΕΙ) τμήματος πληροφορικής τετραετούς φοίτησης
+ Άπταιστα γραπτά και προφορικά Αγγλικά
+ Πάθος για την ασφάλεια πληροφοριακών συστημάτων

Επιθυμητά Προσόντα
==
+ Γλώσσες προγραμματισμού (Python, Perl, PHP, Ruby, Java, Javascript)
+ Λειτουργικά συστήματα (Linux, Unix, Windows, OSX)
+ Δίκτυα υπολογιστών και πρωτόκολλα επικοινωνίας
+ Κρυπτογραφία
+ Βάσεις Δεδομένων
+ Θέματα ανωνυμίας και ιδιωτικότητας

Ειναι σημαντικό ο υποψήφιος να είναι πρόθυμος να εργαστει σκληρά και με επιμέλεια για να μάθει όλες τις τεχνικές και τα εργαλεία που χρειάζονται. Οι υποψήφιοι καλούνται επίσης να αναφέρουν τυχον δραστηριότητες που έχουν σε πεδία εκτός της πληροφορικής, όπως για παράδειγμα τη μουσική, τα ταξίδια, τον αθλητισμό ή την τέχνη.
