Αγαπητά μέλη,

Το OWASP για μια ακόμα χρονιά υποστηρίζει το 5ο Information Security
Conference που θα διαξεχθεί στις 15/2/2018 στο Αμφιθέατρο Μαρούσι Plaza.

Στην πέμπτη διοργάνωσή του, το Information Security Conference αναλύει τις
πολλές διαστάσεις των σύγχρονων ψηφιακών απειλών, οι οποίες
«τροφοδοτούνται» από τα συνεχώς αυξανόμενα user data, προσαρμόζονται και
μεταλλάσσονται βάσει των μέσων αντιμετώπισης αυτών και εμφανίζουν
επεκτατικές τάσεις σε cloud και διασυνδεδεμένα περιβάλλοντα.

Επιπρόσθετα, τα expert sessions της Παρασκευής, 16/2/2018, συμπεριλαμβάνουν
workshop για το OWASP SAMM, που μπορεί να βοηθήσει σημαντικά στην
ενσωμάτωση της ασφάλειας στο development lifecycle.

Περισσότερες πληροφορίες για το πολύ ενδιαφέρον συνέδριο και τα expert
sessions μπορείτε να βρείτε εδώ: http://www.eits.gr/

Παράλληλα, εξασφαλίσαμε 10% έκπτωση για τα μέλη του OWASP. Κατά την εγγραφή
σας μπροείτε να χρησιμοποιείτε τον κωδικό OWASP_GC για να εφαρμοστεί η
έκπτωση.

Θα χαρώ να έχουμε την ευκαιρία να τα πούμε από κοντά στο συνέδριο.

Κ. Παπαπαναγιώτου
