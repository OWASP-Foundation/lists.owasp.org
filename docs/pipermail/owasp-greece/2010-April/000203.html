Οι βασικοί ιατρικοί και επιστημονικοί όροι παγκοσμίως είναι στα Ελληνικά. Κανείς ξένος δεν μπήκε στον "κόπο" να τους μεταφράσει. Δεν νομίζω οτι αυτό έγινε κατά λάθος. Με την ίδια ακριβώς λογική (κατά την ταπεινή μου γνώμη) πρέπει να "κρατήσουμε" τους ξένους όρους ακριβώς ως έχουν. Είτε είναι εύηχοι είτε όχι δεν θα έλεγα οτι είναι δόκιμοι εφόσον αποτελούν πλέον ορολογία. Επομένως δεν θα πρέπει να μεταφραστούν καθόλου.

Καλησπέρα σε όλους,

Αυτό το καιρό γίνεται μία καλή προσπάθεια μετάφρασης του νέου OWASP
Top10 2010, η οποία βρίσκεται στα τελευταία στάδια.

Ο βασικός προβληματισμός της μεταφραστικής ομάδας αφορά κυρίως στην
απόδοση όρων ασφάλειας στα Ελληνικά. Οι δύο ακραίες απόψεις που θα
μπορούσε κανείς να διακρίνει είναι οι εξής: μετάφραση όλων των όρων
στα Ελληνικά και στον αντίποδα διατήρηση όλων των όρων στα Αγλικά.
Ενδεικτικά παραθέτω πιο κάτω απόπειρες μετάφρασης των Top10 ευπαθειών:

A1: Έγχυση (Injection)
A2: Cross-Site Scripting (XSS)
A3: Broken Authentication and Session Management
A4: Ανασφαλής Απευθείας Αναφορά σε Αντικείμενα (Insecure Direct Object
References)
A5: Πλαστογράφηση αίτησης μεταξύ θέσεων (Cross-Site Request Forgery - CSRF)
A6: Λανθασμένες Ρυθμίσεις Ασφάλειας (Security Misconfiguration)
A7: Ανασφαλής Κρυπτογραφική Αποθήκευση (Insecure Cryptographic Storage)
A8: Αποτυχία Περιορισμού της Πρόσβασης URL (Failure to Restrict URL Access)
A9: Ανεπαρκής Προστασία Επιπέδου Μεταφοράς (Insufficient Transport
Layer Protection)
A10: Μη Επαληθευμένες Αναδρομολογήσεις και Προωθήσεις (Unvalidated
Redirects and Forwards)

Όπως καταλαβαίνετε σε κάποιες περιπτώσεις το αποτέλεσμα δεν είναι
πάντα εύηχο, ενώ πολλές φορές δεν γίνεται άμεσα αντιληπτό για ποιο
πράγμα μιλάμε. Απ' την άλλη μεριά υπάρχουν πολλοί, ακόμα και από
μεγάλους οργανισμούς, που ζητάνε μια πιο "επίσημη" απόδοση των όρων
στα Ελληνικά.

Θα θέλαμε τις απόψεις για το θέμα ώστε η τελική μετάφραση του Top10
στα Ελληνικά να αντικατοπτρίζει καλύτερα τις ανάγκες των ειδικών στο
χώρο της ασφάλειας: θα αποδίδατε π.χ. όρους όπως SQL Injection και XSS
στα Ελληνικά -και με ποιο τρόπο- ή θα τους αφήνατε ως έχουν;

Ευχαριστώ εκ των προτέρων για την ανταπόκριση,
