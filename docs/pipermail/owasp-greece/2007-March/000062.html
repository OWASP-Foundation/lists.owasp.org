Ένα πολύ χρήσιμο εργαλείο για πολλούς από μας:

Open-Tran.eu
This service offers an easy access to the translations of open source
projects. It acts as a kind of a central Memory Translation database
where you can look up a translation suggestion or submit a whole .po
file to the server. Just go ahead: find your language below and try it
yourself.

Περισσότερα:
http://www.open-tran.eu/ και
http://el.open-tran.eu/
