    της πάνω στην ασφάλεια των web και mobile εφαρμογών. Το Application
    Security Report είναι μία έκδοση που συνοψίζει τις ευπάθειες που
    εντοπίστηκαν στις  εφαρμογές web και mobile που ελέγχθηκαν από την
    Ethical Hacking ομάδα της SYNTAX.<br>
    <br>
    Κύριος σκοπός της μελέτης αυτής είναι να προάγει την ευαισθητοποίηση
    σε θέματα ασφαλείας εφαρμογών web, να συνεισφέρει στην διεθνή και
    στην τοπική  κοινότητα της ασφάλειας πληροφοριών, και να προσφέρει
    την ενόραση της SYNTAX  σχετικά με τα αντίστοιχα θέματα.
    Συνεπικουρούμενοι από την μεγάλη τεχνογνωσία της SYNTAX πάνω σε
    θέματα ασφάλειας web και mobile εφαρμογών, στοχεύουμε στην προαγωγή
    του ευαίσθητου αυτού τομέα και την αναγωγή του σε επιχειρηματικό
    εργαλείο που θα βοηθήσει αλλά και θα ωθήσει  τον οργανισμό σας  στην
    ευρωστία και την περαιτέρω ανάπτυξή του.<br>
