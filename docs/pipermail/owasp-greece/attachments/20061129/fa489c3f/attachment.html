Kainourio OWASP ergaleio :<br><br><div class="moz-text-plain" style="font-family: -moz-fixed; font-size: 13px;" lang="x-western"><pre>JBroFuzz is an OWASP Project that emerged from penetration testing. It deals with fuzzing stateless network protocols such as HTTP, SOAP, XML, LDAP, etc.
<br><br>Apart from fancy terminology, JBroFuzz has inbuilt the following Generators ready to be used:<br><br>basic cross site scripting checks (XSS)<br>basic SQL injection (SQL)<br>buffer overflows checks (BFO)<br>format string errors checks (FSE)
<br>integer overflows checks (INT)<br><br>Furthermore, a user can specify their own custom fuzz vectors by edititing the contents of the generators file (generators-0.3.jbrf1). This is also a first attempt to practically standardise fuzzing types and structure for network fuzzing.
