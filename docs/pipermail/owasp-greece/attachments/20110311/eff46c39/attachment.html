                        <p>Ήρθε η ώρα για το επόμενο Hackfest! Την <strong>Κυριακή 20 Μαρτίου</strong> και απ&rsquo; τις <strong>11:00</strong> το πρωί θα είμαστε στην <strong>καφετέρια Egalite (Γ.     Αβέρωφ 36, Πευκάκια, Νέα Ιωνία)</strong> και για τουλάχιστον 4 ώρες θα πίνουμε τον καφέ ή την μπύρα μας και θα &ldquo;παίζουμε&rdquo; με αγαπημένα μας projects.</p>


<p>Αυτό το hackfest όμως θα είναι αφιερωμένο και στην <strong>προσπάθεια που ξεκινάει για ανεύρεση χώρου για δημιουργία hackerspace!</strong> Ενώ θα γίνουν και κάποιες τεχνικές ομιλίες. Αναλυτικά:</p>
<p><strong>Workshop:<br>
- <a href="http://hackademic.s3cure.gr/" target="_blank">Hackademic Challenges</a></strong><strong>: Αναστάσης Στασινόπουλος<br>
</strong>Το εργαλείο Hackademic Challenges&nbsp;αποτελεί
 μια δημιουργική αλλά ταυτόχρονα ευχάριστη μέθοδο να μάθει  καποιος τα 
βασικά στοιχεία για την ασφάλεια των web εφαρμογών.</p>
<p><strong> </strong></p>
<p><strong>Lighting Talks:<br>
- <a href="http://ziproxy.sourceforge.net/" target="_blank">Ziproxy</a>/<a href="http://www.privoxy.org/" target="_blank">Privoxy</a>: Ευάγγελος Μπαλάσκας<br>
</strong>Χρήση απλών εργαλείων για να φιλτράρουμε και να περιορίσουμε το bandwidth που καταναλώνουμε.<br>
