Αγαπητοί φίλοι,

Η Ελληνική Ομάδα Εργασίας του Open Web Application Security Project
(http://www.owasp.gr) έχει τη χαρά να σας προσκαλέσει στην παρουσίαση
των δραστηριοτήτων της, στα πλαίσια του 1ου Συνεδρίου Κοινοτήτων
Ελεύθερου Λογισμικού και Λογισμικού Ανοιχτού Κώδικα
(http://www.fosscomm.gr),

το Σάββατο 22 Μαρτίου 2008 και ώρα 17:50 στο
Εθνικό Μετσόβιο Πολυτεχνείο, στην Πολυτεχνειούπολη Ζωγράφου
(αίθουσα εκδηλώσεων στο ισόγειο του κτηρίου Διοίκησης (Πρυτανεία))

Κατά τη διάρκεια της εκδήλωσης θα παρουσιαστεί εργαλείο εντοπισμού
ευπαθειών σε διαδικτυακές εφαρμογές, που σύντομα θα κυκλοφορήσει με τη
μορφή Ελεύθερου Λογισμικού μέσα από το site του OWASP.gr

Η παρουσία σας θα μας τιμήσει ιδιαίτερα και θα μας δώσει την αφορμή να
συζητήσουμε ενδιαφέροντα θέματα σχετικά με την ασφάλεια των εφαρμογών
διαδικτύου, αλλά και πιθανές μελλοντικές δράσεις της ομάδας εργασίας
μας.

Εκ μέρους του OWASP.gr,
Κωνσταντίνος Παπαπαναγιώτου
