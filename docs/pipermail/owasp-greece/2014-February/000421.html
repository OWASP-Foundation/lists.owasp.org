Αγαπητοί,

Στις 14, 17, 18, 19, και 21 Φεβρουαρίου, οργανώνουμε ο Πέτρος
Αγγελάτος κι εγώ ένα σεμινάριο ασφάλειας πληροφοριών και web εφαρμογών
στην Αθήνα. Καθώς θα θίξουμε κάποια θέματα σχετικά με OWASP,
ενημερώνουμε τη λίστα.

Οι ώρες διεξαγωγής είναι 18:00 - 20:00 και το σεμινάριο θα λάβει χώρα
στο The Cube. Τα θέματα παρουσίασης περιλαμβάνουν:
* GPG & κρυπτογραφία
* HTTPS, HSTS, ARP, BREACH
* Εισαγωγικά θέματα σε ασφάλεια web εφαρμογών
* Tor, OTR, Bitcoin
* Κρυπτογράφηση δίσκου, φυσικές επιθέσεις

Θα είναι επίσης μία καλή ευκαιρία να έρθουμε σε επαφή οι άνθρωποι που
ασχολούμαστε με ασφάλεια λογισμικού.

Θα χαρούμε να σας δούμε εκεί.

Το πρόγραμμα, οι ώρες, ο τόπος διεξαγωγής, οι ανακοινώσεις μας και το
υλικό μας υπάρχουν στην ιστοσελίδα του σεμιναρίου:

http://security-class.gr

Με εκτίμηση,
Διονύσης Ζήνδρος.
