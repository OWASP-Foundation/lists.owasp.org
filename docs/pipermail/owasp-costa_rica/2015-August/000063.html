El siguiente código fuente contiene una vulnerabilidad importante, puede
identificar de que vulnerabilidad se trata?, puede identificar los orígenes

El capítulo de OWASP Costa Rica está organizando un taller de Seguridad

software sea inseguro.

Estaremos cubriendo los principales riesgos que afectan a a las
aplicaciones Web,como realizar pruebas en el software para encontrar estas
vulnerabilidades y recomendaciones y contramedidas .
