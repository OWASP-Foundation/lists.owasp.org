Buenos días,

Estoy organzando  el primer evento llamado The First OWASP Day Costa
Rica, un día exclusivo para dar conferencias de Seguridad Informática.
Todavía nos hacen falta tres personas expertas en Seguridad  que
puedan presentar algún tema. Como sugerencia los temas pueden ser

1.Metodologías para el control de Calidad enfocadas al software seguro.
2.Certificados Digitales (SSL Y TLl).
3.Seguridad en el desarrollo de dispositivos Mobiles
4.Ciclo del desarrollo del Software Seguro(SAMM).
5.Obfuscando JavaScript
6.Utilizando las herramientas de OWASP para escribir aplicaciones Seguras.
7. Desarrollando código seguro en lenguajes de programación como Java,
.Net, PHP entre otros.

La idea es que si conocen a algún especialista de alguna empresa o
alguno de ustedes que pueda presentar uno de estos temas o cualquier
otro tema que no esté aquí pero que esté relacionado con la Seguridad
de Aplicaciones, por favos contáctenos.

De igual forma cualquier ayuda en la creación de éste evento será muy
bien recibida.
