Les quería compartir el reporte de estadísticas de seguridad en los sitios
Web desarrollado por WhiteHat.  Este estudio es el resultado de analizar
cerca de 7000 sitios Web. En dicho reporte se pueden encontrar datos
curiosos como  el que indica que Cross Site Scripting (XSS) lidera la lista
de las vulnerabilidades encontradas lo cual indica que ésta vulnerabilidad
está presente en  cerca del 55% de los sitios analizados
