Buenos días,

El próximo jueves 30 de Octubre de 2014, voy a estar presentando un Webinar
en colaboración con la IEEE, en especial con el programa denominado IEEE
Young Professionals.

El título del Webinar es *Application Security for Software Engineers:
Finding an approach to write secure applications.*

En este Webinar voy a cubrir aspectos esenciales del desarrollo seguro de
software y enfoques para utilizar herramientas open source (formalmente de
OWASP) para cambiar el panorama actual. En una época donde los incidentes
en seguridad informática proliferan, es bueno crear un poco de conciencia.
