<p>Hola a todos,</p>
<p><br>Les quiero compartir un poco de información acerca del proyecto OWASP Top Ten. Dicho proyecto se enfoca en dar a conocer las vulnerabilidades más comunes que están expuestas las aplicaciones Web, estos ataques son:</p>

<p><br>1.       Injection o Inyección sobre sistemas legacy(SQL,LDAP,XPATH)<br>2.       Cross Site Scripting(XSS).<br>3.       Broken Authentication and Session Management.<br>4.       Insecure Direct Object References.<br>
5.       Cross Site Request Forgery(CSRF).<br>6.       Security MisConfiguration.<br>7.       Insecure Cryptographic Storage.<br>8.       Failure to Restrict URL Access.<br>9.       Insufficient Transport Layer Protection.<br>
10.     Unvalidated Redirects  Forwards.</p>
<p><br>Cada uno de estos ataques tienen sus características particulares, no obstante todos aprovechan que la Web es el sistema distribuido más escalable. Esto es uno de los muchos materiales que nos proporcina OWASP.</p>
