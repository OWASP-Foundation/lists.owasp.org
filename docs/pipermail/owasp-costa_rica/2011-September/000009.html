Saludos a todos,
El ataque SQL Inyection consiste básicamente en la inserción o inyección de
SQL vía campos de formulario disponibles para el usuario.

Este ataque fue detectado hace 10 años y todavía hoy sigue encabezando la
lista como los ataques más comunes. Que piensan ustedes, que se debe estar
haciendo mal? 10 años es mucho tiempo como para no haber solucionado el
problema.
