TUESDAY, JANUARY 31, 2017, 11:00AM - 3:00PM

More and more software developers are leveraging open source projects to innovate and bring applications to market faster. Join engineering and application security experts from Black Duck at the Centre for Secure Information Technologies (CSIT) for an afternoon seminar on integrating and managing open source in your software projects.
