April OWASP Meeting – Gunnar Peterson
OWASP Top Ten Web Services

Monday, April 27, 2009, 6:00 p.m.

What do Web apps, Web 2.0, Cloud Computing, SOA, and Rest all have in
common? They all use Web services for functionality, data access and
integration. Unfortunately, by default Web services also lack a security
model. The OWASP Top Ten Web Services goes into the technical details of the
vulnerabilities, remediations, and examples of common Web services security
issues like authentication and authorization flaws, how sensitive data is
disclosed, and why security standards like WS-Security and SAML can be your
best friend or your worst nightmare.
 Speaker Bio

Gunnar Peterson Managing Principal Arctec Group, a Twin Cities based
consulting and training firm. He is also Visiting Scientist at Carnegie
Mellon University Software Engineering Institute, editor for IEEE Security &
Privacy Journal "Build Security In," and lead on OWASP Top Ten Web Services.
He maintains a popular information security blog at
http://1raindrop.typepad.com
