The August meeting for OWASP will be August 8th at Metropolitan State
University, Minneapolis.
(This is where we met in June)

Map and directions are below.

Agenda:
6:00 sign in for CISSP Credits
6:10 Sam Buchanan: AJAX security
Ajax itself is not insecure, but without careful attention it can
heighten
the risk of introducing vulnerabilities to a web application. Sam will
explore things to watch out for when using Ajax.

6:45 Pete Palmer: Featured Talk: "Making the Most of Apache Web Server
Security".
7:45 Upcoming events (from the floor)

Directions:
Metro State,Management Education Center, 1300 Harmon Place June
20th.Room
M.1500 or M.1700 look for the event titled: "Open Web Security Meeting"
http://www.metrostate.edu/facilities/mpls.html

Building location:
http://www.metrostate.edu/images/facilities/com.gif

Parking:
Rooms are on first floor of the Management Education Center. They are
really
nice rooms with very comfortable chairs. Street parking is free after 6
pm.
If you get there early it's just a $.25 for a half hour. There is also a
ramp which is $5.00.

This information is also posted at the local chapter web page:
http://www.owasp.org/index.php/Minneapolis_St_Paul#Agenda_August_8
It's a wiki-web. Please add any upcoming events there and/or send them
