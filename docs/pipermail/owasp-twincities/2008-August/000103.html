Greetings MN OWASP Chapter,

WhiteHat Security, the leader in SaaS website security, and F5 Networks, the
global leader in Application Delivery Networking, cordially invite you to a
luncheon on Wednesday, September 10th at the W Hotel Minneapolis.

At this event, you will learn about advancements in protecting and securing
Web applications from attacks. You'll hear from Ken Salchow, Manager,
Technical Marketing, F5 will offer insights into how F5 customers are
driving website security in their organizations. We will then present a live
demonstration of a revolutionary new solution that closes the loop from Web
application vulnerability detection to remediation – an integrated solution
delivering TOTAL website security.

Agenda:
11:30 AM:  Registration
Noon: Lunch served

    * Welcome & Website Security Overview: Stephanie Fohn, CEO, WhiteHat
Security
    * The Challenge of Managing Application Security in Today's Environment:
Ken Salchow, Manager, Technical Marketing, F5 Networks
    * Integrated Web Application Firewall Presentation and Demonstration: F5
Networks Security Systems Architect, Clint Hatton and WhiteHat Security
Founder and CTO, Jeremiah Grossman

WhiteHat CEO, Stephanie Fohn will open the program and provide a snapshot of
the latest website attacks from the WhiteHat Security Statistics Report. F5
Security Systems Architect, Clint Hatton and WhiteHat CTO and Founder,
Jeremiah Grossman will offer a first look at "virtual patching," an exciting
new concept in website vulnerability management.

This solution:

    * Closes the loop from vulnerability detection to remediation
    * Increases security via WhiteHat's rapid identification and reporting
of website vulnerabilities
    * Ensures highly targeted vulnerability remediation (virtual patching)
via F5 BIG-IP Application Security Manager (ASM)
    * Delivers one-step PCI 6.6 Compliance
