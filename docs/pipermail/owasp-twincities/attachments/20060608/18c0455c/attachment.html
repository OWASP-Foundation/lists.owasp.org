<p><strong>Open Source Tools Presentation</strong><br>Tim McGuire – Consultant <br>Will present:&nbsp; Selected open source web application security testing tools. </p>
<div>Tim will demonstrate these tools:<br>1)&nbsp; Using Gforge, a fat target for security scanning. It uses CVS module, file uploads, email and SOAP.</div>
<div>2)&nbsp; Using Wikto, a Web Server Assesment Tool<br>3)&nbsp; Using and customizing WSFuzzer, a penetration testing tool that audits HTTP based SOAP targets. <br>4)&nbsp; Using Oedipus, a web application scanner written in Ruby.<br>
5)&nbsp; Using and customizing Rats, a source code scanner.<br>6)&nbsp; Using and customizing spike proxy, a HTTP proxy for finding security flaws in web sites.</div>
