<p style="MARGIN: 0in 0in 12pt"><span style="FONT-SIZE: 9.5pt"><font face="Times New Roman">Joe Teff is a security group manager at Wells Fargo and a charter member of OWASP-Twin Cities. His knowledge and experience cover a broad range of security domains. 
</font></span></p>
<p style="MARGIN: 0in 0in 0pt"><span style="FONT-SIZE: 9pt"><font face="Times New Roman">Muthu Meyyappan is a Sr. Application Security Architect at United Healthcare </font></span></p>
<p style="MARGIN: 0in 0in 0pt"><span style="FONT-SIZE: 9pt"><font face="Times New Roman">He will describe a new OWASP Project: XML Security Gateway Evaluation Criteria Project </font></span></p>
<p style="MARGIN: 0in 0in 0pt"><span style="FONT-SIZE: 9pt"><font face="Times New Roman">This OWASP Project defines an open standard for evaluating XML Security Gateways. This criteria will provide the OWASP community a set of standard evaluation criteria to assess the functionality and quality of XML Security Gateways. The main driver for this project is to reduce the confusion and complexity in assessing the strengths and weaknesses of solutions in this the XML Security space, and enlightening the community as to the utility of XML Security Gateways to deliver a number of valuable security services. 
</font></span></p>
