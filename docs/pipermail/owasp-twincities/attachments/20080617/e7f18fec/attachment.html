OWASP needs your help with a new important project.<br>
<br>
We&#39;re creating the OWASP Application Security Desk Reference (ASDR) to capture
and organize all the foundational knowledge in application security.&nbsp; Like
the Physicians&#39; Desk Reference for doctors, this book is a well-organized
reference work that anyone working in application security will want on their
desk.<br>
<br>
The OWASP ASDR covers application security principles, threat agents, attacks,
vulnerabilities, controls, technical impacts, and business impacts. A 900-page
alpha draft is available for your review.&nbsp; We need your help to get a
final version created by August 25, 2008!<br>
