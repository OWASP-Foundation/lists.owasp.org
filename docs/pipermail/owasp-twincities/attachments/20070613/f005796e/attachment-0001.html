<p style="MARGIN: 0in 0in 0pt"><b><font face="Times New Roman">OWASP Top Ten 2007</font></b></p>
<p style="MARGIN: 0in 0in 0pt"><span lang="EN"><font face="Times New Roman">This totally re-written edition lists the most serious web application vulnerabilities, discusses how to protect against them, and provides links to more information. 
</font></span></p>
<p style="MARGIN: 0in 0in 0pt"><span lang="EN"><font face="Times New Roman">&nbsp;</font></span></p>
<p style="MARGIN: 0in 0in 0pt"><font face="Times New Roman"><span lang="EN">The primary aim of the OWASP Top 10 is to educate developers, designers, architects and organizations </span><span lang="EN">about the consequences of the most common web application security vulnerabilities. The Top 10 provides basic methods to protect against these vulnerabilities – a great start to your secure coding security program. 
</span></font></p>
<p style="MARGIN: 0in 0in 0pt"><span lang="EN"><font face="Times New Roman">Sam has been using the OWASP Top 10 as a training and awareness tool in his organization. Join us to find out how you can put it to use. </font>
</span></p>

<p style="MARGIN: 0in 0in 0pt"><font face="Times New Roman">Gunnar has compiled a list of Web Services security issues. This comes from his recent, practical experience and feedback from students that have attended his Web Services Security seminars. This session will give you practical advice and a clear sense of which issues are the most important.
