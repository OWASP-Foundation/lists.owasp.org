The University of Minnesota is working with SANS to bring the SANS
"Secure Coding in Java/JEE: Developing Defensible Applications" class to
Minneapolis.  This is a 4-day, hands-on class covering the tools &
techniques to build secure applications using Java/JEE.

This course is being offered as part of SANS' partnership series, which
provides a significant discount on the course costs for students in
government, education, and law-enforcement organizations.

Below are the course details:

DEV541 :: Secure Coding in Java/JEE: Developing Defensible Applications
   Dates: Tuesday, January 5th, 2010 through Friday, January 8th, 2010
   Time: 9:00 AM to 5:00 PM EST
   Instructor: Tanya Baccam, at UMN
   Registration:   https://www.sans.org/registration/ivc.php?lid=19229
   Class Info:     http://www.sans.org/training/description.php?mid=912
