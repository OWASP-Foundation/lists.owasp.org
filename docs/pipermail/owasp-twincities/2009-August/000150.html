Hi all!

Don't forget to register for the OWASP Minneapolis - St. Paul Event on
August 24th.  This year's event is being held on the St. Paul Campus of the

that 150 seats left for this event and expect that we will not be able to
accommodate any unregistered attendees. So make sure to reserve your seat
now. http://owaspmn.eventbrite.com/
