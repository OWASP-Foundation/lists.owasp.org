*Speaker Bios:*
Gunnar Peterson is a Managing Principal at Arctec Group. He is focused on
distributed systems security for large mission critical financial, financial
exchanges, healthcare, manufacturer, and insurance systems, as well as
emerging start ups Mr. Peterson is an internationally recognized software
security expert, frequently published, an Associate Editor for IEEE Security
& Privacy Journal on Building Security In, an Associate Editor for
Information Security Bulletin, a contributor to the SEI and DHS Build
Security In portal on software security, and an in-demand speaker at
security conferences.
Read more at his page which is always up-to-date:
http://1raindrop.typepad.com/1_raindrop/
