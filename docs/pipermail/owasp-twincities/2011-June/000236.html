OWASP MSP meeting tonight. Darren Meyer will be presenting "Selling Static Analysis: How to Start Fast and Finish Strong". Full details and registration at http://staticanalysis.eventbrite.com/

See you tonight!

 

*** SUMMARY ***

This talk addresses how to:

1. Convince management of the value a Static Code Analysis tool can provide 2. Sell process over product 3. Get development teams on your side 4. Get a "quick and dirty" Static Code Analysis program up and running 5. Expand that nascent program into a mature part of a Secure SDLC 5a. Use Static Code Analysis to drive building a Secure SDLC if you don't have one
