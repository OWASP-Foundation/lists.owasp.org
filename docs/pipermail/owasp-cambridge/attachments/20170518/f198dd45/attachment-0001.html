<div dir="ltr">Most of you will now know about the ransomware attack that occurred last Friday. Below is some advice on what to do next (whether or not you were directly affected by the attack):<br><br>There are some direct advice links below for:<br> <br>·        The Home User<br>·        The Enterprise User<br>·        General advice from the NCA.<br> <br>The advice can also be found by going directly to the National Cyber Security Centre’s (NCSC) or Microsoft’s site.<br> <br>Microsoft have advised their customers that “This ransomware can stop you from using your PC or accessing your data. Unlike other ransomware, however, this threat has worm capabilities.” and<br>“The exploit code used by this threat to spread to other computers was designed to work only against unpatched Windows 7 and Windows Server 2008 (or earlier OS) systems. The exploit does not affect Windows 10 PCs.”<br> <br> <br>Prepare:<br>o   Understand the technical estate (network) that you are responsible for, and patch all software on all systems within. Microsoft have also now released a patch for legacy Windows XP systems relevant to this malware.<br>o   NCSC have also released additional defence steps relevant to the enterprise network defender.<br>o   Use Anti-Virus software at all times and ensure that it too is updated.<br>o   Backup your system or critical data to a storage device that is not within the same network. Consider cloud storage options where suitable.<br>o   If you believe that you have been a victim of a ransomware attack, report it to your Local Police and in turn Action Fraud.<br> <br>Master Level Guidance for use as reference:<br> <br>·       Main NCSC Statement: <a href="https://www.ncsc.gov.uk/news/statement-international-ransomware-cyber-attack">https://www.ncsc.gov.uk/news/statement-international-ransomware-cyber-attack</a><br> <br>·        General advice on how to protect yourself from ransomware: <a href="https://www.ncsc.gov.uk/guidance/protecting-your-organisation-ransomware">https://www.ncsc.gov.uk/guidance/protecting-your-organisation-ransomware</a><br> <br>·       General NCSC advice on patching systems: patch your systems <br> <br>·       Specific NCSC work with Tech community on mitigating this malware: <a href="https://www.ncsc.gov.uk/blog-post/finding-kill-switch-stop-spread-ransomware-0">https://www.ncsc.gov.uk/blog-post/finding-kill-switch-stop-spread-ransomware-0</a><br> <br>·       Guidance for home users: <a href="https://www.ncsc.gov.uk/guidance/ransomware-guidance-home-users">https://www.ncsc.gov.uk/guidance/ransomware-guidance-home-users</a><br> <br>·       Guidance for enterprise: <a href="https://www.ncsc.gov.uk/guidance/ransomware-guidance-enterprise-administrators">https://www.ncsc.gov.uk/guidance/ransomware-guidance-enterprise-administrators</a><br><br> <br>Important info from Industry Partners to be used as reference:<br>·        MS description of malware: <a href="https://www.microsoft.com/security/portal/threat/encyclopedia/Entry.aspx?Name=Ransom:Win32/WannaCrypt">https://www.microsoft.com/security/portal/threat/encyclopedia/Entry.aspx?Name=Ransom:Win32/WannaCrypt</a><br> <br>·       Also from MS who have published a relevant patch for XP: <a href="https://blogs.technet.microsoft.com/msrc/2017/05/12/customer-guidance-for-wannacrypt-attacks/">https://blogs.technet.microsoft.com/msrc/2017/05/12/customer-guidance-for-wannacrypt-attacks/</a><br><br> <br></div>
