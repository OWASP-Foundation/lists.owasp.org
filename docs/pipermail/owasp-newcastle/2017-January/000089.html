Hello all,

Just a quick note to let you know that OWASP Security Shepherd
<https://www.owasp.org/index.php/OWASP_Security_Shepherd> has graduated
from an OWASP Lab project to a Flagship project. There are only a small
number of Flagship projects and they all meet a demanding set of qualifying
criteria including a good level of activity, proven quality and high level
of adoption within the security community.

OWASP Security Shepherd is an application security training platform that
provides:

   - Teaching Tool for All Application Security
   - Web Application Pen Testing Training
   - Mobile Application Pen Testing Training
   - Safe Playground to Practise AppSec Techniques
   - Real Security Risk Examples

And obviously, because it's OWASP, it is completely free and open source.
If you haven't looked at it before, give it a go!

Also, if anyone would like to have a demo/talk around this project at a
chapter meetup, please let me know and I'll see what I can do.
