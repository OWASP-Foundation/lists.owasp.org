Dear all,

Hot on the heels of ASVS v3, OWASP Security Shepherd v3 has just been
released:

https://www.owasp.org/index.php/OWASP_Security_Shepherd

Security Shepherd is a training platform for mobile and web application
security. Obviously being OWASP, it is free and open source.

I haven't used it myself yet, but I've heard very good things about it. If
you try it out, it would be great to hear any feedback either via this
mailing list, or our Slack channel. You could even feedback via a talk at
one of our meetings!

Best regards,

Mike
