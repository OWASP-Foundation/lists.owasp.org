Dear all,

trying to solve the Forced Browsing lesson,
http://localhost/WebGoat/config
(as the final hint says) simply returns a 404 error. Looking at the
lesson's
Java source code, that does not even mention "config", but instead
checks
if a parameter "succeeded" is present and has the value "yes". So, I
simply added "&succeeded=yes" to the normal lesson page URL, and 
"Congratulations. You have successfully completed this lesson" appeared
along with the admin settings inputs fields.

Was this the intended way of solving this lesson? If so, the lesson
would
be essentially the same as the "Remote Admin Access" lesson. If not,
please
give me a hint how to solve it beyond trying
http://localhost/WebGoat/config.
