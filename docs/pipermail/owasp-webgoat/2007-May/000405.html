If you downloaded the release version, it should be
 
http://localhost/WebGoat/attack
 
look in the tomcat/webapps directory to confirm the name of the WebGoat directory.  If the war file is named WebGoat.war the url will be the url specified above.  If the war file is named WebGoat-5.0.war then the url should be the url specified in your message.
