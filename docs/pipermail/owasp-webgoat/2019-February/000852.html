I think I have the solution for Page 7 of the XXE Lesson, but the form is
not marking it as solved.

Solution:
Upload a dtd to WebWolf that generates an entity that will send  request to
webwolf/landing?text=<Contents of secret file>

Make a post to http://localhost:8081/WebGoat/xxe/blind from ZAP generating
the new entity and in the text section of the xml comment call the send
entity.

This will make a request to webwolf/landing with the contents of the file.
I have confirmed that the contents match by cat'ing out the file on disk
locally.
