Although you did not have access to the delete button (presentation layer access control), by changing the name of the action you were able to execute the business function (business layer access control) which you were not allowed to see/execute.  This is the condition for moving onto to stage 2.  
 
The fact that you were able to bypass the data level access control and view another person's profile is not relevant at this stage of the lesson.  This will become important at another stage.  You must first fix the broken business layer access control in order to move to stage 3.
