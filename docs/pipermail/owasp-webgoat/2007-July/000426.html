You need to get WebGoat to "reauthenticate" you.  You will type in the
credentials supplied to you on the basic auth lesson page once you've
completed the first stage.  If all your green check marks go away after
you "reauthenticate" then you've succeeded.  Go back to the basic
authentication lesson to see if you get the green check mark for this
lesson.  Remember to switch back to the guest user when you're done.
