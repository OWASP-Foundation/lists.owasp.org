  I've made some improvements to WebGoat. There's a new category based menu on the left (thanks Bruce). Also, a few lessons have been cleaned up a bit, and there are a few new lessons:
   - JavaScript validation
   - Role Based Authentication

  And I've got a bunch more lesson ideas in the todo.txt file in the project root, but please keep the suggestions rolling in.  It really doesn't take long to put a lesson together -- the plugin architecture really works well.

  If you'd like to try it, you can build from source with ant dist, or grab WebGoat-2.1.war from the dist directory and drop it into your webapps directory.

  Oh, I almost forgot. I added support for Microsoft Access (much more realistic SQL injection).  That's how it's set up to run now.  If you want the embedded database back, you can edit web.xml in the WEB-INF directory and uncomment it. Let me know if this works for people as it hasn't been tested anywhere but my machine.  I think it should be easy to use any JDBC database you want now, just put the right strings in the web.xml file.

  -- Jeff
