I've answered my own question.  I added the following:


server.address=x.x.x.x


To WebGoat/webgoat-container/src/main/resources/application.properties


Which was somewhat counter intuitive since it's called during startup for Spring-Boot

mvn -pl webgoat-server spring-boot:run
