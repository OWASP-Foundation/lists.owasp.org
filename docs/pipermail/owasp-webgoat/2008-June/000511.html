Actually, the solution to the first stage is a flaw I find many
applications expose.  They remove functionality from the presentation
layer and don't have the corresponding business layer access control
checks.  The showing of the source code is not the point of the flaw,
the point is the missing access control check.  The first stage of the
challenge is supposed to be challenging and is meant to drive home the
point of access control.  Since these same principles are supplied in
the Labs, perhaps we should beef up the help in the labs to make sure
the access control checks in the different layers are understood upon
completion of the RBAC lab.

I would welcome another challenge lesson but this stage of this
challenge is appropriate.

Bruce
