Hello WebGoaters,

I would like to provide some information regarding decisions made during the last "Core Contributors" conference call on 08/14/15:
    * Agreed to using TravisCI for our development branch automated builds.
    * Agreed to using Github Issues to track bug/defects and feature requests.
    * Agreed to using version-SNAPSHOT for development builds.
    * Agreed to release next version of WebGoat as version 7.0, given the signifant contribution and changes that happend in the current cycle
    * Agreed to start using Maven Central to push WebGoat artifacts.
    * Defined target date for 7.0 release to be just before OWASP AppSec USA conference. As of know, targeted release date is 09/15/15

Now, we would liketo foster the communication between developers, comunity and users.

OWASP has recently started using the " https://slack.com " service for communication and made a tenant available for our use.

Slack provides an "IRC" style of comunication, with history, search and some other goodies, completely free !

You can request a free invite to access the OWASP SLACK tenant at: http://owasp.herokuapp.com/

Once you get your invite, you will be able to join the conversation at https://owasp.slack.com/ . You can also use a Slack Client or your plain old IRC client.

Several channels are available, but we created a #project-webgoat channel. I look forward to talk to you there !

If you have any pending code changes, please try to create a pull request soon, as we are approaching our target release date.

As always, thank you for all the great work and feedback.

Till next week!
