<DIV>The point of this lesson is on the fail-open aspect of the authentication 
mechanism. In this case, you can gain access if you enter a valid username and 
delete the password parameter entirely.&nbsp;The lessons are interlinked a bit, 
since you need to know a valid username.&nbsp; This is realistic, as many sites 
allow valid usernames to be harvested.</DIV>
<DIV>&nbsp;</DIV>
<DIV>Achilles is just one way to send a request without the password parameter 
-- you could craft a special URL to do it as well.</DIV>
<DIV>&nbsp;</DIV>
<DIV>If you think this lesson is overly confusing, let&nbsp;us know.&nbsp; 
Perhaps there's a way to recraft it so that it drives the point home more 
clearly.&nbsp; Let me know if you have any ideas about how it might work 
better.</DIV>
<DIV>&nbsp;</DIV>
<DIV>Also, if you have ideas for other lessons, they are easy to create. Please 
send your ideas to the webgoat list.&nbsp; Or, just code up a lesson -- the 
directions are in the user guide!&nbsp; It's easy.</DIV>
