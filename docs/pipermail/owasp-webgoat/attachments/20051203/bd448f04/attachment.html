<p><b>Lesson Plan Title:</b> How to perform a denial of service attack </p>
</div>
<p><b>Concept / Topic To Teach:</b> </p>
This lesson explores that fact that a denial of service of an application can not
only be accomplished by consuming all resources, but also by making it impossible
to log in.
  <br>
<p><b>General Goal(s):</b> </p>
Make sure nobody else can log in to the application by using all entries in the connection pool.
