<DIV>&nbsp;</DIV>

<DIV>&nbsp;&nbsp;&nbsp; Drag the installer folder ( which contains install.props 
) and </DIV>

install.props </DIV>
<DIV>&nbsp;&nbsp;&nbsp; file to be added into the 
"install_WebGoat-2.0_windows.jar" </DIV>
<DIV>&nbsp;&nbsp;&nbsp; file with a path of installer.&nbsp; </DIV>
<DIV>&nbsp;</DIV>
<DIV>3) Make sure you only have ONE install.props file in the .jar 
file.&nbsp;</DIV>
<DIV>&nbsp;&nbsp;&nbsp; &nbsp;If you have more than one install.props file, the
