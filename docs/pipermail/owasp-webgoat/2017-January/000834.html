Hi Chris,

If you are running WebGoat using maven:

For WebGoat 8 you need to clean the following directory: ~/<user.home>/.webgoat
For WebGoat 7 if you run mvn tomcat7:run there will be a file per lesson in webgoat-container/src/main/webapp/users/guest.org.owasp.webgoat.plugin.HttpBasics.props and you will need to remove the contents of the users directory.

For WebGoat 7 standalone version (using java -jar ...) there is a directory called .extract you will need to remove this directory.
