You should create a bunch of user accounts in tomcat-users.xml (i.e. 
student_1, student_2, ...)  This will prevent the students from stepping on 
each other while doing each lesson.  Some lessons are better than others in 
regards to keeping the user data separate but they all should work in this 
mode.

You can then run WebGoat in enterprise mode (web.xml setting) which will 
allow you to view each student's report card.   You will need to log in as 
webgoat/webgoat in order to see the additional admin features.

You can also use the lesson plans feature to add custom content to each 
lesson for teaching purposes.  All you need to do is overwrite the existing 
lesson_plans/lesson_name.html file with your content

Send me mail if more information is required

Bruce
