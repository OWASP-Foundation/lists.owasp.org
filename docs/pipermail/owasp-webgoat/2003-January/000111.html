For the 2.0 release, we just need to update the links on the main page,
the webgoat page, and the downloads page.  There has been some
confusion with these links. I downloaded a copy from the downloads
page TODAY and got the wrong installer....


Some of the things I would like to see for a 2.x release:

    1) Categorize the lessons using the "Top Ten"
    2) Create at least one lesson for each Category.
    3) Create more complex challenges.  WebGoat currently has a whole
        bunch of "admin" features that haven't been discovered.  WebGoat
        also has the ability to enforce "roles" for each screen.   We could
        create some fun challenges with these features.

For the v3 release:

    1) Redesign of the look and feel ( We may have a volunteer for that )
    2) Create the "Test" architecture
            - extend the lesson architecture so that a test will know if a
"user"
                defeated a lesson.
            - The "Test" can ( but doesn't have to ) stand on it's own
without a lesson.
                In this case, the test would be filtered from the menu's but
would still
                be executable. This would allow some complex text cases to
be created
                without worrying about an end user.
    3) Create the "Tester" client.
            - Client will read url data from a flat file, execute the link,
parse the results,
                and generate a score card. We could get pretty complex in
the scoring
                by weighting the different tests.


Just some ideas...


bruce.
