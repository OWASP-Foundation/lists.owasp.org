After installing WebGoat...


Start the tomcat webserver

Browse to http://localhost:8080/WebGoat/attack


Note:

There is a problem with the unix installer. It does not correctly copy the
WebGoat war file into the webapps directly.

Solution:
    1a) Pull the webgoat war file from the cvs webgoat/dist directory
    1b) or extract warfile from installer jar: jar -xvf
install_WebGoat-2.0_unix.jar installer/WebGoat-2.0.war
    2) Copy war file to <tomcat_root>/webapps/WebGoat.war
