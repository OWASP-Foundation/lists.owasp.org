After installing WebGoat 3.0
Browse to http://localhost:8080/WebGoat/attack

A few things to note:

The port number depends on how you installed Tomcat.
The url is case sensitive
The WebGoat portion of the url should match the war file name in the
<Tomcat_home>/webapps directory.  The installer will do the rename
automatically.
The login username and password is "guest", specified in
<Tomcat_home>/conf/tomcat-users.xml
