Buon pomeriggio,
il 21 Giugno OWASP-Italy, in collaborazione con il CLUSIT, terrà un seminario
all'interno di Infosecurity Roma 2006 dal titolo: "Web Application
Security: linee guida per la progettazione e l'audit di sicurezza
degli applicativi".

Luogo: Sheraton Roma Hotel - Viale Del Pattinaggio, 100
Orari: Registrazione 10,15 - Inizio Seminario 10,30 - Fine lavori 17,00
Docenti: Matteo Meucci e Alberto Revelli (OWASP-Italy)
Riferimenti: http://www.infosecurity.it/Roma/programma.php

Programma del seminario:
PARTE I
Introduzione alla Web Application Security
• Quali sono i rischi a cui è esposto un servizio web
• Risk assessment di un applicativo web
• Linee guida di design architetturale e principi di sicurezza
  Come sviluppare applicazioni web "sicure".
• Linee guida ed esempi

PARTE II
Come realizzare un'analisi di sicurezza per gli applicativi web
• La metodologia OWASP PenTesting
• Gli strumenti: OWASP WebScarab
   Apprendimento delle più comuni vulnerabilità
• Sessione pratica tramite OWASP WebGoat
• Tecniche avanzate di SQL Injection.
• Case-study di applicativi web vulnerabili.

Siete tutti invitati a partecipare: InfoSecurity è gratuito previa
registrazione (http://www.infosecurity.it/Roma/form.php), il seminario
invece facendo parte del percorso formativo CLUSIT è riservato ai
soci.
OWASP-Italy è socio CLUSIT grazie alla cross-membership stipulata a
gennaio scorso e quindi sono disponibili 3 ingressi al seminario.

Per registrarsi:
http://www.clusit.it/edu/index.htm (WG01)
Se siete interessati, vi prego prima di verificare se la vostra
azienda è socia CLUSIT.
Nel caso in cui non lo siate potete utilizzare il modulo di
registrazione del CLUSIT indicando: WG01 (titolo seminario) e
OWASP-Italy (azienda).

Vi prego di notificare in lista se vi siete iscritti o meno, in modo
da evitare l'inoltro di più di tre richieste al CLUSIT.

Grazie,
Matteo
