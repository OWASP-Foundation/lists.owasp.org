Buona sera a tutti,
vorrei fare insieme a voi il punto della situazione delle attività di
OWASP-Italy al fine di mettere tutti a conoscenza dei progetti in
essere e futuri e di coninvolgere, se possibile, altre persone.


*****************************************************************************
1) Redazione del quaderno CLUSIT sulla Web Application Security
*****************************************************************************
Obiettivo: Realizzare un documento per i soci del CLUSIT che introduca
le tematiche di Web Application Security ed il progetto OWASP.
Team: Raoul Chiesa, Matteo Meucci, Lorenzo De Santis, Alberto Revelli,
Luca Legato,
Deadline: 28 Febbraio 2006
SAL: Stiamo ultimando la redazione del doc. (95%).

*****************************************************************************
2) Scrittura di un articolo sulla sicurezza dei Web Services
*****************************************************************************
Obiettivo: Realizzare un articolo che tratti le problematiche di
sicurezza dei Web Service da pubblicare possibilmente su ICTSecurity
magazine.
Team: Marco Graia
Deadline: (?)
SAL: (?)

*****************************************************************************
3) Traduzione della OWASP Guide v2.0.1
*****************************************************************************
Obiettivo: Rendere disponibile la traduzione italiana dell'ultima
versione della OWASP-Guide
Team: Matteo Paolelli, (?)
Deadline: (?)
SAL: (?)

*****************************************************************************
4) OWASP-Legal
*****************************************************************************
Obiettivo: Cercare di dare un senso al progetto OWASP-Legal
comparata con la giurisdizione italiana.
Team: Marco Scialdone, Dario Vaccaro
Deadline: (?)
SAL: (?)

*****************************************************************************
5) OWASP Web Application Penetration Testing (WAPT) v2
*****************************************************************************
Obiettivo: Il progetto OWASP vuole creare un insieme di linee
guida per realizzare un WAPT.
Team: Alberto Revelli, Matteo Meucci
Deadline: Abbiamo consegnato la nostra parte ma il documento
completo non è stato ancora pubblicato.
SAL: (100%)

Prego chi è coinvolto in una o più attività di comunicare alla lista
le parti indicate con (?).
Chiunque abbia voglia è invitato a partecipare alle iniziative sopra o
di proporne di nuove.

Grazie,
