Buongiorno,
anche quest'anno stiamo collaborando con il capitolo veneziano di ISACA
per creare una conferenza sui temi di Web Application Security e
Governance IT.

La conferenza si terrà a Venezia il prossimo 27 Settembre 2014 (da
confermare).

Siete tutti invitati quindi ad inviarci le vostre idee a:
owasp-italy <at> owasp.org.

I temi di interesse saranno:
- Mobile Security
- Application Security
- Cyber Security
- Tutto ciò che ha a che fare con il mondo OWASP, quindi nuovi progetti,
nuove ricerche, ecc.

Subject della mail: OWASP-ISACA Venice 2014 CFP

* Name and Surname
* Email address
* Telephone number
* Company name and role
* Short biography(max 100 words)
* List of the author's previous papers/articles/speeches on the same topics
* Title of the contribution
* Type of contribution: Technical or Informative
* Abstract

Tempo per il talk: 35-40 minuti
Deadline: 15 Giugno 2104

Grazie e buon lavoro!
