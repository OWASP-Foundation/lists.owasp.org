Buongiorno a tutti,
vi inoltro l'invito al prossimo OWASP Day creato appositamente per la
presenza di Gary McGraw in Italia.

Genova, 14 Maggio 2014
DIBRIS Università di Genova - Via Dodecaneso 35 - Sala Conferenze III
piano, stanza 326bis.

L'OWASP (Open Web Application Security Project) è un'associazione
internazionale open source dedicata allo studio della sicurezza del
software.

OWASP Italy ed il Prof. Alessandro Armando (docente di Sicurezza
Informatica presso il DIBRIS dell'Università di Genova) organizzano
l'OWASP Day 2014 sulla Cyber Security a Genova.

Il key note sarà tenuto da Gary McGraw, esperto di fama internazionale
ed autore di ben 8 libri best-sellers sulla Sicurezza del Software.

Nel suo intervento intitolato "Cyber War, Cyber Peace, Stones, and Glass
Houses", McGraw metterà analizzerà con un occhio critico l'approccio
alla Cyber Security utilizzato dal governo americano ed evidenzierà
l'esigenza di rifocalizzare l'attenzione verso la costruzione di
software sicuro.

L'intervento di McGraw sarà seguito da interventi di Stefano Di Paola
(OWASP-Italy), Matteo Meucci (OWASP-Italy) e del Prof. Alessandro
Armando (Università di Genova).

Il programma dell'evento gratuito è disponibile all'indirizzo
https://www.owasp.org/index.php/Italy_OWASP_Day_2014_Genova

Grazie,
Matteo
