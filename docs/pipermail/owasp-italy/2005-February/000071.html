Per Front End applicativo intendo la parte visibile dall'utente
Internet. Per capirci quella su cui finisce la connessione HTTP/HTTPS,
ovvero il web server. Come indicato nella letteratura in materia
infatti una applicazione Web si divide in tre livelli logici:
	- presentation (web server)
	- logica (application server)
	- Dati (db server)
