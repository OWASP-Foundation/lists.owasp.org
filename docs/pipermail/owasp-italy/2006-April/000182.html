Buondi' a tutti,

visto che qualche giorno addietro e' stata fatta una richiesta di
informazioni sullo sviluppo di un web scanner, colgo l'occasione per
reclutare sviluppatori java.

Il progetto a cui lavoro ormai da diverso tempo si chiama WebSS (Web
Security Scanner) un software opensource di web vulnerability assessment.

WebSS si basa su una struttura a plugin, in cui ogni plugin ha il
compito di effettuare delle operazione atte a verificare una specifica
vulnerabilita' web (come ormai succede in tutti i software di questo
genere). Il problema e' che a causa della vastita' del software, tutto
il mio tempo lo impegno nello sviluppo dell'applicazione in se, senza
avere tempo per poter sviluppare nuovi plugin.

Da qui la mia richiesta, chinque fosse interessato a scrivere qualche
plugin e' il ben venuto, fornendo un aiuto non indifferente a me e anche
ai futuri utenti.

Chi e' interessato non esiti a contattarmi, scrivere plugin per webss e'
semplicissimo e non e' necessario essere dei java guru.

Saluti,
