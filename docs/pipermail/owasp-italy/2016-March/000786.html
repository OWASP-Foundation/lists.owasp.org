Buongiorno,
è possibile inoltrare il vostro light training alla prossima OWASP
AppSec EU a Roma: http://2016.appsec.eu

Lighting Trainings sono corsi di formazione gratuiti di un'ora tenuti
durante la conferenza il Giovedi, 30 giugno e Venerdì 1 luglio 2016.

Esempi di proposte:

Introduzione all'utilizzo di OWASP ZAP
Introduzione a Python per i test di sicurezza
Risolvere le lezioni x,y,z di WebGoat
Utilizzare OWASP ZAP Proxy per API Testing

Fate la vostra submission qui:

https://docs.google.com/a/owasp.org/forms/d/1xPOlBu2TYkqLxGAbpq2TO-MjRYUZxnuie8CsSsOExo4/viewform

Il termine ultimo per la richiesta di fulmine Training è il 30 aprile.
