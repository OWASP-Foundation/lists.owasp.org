OWASP WebGoat mostra attualmente le seguenti vulnerabilità:
* 	   - Http Basics (WebScarab)
* 	   - HTML Clues
* 	   - Hidden Field Tampering (WebScarab alteri un parametro)
* 	   - Role Based Access Control 
* 	   - Weak Authentication Cookie  
*	   - Stored XSS 
*	   - Command Injection
*	   - SQL Injection
*	   - Fail Open Authentication
* 	   - Forced Browsing
Gli attacchi più interessanti sono "Weak Authentication Cookie" e
"Command Injection".
