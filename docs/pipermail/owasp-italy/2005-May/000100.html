grazie alla paziente traduzione di Massimiliano Graziani il documento
OWASP Pen Test Checklist v.1.1 è ora disponibile in italiano alla
seguente URL: http://www.owasp.org/documentation/testing.html

Grazie alla collaborazione con il CLUSIT il documento è disponibile
anche alla seguente URL: http://www.clusit.it/whitepapers.htm

Tale documento fornisce (insieme al testing framework) una metodologia
per il Web Application Vulnerability Assessment, ovvero per l'analisi
delle vulnerabiltà di un applicativo web.
