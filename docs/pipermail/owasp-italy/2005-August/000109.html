Caro/a relatore/trice,

grazie per la tua pazienza. La lunga attesa prima dell'accettazione e'
stata dovuta agli inevitabili contraccolpi della fusione tra Webbit e
SMAU. Come saprai, quest'anno i seminari di Webbit sono la componente
"e-Academy" di SMAU. Questo significa l'occasione di raggiungere un
pubblico qualificato che non abbiamo mai raggiunto con i precedenti
Webbit, per cercare di portare una "ventata di novita' " in SMAU.

Sono lieto di poterti comunicare che la tua proposta di seminario e'
stata accettata nel settore tematico "Sicurezza delle Applicazioni"

Ecco l'elenco dei seminari collegati a questo settore:

- Security AND Tuning for Apache PHP MySQL (Lorenzo Barone)

- Web Security Through Examples (Fedon Giorgio)

- Sicurezza delle applicazioni WEB (Vincenzo Ciaglia)

- Progetto OWASP: case-study di applicativi web vulnerabili (Matteo
Meucci)

- Individuazione automatica di vulnerabilita' nelle applicazioni Web
(Stefano Zanero)

- 0-days: how hacking really works (Dave Aitel)

Come sempre, siamo a tua disposizione per ogni domanda relativa a
Webbit/SMAU o al tuo intervanto. Seguiranno ulteriori comunicazioni, ma
ti prego, nel caso ravvisassi un possibile problema riguardo la tua
partecipazione (impegni di lavoro, personali o altro) di dircelo il
prima possibile, perche' dovremo adoperarci per trovare un sostituto
altrettanto qualificato... e non e' assolutamente un lavoro semplice ;)

Grazie di nuovo, e a presto.
