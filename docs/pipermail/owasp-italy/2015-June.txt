Come promesso, dopo un mese di attivit? sui progetti di AdoptOSS (
https://www.owasp.org/index.php/Italy#Adopt_OSS._First_Edition), volevamo
aggiornarvi sul progresso fatto sino ad ora.

[Ntopng]

In questo primo mese, dopo una call introduttiva con i mantainers, abbiamo
preso confidenza con ntopng ed abbiamo diviso le attivit? in Code Review e
Web Application Penetration Test.
Mattia si ? focalizzato sull'analisi del codice relativo al servizio su
Windows e Giuseppe, grazie alle sue precedenti esperienze di sviluppo, gli
dar? un prezioso aiuto. Alessio sta analizzando l'applicazione web e
preparando il template per il report finale.
La prossima settimana organizzeremo un'altra call con gli sviluppatori per
chiarire alcuni dubbi sul codice e sul comportamento del software.


[WordPress]

Nel primo mese abbiamo condotto e pressoch? concluso le code review delle
app mobile Android e Windows phone. Abbiamo individuato l'elenco di plugin
e temi da installare nei 3 filoni di Wordpress che intendiamo prendere in
considerazione ed abbiamo iniziato a lavorare sullo script per il setup del
laboratorio di test (scaricabile suhttps://github.com/thesp0nge/lab-o-matic
).

Ricordo il link pubblico della nostra roadmap con gli avanzamenti:
https://docs.google.com/spreadsheets/d/1o9DbfEF6y7Am3FhfQcCvCGmwEw9N01tLdN7pMHCzXco/edit?usp=sharing
Come criticit? segnalo che ci mancano risorse per la review della versione
mobile per iOS.

[GlobaLeaks]

Dopo una conversazione introduttiva con i mantainers del progetto, abbiamo
iniziato a rivedere tutta la documentazione pubblica per familiarizzarci
con il design e thread modeling specifico di GL. Giovanni ha creato
un'immagine virtuale che utilizzeremo per effettuare i nostri test - in
maniera da avere una configurazione consistente per tutti i volontari. Luca
ha preparato un template per il report finale che vogliamo utilizzare per
includere i nostri risultati. Nei prossimi giorni contiamo di aiutare il
progetto a lanciare una nuova iniziativa di bug bounty. Nelle prossime
settimane, dovremmo finalmente iniziare a fare un po' di security testing.
