===================================================================
OWASP Day III - "Web Application Security: research meets industry"
   Centro di Competenza ICT-Puglia - Dipartimento di Informatica
                   Università degli Studi di Bari
                   23 Febbraio 2009 - Bari (Italy)
===================================================================

Buongiorno,
anche quest'anno siamo riusciti ad organizzare una conferenza di una
giornata per parlare di OWASP e di Web Application Security.
Vi invito tutti a partecipare all'OWASP Day III, un evento nato dalla
collaborazione con il Centro di Competenza ICT-Puglia dell'Università
degli Studi di Bari, polo di riconosciuta rilevanza internazionale
sull'Ingegneria del Software.

La conferenza rappresenta una giornata di dibattito sulla Web
Application Security, ed è organizzata con interventi tecnici e di
alto livello tenuti da autorevoli professionisti del mondo Accademico
ed Aziendale che relazioneranno sulle seguenti tematiche:
* L'evoluzione degli attacchi e delle contromisure nel campo della Web
Application Security
* Casi di studio di aziende che hanno implementato le linee guida
OWASP nel proprio ciclo di vita di sviluppo del software.
* Application Security Assessment Model
* Data Privacy Enabling Technology

Agenda:
=======
