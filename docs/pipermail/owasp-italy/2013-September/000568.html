Su https://www.pentesterlab.com/exercises/ puoi trovare degli esercizi di
penetration testing, con allegate delle web app/servizi vulnerabili scritti
dall'autore, da cui potresti prendere spunto.

Se sei orientato alle vulnerabilità delle applicazioni web, sono stati
realizzati due esercizi "Web For Pentester" e "Web For Pentester II" che
trattano e portano avanti esempi pratici di XSS, SQL injection, Directory
traversal, File include, Code/Command Injection, LDAP attacks e XML attacks.
<https://www.pentesterlab.com/exercises/web_for_pentester/>
