Buon pomeriggio a tutti,
vi inoltro il Call For Paper del prossimo OWASP Day che stiamo
organizzando in collaborazione con la Fondazione Bruno Klessler.


=============================================================

     		                OWASP-Italy Day V
                                           Trento - 5th November 2010

=============================================================

Introduction
============
Following on from the great successes of last OWASP Days and thanks to
the collaboration with the FBK (Fondazione Bruno Kessler), the new
conference will take place next 5th November 2010 in Trento.


Organization and goals:
=======================

* The event will show several points of discussion: we will present
the state of the art of OWASP projects and technical
speeches about the new researches in Application Security.
* As conclusion of the day, we organize a round table discussing the
most interesting subjects came out during the event.
* Conference goal is creating a debate on which will be the evolution
of the research for the Web Application Security, and how to start a
secure software initiative.

Call For Paper:
==============

OWASP solicits contributions on the above topics, or general matters
of interest to the community. Those who are interested in
participating as speakers to the conference can submit an abstract of
the speech to the OWASP-Italy Board by email at:
owasp-italy <at> owasp.org.
The email subject must be “OWASP Day V: CFP” and  the email body must
contains the following information/sections:

* Name and Surname
* Email address
* Telephone number
* Company name and role
* Short biography(max 100 words)
* List of the author’s previous papers/articles/speeches on the same topics
* Title of the contribution
* Type of contribution: Technical or Informative
* Abstract (max one A4 style page)
* Why the contribution is relevant for OWASP-Italy Day V

We are seeking people that want to present on any of the following
topics (in no particular order):
 - Business Risks with Application Security.
 - Starting and Managing Secure Development Lifecycle Programs.
 - Web Services-, XML- and Application Security.
 - Metrics for Application Security.
 - Application Threat Modeling.
 - Hands-on Source Code Review.
 - Web Application Security Testing.
 - OWASP Tools and Projects.
 - Secure Coding Practices (J2EE/.NET).
 - Privacy Concerns with Applications and Data Storage
 - Web Application Security countermeasures
 - Technology specific presentations on security such as AJAX, XML, etc.
 - Anything else relating to OWASP and Application Security.

The submission will be reviewed by the Program Commitee and the 8-9
most interesting ones will be selected and invited for presentation.
The duration of each talk is 40 minutes.

Important dates:
============

* Contributions submission deadline:                    12th September 2010
* Communication of acceptance for contributions:        30th September 2010
