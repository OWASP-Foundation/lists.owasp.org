Hi

Very Nice summary of the Metasploit framework. I like the SQLMAP section
--  don't forget about "armitage" which is the GUI interface for the
MSFconsole, quite nicely done with the capability to easily "Hail Mary" the
exploits and payloads (not always ideal, but handy in some cases).

As a side note --
if you look at using Metasploit to attack OS and Applications consider
targeting the Network Device infrastructure using "Yersinia", nice options
for MITM and DOS  --  but of course best results with Yesrinia is achieved
if you are physically on the target network already, which isn't always the
case.
