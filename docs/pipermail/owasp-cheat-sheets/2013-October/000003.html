AppSec USA <http://appsecusa.org/2013/activities/owasp-project-summit/> is
just a month away and there is so much more going on than just the
conference. From November 18-21 the Project
Summit<https://www.owasp.org/index.php/Projects_Summit_2013/Home>will
be held in tandem with AppSec USA. For those unfamiliar with the
Project Summit, it is a smaller version of the much larger OWASP Summits.
The Project Summit gives project leaders a chance to showcase their project
process, as well as allow attendees to contribute to the project.

Featured at this year’s Project Summit are nine sessions, two of which will
be held over the course of the four days. You can sign up for each session
below:
