Hello Everyone,

In recent discussions on vulnerability disclosure, I did not see the reference of CVE id and website so just wanted to share the information.

Regarding vulnerability disclosure, CVE (Common Vulnerabilities and Exposures, http://cve.mitre.org/about/index.html) is the most referenced site these days. Every other vulnerability reporting site like: securityfocus.com, secunia, NVDB, OSVDB. All have a reference to a CVE id. Also vulnerability scanners also reference the CVE for any vulnerablity.

One can obtain a CVE ID using http://cve.mitre.org/cve/obtain_id.html
