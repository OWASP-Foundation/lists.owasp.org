Hi Gunwant,
This method is good for a run-of-a-mill bots. The bots which simply scans
webpages for comments form and submit their spammy link ther filling all the
details asked. so yes you can make a plugin for off-the-shelf
CMS(WP/Drupal/Joomla) by this way. "A non intrusive spam protection"

But that's not what all the bot do, rather people customize bots as per web
pages to exploit the features.
Say a bot for web services like cbl.abuseat.org or iptools.com. Even for non
standard comments page like one at blogoscoped.com

In any such case where a script is modified for the target webpage, this won't work

thanks
./Rohit11
