Couple of approaches

1. Start the application.
2. Start ollydbg/your favourite debugger
3. Attach the process.
4. See where all can you input data.
5. Automate entering of data using autohotkey or something like that.
6. If it crashes try this http://www.codeplex.com/msecdbg

Or

1. What does it use for IPC
2. Does it rely on a network socket.
3. Can you send data to that ?

Or maybe

1. Find out what it was written in.
2. Find a fuzzing tool for that
