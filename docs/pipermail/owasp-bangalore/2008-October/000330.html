Guys,

DirBuster is one of the OWASP active projects. This tool can be used
to map the directory structure of any web application. Often one can
see the directories which are exposed only to the public, but using
this we can expose all the directories in a webroot. Please find more
information at the following link And yeah, it is purely based on Java.

http://www.owasp.org/index.php/Category:OWASP_DirBuster_Project#Overview
