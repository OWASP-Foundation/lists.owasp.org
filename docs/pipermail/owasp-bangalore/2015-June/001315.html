Between WAHH and Beginners Guide the latter does a better job of explaining 
context. While WAHH is brilliant at covering all aspects of app sec for web 
sometimes, the reader does need to be able to understand the concept with 
the whole picture. 

For the basics and getting started the Beginners Guide is definitely more 
suitable IMHO.
