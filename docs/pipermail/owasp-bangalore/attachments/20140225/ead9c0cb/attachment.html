> *Please note that all Bangalore null meets are free for anyone to attend.<br>
> There are absolutely no fees. Just come with an open mind and willingness<br>
> to share and learn.*<br>
><br>
> The schedule for this month's meet is as outlined below:<br>
><br>
> 09:30 - 10:10:  Web Application Security for Beginners: CSRF - Jayesh Singh<br>
> This is a multipart series on Web Application Security. This session will<br>
> cover Cross Site Request Forgery (CSRF) attacks, the identification and<br>
> concepts behind it. The speaker will also cover the creation of a PoC and<br>
> the mitigation for this vulnerability.<br>
><br>
> 10:10 - 10:20:  A Introductions<br>
><br>
> 10:20 - 10:50: Getting started with iptables - Sriram Narayanan<br>
> The Linux iptables firewall is usually considered either a scary or a<br>
> complicated topic. Ram will cover some basics, introduce patterns of<br>
> thinking and approaching iptables, and show some examples of how this is<br>
> used. This should get you started with exploring more about this tool.<br>
><br>
> 10:50 - 11:20:  Struts Validation Framework: Part 3 - Satish<br>
> This session is the third part of the talk on Validation frameworks. These<br>
> frameworks are used to secure information from entering business model in<br>
> an MVC architecture. ?Struts Validation framework? is a set of predefined<br>
> plugin codes which have proven best practices in Data validation. We will<br>
> take a look at the working of the framework and understand how malicious<br>
> data is treated.<br>
><br>
> 11:20 - 11:40:  Networking session followed by a break<br>
><br>
> 11:40 - 12:20:  The fall of a domain - Riyaz Walikar<br>
> This is a demo of a simulated environment that was akin to a domain<br>
> controller compromise from the network during a pentest that Riyaz<br>
> performed. He will show the tools and the tricks employed to go from a<br>
> local user to a domain administrator in this fast paced demo.<br>
><br>
> 12:20 - 12:40: XMLChor - XPath Injection Exploitation Tool (Demo) -<br>
> Harshal Jamdade<br>
> XMLChor is a web based tool which will allows attackers to find and<br>
> exploit XPATH injection vulnerabilities in vulnerable web applications.<br>
> This tool automates XPATH injection.<br>
><br>
> 12:40 - 13:10: Web Application Security: The pitfalls and the brickwalls,<br>
> a developer perspective - Insecure Direct Object Reference - Vamsi Krishna<br>
> This is a multi-part series on common developer mistakes that result in<br>
> major security vulnerabilities. This month we will see how authentication<br>
> and authorization are implemented in current systems where a developer does<br>
> not anticipate a permission model for objects resulting in unauthorized<br>
> access to data resulting in an Insecure Direct Object Reference.<br>
><br>
> 13:10 - 13:30:  Feedback and Topic discussion for next month meet<br>
><br>
