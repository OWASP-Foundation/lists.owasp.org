HTML encoding makes a lot of sense, when (as Akash mentioned) the
character-set usage is proper.<br>
<br>
For example, in UTF-7, "+ADw-" implies "&lt;" and "+AD4-" implies
"&gt;". Search for Google UTF-7 XSS Vulnerability. It'll make things a
bit more clear.<br>
