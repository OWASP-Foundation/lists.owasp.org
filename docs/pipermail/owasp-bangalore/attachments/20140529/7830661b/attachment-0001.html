                            <h3><font size="4" color="#D85D33" face="Helvetica, Arial, sans-serif">Featured OWASP Projects</font></h3>
                            <p><font size="3" face="Helvetica, Arial, sans-serif"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/7c4dfa1834/0307c7c15a/9ae0b3ebce" target="_blank">OWASP STING Game Project</a></font></p>


                            <p><font face="Helvetica, Arial, sans-serif">The OWASP STING Game Project is a card game in downloadable format or if funded, printed and distributed at OWASP events.</font></p>
                            <p><font face="Helvetica, Arial, sans-serif"> STING is a combative card game in the style of Magic the Gathering designed to teach application security attack and defense. Players will simultaneously attack other players apps while defending their own and supporting game business objectives. </font></p>

                            <p><font face="Helvetica, Arial, sans-serif">For more information, please contact the Project Leader, Tony Turner</font></p>

                            <p><font size="3" face="Helvetica, Arial, sans-serif"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/7c4dfa1834/0307c7c15a/dc356840fd" target="_blank">OWASP GoatDroid</a></font></p>
                            <p><font face="Helvetica, Arial, sans-serif">OWASP GoatDroid is a fully functional and self-contained training environment for educating developers and testers on Android security. GoatDroid requires minimal dependencies and is ideal for both Android beginners as well as more advanced users. The project currently includes two applications: FourGoats, a location-based social network, and Herd Financial, a mobile banking application. There are also several feature that greatly simplify usage within a training environment or for absolute beginners who want a good introduction to working with the Android platform.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif">For more information, please contact the Project Leader, Jack Mannino</font></p>

                            <h3><font size="4" color="#D85D33" face="Helvetica, Arial, sans-serif">New OWASP Projects</font></h3>

                            <p><font size="3" face="Helvetica, Arial, sans-serif">OWASP PHP Security Training Project</font></p>
                            <p><font face="Helvetica, Arial, sans-serif">The goal of this project is to create an interactive training system, consisting of several units, for PHP developers. Every unit is divided in an attack and a defense part. For more information, please contact the Project Leader, <a href="mailto:timo.pagel@owasp.org" target="_blank">Timo Pagel.</a></font></p>

                            <p><font size="3" face="Helvetica, Arial, sans-serif">OWASP Hardened Phalcon Project</font></p>
                            <p><font face="Helvetica, Arial, sans-serif">The Phalcon Framework is the world's fastest PHP Framework, however, like most frameworks it is not 'hardened' by default. OWASP Hardened Phalcon aims to help developers harden their Phalcon applications in-line with the published OWASP guidelines. For more information, please contact the Project Leader, <a href="mailto:rhodry.korb@owasp.org" target="_blank">Rhodry Korb.</a></font></p>

                            <h3><font size="4" color="#00549E" face="Helvetica, Arial, sans-serif">OWASP Committees 2.0</font></h3>
                            <p><font face="Helvetica, Arial, sans-serif">OWASP is an organization that has been built on collaboration and community involvement. I also hope that OWASP is an organization that can support and innovation - encouraging the community to try new things and be willing to look frequently and assess what is working and what isn't.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif">We have grown to the point where an improved process needs to be implemented where our leaders can lead and those who wish to participate can do so easily and productively.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif">In 2008, the Foundation created committees.  These committees were successful in that they pushed forward some much needed guidelines and put some structure around areas that were undefined.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif">Unfortunately, over time, there were built in flaws with the committee design that created roadblocks and eventually their failure.</font></p>
                            <p><font face="Helvetica, Arial, sans-serif">We would like to propose a revamped committee structure based on a solid foundation that provides the voice and opportunities to the community.  This structure will depend on a high level of community engagement.</font></p>
