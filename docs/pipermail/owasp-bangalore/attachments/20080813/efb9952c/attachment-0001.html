  <DIV dir=ltr align=left><FONT face=Corbel color=#0000ff><SPAN 
  class=931470314-12082008><FONT face=Corbel><FONT color=#000000><SPAN 
  style="FONT-FAMILY: 'Calibri','sans-serif'">A new threat has been discovered 
  against user supplied content.&nbsp; Basically you can concatenate an image 
  and a jar together and then upload it to a server that allows for image 
  upload.&nbsp; This type of file is called a gifar (gif + jar), but can be a 
  jpg or other file types and a jar.</SPAN><o:p></o:p></FONT></FONT></DIV>
  <DIV dir=ltr align=left>
  <DIV class=Section1>
  <P class=MsoNormal><FONT face=Corbel><FONT color=#000000><SPAN 
  style="FONT-FAMILY: 'Calibri','sans-serif'">The server can perform fairly 
  comprehensive data validation on the gifar beyond simply checking the file 
  name extension, including&nbsp;actually looking at the image header, etc. and 
  conclude that it is an image, completely missing the jar tacked onto the 
  end.&nbsp; You can unzip the file and see the jar, unzip will ignore the image 
  part of the file.</SPAN><o:p></o:p></FONT></FONT></P>
  <P class=MsoNormal><FONT face=Corbel><FONT color=#000000><SPAN 
  style="FONT-FAMILY: 'Calibri','sans-serif'">A browser (IE and Firefox both) 
  will render the image portion of the file via an image tag, and will also 
  execute the jar portion of the file via the applet tag.&nbsp; Note that the 
  applet tag can be on any page, the important thing is to be able to upload the 
  applet (embedded in the image) and be able to reference its location (must be 
  predictable).</SPAN><o:p></o:p></FONT></FONT></P>
  <P class=MsoNormal><FONT face=Corbel><FONT color=#000000><SPAN 
  style="FONT-FAMILY: 'Calibri','sans-serif'">Once the applet is running, it can 
  contact the server, so the applet can send any arbitrary and arbitrarily 
  complex HTTP and HTTPS requests back to the server the applet was retrieved 
  from.&nbsp; And, the kicker is that the applet has access to the cookies 
  related to the server it was retrieved from, so it can masquerade as the 
  user.&nbsp; Evil.</SPAN><o:p></o:p></FONT></FONT></P>
