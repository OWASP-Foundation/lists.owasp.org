About no-techie users :<br>
If they are in US, they most likely have AOL account. AOL provides<br>
OpenID, and so does Yahoo. More likely than not, they already have<br>
Yahoo account. So it might be worth the try.<br>
<br>
Wordpress allows the comments to be OpenID enabled, but in case user<br>
doesn&#39;t have OpenID, s/he can go through normal routine of providing<br>
name/email when posting comment. You can try similar option, enable<br>
OpenID access, but don&#39;t make that the &quot;only&quot; option. Allow the users<br>
to create &quot;yet another username password pair&quot; if they are OK with it.<br>
