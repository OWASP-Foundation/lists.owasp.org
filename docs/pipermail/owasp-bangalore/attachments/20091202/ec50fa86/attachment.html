<h1><b><span style="font-size: 14pt; font-family: sans-serif;">India’s
own International Hackers’ Convention</span></b><span style="font-size: 10pt;"> </span></h1>
<p><b><span style="font-size: 11pt; color: rgb(118, 148, 247);">Club</span><span style="font-size: 11pt; color: rgb(249, 90, 4);">Hack</span></b><span style="font-size: 11pt;">
</span><span style="font-size: 11.5pt; font-family: sans-serif;">is
proud to welcome you for India’s OWN, ONLY and ONE of its kind
international hackers’ convention on 5th, 6th and 7th December 2009 in
Pune, India. </span></p>
<p>
<span style="font-size: 11.5pt; font-family: sans-serif;">Started in
Dec 2007, this convention brings together hackers from all over the
country &amp; some of the celebrated international hackers. The sole
purpose of this event is “making security a common sense”. Technical
community in India has already started naming </span><b><span style="font-size: 11pt; color: rgb(118, 148, 247);">Club</span><span style="font-size: 11pt; color: rgb(249, 90, 4);">Hack</span></b><span style="font-size: 11.5pt; font-family: sans-serif;"> as India’s own
version of “Defcon” or “Hack in the box”. </span></p>
<p> 
<span style="font-size: 11.5pt; font-family: sans-serif;"><br>
December 2007, India witnessed its FIRST ever international hackers’
convention and 2008 was the sequel. </span></p>
<p>
<span style="font-size: 11.5pt; font-family: sans-serif;">The IT hub of
the country, Pune, provides perfect environment for this unique
convention.<br>
The audience largely includes, C-level executives, Sys Admins, IT
Managers, Developers, QA Engineers, Lawyers, Cyber Crime investigators
&amp; “Ethical” hackers.<br>
 </span></p>
<p>
<span style="font-size: 11.5pt; font-family: sans-serif; color: black;">In
2009, </span><b>
<span style="font-size: 11pt; font-family: &#39;Trebuchet MS&#39;,sans-serif; color: rgb(118, 148, 247);">Club</span><span style="font-size: 11pt; font-family: &#39;Trebuchet MS&#39;,sans-serif; color: rgb(249, 90, 4);">Hack</span><span style="font-size: 10pt; font-family: &#39;Trebuchet MS&#39;,sans-serif; color: black;">
</span></b><span style="font-size: 11.5pt; font-family: sans-serif; color: black;">is
proud to announce alliance with UK Science and Innovation Network,
British High Commission in India and City University. </span></p>
<p><b>
<span style="font-size: 11pt; font-family: &#39;Trebuchet MS&#39;,sans-serif; color: rgb(118, 148, 247);">Club</span><span style="font-size: 11pt; font-family: &#39;Trebuchet MS&#39;,sans-serif; color: rgb(249, 90, 4);">Hack</span></b><span style="font-size: 11.5pt; font-family: sans-serif; color: black;">2009
is a 3 day event with below mentioned high level agenda. </span></p>
<p>

</span><span style="font-size: 11.5pt; font-family: sans-serif; color: black;">Dec
