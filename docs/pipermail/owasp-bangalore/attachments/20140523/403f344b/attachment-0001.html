                            <h3><font color="#D85D33" face="Helvetica, Arial, sans-serif" size="4">Featured OWASP Project</font></h3>
                            <p><font face="Helvetica, Arial, sans-serif" size="3"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/ed8328d802/0307c7c15a/0a3133320f" target="_blank">OWASP Bricks</a></font></p>

                            <p><font face="Helvetica, Arial, sans-serif">OWASP Bricks is a deliberately vulnerable web application built on PHP & MySQL focuses on variations of commonly seen application security vulnerabilities & exploits, which can be exploited using tools like Mantra and ZAP. OWASP Bricks provides a platform for learning web application security and a test bed for analyzing the performance of web application security scanners.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif">For more information, please contact the Project Leader, Abhi Balakrishnan</font></p>

                            <h3><font color="#D85D33" face="Helvetica, Arial, sans-serif" size="4">New OWASP Projects</font></h3>

                            <p><font face="Helvetica, Arial, sans-serif" size="3">OWASP Code Pulse 2.0</font></p>
                            <p><font face="Helvetica, Arial, sans-serif">The OWASP Code Pulse team is proud to announce version 1.0 of their real-time coverage tool! Are you a penetration tester or a user of tools like ZAP? Then we think we have something that is going to make your life as a pen-tester easier. Code Pulse is a real-time code coverage tool that lets you visually see coverage gaps in your testing activity. To find out more about it and to download it please visit <a href="http://cts.vresp.com/c/?TheOWASPFoundation/ed8328d802/0307c7c15a/13787e3e6f" target="_blank">Code Pulse</a><a></a></font></p>

                            <p><font face="Helvetica, Arial, sans-serif">For more information, please contact the Project Leader, <a href="mailto:Hassan.Radwan@owasp.org" target="_blank">Hassan Radwan.</a></font></p>

                            <h3><font color="#D85D33" face="Helvetica, Arial, sans-serif" size="4">Project Announcements</font></h3>


                            <p><font face="Helvetica, Arial, sans-serif" size="3">Open Source Showcase</font></p>
                            <p><font face="Helvetica, Arial, sans-serif">The AppSec EU Conference Team is happy to announce that there will be ten projects participating in this year’s Open Source Showcase at AppSec EU this summer. The Open Source Showcase is an unique event module that allows project leaders and/or project contributors to showcase their work in a demo setting and gain exposure for their projects without to conduct a full session. The Showcase affords a more personal view of project between attendees.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif">Throughout the conference, these project will be demoing at the Open Source Showcase space within the conference venue. Join us at the Open Source Showcase June 23-26. Demo times to be announced closer to the conference.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif">See you in Cambridge!</font></p>
                            <p><font face="Helvetica, Arial, sans-serif">Below is a list of all the participating projects.</font></p>
                            <p><font face="Helvetica, Arial, sans-serif"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/ed8328d802/0307c7c15a/06bd801fae" target="_blank">Bywaf</a> - ByWaf is a web application penetration testing framework (WAPTF). It consists of a command-line interpreter and a set of plugins.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/ed8328d802/0307c7c15a/95b2008fab" target="_blank">OWASP Python Security Project</a> - Python Security aims at creating a hardened version of python that makes it easier for security professionals and developers to write applications more resilient to attacks and manipulations.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/ed8328d802/0307c7c15a/cbd003a104" target="_blank">OWASP Ninja PingU Project</a> - is a high performance network scanner tool for large scale analyses. It has been designed with performance as its primary goal and developed as a framework to allow easy plugin integration.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/ed8328d802/0307c7c15a/e537546277" target="_blank">OWASP PCI Toolkit</a> - OWASP PCI toolkit is a c# Windows form project, that will help you to scope the PCI-DSS requirements for your System Components. Beta version of this tool will be released May 2014. </font></p>

                            <p><font face="Helvetica, Arial, sans-serif"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/ed8328d802/0307c7c15a/2cbd5bcc6f" target="_blank">WPScan</a> - WPScan is a black box WordPress vulnerability scanner.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/ed8328d802/0307c7c15a/c2e0e2d78d" target="_blank">OWASP Hackademic Challenges Project</a> - The Hackademic Challenges implement realistic scenarios with known vulnerabilities in a safe, controllable environment. Users can attempt to discover and exploit these vulnerabilities in order to learn important concepts of information security through the attacker's perspective. Currently, there are 10 web application security scenarios available.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/ed8328d802/0307c7c15a/b3c569852f" target="_blank">OWASP OWTF</a> - OWASP OWTF is an OWASP+PTES-focused try to unite great tools and make pen testing more efficient.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/ed8328d802/0307c7c15a/8890b6746c" target="_blank">OWASP WTE</a> - The OWASP WTE project is an enhancement of the original OWASP Live CD Project and expands the offering from a static Live CD ISO image to a collection of sub-projects. Its primary goal is to make application security tools and documentation easily available and easy to use.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/ed8328d802/0307c7c15a/8016cd44ff" target="_blank">OWASP ZAP</a> - The Zed Attack Proxy (ZAP) is an easy to use integrated penetration testing tool for finding vulnerabilities in web applications. It is designed to be used by people with a wide range of security experience and as such is ideal for developers and functional testers who are new to penetration testing. ZAP provides automated scanners as well as a set of tools that allow you to find security vulnerabilities manually.</font></p>

                            <p><font face="Helvetica, Arial, sans-serif"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/ed8328d802/0307c7c15a/2317a05e09" target="_blank">ThreadFix</a>ThreadFix is a software vulnerability aggregation and management system that helps organizations aggregate vulnerability data, generate virtual patches, and interact with software defect tracking systems.</font></p>
