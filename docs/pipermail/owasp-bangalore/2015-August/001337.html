Hello All,

We are proud to announce the CFP for X0RC0NF 2015. 

X0RC0NF is an annual international security conference conducted in God's Own Country, Kerala. It is a platform for security researchers, hackers, enthusiasts, professionals and students. We promote security research and provide deep technical trainings on various security domains. X0RC0NF is the successor of DEFCON DCG Kerala and is run by the same team. 


Venue: Hotel Travancore Court, Kochi, India
Trainings: 17th Oct
Conference: 18th Oct

X0RC0NF CFP
--------------------
We are looking forward to technical talks on the following domains:
IoT Security and Exploitation

New Security Tools, Exploits, Vulnerabilities and Zero Days

Web Application and Server side Security

HTML5, ES6 and Client Side Security

Antivirus/IDS/Firewall/WAF/filter evasion techniques

Malware Techniques and Detection

Forensics and AntiForensics

Fuzzing OS, Applications, Browser and Exploitation

Mobile Application Security and Exploitation

Network and Wireless Security

Hardware Hacking, Physical Security and SCADA Hacking

Breaking SSL and Cryptography

Anonymity in Internet, Carding and Black Market Analysis

Last date of CFP submissions is 17th September 2015. Send in your papers to cfp <AT> xorconf.com



For details on paper formatting please visit: https://www.xorconf.com/cfp.html
