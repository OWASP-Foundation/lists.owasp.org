Leaders,

The Global Projects Committee is in the process of improving the OWASP
Project structure by determining orphaned projects, updating and
augmenting the existing project criteria, and reclassifying projects
to help users find OWASP Projects that are relevant and important to
them. To that end, we have sent out a Project Update survey to all
OWASP Project Owners to fill out before April 1st, 2009.

In addition, we would like to give the OWASP Community the opportunity
to provide feedback for any particular OWASP Project.

If there's a particular project that you enjoy using or have some
suggestions for, please fill out the feedback form here:
https://spreadsheets.google.com/viewform?hl=en&formkey=cEp6TlUxeU5KZDdXaTdlNk5PRUgxWVE6MA
..

Please keep feedback constructive and note that all feedback will be
made public.

Feel free to share this feedback form with other OWASP Community
members, OWASP Project users, etc.

You can read more about the Global Project Committee's efforts here:
https://www.owasp.org/index.php/GPC_Project_Surveys_2009
