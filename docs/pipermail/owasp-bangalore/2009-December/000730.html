Hi Srikanth

just learn about how html, http, tcp/ip socket programming works, Cookie etc
+ http header info etc + how basic Ajax will work.

Post that try to learn how you can develop basic html/form/search/query
string + simple ajax. with one or more serverside language.

exploit some stuff.  After that it is wise to use the tools... Many people
right now using tools and telling some thing is vulnerable, but they dont
know how and why....Just using tool is not enough, even tools are easy one
should know basic.
