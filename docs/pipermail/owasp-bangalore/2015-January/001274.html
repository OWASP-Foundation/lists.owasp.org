Open Source

   - SpiceWorks (agentless, easy to set up but requires a common account
   across all systems), Runs on Windows machine. With plugins you can get
   license information of most applications. Ad based but for few bucks you
   can remove the ad and put your logo. Has Helpdesk module as well, which can
   be integrated with a portal or mail-based ticketing. Downside sometimes WMI
   issues are tough to troubleshoot
   - OcsInventory (agent based, client-server architecture, has a client
   for Windows as well as Linux machines & android phones not sure about mac)
   can run on Windows or linux


Commercial:

   - Microsoft SCCM (very robust, if you are already in a Win Environment)
   - Kaspersky Endpoint Security v10 (also does a good job of asset
   inventory in the network, requires installation of Network Agent apart from
   providing end point protection)
