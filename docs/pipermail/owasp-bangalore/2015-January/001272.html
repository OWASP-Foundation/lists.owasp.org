How about SpiceWorks http://www.spiceworks.com/ ? Does well with
Software/Hardware Inventory, Licence Management, plus supporting a myriad
of other features. SpiceWorks worth with WMI for all client interactions. I
don't know if spice works can handle 3rd party libraries as well.
SpiceWorks also detects some malicious activities happening on your subnet
such an a computer trying to communicate with a potentially malicious IP
address.

Spice works is free (but closed source)

If you have open source code, there are some commercial cloud based
offerings for Static Code Analysis. https://scan.coverity.com/ is free for
open source.
