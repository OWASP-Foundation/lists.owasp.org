Hi All,

To make sure our update broadcasts are send quickly and reach all, we can
use Google Labs SMS Channels at http://labs.google.co.in/smschannels/browse,
which helps us like the google docs share.
As we already have google accounts, we can create a private channel and get
ourselves subscribed to it and the OWASP blr moderator(s) can post messages
which we receive as SMS on our mobiles.

Hope it helps.
