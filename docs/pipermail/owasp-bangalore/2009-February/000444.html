Saurabh,

Please go this presentation regarding the risks with OpenID :
http://www.slideshare.net/rohit11/risks-with-openid-presentation

About no-techie users :
If they are in US, they most likely have AOL account. AOL provides
OpenID, and so does Yahoo. More likely than not, they already have
Yahoo account. So it might be worth the try.

Wordpress allows the comments to be OpenID enabled, but in case user
doesn't have OpenID, s/he can go through normal routine of providing
name/email when posting comment. You can try similar option, enable
OpenID access, but don't make that the "only" option. Allow the users
to create "yet another username password pair" if they are OK with it.
