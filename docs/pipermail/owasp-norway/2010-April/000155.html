God påske alle sammen!

Torsdag 8. april er det tid for neste medlemsmøte. Vi kjører denne gangen 
et praktisk møte der vi jobber med Web Security Dojo.

Web security Dojo er en virtuall maskin som har flere sårbare 
webapplikasjoner og sikkerhetsverktøy installert. Den kan kjøres på
virtualbox eller på vmware (kan mao. kjøres på windows, linux eller mac).

Dette betyr at dere må ta med (eller alliere dere med noen som tar med)
laptop. Last ned og installer virtualbox/vmware og imaget på forhånd:
http://www.mavensecurity.com/web_security_dojo/

Meld dere på her: http://www.doodle.com/te2q448nsp4epy73

Kommer tilbake med lokasjon tidlig neste uke.
