Dear all,

I am happy to announce that we *published the countermeasures for the OWASP
Top 10 Privacy Risks* today. In this document
<https://www.owasp.org/images/0/0a/OWASP_Top_10_Privacy_Countermeasures_v1.0.pdf>
you can find many best practices and hints how to improve web application
privacy.

application privacy". Maybe I see you there.
