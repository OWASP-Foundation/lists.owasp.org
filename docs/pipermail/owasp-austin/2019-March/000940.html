Hello everyone!


*This email will be the last announcement sent to the old OWASP Austin
mailing list*, as it will be discontinued in a couple of days. If you have
not already done so, please sign up to be on the new Google Groups list.


Here is the link you can use to submit your request
https://groups.google.com/a/owasp.org/forum/#!forum/austin-chapter/join.

Additional help information can be viewed at
https://drive.google.com/open?id=1TBzgvB8Tb0aAZnEy2qYzLnrzJuRzK_T91Em09DVf5YA.



As a reminder, the Google Groups list, at least for our chapter's purpose,
will only be used for sending event notifications and reminders like we
have done in the past. If for whatever reason you do not want to sign up,
then please know that our wiki page (https://austin.owasp.org) will always
have the upcoming events posted.


----------


As a reminder, this month's OWASP Austin Chapter monthly meeting is coming
up next week on *Tuesday, March 26, **2019*.  Also, details about using
Zoom are below.


If you plan to attend and have not already done so, please register on
Eventbrite <https://owasp-austin-2019-march.eventbrite.com/> so that we can
be sure to order enough food for everyone.


*OWASP Austin Chapter Meeting for March*

*When: *Tuesday, March 26th @ 11:45 AM - 1:00 PM

*Where: *National Instruments, 11500 N. Mopac.Building C

*Title: *Browser Hardening, Personal Security and Privacy Measures


*In this day and age, it is becoming increasingly difficult to stay secure
and private online. In this talk, we will show you how to harden your
browser, along with a set of best practices aimed at improving one's
security and privacy**.*


Speaker: Héctor Quartino


*Héctor is the manager of the Product Security Engineering team at
Oracle+NetSuite. He has been a software developer for more than 15 years in
multiple technologies (Java, .NET and Web)**.*

enough for all!
