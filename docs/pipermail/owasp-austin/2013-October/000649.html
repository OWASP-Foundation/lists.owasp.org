There may have already been a note that's gone out so if I missed that and
I'm sending this a second time, I apologize.  In any case, I wanted to
quickly remind everyone that the "Web Application Defender's Cookbook"
study group starts today (Wed Oct 2) from 12-1 PM at the National
Instruments, Building A (smallest 2-story building on campus).  Today will
mostly just be looking at the book and trying to assign out some of the
first chapters so if you cannot make it, there is still plenty of time to
get your book ordered and start reading before next Wednesday's meeting.

A little bit about this book....it goes through various types of web-based
application attacks and then talks about how to defend against them using
the free and open source ModSecurity Apache module.  If you have web apps
and use Apache, then this is an excellent way to help defend against these
attacks without actually purchasing a WAF.  Come and learn about
ModSecurity with the group!
