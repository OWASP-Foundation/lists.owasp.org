Remember from Branden's talk that while there are many SD cards, some are
not usable with a Raspberry Pi, so make sure to check out --
http://elinux.org/RPi_SD_cards

And that the power adapter and USB cable for powering the Pi are typically
identical to a phone charger/micro USB, so that part is easily
purchased/procured elsewhere.  See the Raspberry Pi FAQ for more
information about power requirements --
http://www.raspberrypi.org/help/faqs/#powerReqs
