<p><span style="text-decoration: underline;"><strong>About Our Sponsor, Veracode</strong></span></p>
<p>Veracode provides the world’s leading Application Risk Management Platform. Veracode SecurityReview‘s patented and proven cloud-based capabilities allow customers to govern and mitigate software security risk across a single application or an enterprise portfolio with unmatched simplicity. Veracode was founded with one simple mission in mind: to make it simple and cost-effective for organizations to accurately identify and manage application security risk. </p>
<p>Why has Veracode revolutionized the way in which software is being secured? Because our automated, SaaS service - SecurityReview - is a fundamentally better way to manage application security risk: </p>
<ul class="bullets">
<li><strong>Automated</strong>. SecurityReview is completely automated and delivers a more consistent and efficient deployment of secure coding best practices and offers the fastest path to compliance over any competing application security testing solution. </li>
<li><strong>Security-as-a-Service (SaaS)</strong>. Our cloud-based model for application risk management scales globally across teams and geographies without need for any hardware or software leading to lower operational expenditures. Additionally, our service is more dynamic than any “on-premise” tool as it gets smarter the more code it processes, resulting in higher quality results, reduced risk and significantly improved developer productivity. </li>
<li><strong>Multiple Analysis Techniques.</strong> A single subscription provides access to <a href="/solutions/application-security-analyze-your-app-statically.html"><font color="#0066cc">static (white box)</font></a> and <a href="/solutions/web-application-security-analyze-your-app-dynamically.html"><font color="#0066cc">dynamic (black box)</font></a> security testing capabilities and the knowledge of our world-class security experts which negates the need to deal with multiple security vendors. </li>
<li><strong>Flexible.</strong> Our service can be used to assess both internally developed applications or assess software risk in externally sourced components or applications. </li>
<li><strong>Trusted and Independent.</strong> Veracode provides a standards-based "Verified by Veracode" rating that helps organizations achieve the fastest path to compliance and demonstrate "visible proof" of security best practices. </li>
