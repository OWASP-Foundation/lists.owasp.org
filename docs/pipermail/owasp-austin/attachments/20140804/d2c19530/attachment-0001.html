</div><div><br></div><div><p style="margin:0.4em 0px 0.5em">Title: Identifying Web Attacks via Data Analysis<br><br>Abstract: This presentation will look at detection of SQL injection using Machine Learning as well as profiling web traffic to find misbehaving hosts. The goal is to get beyond "Top N" types of analysis and begin using multiple features to guide us towards interesting traffic. With these techniques multiple log types can be used, everything from web server logs to proxy logs.<br>
