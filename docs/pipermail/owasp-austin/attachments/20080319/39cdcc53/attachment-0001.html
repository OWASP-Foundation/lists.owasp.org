<p style="margin:0 0 1em;color:#0; white-space:pre-wrap !important; white-space:-moz-pre-wrap !important; white-space:-pre-wrap !important; white-space:-o-pre-wrap !important; white-space:pre; word-wrap:break-word;">&lt;p&gt;&lt;b&gt;Topic: &lt;/b&gt;Static Analysis Techniques for Testing Application Security
<br>&lt;/p&gt;
<br>&lt;p&gt;Static Analysis of software refers to examining source code and
<br>other software artifacts without executing them. This presentation
<br>looks at how these techniques can be used to identify security defects
<br>in applications. Approaches examined will range from simple keyword
<br>search methods used to identify calls to banned functions through more
<br>sophisticated data flow analysis used to identify more complicated
<br>issues such as injection flaws. In addition, a demonstration will be
<br>given of two freely-available static analysis tools: FindBugs for the
<br>Java platform and FXCop for the .NET platform. Finally, some approaches
<br>will be presented on how organizations can start using static analysis
<br>tools as part of their development and quality assurance processes.
<br>&lt;/p&gt;&lt;b&gt;&lt;/b&gt;&lt;p&gt;&lt;b&gt;When:&lt;/b&gt; March 25th, 2008, 11:30am - 1:00pm 
<br>&lt;/p&gt;&lt;p&gt;&lt;b&gt;Who:&lt;/b&gt; Dan Cornell, Principal of Denim Group, Ltd., OWASP San Antonio Leader, Creator of Sprajax
<br>&lt;/p&gt;&lt;p&gt;Dan Cornell has over ten years of experience architecting and
<br>developing web-based software systems. He leads Denim Group&#39;s security
<br>research team in investigating the application of secure coding and
<br>development techniques to improve web-based software development
<br>methodologies.
<br>&lt;/p&gt;&lt;p&gt;Dan was the founding coordinator and chairman for the Java
<br>Users Group of San Antonio (JUGSA) and is currently the San Antonio
<br>chapter leader of the Open Web Application Security Project (OWASP). He
<br>is a recognized expert in the area of web application security for
<br>SearchSoftwareQuality.com and the primary author of Sprajax, OWASP&#39;s
<br>open source tool for assessing the security of AJAX-enabled web
<br>applications.
