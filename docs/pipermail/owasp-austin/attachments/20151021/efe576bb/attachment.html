        <h1 class="">Information Security Specialist</h1>

        <div class="">
            <span class="">Posted on <a href="http://www.planview.com/company/careers/information-security-specialist/" rel="bookmark">September 16, 2015</a></span>        </div>
    

    <div class="">
        <p>Planview, the world leader in software for portfolio and 
resource management, project collaboration, and enterprise architecture 
is growing quickly. Currently, we are looking for an Information 
Security Specialist to join our team! This new member of the team will 
work in a comfortable-paced environment, collaborating with both 
infrastructure and development teams as an independent role reporting 
directly to the Chief Security Officer.</p>

<p>The position will be responsible in providing support to plan, 
coordinate, and implement information security operations for Planview 
applications, networks, and systems. Our ideal candidate must have a 
desire to learn new concepts and continue developing their security 
skills.</p>

<h2>Responsibilities</h2>

<ul>
        <li>Support risk assessment process by identifying assets, threats and vulnerabilities</li>
        <li>Support documentation and implementing information security policy and procedures in accordance with the industry best practices</li>
        <li>Assist to define and implement security configuration standards for Planview systems</li>
        <li>Administrate automated vulnerability scans and assist in reviewing results</li>
        <li>Assist to define access privileges, control structures, and resources. Perform access reviews</li>
        <li>Monitor system security logs and application audit trails to identify and help respond to information security incidents</li>
        <li>Communicate information security policy and procedures, help increase awareness within organization</li>
        <li>Maintain and improve technical knowledge by attending educational workshops; reviewing publications</li>
</ul>

<h2>Qualifications</h2>

<ul>
        <li>Moderate-to-strong knowledge of information security concepts and fundamentals</li>
        <li>A working knowledge of several of the following areas is required:
    <ul>
        <li>Understanding of information security practices and procedures</li>
        <li>Knowledge of current security tools available</li>
        <li>Hardware/software security implementation</li>
        <li>Different communication protocols</li>
        <li>Encryption techniques/tools and current Internet technology</li>
    </ul>
</li>
        <li>Ability to apply processes, practices, and procedures on technical assignments</li>
        <li>Ability to research security related issues, new technologies and effectively communicate findings</li>
        <li>Strong command in English language with excellent writing skills</li>
        <li>Autonomous in nature and ability to propose new ideas</li>
        <li>Knowledge of security tools (e.g. QualysGuard, Dell SecureWorks IDS and Akamai WAF or similar) preferred</li>
        <li>Information security education at college or University level is desirable</li>
        <li>Professional certifications such as CISA, CISM or CISSP are a plus</li>
</ul>

<p>Planview is an Equal Opportunity/Affirmative Action Employer</p></div></div></div>
