<p class=MsoNormal><span style='font-size:10.0pt;font-family:"Arial","sans-serif"'>We
are&nbsp;hosting an&nbsp;online presentation&nbsp;on the TVA (Topological

Information Systems (CSIS)&nbsp;department at George Mason University. <o:p></o:p></span></p>

<p class=MsoNormal><span style='font-size:10.0pt;font-family:"Arial","sans-serif"'><o:p>&nbsp;</o:p></span></p>

<p class=MsoNormal><span style='font-size:10.0pt;font-family:"Arial","sans-serif"'>Currently
used by DHS, NSA, AFRL, FAA, DISA, NIST &amp; US Strategic Command, <span
class=text>CAULDRON, allows for the transformation of raw security data into
road maps that allow users to proactively prepare for attacks, manage
vulnerability risks and have real-time situational awareness. CAULDRON provides
informed risk analysis, analyzes vulnerability dependencies and shows all
possible attack paths into a network. In this way, it accounts for
sophisticated attack strategies that may penetrate an organization&#8217;s
layered defenses.</span></span><span class=text><o:p></o:p></span></p>
