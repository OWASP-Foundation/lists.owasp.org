<p class=MsoNormal style='text-autospace:none'><span style='font-size:10.0pt;
font-family:"Arial","sans-serif"'>The Open Web Application Security Project
(OWASP) &lt;<a href="http://www.owasp.org/index.php/Austin">http://www.owasp.org/index.php/Austin</a>&gt;
is an open community dedicated to enabling organizations to develop, purchase,
and maintain applications that can be trusted.&nbsp; We advocate approaching
application security as a people, process, and technology problem because the
most effective approaches to application security includes improvements in all
of these areas.<o:p></o:p></span></p>

<p class=MsoNormal style='text-autospace:none'><span style='font-size:10.0pt;
font-family:"Arial","sans-serif"'><o:p>&nbsp;</o:p></span></p>

font-family:"Arial","sans-serif"'>Each month during our chapter meeting, a
subject matter expert will present and discuss one of the OWASP top 10 topics
&lt;<a href="http://www.owasp.org/index.php/OWASP_Top_Ten">http://www.owasp.org/index.php/OWASP_Top_Ten</a>&gt;
. OWASP chapter meetings are free and open to anyone interested in application
security. <o:p></o:p></span></p>

<p class=MsoNormal><span style='font-size:10.0pt;font-family:"Arial","sans-serif"'>Abstract:<o:p></o:p></span></p>

<p class=MsoNormal><span style='font-size:10.0pt;font-family:"Arial","sans-serif"'><a
href="http://www.owasp.org/index.php/Cross-Site_Request_Forgery"><span
style='color:windowtext'>Cross Site Request Forgery</span></a> (CSRF)
vulnerabilities occur when applications allow an attacker to force a
victim&#8217;s browser to make requests detrimental to the victim&#8217;s
interests. &nbsp;Based on MITRE statistics CSRF vulnerabilities are on the
rise, and the inclusion of CSRF on the updated OWASP Top 10 2007 has cemented
their position as a serious concern for organizations building secure web
applications. &nbsp;This presentation will examine CSRF vulnerabilities in
detail, using several examples to illustrate their power and impact. &nbsp;In
addition, safeguard measures will be discussed with code examples in both Java
and .NET.</span><span style='font-family:"Arial","sans-serif"'> <br>
