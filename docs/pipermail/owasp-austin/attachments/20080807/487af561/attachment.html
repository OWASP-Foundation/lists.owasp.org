</p><p><b>Topic: </b> OWASP Live CD 2008 - An OWASP Summer of Code Project
</p><p>The OWASP Live CD 2008 project is an OWASP SoC project to update
the previously created OWASP 2007 Live CD. As the project lead, I&#39;ll
show you the latest version of the Live CD and discuss where its been
and where its going. Some of the design goals include:
</p>
<ol><li> easy for the users to keep the tools updated
</li><li> easy for the project lead to keep the tools updated
</li><li> easy to produce releases (I&#39;m thinking quarterly releases)
</li><li> focused on just web application testing - not general Pen Testing
</li></ol>
