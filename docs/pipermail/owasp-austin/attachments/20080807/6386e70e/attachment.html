<br>&lt;/p&gt;&lt;p&gt;&lt;b&gt;Who:&lt;/b&gt; Matt Tesauro
<br>&lt;/p&gt;&lt;p&gt;Matt&#39;s Bio: Matt Tesauro has worked in web application
<br>development and security since 2000. He&#39;s worn many different hats,
<br>from developer to DBA to sys admin to university lecturer to pen
<br>tester. Currently, he&#39;s focused on web application security and
<br>developing a Secure SDLC for TEA. Outside work, he is the project lead
<br>for the topic of this talk: OWASP Live CD 2008.
<br>&lt;/p&gt;&lt;p&gt;&lt;b&gt;Topic: &lt;/b&gt; OWASP Live CD 2008 - An OWASP Summer of Code Project
<br>&lt;/p&gt;&lt;p&gt;The OWASP Live CD 2008 project is an OWASP SoC project to update
<br>the previously created OWASP 2007 Live CD. As the project lead, I&#39;ll
<br>show you the latest version of the Live CD and discuss where its been
<br>and where its going. Some of the design goals include:
<br>&lt;/p&gt;
<br>&lt;ol&gt;&lt;li&gt; easy for the users to keep the tools updated
<br>&lt;/li&gt;&lt;li&gt; easy for the project lead to keep the tools updated
<br>&lt;/li&gt;&lt;li&gt; easy to produce releases (I&#39;m thinking quarterly releases)
<br>&lt;/li&gt;&lt;li&gt; focused on just web application testing - not general Pen Testing
<br>&lt;/li&gt;&lt;/ol&gt;
