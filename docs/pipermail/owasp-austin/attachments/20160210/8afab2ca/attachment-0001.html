<p style="margin:0px;font-size:13px;line-height:normal;font-family:Arial;color:rgb(35,35,35)"><b>Rugged DevOps Using Gauntlt</b></p>
<p style="margin:0px;font-size:13px;line-height:normal;font-family:Arial;color:rgb(35,35,35);min-height:15px"><br></p>
<p style="margin:0px;font-size:13px;line-height:normal;font-family:Arial;color:rgb(80,13,80)">Writing code that works is hard. Writing rugged code that can stand the test of time is even harder. This difficulty is often compounded by crunched timelines and fast cycles that prioritize new features. Add in evolving business needs and new technology and it becomes confusing to know what to do and how to integrate security into your application.</p>
<p style="margin:0px;font-size:13px;line-height:normal;font-family:Arial;color:rgb(80,13,80);min-height:15px"><br></p>
<p style="margin:0px;font-size:13px;line-height:normal;font-family:Arial;color:rgb(80,13,80)">This workshop brings in some of the top developers and application security practitioners to help you ruggedize your end-to-end development lifecycle from code commit to running system.</p>
<p style="margin:0px;font-size:13px;line-height:normal;font-family:Arial;color:rgb(80,13,80);min-height:15px"><br></p>
<p style="margin:0px;font-size:13px;line-height:normal;font-family:Arial;color:rgb(80,13,80)">Three Takeaways: </p>
<ol>
<li style="margin:0px;font-size:13px;line-height:normal;font-family:Arial;color:rgb(80,13,80)"><span style="font-size:12px;line-height:normal;font-family:Helvetica;color:rgb(0,0,0)"></span>You will learn pragmatic approaches and tooling that will affect your development processes and delivery pipelines.</li>
<li style="margin:0px;font-size:13px;line-height:normal;font-family:Arial;color:rgb(80,13,80)"><span style="font-size:12px;line-height:normal;font-family:Helvetica;color:rgb(0,0,0)"></span>Armed with tools and ideas for monitoring your operational and runtime security.</li>
<li style="margin:0px;font-size:13px;line-height:normal;font-family:Arial;color:rgb(80,13,80)"><span style="font-size:12px;line-height:normal;font-family:Helvetica;color:rgb(0,0,0)"></span>You will walk away with code examples and tools that you can put into practice right away for security and rugged testing.</li></ol>
<p style="margin:0px;font-size:13px;line-height:normal;font-family:Arial;color:rgb(80,13,80)">Bring a laptop (mac or linux) that you can install software on and a github account.</p>
