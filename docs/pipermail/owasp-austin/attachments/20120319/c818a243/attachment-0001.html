      <h3><big> <span class="mw-headline" id="March_Chapter_Meeting.">
            March Chapter Meeting.</span></big></h3>
      <p><b>When:</b> March 27th, from 11:30a-1:00pm </p>
      <p><b>Topic:</b>: Cucumber and friends: Tools for Security that
        Matter </p>
      <p>Behavior-Driven Development (BDD) helps focus software
        development on delivering prioritized, verifiable business value
        by providing a common vocabulary that spans the divide between
        Business and Technology. Cucumber is a widely used tool in Ruby
        community for implementing BDD and it executes plain-text
        functional descriptions as automated tests. In this talk, Mani
        and Tin will discuss how Cucumber and related tools can be used
        to define and verify security features that matter in software.
      </p>
      <p><br>
        <b>Speakers:</b> Mani Tadayon and Tin Zaw </p>
      <p><br>
        <b>Mani Tadayon, Senior Software Engineer, AT&T Interactive</b>
      </p>
      <p>At AT&T Interactive, Mani is part of the team responsible
        for YP.com. Mani studied foreign languages at UC Berkeley,
        computer science at Cal State Hayward and is now a graduate
        student in Geography at Cal State Northridge. He has been
        developing web applications using open source tools for over 10
        years. Currently, his focus is on behavior-driven development
        with Ruby. </p>
      <p><b>Tin Zaw, OWASP and AT&T Interactive</b> </p>
      <p>Tin is currently the president of OWASP Los Angeles chapter.
        During day time, he works with Mani at AT&T Interactive as
        an application security architect. Before AT&T, he worked as
