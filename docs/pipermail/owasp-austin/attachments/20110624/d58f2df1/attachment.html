Summary of Responsibilities:<br>
<br>
Responsible for ensuring that ERCOT&#39;s internal and external security 
posture is monitored for the latest threats and risks. Provides 
recommended actions and implements them as needed. Responsible for 
management and administration of Information Security technologies at 
ERCOT.<br>
<br>
Essential Job Duties:<br>
<br>
•  Monitors computing resources for evidence of compromise and report security incidents.<br>
•  Responds to, and report on, IT security incidents, providing initial 
assessment of impact severity and types of incidents being addressed.<br>
•  Analyzes compromised computing resources of routine to moderate 
complexity, to assist in improving security design and policy 
compliance, and to understand and document new threat profiles.<br>
•  Develops and implements mitigation strategies for threat reduction based on monitoring of IT systems.<br>
•  Researches, analyzes, designs, tests and implements new or vendor supplied security software solutions.<br>
•  Provides up to date documentation and procedures on security software product administration.<br>
•  Assists in the development of technical security standards to support
 policies, including creating, coordinating and monitoring standards and
 incident investigation procedures<br>
•  Work as a member of the Incident Response Team and perform analysis of security breaches.<br>
•  Studies and maintains current knowledge of security issues<br>
•  Performs other duties as assigned.<br>
