Security Engineer

Job Description:
HomeAway is looking for a motivated an experienced Security Engineer who will be a part of the Global IT Security Team.  Emphasis for the position will be on implementing & supporting technologies which are critical for protecting system boundaries, keeping computer systems and network devices hardened against attacks and securing sensitive data.  This is an excellent opportunity for a person with experience in the information security field to gain invaluable career experience. The position has great growth potential within the Security Team.

Key Responsibilities:
- Design, Engineer, implement and monitor security measures for the protection of computer systems, networks and information.
- Serve as a deeply skilled and knowledgeable resource within the Security Operation team.
- Lead and participate in project teams providing consultation on information security designs.
- Work on improvements including the development of new tools and supporting assets.
- Participate on automation of incident prioritization, false positives identification, correlation, vulnerability remediation.
- Participate on interconnecting tools with sources contributing to security events and incidents - e.g. logs from servers, network and security devices, Vulnerability Management system, Antivirus system, etc.
- Ensure that the company knows as much as possible, as quickly as possible about security vulnerabilities and incidents.
- Hands on experience in security systems, including firewalls, intrusion detection systems, malware prevention, authentication systems, log management, data loss prevention, cloud security, etc.
- Thorough understanding of the latest security principles, techniques, and protocols.
- Analyze infrastructure security events to determine if the event qualifies as a legitimate security breach.
- Assist in coordinating security efforts, processing security-related services, change control, security research.
- Appropriately inform and advise management on incidents and incident prevention.
- Document and conform to processes related to security monitoring and operations.
- Ability to adapt to fluid client infrastructures and scopes and to learn and support new technologies.
- Strong analytical skills and problem solving ability.

Job Requirements:
- Knowledge of Security principles, techniques, and technologies.
- Understanding the technical aspects of the Information Security.
- Advanced scripting and integration skills (Python, Ruby, Perl, etc.).
- Analytical and problem solving skills.
- Ability to manage multiple tasks.
- Must understand various attack vectors and vulnerabilities.
- Knowledge of IT technologies and endpoint solutions related to Security.
- Interest in Security, Network and System Administration & Monitoring technologies.
- Strong analytical and problem solving skills.
- Ability to manage multiple tasks and resources.
- Self-starter.
- Knowledge of Windows and Unix/Linux operating systems.
- Proficiency in written and spoken English.
- Willingness to learn and improve.

Good to have Skills:
- Experience with Security Information and Event Management systems.
- Application Security.
- Experience with Vulnerability scanners like Nessus, Qualys, etc.
- Experience with Web Application Firewalls, appliances or cloud security solutions.
- Experience with advanced endpoint security tools.
- Natural ability to work in a fast pace, ever changing, agile environment with minimum oversight.
