Keeping Your Web Apps Secure: The OWASP Top 10 & Beyond
December 7, 2010 @ ACSAC Conference, Four Seasons Hotel, Austin

The Open Web Application Security Project (OWASP) Top 10 provides an overview
of the most critical web application security risks. This tutorial introduces
the OWASP Top 10 (2010 edition) along with other risks and vulnerabilities,
and discusses the techniques and practices to protect against them. References
to software tools and other secure coding resources will also be provided.
This tutorial is a must if you are developing web applications, managing
developers, researching web security, or simply are a security enthusiast.

Full details are available at:
