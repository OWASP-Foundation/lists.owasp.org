A couple of interesting job opportunities have come across my e-mail 
lately if anyone is interested:

information security professional for a full time application security 
evaluation position.  I'm not sure what all that entails, but if it sounds 
interesting you should contact Auston Hold at holt at austonhold.com.

2) Zeb Blalock contacted me about an opportunity with Amazon in Seattle

security reviews across all elements of Amazon.com's software systems.  If
