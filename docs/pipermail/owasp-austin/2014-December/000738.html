We had planned to start the next study group topic - See the forum
titled "Study
Group - The Browser Hacker's Handbook
<http://myowasp.ning.com/groups/austinowaspstudygroup/forum/study-group-the-browser-hacker-s-handbook>"

would suggest signing up for the Ning site, then joining the "OWASP Austin
Study Group" group.  A new forum discussion thread has been created for
this next topic -
http://myowasp.ning.com/groups/austinowaspstudygroup/forum/study-group-the-browser-hacker-s-handbook,
which you would want to "follow" to receive email updates on status of the
specific study group topic. If you want to participate, then you will find
the book information noted in the discussion forum description.


If any questions or concerns, please let me know.
