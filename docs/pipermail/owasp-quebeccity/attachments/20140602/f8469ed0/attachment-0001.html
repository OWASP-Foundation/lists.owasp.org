<p class="MsoNormal" style="margin-bottom:0.0001pt"><b><i><span lang="FR-CA">"Avez-vous dit XSS?"</span></i></b><b><span lang="FR-CA">:</span></b></p>

<p class="MsoNormal" style="margin-bottom:0.0001pt"><b><span lang="FR-CA"> </span></b></p>

<p class="MsoNormal" style="margin:0cm 0cm 0.0001pt 36pt"><i><span lang="FR-CA">La faille dite de <b>Cross-site scripting</b> (XSS) est une
faille web bien connue des pirates informatiques.  Sur un site web vulnérable, elle permet d’y
injecter du code malicieux qui sera exécuté à partir du navigateur des
visiteurs, ou plutôt, des victimes...</span></i></p>

<p class="MsoNormal" style="margin:0cm 0cm 0.0001pt 36pt"><i><span lang="FR-CA"> </span></i></p>

<p class="MsoNormal" style="margin:0cm 0cm 0.0001pt 36pt"><i><span lang="FR-CA">Ces failles peuvent sembler bénignes de prime abord, mais lorsqu'elles
sont exploitées malicieusement, elles deviennent un vecteur d'attaque
puissant pour d'éventuels attaquants.</span></i></p>

<p class="MsoNormal" style="margin:0cm 0cm 0.0001pt 36pt"><i><span lang="FR-CA"> </span></i></p>

<p class="MsoNormal" style="margin:0cm 0cm 0.0001pt 36pt"><i><span lang="FR-CA">Cette présentation se veut une
introduction aux différents types de failles XSS de base.</span></i></p>

<p class="MsoNormal" style="margin:0cm 0cm 0.0001pt 36pt"><i><span lang="FR-CA"> </span></i></p>

<p class="MsoNormal" style="margin:0cm 0cm 0.0001pt 36pt"><i><span lang="FR-CA">Nous couvrirons plusieurs sujets
relatifs aux failles XSS:</span></i></p>
