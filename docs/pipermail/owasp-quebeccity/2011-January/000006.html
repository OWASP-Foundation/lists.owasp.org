ConFoo : Dévoilement de l'horaire

Nous sommes heureux de dévoiler le programme de la conférence ConFoo qui
aura lieu du 7 au 11 mars 2011. Avec près de 150 présentations réparties
dans 8 salles et avec 3 formations, ConFoo vous apporte le meilleur du
développement Web.

Formations (7 et 8 mars)
  * Advanced PHP Development (en anglais par Sebastian Bergmann, Arne
Blankerts et Stefan Priebsch de PHP.cc)
  * Démarrer avec Symfony2 (en français par Hugo Hamon de Sensio Labs)
  * Standards Web - Html5, Css3, Accessibilité et JavaScript (en français
par W3QC)

Conférences (9, 10 et 11 mars)
Nous sommes fiers d'accueillir plus de 100 spécialistes qui viendront
partager leur expertise lors des présentations. Parmi eux, nous comptons:

  * Antonio Fontes (OWASP Geneva / L7 Sécurity)
  * Philippe Gamacge (OWASP Montréal / Parler Haut, Interagir Librement)
  * Sébastien Gioria (OWASP France)
  * Sylvain Maret (MARET Consulting / OpenID Switzerland)
  * David Mirza (Subgraph)
  * Arne Blankerts (thePHP.cc)
  * Derick Rethans (PHP Team)
  * Éric Araujo (Python Team)
  * Fabien Potencier (Sensio Labs)
  * Joshua Hull (Twitter)
  * Scott MacVicar (Facebook)

Vous ne voudrez définitivement pas manquer les présentations suivantes:

  * OWASP Enterprise Security API
  * OWASP Top Ten 2010: understanding the attacks
  * Multi-factor authentication
  * Opensource Authentication and Authorization
  * Strong Authentication in Web Application
  * Distributed Identities with OpenID
  * Debugging applications with network security tools
  * DISTCHA : une tentative pour exterminer CAPTCHA
  * Hacking your website with Vega
  * Introduction à la sécurité des WebServices
  * Le bon, la brute et le truand dans les nuages
  * Security 202 - Are you sure your site is secure?
  * Threat Modeling: detecting threats before coding
  * HTML5 - Moving from hacks to solutions
  * Scalable Architecture 101
  * Graphs, Edges & Nodes: Untangling the Social Web
