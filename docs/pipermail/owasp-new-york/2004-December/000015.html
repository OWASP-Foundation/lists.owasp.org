NEXT NYC OWASP CHAPTER MEETING
 
Date: January 25, 2005
Time: 6PM - 9PM
Location: 
Time Inc.
1271 Sixth Avenue 
(between 50th & 51st Streets)
 
To attend this free event: please RSVP: pstern100 at yahoo.com with Names and Contact Info of attendees
 
Subject: ASP.NET Security
 
Presenter Background

 

David Raphael, Foundstone Senior Software Security Consultant will be talking about ASP.NET security.  David is an expert ASP.NET and Java developer and performs hundreds of security code reviews per year. Topics that David will be discussing include:

   using Microsoft's new Enterprise Library to build security into your applications

   the dangers of Full Trust in an ASP.NET world (including live demos of how hackers break applications running in Full Trust mode)

   security considerations of ASP.NET 2.0.

Schedule:
 
6:00PM – 6:30PM Check-In/Social

6:30PM – 6:45PM Chapter Details – Stan Guzik

6:45PM – 8:45PM Presentation – David Raphael

8:45PM – 9:00PM Meeting Wrap-up
