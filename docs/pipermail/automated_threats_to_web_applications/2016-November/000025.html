Project list members

Further to the work that has been undertaken over the summer, we are
pleased to announce the release of v1.1 of the OWASP Automated Threat
Handbook - Web Applications, which can be downloaded from the usual place:

   https://www.owasp.org/index.php/File:Automated-threat-handbook.pdf

We will update the document on Lulu, and also release all the source files
in due course. Tin is speaking about the project at 12:00 hrs today at
LASCON.

The changes were previously notified to this list, but primarily include
almost 9,000 extra words about relevant countermeasures.

We look for ward to working with you on v1.2 over the next few months,
which might include one or two new threat events.

Colin Watson and Tin Zaw
Project leaders
