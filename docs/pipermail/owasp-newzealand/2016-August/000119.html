Hi,

We're having an OWASP meetup in Wellington on Monday 29 August, and it
would be great if you could come -- and spread the word to your friends and
colleagues!

http://www.meetup.com/OWASP-New-Zealand-Chapter-Wellington/events/232212284/


It would be great to have a wide range of mobile developers attend, so if
you know any in Wellington I'd appreciate if you can forward the link to
them.

We also intend to try and stream the event via a Youtube livestream, so at
approx 6pm on the 29th, this link _should_ give you access to Mike's video
stream:

*https://www.youtube.com/watch?v=SbXO6wNvOM4
<https://www.youtube.com/watch?v=SbXO6wNvOM4>*

A recording will be at the same location afterwards.

*Date:* Monday 29 August

*Time: *5:45pm for a 6pm start

*Title: *Into to the OWASP Mobile Top 10

*Audience:* Anyone involved in mobile app development

*Location:* To be determined - keep an eye on Meetup

*Synopsis:*

If you're a mobile application developer, or interested in how to secure
applications, then this session is for you!

Mike Haworth from Insomnia will be introducing us to the OWASP Top 10 for
Mobile - a collection of mobile security issues that commonly occur in
applications.

This will be an introductory talk on the topic of mobile security, and so
should be suitable for anyone interested in the topic. Some concepts may
assume a bit of development knowledge, but Mike can elaborate on them if
they are unclear.

*Signup:*

http://www.meetup.com/OWASP-New-Zealand-Chapter-Wellington/events/232212284/

Cheers!
