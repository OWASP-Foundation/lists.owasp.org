Application Security for Web Developers<br>
<br>
Course Abstract:<br>
<br>
The 2 days hands-on course targets web developers, security auditors,<br>
penetration testers, security managers and anyone else who wants to<br>
learn to write secure code or to audit code against security flaws.<br>
<br>
The course covers each and every vulnerability in depth and talks  about<br>
a variety of best security practices and defence-in-depth approach which<br>
developers should keep in mind while developing applications.<br>
<br>
The attendees will be provided access to infrastructure on which they<br>
will be practising to identify vulnerable code and subsequently discuss<br>
patching approaches.<br>
<br>
While the course covers industry standards such as OWASP Top 10 and SANS<br>
top 25 security issues, it also talks about real world issues which<br>
don't find a mention in these lists.<br>
<br>
The course does not focus on any particular web development language  or<br>
technology but focus on the principles. It includes examples from  PHP,<br>
.NET, classic ASP and Java.<br>
