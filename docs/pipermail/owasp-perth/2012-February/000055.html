Hi All,

There's been quite a few updates to the OWASP AppSec AsiaPac 2012 website,
least of all the announcement of the keynote speakers. It's going to be a
great few days all.

Keynote speakers locked in so far:

- Alastair MacGibbon, the founding director of the AFP's High Tech Crime
Centre.
- Dr Jason Smith from CERT Australia
- Jacob West, Director of Software Security Research for the Enterprise
Security Products division in HP.
- Jeremiah Grossman, founder and CTO of WhiteHat Security (For those who
missed him at Ruxcon 2011, perhaps this is your chance to see this
fantastic speaker)
- Rafal Los (@Wh1t3Rabbit), Chief Security Evangelist for HP

The pricing has also been announced, starting at $445 (early bird) for
OWASP members, or $495 (early bird) to include a new OWASP membership.
These early bird prices are valid for the next 13 days so get those
requests in quick.

You can read more about the event here:
https://www.owasp.org/index.php/AppSecAsiaPac2012

And register here:
http://www.regonline.com/appsecapac2012

If you have any questions don't hesitate to let us know.

Cheers,
