<br>
* OWASP Project Presentation (i.e Tool Updates/Project Status etc)<br>
* Threat modelling of web applications<br>
* Privacy Concerns with Applications and Data Storage<br>
* Vulnerability analysis of web applications (code review, pentest,<br>
static analysis, scanning)<br>
* Baseline or Metrics for Application Security<br>
* Countermeasures for web application vulnerabilities<br>
* Web application security<br>
* Platform or language (e.g. Java, .NET) security features that help<br>
secure web applications<br>
* Secure application development<br>
* How to use databases securely in web applications<br>
* Security of Service Oriented Architectures<br>
* Access control in web applications<br>
* Web services security<br>
* Browser security<br>
* PCI<br>
<br>
The official closing date for receiving a synopsis of the presentation<br>

week of June 2010. Complete presentations will need to be submitted by<br>

* Title of the contribution<br>

* Abstract (up to 500 words)<br>
