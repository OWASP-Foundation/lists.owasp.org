Hey All,

There's an event coming up this Thursday that you might be interested in.
Perth's inaugural #CryptoParty!

You can find out more information here:

https://cryptoparty.org/wiki/Perth

The background behind the event:

  What is CryptoParty? Interested parties with computers, devices, and
  the desire to learn to use the most basic crypto programs and the
  fundamental concepts of their operation! CryptoParties are free to
  attend, public, and are commercially non-aligned.

Whilst not a traditional web security event, you might walk away learning

linked to from the wiki)
