Hi All,

The Perth OWASP Chapter is pleased to announce the first meeting of the
year. And lo' and behold, it's not being presented by one of us ;)

We're super happy to have Chris Arnold, a seasoned software architect,
present to us on Secure Coding and OOAD ..

Talk Overview:
Building on the OWASP secure coding priniciples, this session will
introduce Robert C Martin's 5 OO principles in order to show how well
designed code can help with the implementation and enforcement of secure
coding principles, as well as alleviating maintenance headaches.

Chris' Bio:
Chris doesn't like writing bios in the 3rd person, but he's always held an
interest in information systems security, and believes that it is becoming
increasingly important for developers to accept responsibility for their
product's security. He also doesn't like making unjustified statements so
if you're interested, feel free to ask him why. Chris has 13 years'
experience as a software engineer, and currently holds a position as an
architect for Swiss power and automation provider ABB.

The meeting details:
Date: Wednesday, the 21st of March 2012
Time: Sign in at 5.15pm for a kick off of 5.30pm
Location: Level 46 of the Bankwest Tower, 108 St Georges Tce, Perth

The event details are available here:
https://www.owasp.org/index.php/Perth_Australia#Secure_Coding_and_OOAD

And can you please RSVP via here:
http://owaspperthmar2012.eventbrite.com/

Looking forward to seeing all next week!
