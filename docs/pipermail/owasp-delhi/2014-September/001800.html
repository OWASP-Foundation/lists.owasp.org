Folks,

The OWASP Testing Guide Version 4 has been publicly released

There are several changes between v3 and v4. Several new chapters have been
introduced and the existing chapters improved. The test cases have been
expanded to 87 (64 test cases in v3)

This v4 of the guide also integrates with the two other flagship OWASP
documentation products: The Developers Guide and The Code Review Guide.

The Guide is a "live" document and your feedbacks are always accepted and
appreciated. Please join the mailing list and share your ideas
http://lists.owasp.org/mailman/listinfo/owasp-testing

The OWASP Testing Guide includes a "best practice" penetration testing
framework which users can implement in their own organizations and a "low
level" penetration testing guide that describes techniques for testing most
common web application and web service security issues.

Download or browse the Guide from:

- https://www.owasp.org/images/1/19/OTGv4.pdf

- https://www.owasp.org/index.php/OWASP_Testing_Guide_v4_Table_of_Contents

Thanks
