Hello,

 

Alessio has put a new version of the OWASP code review tool, Code crawler,
available for download.

It has a totally new look and feel and also runs SQL Server express for the
definitions.

The source is available also. Suggestions on new functionality are welcome.

 

Code crawler is a complementary tool associated with definitions defined in
the code review guide. It works best using both the guide and tool together.

Remember this is a beta tool, stable, but much functionality to implement.

It can be downloaded from the OWASP code review project site:

 

https://www.owasp.org/index.php/Category:OWASP_Code_Review_Project
