Old saying and a song – “It happens only in India”. Till the time laws aren’t enforced with a force, no one is bothered about security. Even laws can be bypassed here, but atleast few would be trapped to set the example for others. 



When I explain India InfoSec to the friends abroad, I generally blame the Indian mindset. Bike Riders here don’t care about their personal security and wear helmets only when there is strict police checking. Same is with the car drivers for their seat belts. In Delhi, we find better police checking so people are serious, but in other areas like Noida, no one cares. In a nutshell, only strict laws can help Corporate India to be secure and deliver secure.
