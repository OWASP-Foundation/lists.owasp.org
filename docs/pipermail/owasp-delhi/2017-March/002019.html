*Abstracts:*
*-----------------*

*Pentesting ICS 101 by Sandeep Singh:* This is an attempt of the speaker to
understand ICS Pentesting and explain it to the community. The talk
includes:

- Introduction to ICS
- ICS jargons simplified
- ICS/SCADA/DCS Architecture
- PLC: The Heart of ICS
- An overview of ICS threats and attacks
- Securing ICS

*Introduction to Buffer Overflows w/ Demo by Prashant Shukla: *The talk
contains introduction to buffer overflow attacks and all about writing a
basic first exploit for exploiting buffer overflow vulnerabilities.
