Dear Gunwant

 

The attack vector is designed to redirect a potential victim to one of the
sites configured by the bot-herder (gumblar.cn and some other sites) and
then infect the victim's system with the worm. The gumblar.cn site has been
closed by the authorities, so have some of the others, but there still might
be some more active out there. 

 

I suggest you not export the worm in a raw form. The one which I have posted
in the blog has been appropriately padded. The reason why it has been posted
is for web admins or webmasters to use it as a reference to check for any
malicious Javascript in their own webservers, etc.

 

Regards

Abhay
