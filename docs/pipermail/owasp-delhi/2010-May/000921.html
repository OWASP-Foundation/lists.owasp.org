The WASC 26 is an alias for the Threat Classification v1 which can be found
at
http://projects.webappsec.org/Threat-Classification-Previous-Versions

The latest version of the Threat Classification (v2.0) can be found at
http://projects.webappsec.org/Threat-Classification
and
http://projects.webappsec.org/f/WASC-TC-v2_0.pdf
and
http://projects.webappsec.org/Threat-Classification-Reference-Grid

Let me know if you have any questions.
