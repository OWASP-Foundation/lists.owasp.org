*Abstracts:*
*------------------------*

*Escape Plan - Evading Web Application Firewalls (WAF) for XSS Attacks by
Divya: *The talk will cover introduction to different aspects of WAF
Evasion for XSS and a case study from a previous client engagement of the
speaker.

*Machine Learning in Information Security - The Relationship by Mohammed
Zuber:* Big Data, Data Science, Machine Learning and Analytics are a few of
the new buzzwords now prevailing in the industry. In this talk the speaker
will give general overview of the Big Data ecosystem and how it can help in
information security problems.

*Pwning Windows Mobile Applications by Ankit Giri:* The talk will cover the
following topics.
- Mobile Platform Operating Systems
- Windows Phone Overview
- What we can test?
- Challenges
- Approach & Prerequisites
- Methodology
- Application File Structure
- Tools for Penetration Testing
- Security Features
