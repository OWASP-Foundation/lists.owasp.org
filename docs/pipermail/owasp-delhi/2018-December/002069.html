Hi All,

Announcing the upcoming OWASP Delhi Chapter December meetup on December 15,
2018 (Saturday) - 11:00 am onwards.

Note - All OWASP chapter meets are free and open to everyone. Prior RSVP is
mandatory in order to be allowed to attend the meet.

Note - This is a combined null Delhi and OWASP Delhi meet up

*More Details and RSVP here: *
https://null.co.in/events/525-delhi-null-delhi-meet-15-december-2018-combined-null-owasp-meet

*Details: *https://www.owasp.org/index.php/Delhi_NCR

*Agenda:*
11:15 am - 11:30 am: Introductions and Chapter Updates
11:30 am - 11:45 am: News Bytes & Discussions on Latest Infosec happenings
by Pratul
11:45 am - 12:15 pm: Notable APT activities of 2018 and their analysis by
Ankit
12:15 pm - 01:00 pm: Getting Started with Threat Hunting by Sandeep
01:05 pm - 01:25 pm: Networking Breaks
01:25 pm - 01:55 pm: Setting up and using VPS Infrastructure for BugBounty
and Pentesting by Narendra
02:00 pm - 02:20 pm: Offensive OSINT by Tushar
02:20 pm - 02:30 pm: Feedback and Next Meet Discussion

*When: *December 15, 2018 (Saturday) - 11:00 am onward (Please be on time)

*Where: *Airtel Center
Plot No 16, Udyog Vihar Phase IV,
Gurgaon - 122001

*Landmark:* Airtel Office location is Atlas Chowk

- IFFCO Chowk Metro / MG Road Metro / IndusInd Bank Rapid Metro Station

*Session Briefs*
*------------------------*

*1. Notable APT activities of 2018 and their analysis**:* This talk will
cover the most interesting developments in terms of APT activity throughout
the year and what can we learn from them

*2. Getting Started with Threat Hunting: *The talk will cover the following:

   - Some background on Detection and Threat Hunting
   - What is it? Why do organizations need it?
   - Becoming a Threat Hunter - Skills needed - Where does the role fit in?
   - Useful frameworks and projects
   - Example Threat Hunting Scenario
   - Open Source Tools to Assist in Threat Hunting
   - Further Learning Resources

Note: The talk does not cover how to set up hunt teams and programs in
organizations, rather it focusses on individuals interested in the Threat
Hunting

*3. Setting up and using VPS Infrastructure for BugBounty and Pentesting: *This
talk will cover the following:

   - What is a VPS
   - How much would it cost to setup a VPS
   - How much config you need
   - Setting up a simple VPS and tools
   - Best use cases
   - Some other use cases
   - How can you get a ready to use VPS for pentests and bug bounty

*4. Offensive OSINT:* The talk covers the offensive uses of OSINT

See you there.

Thanks
