<div>The split happens due carriage return i.e. CRLF Injection. HTTP Response Splitting is one of the attacks originating from CRLF Injection. The other one is Log Forging refer <a href="http://www.acunetix.com/websitesecurity/crlf-injection.htm">http://www.acunetix.com/websitesecurity/crlf-injection.htm</a>.</div>
