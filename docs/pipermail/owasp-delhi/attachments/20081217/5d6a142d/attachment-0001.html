<br>
ANNOUNCING THE NEW &quot;OWASP TESTING GUIDE v3<br>
<br>
OWASP is announcing the new OWASP Testing Guide v3. The project as<br>
part of the OWASP Summer of Code, started on April 2008 reviewing the<br>
version 2, improving it.<br>
OWASP Testing Guide v3 is a 349 page book; we have split the set of<br>
active tests in 9 sub-categories for a total of 66 controls to test<br>
during the Web Application Testing activity.<br>
Each control has an OWASP name, so for example a SQL Injection is<br>
called: OWASP-DV-005, meaning that it is the 5th control of the Data<br>
Validation category.<br>
We got a dream team of 21 authors and 4 reviewers: after 6 months of<br>
hard work and great team work we realized the v3.<br>
<br>
We'd like to ask you to support OWASP to reach the following goals:<br>
<br>
*** Continuously improve the guide.<br>
The Guide is a &quot;live&quot; document: we always need your feedback!<br>
Please join our testing mailing list and share your ideas:<br>
<a href="http://lists.owasp.org/mailman/listinfo/owasp-testing" target="_blank">http://lists.owasp.org/mailman/listinfo/owasp-testing</a><br>
<br>
*** Promote the Testing Guide.<br>
We would like to have some more media coverage on the guide, so<br>
please, if you know somebody in there put them in touch.<br>
If you have the chance, you can write an article about the Testing<br>
Guide and the new OWASP Projects.<br>
Also you can pick up the OWASP Testing Guide presentations and talk<br>
about it in local conferences and Chapter meetings.<br>
<a
href="http://www.owasp.org/images/2/2c/OWASP_EU_Summit_2008_OWASP_Testing_Guide_v3.ppt"
target="_blank">http://www.owasp.org/images/2/2c/OWASP_EU_Summit_2008_OWASP_Testing_Guide_v3.ppt</a><br>
<br>
*** Add 'quotes' to the Guide.<br>
We made a special 'quotes' pages for the Testing Guide.<br>
Here we'd want to add all the comments and references to the Guide.<br>
<a href="http://www.owasp.org/index.php/OWASP_Testing_Guide_Quotes"
target="_blank">http://www.owasp.org/index.php/OWASP_Testing_Guide_Quotes</a><br>
<br>
The OWASP Testing Guide includes a &quot;best practice&quot; penetration
testing<br>
framework which users can implement in their own organizations and a<br>
&quot;low level&quot; penetration testing guide that describes techniques for<br>
testing most common web application and web service security issues.<br>
