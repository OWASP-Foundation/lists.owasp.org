<p class=MsoNormal><font size=2 color="#000066" face=Verdana><span

to make it quick launch to market. Moreover, they weren't really sure about
being so lucky and getting so many followers in a quick span of time. But idea
got back to basics after few months from its launch. Using RoR, one you can
build applications approx. 4-5 times faster in comparison to other web
programming languages like PHP/ASP.Net/J2EE/etc. <br>
<br>
If I remember it correctly, then last year Twitter accepted the scalability
limitations and raised venture funding to port entire application from RoR to
php. I am not really sure, if they are still on to it but I am sure it would be
very difficult to port it due to open APIs those have already been published
and used by different applications/portals. Having said that, the recent
