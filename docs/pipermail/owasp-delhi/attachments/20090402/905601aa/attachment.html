. The problem with Black box testing is that it cannot really show all the
vulnerabilities that are inherent in an application, nor can anybody quantify
how much of testing has been done on the application. Thus, leaving an
application vulnerable, post- testing. An inside out approach is what is
required to assure that all paths have been tested. A solution like Fortify PTA
earlier known as Fortify Tracer provides deeper insight into application
testing. Of course you can&#8217;t do away with black box vulnerability
assessment solutions as they are the only way to find infrastructure based
vulnerabilities. There is also a report in the ISACA&#8217; Information System
Control Journal (Vol 6. 2008) on black box testing statistics which would help
you decide in choosing a solution. <o:p></o:p></span></font></p>
