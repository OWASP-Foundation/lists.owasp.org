Whether cookies are shared between browser windows depends on various<br>
factors like browser vendor and how the new browser windows is launched.<br>
Due to these inconsistencies it is good for web application security<br>
professionals to assume that cookies will be shared across browser windows<br>
and tabs. As you might all know, cookies will not be shared across browsers<br>
from different vendors.<br>
<br>
Examples:<br>
<br>
In Win XP SP3 IE7:<br>
a. Cookies will be shared if the different browser windows belong to the<br>
same process. During normal use, this happens when one right-clicks a link<br>
and then chooses new tab or new window.<br>
b. Cookies will not be shared if the browser windows belong to separate<br>
processes. During normal use, this happens when Internet Explorer icon (in<br>
desktop or startmenu) is double clicked.<br>
<br>
In Win XP SP3 Firefox 3.0.x:<br>
Firefox maintains only one process, so cookies are shared between all<br>
windows and tabs.<br>
