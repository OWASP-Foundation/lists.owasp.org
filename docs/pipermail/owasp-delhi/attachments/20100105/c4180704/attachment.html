<p><span lang="EN-US"> </span></p>
<p><span lang="EN-US">The Web Application Security Consortium (WASC) is pleased to announce the long awaited release of the WASC </span></p>
<p><span lang="EN-US">Threat Classification v2.0. The Threat Classification is an effort to classify the weaknesses, and attacks </span></p>
<p><span lang="EN-US">that can lead to the compromise of a website, its data, or its users. This document&#39;s primarily purpose is </span></p>
<p><span lang="EN-US">to serve as a reference guide for common attacks and weaknesses.</span></p>
<p><span lang="EN-US"> </span></p>
<p><span lang="EN-US">Main goals</span></p>
<p><span lang="EN-US">- Refine document scope, terminology, and purpose</span></p>
<p><span lang="EN-US">- Update existing sections when applicable</span></p>
<p><span lang="EN-US">- Add missing attacks and weaknesses</span></p>
<p><span lang="EN-US">- Creation of a firm, scalable base foundation allowing for the introduction of data views allowing for various </span></p>
<p><span lang="EN-US">  forms of data representation</span></p>
<p><span lang="EN-US">- Addition of attack and weakness reference identifiers (WASC-&lt;xx&gt;)</span></p>
<p><span lang="EN-US">- Publication of two data views</span></p>
