OK, If you are looking for a piece of cake, then I would suggest you to
&quot;include&quot; functions in your code that have already been built by
experts for input/output validation anterior to be included in the PHP
distributions. You would not be able to find any built-in security
input/output validation functions in standard PHP distributions as yet.<br>
<br>
Instead you can download and include functions in your code like the
famous &quot;field_validator&quot;  function available here:
<a href="http://www.osix.net/modules/article/?id=218">http://www.osix.net/modules/article/?id=218</a>.  By doing this you still will be using Regexs but you do not have to include regexs on each field. You will use the function call instead.<br>
