<p><u><font face="Arial" color="navy" size="2"><span style="FONT-SIZE: 11pt; COLOR: navy; FONT-FAMILY: Arial">What are Magic Quotes?</span></font></u></p>
<p><u><font face="Arial" color="navy" size="2"><span style="FONT-SIZE: 11pt; COLOR: navy; FONT-FAMILY: Arial"><span style="TEXT-DECORATION: none">&nbsp;</span></span></font></u></p>
<p><font face="Arial" color="navy" size="2"><span style="FONT-SIZE: 11pt; COLOR: navy; FONT-FAMILY: Arial">When on, all <i><span style="FONT-STYLE: italic">&#39;</span></i> (single-quote), <i><span style="FONT-STYLE: italic">&quot;</span></i> (double quote), <i><span style="FONT-STYLE: italic">\</span></i> (backslash) and <i><span style="FONT-STYLE: italic">NULL</span></i> characters are escaped with a backslash automatically. This is identical to what <a href="http://in2.php.net/manual/en/function.addslashes.php" target="_blank">addslashes()</a> does. </span></font></p>

<p><font face="Arial" color="navy" size="2"><span style="FONT-SIZE: 11pt; COLOR: navy; FONT-FAMILY: Arial">There are three magic quote directives: </span></font></p>
<ul style="MARGIN-TOP: 0in" type="disc">
<li style="COLOR: navy"><font face="Arial" color="navy" size="2"><span style="FONT-SIZE: 11pt; FONT-FAMILY: Arial"><a href="http://in2.php.net/manual/en/info.configuration.php#ini.magic-quotes-gpc" target="_blank">magic_quotes_gpc</a> Affects HTTP Request data (GET, POST, and COOKIE). Cannot be set at runtime, and defaults to <i><span style="FONT-STYLE: italic">on</span></i> in PHP. See also <a href="http://in2.php.net/manual/en/function.get-magic-quotes-gpc.php" target="_blank">get_magic_quotes_gpc()</a>. </span></font></li>

<li style="COLOR: navy"><font face="Arial" color="navy" size="2"><span style="FONT-SIZE: 11pt; FONT-FAMILY: Arial"><a href="http://in2.php.net/manual/en/info.configuration.php#ini.magic-quotes-runtime" target="_blank">magic_quotes_runtime</a> If enabled, most functions that return data from an external source, including databases and text files, will have quotes escaped with a backslash. Can be set at runtime, and defaults to <i><span style="FONT-STYLE: italic">off</span></i> in PHP. See also <a href="http://in2.php.net/manual/en/function.set-magic-quotes-runtime.php" target="_blank">set_magic_quotes_runtime()</a> and <a href="http://in2.php.net/manual/en/function.get-magic-quotes-runtime.php" target="_blank">get_magic_quotes_runtime()</a>. </span></font></li>

<li style="COLOR: navy"><font face="Arial" color="navy" size="2"><span style="FONT-SIZE: 11pt; FONT-FAMILY: Arial"><a href="http://in2.php.net/manual/en/sybase.configuration.php#ini.magic-quotes-sybase" target="_blank">magic_quotes_sybase</a> If enabled, a single-quote is escaped with a single-quote instead of a backslash. If on, it completely overrides <a href="http://in2.php.net/manual/en/info.configuration.php#ini.magic-quotes-gpc" target="_blank">magic_quotes_gpc</a>. Having both directives enabled means only single quotes are escaped as <i><span style="FONT-STYLE: italic">&#39;&#39;</span></i>. Double quotes, backslashes and NULL&#39;s will remain untouched and unescaped. See also <a href="http://in2.php.net/manual/en/function.ini-get.php" target="_blank">ini_get()</a> for retrieving its value. </span></font></li>
</ul>

<p><u><font face="Arial" color="navy" size="2"><span style="FONT-SIZE: 11pt; COLOR: navy; FONT-FAMILY: Arial">magic_quotes_gpc</span></font></u></p>
<p><font face="Arial" color="navy" size="2"><span style="FONT-SIZE: 11pt; COLOR: navy; FONT-FAMILY: Arial">Sets the magic_quotes state for GPC (Get/Post/Cookie) operations. When magic_quotes are on, all &#39; (single-quote), &quot; (double quote), \ (backslash) and NUL&#39;s are escaped with a backslash automatically.</span></font></p>
