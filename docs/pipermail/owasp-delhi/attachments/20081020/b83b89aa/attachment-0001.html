<p class=MsoNormal><u><font size=2 color=navy face=Arial><span
style='font-size:11.0pt;font-family:Arial;color:navy'>What are Magic Quotes?<o:p></o:p></span></font></u></p>

<p class=MsoNormal><u><font size=2 color=navy face=Arial><span
style='font-size:11.0pt;font-family:Arial;color:navy'><o:p><span
 style='text-decoration:none'>&nbsp;</span></o:p></span></font></u></p>

<p class=MsoNormal><font size=2 color=navy face=Arial><span style='font-size:
11.0pt;font-family:Arial;color:navy'>When on, all <i><span style='font-style:
italic'>'</span></i> (single-quote), <i><span style='font-style:italic'>&quot;</span></i>
(double quote), <i><span style='font-style:italic'>\</span></i> (backslash) and
<i><span style='font-style:italic'>NULL</span></i> characters are escaped with
a backslash automatically. This is identical to what <a
href="http://in2.php.net/manual/en/function.addslashes.php">addslashes()</a>
does. <o:p></o:p></span></font></p>

<p class=MsoNormal><font size=2 color=navy face=Arial><span style='font-size:
11.0pt;font-family:Arial;color:navy'>There are three magic quote directives: <o:p></o:p></span></font></p>

<ul style='margin-top:0in' type=disc>
 <li class=MsoNormal style='color:navy;mso-list:l0 level1 lfo1'><font size=2
     color=navy face=Arial><span style='font-size:11.0pt;font-family:Arial'><a
     href="http://in2.php.net/manual/en/info.configuration.php#ini.magic-quotes-gpc">magic_quotes_gpc</a>
     Affects HTTP Request data (GET, POST, and COOKIE). Cannot be set at runtime,
     and defaults to <i><span style='font-style:italic'>on</span></i> in PHP.
     See also <a
     href="http://in2.php.net/manual/en/function.get-magic-quotes-gpc.php">get_magic_quotes_gpc()</a>.
     <o:p></o:p></span></font></li>
 <li class=MsoNormal style='color:navy;mso-list:l0 level1 lfo1'><font size=2
     color=navy face=Arial><span style='font-size:11.0pt;font-family:Arial'><a
     href="http://in2.php.net/manual/en/info.configuration.php#ini.magic-quotes-runtime">magic_quotes_runtime</a>
     If enabled, most functions that return data from an external source,
     including databases and text files, will have quotes escaped with a
     backslash. Can be set at runtime, and defaults to <i><span
     style='font-style:italic'>off</span></i> in PHP. See also <a
     href="http://in2.php.net/manual/en/function.set-magic-quotes-runtime.php">set_magic_quotes_runtime()</a>
     and <a
     href="http://in2.php.net/manual/en/function.get-magic-quotes-runtime.php">get_magic_quotes_runtime()</a>.
     <o:p></o:p></span></font></li>
 <li class=MsoNormal style='color:navy;mso-list:l0 level1 lfo1'><font size=2
     color=navy face=Arial><span style='font-size:11.0pt;font-family:Arial'><a
     href="http://in2.php.net/manual/en/sybase.configuration.php#ini.magic-quotes-sybase">magic_quotes_sybase</a>
     If enabled, a single-quote is escaped with a single-quote instead of a
     backslash. If on, it completely overrides <a
     href="http://in2.php.net/manual/en/info.configuration.php#ini.magic-quotes-gpc">magic_quotes_gpc</a>.
     Having both directives enabled means only single quotes are escaped as <i><span
     style='font-style:italic'>''</span></i>. Double quotes, backslashes and
     NULL's will remain untouched and unescaped. See also <a
     href="http://in2.php.net/manual/en/function.ini-get.php">ini_get()</a> for
     retrieving its value. <o:p></o:p></span></font></li>
</ul>

<p class=MsoNormal><font size=2 color=navy face=Arial><span style='font-size:
11.0pt;font-family:Arial;color:navy'><o:p>&nbsp;</o:p></span></font></p>

<p class=MsoNormal><font size=2 color=navy face=Arial><span style='font-size:
11.0pt;font-family:Arial;color:navy'>Source: <a
href="http://in2.php.net/magic_quotes">http://in2.php.net/magic_quotes</a> <o:p></o:p></span></font></p>

<p class=MsoNormal><u><font size=2 color=navy face=Arial><span
style='font-size:11.0pt;font-family:Arial;color:navy'>magic_quotes_gpc<o:p></o:p></span></font></u></p>

<p class=MsoNormal><font size=2 color=navy face=Arial><span style='font-size:
11.0pt;font-family:Arial;color:navy'>Sets the magic_quotes state for GPC
(Get/Post/Cookie) operations. When magic_quotes are on, all ' (single-quote),
&quot; (double quote), \ (backslash) and NUL's are escaped with a backslash
automatically.<o:p></o:p></span></font></p>
