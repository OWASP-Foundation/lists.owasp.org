<div>Another aspect of the Session ID analysis is to fuzz the Session-Header value i.e. supplying malicious values in the Session Header field. Fuzzing is a great option to catch any exceptions w.r.t SIDs. Popular free fuzzers include SPIKE Proxy, Peach Fuzzer Framework. There is a Fuzzer-UI available in WebScarab besides the Session-ID analysis tab. </div>
