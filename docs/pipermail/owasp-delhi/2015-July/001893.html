Hi Vaibhav,
You cannot create or set or read cookie for another domain from your domaim.

So CSRF of this kind wouldn'tbe possible.
Not sure, if I could understand it fully. Have written quick code for
you to assess my understanding:

==poc.html==
<!DOCTYPE html>
<html>
  <head>
    <script>
      function myFunction() {
        document.cookie = "csrfToken=token123";

document.createElement('form').submit.call(document.getElementById('myForm'));
      }
    </script>
  </head>
  <body onload="myFunction()">
    <form id="myForm" name="myForm" action="http://localhost/poc.php"
method="POST">
      <input type=hidden name="username" id="username"
value="someusername"/>
      <input type=hidden name="csrfToken" id="val3" value="token123"/>
      <input type=hidden name="submit" id="submit" value="Continue"/>
    </form>
  </body>
</html>

==poc.php==
<?php
  var_dump($_POST);
  var_dump($_COOKIE);
