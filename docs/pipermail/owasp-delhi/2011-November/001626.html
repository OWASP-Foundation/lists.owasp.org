Vaibhav,

In java, we can do in following way:

session.invalidate();
session=request.getSession(true);

Above code will destroy the previous session and regenerate a new session
id. Same 'concept' could be used in .Net. I am not sure with the code
though. You may want to view the following link by MSDN as well:

http://msdn.microsoft.com/en-us/library/ms178582.aspx

Let me know if it works! :-)
