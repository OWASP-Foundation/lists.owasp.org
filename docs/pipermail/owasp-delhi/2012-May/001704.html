Hi Karn,

 

I think Waseem has highlighted a good point. You should consider smooth
integration of Code Review tool with WAF so that virtual patching is not a
nightmare. There are few code review tools which have easy integration
support with known WAFs. Waseem has already mentioned about Armorize
products and same support you might find for other products as well. But
"integration" should be part of selection criteria during evaluation it
helps when you are reviewing apps already in production. Reviewing apps
under development, it doesn't value too much though.
