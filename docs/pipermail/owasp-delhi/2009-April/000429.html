Mod-security is the most popular OS/free WAF around.
It not really that hard to set-up, install it as an Apache module, 
configure apache to act in proxy mode and you are almost there.
Not that hard to do, just takes sometime and googling, there are plenty of 
tutorials available online which explain the steps clearly.
Get the help of your Apache administrator.

There is a lot happening with mod-sec, stuff like virtual-patching are 
gaining momentum, you dont want to miss the fun do you???
