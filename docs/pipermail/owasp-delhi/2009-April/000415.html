To answer you question Venkat the cookie will contain a test variable
to check whether or not cookies are enabled. Beside that we could
include some random variable which would be required beside user name
and password. This could be used for counting the number of attempts.

We are using RDBMS at the backend. What Lava suggests makes sense that
we count the number of failed logins on the basis of username and
prompt with a captcha after X number of fails.
