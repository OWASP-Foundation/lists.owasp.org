Here is the sequence that I observed:

 

1.     www.tcs.com and tcs.com were found to be pointing to different IPs.
www.tcs.com was working properly with
http://www.tcs.com/homepage/Pages/default.aspx as default home page and
tcs.com was available for sale. 

2.     After a while, Rakshit pointed that both the domains had same IP
address but tcs.com was still "available for sale" while www.tcs.com was
working. Till this time, it appeared to be some configuration error.

3.     Waited for some more time, both the domains were available for sale.
With all the files deleted from the web root of www.tcs.com. I tried
http://www.tcs.com/homepage/Pages/default.aspx  and
http://www.tcs.com/careers to get runtime error and page not found error
respectively. Couple of their other portals like nextstep etc. were throwing
error to the browser while accepting ssl certificates. Not sure, if it is
usual or appeared today only.

 

The domain sale page had a contact listed abed_uk at hotmail.com. I googled
this email account to find it listed as technical contact for many domains
with a similarity that all those domains were registered at Network
Solutions and obviously had same "available for sales" page set as home
page. Seems that their server(s) was compromised for few hours today.
