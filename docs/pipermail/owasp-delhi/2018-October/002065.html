Hi OWASP Community Members,



We are glad to invite you to the second edition of Security BSides Delhi
on 26th October 2018.


We would be happy to provide a 15% discount to all OWASP Delhi Members



The link to register for the conference is
https://bsidesdelhi.in/registration/

OWASP Delhi 15% Discount Code - OWASPDEL152018



*About BSides Delhi 2018:*


   - 1-day conference packed with Technical Talks, Workshops, villages, CTF
   and Lightning Talks
   - Community organised, volunteer-driven, and sponsor funded
   - There are currently more than 60+ individually organized BSides events
   worldwide which are organized every year
   - *Date: *26th October 2018 (Friday)
   - *Venue: *Vivanta by Taj Hotel, Dwarka, New Delhi



*Few highlights of the conference-*


*Keynote Speakers:*


   - Dave Lewis- Advisory CISO - Global, Duo Security (Talk Title: Security
   and the Eternal Return)
   - Pukhraj Singh, Cyber Intelligence Analyst (Talk Title: Politics &
   Power in Cybersecurity)



*Few Workshops/Village/Sub-Events:*


   - Active Defense using Honeypots By Rahul Binjve
   - Attacking Web Applications with Burp Suite by Riddhi Shree & Vandana
   Verma
   - Hunting Malware in Memory by Malla Reddy
   - Discerning high impact mobile apps vulnerabilities by Abhinav Mishra



*Some of the Talks:*


   - SITH Happens: Attack of Malicious WASMs by Kaizhe Huang
   - DomGoat – The DOM Security Playground by Lava Kumar
   - Finding memory bugs with the Address Sanitizer by Siddharth Muralee
   - Self-host like it’s 2018 and reclaim your privacy by Subash S N
   - Weaponizing a raspberry pi for Red Team Assessments by Sachin S Kamath
   - Garbage Collection Internals of Jscript by Sudhakar Verma
   - Securing the supply chain with risk-based framework by Prithvinder
    Singh
   - Headshot: Game Hacking on macOS by Jai Verma

*Other:*

   - Full Day CTF Hosted by Team bi0s
   - Career Village (Resume Review Sessions, Talks, etc.)

For the complete list of activities and other details visit:
