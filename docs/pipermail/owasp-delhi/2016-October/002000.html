Hi all,

Announcing the *OWASP Delhi Special Workshop on Web Hacking* by Jitendra
Kumar and Narendra Kumar on 05th November, 2016 (Saturday)

This workshop is free to attend but due to limited number of seats
available we will shortlist a fix number of participants based on the
registrations recieved.

*Please register for the workshop here *-
https://docs.google.com/a/owasp.org/forms/d/e/1FAIpQLSdSH1JN32q18J2oBNHdE2WhL7ZNR5a0-LeWY1xYSrfjeJUQnw/viewform

*Details:* https://goo.gl/1mBbRU

Note - All shortlisted participants will be sent a separate invite for the
workshop with venue and other details.

*Date: *05th November, 2016
*Duration: *10 am - 06 pm

*Agenda:*

   - Lab Setup
   - Introduction to OWASP Top 10
   - OWASP Top 10 Hands on
   - Task solving

*Prerequisites:*

   - Basic knowledge of Web Application
   - Interested in learning Web Application Security

*Lab requirements:*

   - Operating System: Windows 7 or above (with admin privilege)
   - RAM: Minimum 4GB
   - Hard disk: Minimum 50GB free space
   - Virtual Box [Download <https://www.virtualbox.org/wiki/Downloads>]
   - OWASP BWA Project 1.2 image file [Download
   <http://downloads.sourceforge.net/project/owaspbwa/1.2/OWASP_Broken_Web_Apps_VM_1.2.7z>
   ]
   - BurpSuite proxy [Download <https://portswigger.net/burp/download.html>]
   - Wireshark [Download <https://www.wireshark.org/download.html>]
   - Mozilla Firefox Browser [Download
   <https://www.mozilla.org/en-US/firefox/new/>]
   - 7-Zip file archiver [Download <http://www.7-zip.org/download.html>]

*Bio:*

*Jitendra Kumar:* is an Information Security Engineer. He has worked on
Source code review, Mobile Application, Web Application, Network, Thick
Client and Wireless Security Testing. He is an  Information Security
trainer and conducts trainings on Secure SDLC, Web Application Security,
Linux Server Security and Information Security awareness programs.

*Narendra Kumar:* is a final year engineering student. Pursuing his B.Tech
in Computer Science. He is a programmer with interest in information
security. He mostly codes in C and C++ language. He conducts sessions and
runs a "Student Tech Cell" at college level to share his knowledge about
information security and programming.

For any queries please reach out to the chapter leaders

thanks
Sandeep
