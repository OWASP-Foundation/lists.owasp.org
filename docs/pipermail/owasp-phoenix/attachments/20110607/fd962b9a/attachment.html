 also serves on the board of directors for HeatSync Labs, a hackerspace in Chandler.<p> <p> <p>OWASP O2 Platform<p>Dinis Cruz<p> <p> <p>The O2 Platform is focused on automating application security knowledge and workflows. It is specifically designed for developers and security consultants to be able to perform quick, effective and thorough source code-driven application security reviews (blackbox + whitebox). In addition to the manual findings created/discovered by security consultants, the OWASP O2 Platform allows the easy consumption of results from multiple OWASP  projects and commercial scanning tools. This allows security consultants to find, exploit and automate (via Unit Tests) security vulnerabilities usually dismissed by the community as impossible to find/recreate. More importantly, it provides security consultants a mechanism to: (a) &quot;talk&quot; with developers (via UnitTest),  (b) give developers a way to replicate + &quot;check if it&#39;s fixed&quot; the vulnerabilities reported and (c) engage in a two-way conversion on the best way to fix/remediate those vulnerabilities. For more details see <a href="http://www.google.com/url?q=https%3A%2F%2Fwww.owasp.org%2Findex.php%2FOWASP_O2_Platform&amp;usd=2&amp;usg=AFQjCNHdc3QOdmzkDrXtY5I5gK6gYV14Mw" target="_blank">https://www.owasp.org/index.php/OWASP_O2_Platform</a>, to download binary or source goto <a href="http://code.google.com/p/o2platform/downloads/list" target="_blank">http://code.google.com/p/o2platform/downloads/list</a><p> <p>Bio<p>Dinis Cruz is a Security Consultant based in London (UK) and specialized in: <a href="http://www.google.com/url?q=http%3A%2F%2FASP.NET%2FJ2EE&amp;usd=2&amp;usg=AFQjCNFmjOiR-1LjdzR_-oKN1TOFRHq5SA" target="_blank">ASP.NET/J2EE</a> Application Security, Application Security audits and .NET Security Curriculum Development.<p>For the past couple years Dinis has focused on the field of Static Source Code Analysis and Dynamic Website Assessments (aka penetration testing), and is the main developer of the OWASP O2 Platform which is an Open Source project that is focused on &#39;Automating Security Consultants Knowledge/Workflows&#39; and &#39;Allowing non-security experts to access and consume Security Knowledge&#39;. Dinis is currently focused on making the O2 Platform the industry standard for consuming, instrumenting and data-sharing between: the multiple WebAppSec tools, the Security consultants and the final users (from management to developers).<p>(<a href="http://www.google.com/url?q=https%3A%2F%2Fwww.owasp.org%2Findex.php%2FUser%3ADinis.cruz&amp;usd=2&amp;usg=AFQjCNH4e5Qxx-bkoBBl9NnVpWx_Aee_hQ" target="_blank">https://www.owasp.org/index.php/User:Dinis.cruz</a>)<p> <p> <p> <p> <p>Drinks will be found afterward at Doc &amp; Eddies.<p>Doc &amp; Eddy&#39;s, 909 East Minton Drive, Tempe, AZ 85282-7021<p> <p> <p>Please forward this announcement to others that may be interested.<p>Join the mailing list: &gt;&gt;&gt;&gt;  <a href="http://www.google.com/url?q=https%3A%2F%2Flists.owasp.org%2Fmailman%2Flistinfo%2Fowasp-phoenix&amp;usd=2&amp;usg=AFQjCNFB16gnf4-PEyD93lMP3GemRrMQiQ" target="_blank">https://lists.owasp.org/mailman/listinfo/owasp-phoenix</a><p> </p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></p></div><table cellpadding="0" cellspacing="0" border="0" summary="Event details"><tr><td style="padding:0 1em 10px 0;font-family:Arial,Sans-serif;font-size:13px;color:#888;white-space:nowrap" valign="top"><div><i style="font-style:normal">When</i></div></td><td style="padding-bottom:10px;font-family:Arial,Sans-serif;font-size:13px;color:#222" valign="top">Tue Jun 7 6:30pm – 7:30pm <span style="color:#888">Pacific Time</span></td></tr><tr><td style="padding:0 1em 10px 0;font-family:Arial,Sans-serif;font-size:13px;color:#888;white-space:nowrap" valign="top"><div><i style="font-style:normal">Where</i></div></td><td style="padding-bottom:10px;font-fami
