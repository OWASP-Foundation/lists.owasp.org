BIO:<br>
Andre works for the HP Application Security Center (ASC) doing<br>
application penetration-testing, secure code review, and reverse<br>
engineering. He has 9 years of direct experience with application<br>
security topics, has been using Burp Suite on pen-tests since early<br>
2005, and runs his own tool benchmarks at home in Tempe, AZ.<br>
<br>
Abstract:<br>
Every tool you should leverage during an app pen-test or secure code<br>
review will be discussed. The two best web proxies, Burp Pro<br>
(@portswigger) and Fiddler (@ericlaw) will be demonstrated along with<br>
the two best crawlers from @netsparker and WebInspect. The results<br>
from @sectooladdict will be discussed and the analysis demonstrated on<br>
@owaspbwa. Additional topics will be discussed, such as executive<br>
management reporting using <a href="http://dradisframework.org/" target="_blank">dradisframework.org</a> by way of imports from<br>
@w3af. There will also be topics for application developers, such as<br>
the new OWASP Data Exchange Format Project, as well as using <a href="http://cat.net/" target="_blank">CAT.NET</a>,<br>
RIPS, LAPSE+, and Fortify to go from vulnerable sources to runtime<br>
analysis to full exploitation. Even esoteric tools from long-ago that<br>
have held their value will be discussed and potentially demonstrated<br>
if there is time<br>
