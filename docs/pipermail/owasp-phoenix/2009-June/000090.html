https://addons.mozilla.org/en-US/firefox/collection/webappsec

Firefox now supports "collections," so I created a collection of the most
commonly used web app security testing add-ons.

If you add the "Add-On Collector" add-on (link on the page above), create a
Mozilla.org acct if you don't already have one, go to the Add-On window, and
enter your Mozilla.org account, it will let you subscribe to your favorite-d
"Collections," which will make adding a bunch of add-on's at once super
easy.

There are currently 16 plugins in this collection. If I missed anything...
let me know.
