Our chapter is set to participate.


The topic of the event is "Privacy in the 21st Century", so all talks
should be related to it (we should be addressing the Web Application
side of Privacy (for example what happens to Privacy with SQL Injection,
XSS and issues like pdp's Snoop onto Them as they Snoop onto us)
      * The event will have 4 to 5 speaking slots (can be 30m if
        required)
      * All events are recommended to have the same panel discussion on
        the subject "What is the current state of Privacy on Web
        Application Security? and what should we be focusing on?").
        After the panel discussion, each local chapters is invited to
        create a summary of its conclusions for publishing on the OWASP
        website

Looking for volunteers to speak and for the panel discussion!

We will invite some local press so this is a good opportunity for some
web app sec exposure!
