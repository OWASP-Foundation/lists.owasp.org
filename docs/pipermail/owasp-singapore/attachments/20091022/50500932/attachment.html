<br>
<b>JBroFuzz</b> is a web application fuzzer for requests being made
over <a href="http://en.wikipedia.org/wiki/HTTP" class="external text"
 title="http://en.wikipedia.org/wiki/HTTP" rel="nofollow">HTTP</a>
and/or <a href="http://en.wikipedia.org/wiki/Https"
 class="external text" title="http://en.wikipedia.org/wiki/Https"
 rel="nofollow">HTTPS</a>. Its purpose is to provide a single, portable
application that offers stable web protocol fuzzing capabilities.
