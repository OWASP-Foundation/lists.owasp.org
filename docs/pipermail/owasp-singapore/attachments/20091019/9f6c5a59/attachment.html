<h1>Programmer slip-up produces critical bug, Microsoft admits</h1>
<br>

<br>
<br>
Back to my constant pitch that most, if not all, security issues
originate from software.<br>
<br>
Hence, we need the next generation of IS pros to be software experts in
being about to <b>prevent</b>, not just mitigate, security
vulnerabilities.<br>
<br>
The "a-peh" or older generation of IS pros had been playing a reactive
game, mainly restricting ourselves to the OS and network.<br>
We need to change how we play to better survive in the new era.<br>
<br>
I hope this is a wake-up call for IS pros to engage more in the
software development circle.
<br>
<br>
Just my $0.02.<br>
