Given the centralised monoculture that is IT in Singapore, combined with the banks&#8217; (and Government&#8217;s) general attitudes, I&#8217;m amazed that it hasn&#8217;t happened already... if it truly hasn&#8217;t.<BR>
<BR>
There are flaws in <B>any</B> system. When an excessively large proportion of a critically important industry (like, say, finance) is dependent upon a <B>single</B> system with minor-at-best variation, the when-not-if meter goes to 11....especially in a top-down, reactive management environment. Unfortunately, there&#8217;s only so much of the problem that can be solved by technical means within the status quo otherwise.<BR>
