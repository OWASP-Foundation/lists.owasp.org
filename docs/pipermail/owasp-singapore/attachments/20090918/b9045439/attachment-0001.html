<p class="first">Microsoft still does not acknowledge a <a
 href="https://www.networkworld.com/news/2009/073009-more-holes-found-in-webs.html">weakness</a>
in its Internet Explorer browser that was pointed out seven weeks ago
and enables attackers to hijack what are supposed to be secure Web
sessions.
</p>
<p>The company says it is still evaluating whether the weakness exists,
but Apple, which bases its Safari for Windows browser on Microsoft
code, says Safari for Windows has the weakness and the Microsoft code
is the reason. If Microsoft doesn't fix the problem, Apple can't fix it
on its own, Apple says. Apple has fixed the problem for <a
 href="http://www.networkworld.com/news/2009/080409-mozilla-patches-three-public-firefox.html">Safari
for Macs</a>.<br>
</p>
