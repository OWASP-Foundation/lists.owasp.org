 moz-do-not-send="true"
 href="http://www.owasp.org/index.php/Category:OWASP_Guide_Project">OWASP
Development Guide Project</a></font></span>/&nbsp;-- After many months of
planning and preparation, the OWASP Development Guide project&nbsp;announced
today that&nbsp;it is ready to begin work on the next revision of the Guide,
and that that the project is looking for volunteers to do the work,
both individuals and organizations. </font></span></div>
<div><span class="953223619-10022010"></span>&nbsp;</div>
<div><span class="953223619-10022010"><span class="953223619-10022010"><font
 size="2"><font face="Arial">The OWASP Development Guide&nbsp;is aimed at
architects, developers, consultants and auditors and is a comprehensive
manual for designing, developing and deploying secure web applications.
The original OWASP Development Guide has become a staple diet for many
web security professionals. Since 2002, the initial version was
downloaded over 2 million times. Today, the Development Guide is
referenced by many leading government, financial, and corporate
standards and is the Gold standard for Web Application and Web Service
security.</font></font></span></span></div>
<div><span class="953223619-10022010"><span class="953223619-10022010"></span></span>&nbsp;</div>
<div><span class="953223619-10022010"><span class="953223619-10022010"><font
 size="2" face="Arial">The next version of the OWASP Development Guide
will be in effect the detailed design guide for the requirements of the
OWASP Application Security Verification Standard (ASVS), which can be
found here:&nbsp;</font><a moz-do-not-send="true"
 href="http://www.owasp.org/index.php/ASVS"><font size="2" face="Arial">http://www.owasp.org/index.php/ASVS</font></a><font
 face="Arial"><font size="2">. Key features of the next Guide will
include use of the new OWASP common numbering scheme.&nbsp;The new numbering
scheme will be common across OWASP Guides and References, more
information can be found here:<span class="281022700-11022010"> </span></font></font><a
 moz-do-not-send="true"
 href="http://www.owasp.org/index.php/Common_OWASP_Numbering"><font
 size="2" face="Arial">http://www.owasp.org/index.php/Common_OWASP_Numbering</font></a><font
 face="Arial"><font size="2">. Additional key features will be the
inclusion of worksheets and checklists<span class="281022700-11022010">,
such as the sample input validation worksheet which can be found here: <a
 moz-do-not-send="true"
 href="http://code.google.com/p/owasp-development-guide/wiki/WebAppSecDesignGuide_D5_2_1_1">http://code.google.com/p/owasp-development-guide/wiki/WebAppSecDesignGuide_D5_2_1_1</a></span></font></font></span></span></div>
