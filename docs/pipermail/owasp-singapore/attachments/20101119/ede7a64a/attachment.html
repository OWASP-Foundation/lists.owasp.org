    <h2 id="DownloadName" class="header details-h2"> Microsoft Security
      Development Lifecycle (SDL) - Version 5.0 </h2>
<a class="moz-txt-link-freetext" href="https://www.microsoft.com/downloads/en/details.aspx?FamilyID=7d8e6144-8276-4a62-a4c8-7af77c06b7ac&amp;displaylang=en">https://www.microsoft.com/downloads/en/details.aspx?FamilyID=7d8e6144-8276-4a62-a4c8-7af77c06b7ac&amp;displaylang=en</a><br>
    <br>
    The Microsoft Security Development Lifecycle (SDL) process guidance
    illustrates the way Microsoft applies the SDL to its products and
    technologies. It includes security and privacy requirements and
    recommendations for secure software development at Microsoft. It
    addresses SDL guidance for Waterfall and Spiral development, Agile
    development, web applications and Line of Business applications. IT
    policy makers and software development organizations can leverage
    this content to enhance and inform their own software security and
    privacy assurance programs.<br>
    <br>
