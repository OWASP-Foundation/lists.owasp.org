Leaders,

 

I am glad to announce we've just set a new project up -- the */OWASP
Hatkit Proxy Project,/* led by *Martin Holst Swende*. Please join me to
congratulate him!

 

http://www.owasp.org/index.php/OWASP_Hatkit_Proxy_Project#tab=Project_About

 

The primary purpose of the Hatkit Proxy (which is based on the Owasp
Proxy Project) is to create a minimal, lightweight proxy which stores
traffic into an offline storage where further analysis can be performed,
e.g. all kinds of analysis which is currently implemented by the proxies
themselves (webscarab/burp/paros etc).

 

As always, your suggestions and contributions would be greatly
appreciated.   

 

Thanks,
