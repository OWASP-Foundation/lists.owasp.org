Hi,

Once again, OWASP needs your help with an important project.

As you probably know, the OWASP Testing Guide V2.0 
<http://www.lulu.com/content/1375886> was a great success, with thousand 
of downloads and a great deal of companies adopting it as standard for a 
Web Application Penetration Testing. Now it is time to begin a new 
project based on the previous V2.0.

 

The project’s goal is to create a "best practices" penetration-testing 
framework, which users can implement in their own organizations and a 
"low level" penetration-testing guide that describes how to find certain 
issues.
