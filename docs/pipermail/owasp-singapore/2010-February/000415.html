The CSSLP CBK^® domains detail security measures that need to take place
within each phase of the software lifecycle. It is for software
stakeholders who have four years of direct full-time professional
experience in the software development lifecycle (SDLC) security that
include requirement managers, software developers, engineers and
architects, project managers, software QA, QA testers, business
analysts, auditor and the professionals who manage these stakeholders. 

*The CSSLP CBK^® domains include:* 
 * Secure Software Concepts
 * Secure Software Requirements
 * Secure Software Design
 * Secure Software Implementation/Coding
 * Secure Software Testing
 * Software Acceptance
 * Software Deployment, Operations, Maintenance and Disposal
