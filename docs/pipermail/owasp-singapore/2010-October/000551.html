As discussed earlier, we will like to build a rich media, (videos, PDFs,
PPTs, web articles) repository of developer-friendly educational content
based on OWASP Top 10 2010.

free for all to refer to.

The objective is to provide a good quality reference page with the
content friendly and easy to understand for developers.

Please actively contribute to this OWASP SG initiative to help bring the
secure development message to developers.
