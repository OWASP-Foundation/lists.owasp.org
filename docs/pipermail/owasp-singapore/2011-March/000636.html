Not going to comment about the choice of IE or any other browser.

But will like to bring up the merit of dual browser approach in view of
security benefits.

"They [Google security team] reiterated Microsoft's advice that users
install the temporary fixit <http://support.microsoft.com/kb/2501696> as
soon as possible. A better course of action is to halt all usage of the
Internet Explorer browser, since that is the sole vector for exploits.
This may not be possible for users with custom applications that rely on
the Microsoft browser, but *even avoiding it while on social networking
sites is a good idea,* if at all possible."

Though I am sure that Google is not on these mailing lists, Google is
echoing the dual browser option too. ;-)
