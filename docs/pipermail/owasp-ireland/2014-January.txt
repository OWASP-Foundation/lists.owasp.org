ICON Plc. is a leading global provider of outsourced development services
to the pharmaceutical, biotechnology and medical device industries
operating in 40 countries with 10,000 staff worldwide. At our corporate
headquarters in Leopardstown, we currently have over 680 employees with
plans to grow significantly over the next few years. We are looking to fill
the following position:



*Senior IT Security Analyst (manager level- no direct reports)*



This is diverse hands on role with global scope reporting to the Head of IT
Security. We are looking for an experienced candidate who can assist in the
successful execution of ICONs IT Security Strategy.



*Key Duties*



-          Assist in the development and application of IT Security
Policies, Procedures, Standards and Guidelines Framework in line with
ISO27001 and best practices

-          Develop and manage User Awareness Programs

-          Develop and assist in application of Technical Security
baselines (Server Hardening, Operating Systems, Wireless, End Point
Protection, Network Devices, Remote Access, Databases, Mobile Devices
Management)

-          Conduct Risk Assessments of Cloud Computing, Network
Infrastructures, Business Applications, Third Party Hosting Environments,
Customer facing systems and infrastructure

-          Schedule and review results of Security Audits,  Penetration
Tests, Vulnerability Scanning

-          Assist in selection and design of security related technologies
e.g. Identity and Access Management, IDS/IDP, SIEM, Vulnerability Scanning,
Configuration Management, Mobile Device Management etc.

-          Develop processes for reporting and managing IT Security risks
using the output of assessments, penetration tests and security tools,
audit reports

-          Conduct Gap Analysis of IT security controls against contractual
obligations, industry regulations and ICON IT controls framework

-          Develop processes for embedding security into SDLC / PMLC

-          Develop Incident Response Plans

Full job specification available on ICON website: Job ID *006155*
