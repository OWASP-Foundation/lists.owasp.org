The OWASP testing project for which I am the lead have developer a "Live 
CD" for application penetration testing. The aim of this CD is to have a 
bootable Application pen testing suite "in your pocket" ala 
Auditor/Knoppix.

The CD (When final) shall contain all the OWASP web application tools, The 
.NETtools (utilising Mono to run .NET on Linux) and the application 
testing guide in HTML format.

I shall be giving out a limited amount of CD's at the OWASP AppSec 2006 in 
Belgium. when Final it shall be available on the OWASP site to the public.
