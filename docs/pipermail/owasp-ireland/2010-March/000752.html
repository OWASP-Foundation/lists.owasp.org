>> *== Talk ==*
>>
>> *Title:* OWASP Live CD - An open environment for Web Application 
>> Security
>>
>> *Abstract* 
>> This CD collects some of the best open source security projects in a 
>> single environment. Web developers, testers and security 
>> professionals can boot from this Live CD and have access to a full 
>> security testing suite. This presentation aims to provide a showcase 
>> for the great OWASP tools and documentation materials available in 
>> the CD, tips and tricks, and also some introductory stuff regarding 
>> code review and penetration testing.
>> Training is aimed at introductory /intermediate level in terms of pen 
>> testing, code review and tools.
>>
