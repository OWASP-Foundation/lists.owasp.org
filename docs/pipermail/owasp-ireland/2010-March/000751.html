> == Talk ==
>
> Title: OWASP Live CD - An open environment for Web Application  
> Security
>
> Abstract
> This CD collects some of the best open source security projects in a  
> single environment. Web developers, testers and security  
> professionals can boot from this Live CD and have access to a full  
> security testing suite. This presentation aims to provide a showcase  
> for the great OWASP tools and documentation materials available in  
> the CD, tips and tricks, and also some introductory stuff regarding  
> code review and penetration testing.
> Training is aimed at introductory /intermediate level in terms of  
> pen testing, code review and tools.
>
> == Presenters ==
>
> Rahim Jina
> Rahim Jina currently works as a senior consultant for Ernst &  
> Young's Risk Advisory Services in Dublin. He has worked there for  
> nearly four years primarily delivering penetration testing services

> review.
> He has been involved with OWASP for the past two years, being

> Review Guide 2009. He has also made contributions to the SAMM

> degree in computer science from Trinity college.
>
> Eoin Keary
> Eoin is a long time member of OWASP and have contributed year on
