Dear All,

We are planning to deliver a FREE training about the OWASP Live CD around 
April/May (date to be confirmed).

As you might know, this Live CD collects some of the best open source 
security projects in a single environment. 

Web developers, testers and security professionals can boot from this Live 
CD and have access to a full security testing suite.

The idea would be to provide you with a copy of the CD and then go through 
the different tools in an interactive fashion.

You could find more information about the OWASP Live CD here: 
http://appseclive.org/

As there are going to be some costs involved (eg. burning cds, preparing 
materials, etc), we will appreciate if you could support the event either 
by becoming an OWASP individual member or by asking your employer for a 
discretionary donation.

Any help will be really appreciated.
