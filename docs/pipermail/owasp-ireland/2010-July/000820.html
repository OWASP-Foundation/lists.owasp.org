Hello All

The uploaded presentation also includes the slides on the User Life 
Cycle steps which I never got to during the talk.

The User Life Cycle section suggests good practices around the various 
stages:

    * Identification
    * Registration/Enrolment
    * Logon/Logoff
    * Forgotten Password Mechanism
    * Logoff/Account Deletion


The exact link to the presentation is:

    * http://www.owasp.org/images/4/45/OWASP_Ireland_June10.pdf

If any of you have questions or other feedback, please feel free to 
contact me.

Chapter for arranging it.
