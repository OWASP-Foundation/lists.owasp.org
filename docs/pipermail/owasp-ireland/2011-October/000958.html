Dear all,

For those who are attending the Zap Workshop this Thursday, please find
below instructions on how to setup your laptop to participate in the
hands-on.

Download and install the following software:

   - Java (latest)
   - Tomcat 5.5 (or above)
   - ZAP 1.3.3
   - The BodgeIT Store
(http://code.google.com/p/**bodgeit/downloads/list<http://code.google.com/p/bodgeit/downloads/list>
   )

Put the bodgeit.war file in the Tomcat webapps directory.

Optional is the foxyproxy plugin to switch proxies.

Any questions, please let me know.

Thanks,

Fabio
