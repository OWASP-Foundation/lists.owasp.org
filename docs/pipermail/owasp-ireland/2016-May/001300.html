Folks,

If interested in a career in software development of SaaS, RASP & Threat
Analytics tech please read below!!:

edgescan (www.edgescan.com) is a cloud based SaaS with its corporate HQ in
Ireland. We deliver continuous vulnerability management,  security
integration & and client-side RASP services to our clients and assess
thousands of endpoints for security vulnerabilities every day.

This position is for a developer with the ability to develop and maintain
security solutions using modern software development technologies.

We are looking for candidates with 2-5 years of experience in software
development, preferably in a web development (either front or back end)
role. An EU Work permit is required to apply for this position.

The candidate should be proficient in at least one of the following
programming languages: C, C++, C#, Java, Ruby, Javascript or Python.
Knowledge of multiple languages is a bonus.

The following skills are given as an example of what successful candidates
will use on a day to day basis. Preexisting knowledge of these skills is a
bonus, but is not required. However candidates will be expected to learn
some of these skills on the job.

- Ruby, Javascript, HTML, CSS

- Knowledge of Angular and HTML5

- Use of Linux OS, and familiarity with the command line

- Use of AWS cloud computing technologies, especially EC2

- DevOps tools, especially using Chef for automating configuration

- Secure programming techniques

Interest in web application security, risk analytics and threat
intelligence is a bonus.

*Benefits include:*

- Competitive pay

- Health contribution

- Bonus

- Training

- Beer?##!@#$%^&

- Sponsored staff ‘away days’

- Flexible working environment

This is an opportunity for a talented software developer to join a rapidly
expanding company and enjoy the benefits that brings!
