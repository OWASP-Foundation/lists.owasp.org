Hi Everyone,

Due to demand for this event we've had to change the venue to accommodate
more people. The meeting will now be in the *Hilton Hotel, Charlemont
place* (opposite
the Barge pub) in Ranalegh (Chalemont Luas stop). The Hilton is
approximately a 10 minute walk along the canal from the original location
and a short walk from the city center.

https://www.google.ie/maps/place/Hilton+Dublin/@53.3309682,-6.2614357,17z/data=!3m1!4b1!4m5!3m4!1s0x48670ea16eb347bb:0x5b92598f4735994e!8m2!3d53.330965!4d-6.259247

Courtesy of Ward Solution, Kontex Security and Symantec, we have a number
of gifts that we will be giving away as part of the event which include an
iPad, Samsung tablet, Play Station 4 and some smaller items.

Please note that currently there is a Luas strike planned for the evening
of the 26th May, so we will start at 7pm.

We're really looking forward to the event so hope to see you there. Please
share the event with friends and colleagues as it's an open event and the
more the merrier.
