Fidelity Investments - Technology Risk Analyst - 1300889
Description
 
The Information Security & Risk (IS&R) group provides enterprise-wide security services for Fidelity’s business units.  The Application Security Group within IS&R is responsible for providing a wide range of application security services including Secure Code Review and Secure Application Development training.
 
We are currently seeking a Technology Risk Analyst to review applications for theMalicious Code Detection team, which is a sub team within the Application Security Group. The position will be located in Dublin, Ireland. 
 
RESPONSIBILITIES:
 
The main objective of the Malicious Code Detection team is to identify malicious code (such as time bombs, logic bombs and backdoors) within applications developed in Fidelity to mitigate the risk of the insider threat. The Technology Risk Analyst is responsible for learning and applying best practices for malicious code detection on Fidelity’s applications, while working closely with the Application Security Group leaders and development leaders across the enterprise. 
 
Key job responsibilities include:
 
Review applications and systems for malicious code or inadequate access management
Research industry best practices in the area of Malicious Code Detection.
Continuous improvement to the procedures used by the Malicious Code Detection team.
Update Prescriptive Security Checklists (a knowledge base of security vulnerabilities).
Build strong relationships with key partners (Enterprise Development Team Leads).
Qualifications
 
3+ years of software development experience
Java development skills with experience of Java EE, Websphere, WebLogic, Tomcat
Hands on experience of working on one or more of the following frameworks and technologies: Spring, Struts, MVC, JavaScript
Strong customer management and relationship building skills.
Strong oral and written communication skills.
 
The following qualifications / skills are not required but would be advantageous for this role:
 
.NET development skills with experience of the .NET framework, Microsoft Visual Studio and IIS
Knowledge of Secure Application Development.
Knowledge of techniques for exploiting security vulnerabilities in applications.
Bachelor degree in a software development discipline.
Knowledge of the UNIX and Linux operating systems
Oracle Certified Professional Java SE Programmer (OCPJP) - formerly known as Sun Certified Java Programmer
CISSP (Certified Information Systems Security Professional)
If you are interested in applying for this position, please send your CV tofidelityireland at fmr.com
