The Smart Card Alliance Mobile and NFC Council is partnering with
GlobalPlatform and the NFC Forum to host a four-part webinar series on
mobile and NFC security.

Entitled “Mobile and NFC Security Fundamentals,” the series is geared
toward mobile app developers and the mobile community to help them
understand the various types of security for NFC applications.

The series will cover the security options for mobile devices, secure
elements and NFC tags, and it intends to educate attendees on the
appropriate levels of security for different aspects of NFC technology.

The series’ first installment will be called “Anatomy of a Mobile Device:
Security Architecture and Secure Provisioning,” and is scheduled for *March
5 at 2:00 p.m. EST*. The session will discuss mobile architecture and its
security interfaces, how it uses secure elements and the Trusted Execution
Environment and secure provisioning. Attendees are asked to
register<https://www1.gotomeeting.com/register/215137336>
 in advance.

The remaining webinars are “Secure Elements 101” on March 28, “NFC Forum
Tags and Security Considerations” on April 18 and “NFC Application Use
Cases: Security Perspectives” on May 9
