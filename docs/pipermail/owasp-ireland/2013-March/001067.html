Another Job op below:

Ward Solutions are a well-established Security Systems Integration and Consultancy Provider located in Citywest Dublin, Ireland. We have a number of openings for experienced and highly qualified Information Security specialist to work in our security practice. Candidates will be required to work for clients on a hybrid client/employer site basis - project dependent. 

The candidate will report to the security practice manager and will be responsible for providing a wide range of security services to existing and new national and international customers, including: 
- Application and network penetration testing and strategic risk assessment services 
- Security design and review of systems architecture(s) and solutions as part of established SDLCs
- Organization strategic risk assessments
- Design and implementation of security & compliance frameworks and management systems (ISO-27001, PCI-DSS, Data Protection etc)
- Information security policy development and review
- Miscellaneous security advice and consultancy services

Ideally the candidate should have a minimum of 5+ year's experience in information security, with as many of the following key skills as possible (not all are required): 
- Hands-on experience with application and network penetration testing
- Information security risk assessments, risk management and reporting, and related standards (ISO-27001, Data Protection, PCI DSS etc)
- In-depth working knowledge of systems and application architectures
- In-depth working knowledge of TCP/IP, switching, routing, and associated network protocols
- Previous development experience (Java/C/.NET/PL-SQL), to aid application penetration testing and source code review is a plus but not a requirement

If you are interested in working in a challenging and innovative environment, please email: recruitment at ward.ie
