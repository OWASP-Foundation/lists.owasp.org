The XSS vulnerable function below allows anyone to send un-validated user 
input (which could contain arbitrary code) and being output back to the 
browser. To mitigate against this, you need to implement proper input 
validation and output encoding. More info about XSS and how to prevent it 
could be found here: 

http://www.owasp.org/index.php/XSS_(Cross_Site_Scripting)_Prevention_Cheat_Sheet
