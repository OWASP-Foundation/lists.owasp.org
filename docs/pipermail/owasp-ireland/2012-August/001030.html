hi there,

The OWASP Vicnum Project is an open source project that helps you test
your knowledge on web application security. The AppSec Ireland 2012
challenge is to see if you can 'union' the two Vicnum games, Guessnum
and Jotto, using the same browser as I have used.

Players that solve the challenge should contact ireland at owasp.org

The competition will run until Friday 31st August.

Once the competition is over, the first three winners will get a FREE
ticket to the conference.

http://vicnum.ciphertechs.com/appsec2012.html

Please be aware the use of “scripts”, “brute force” or any other
automated or programmed means, will inmediately void any prize award
and the participant will be refused further participation.

A big thanks goes to Mordecai and the Ciphertechs team for setting up
this challenge.

Best of luck everyone!
