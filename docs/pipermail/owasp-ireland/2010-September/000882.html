== Talk ==

Title: The OWASP Code Review Guide by Eoin Keary

Abstract:
The OWASP Code Review Guide, one of the "Trinity" of OWASP guides is about
to be rebooted, rewritten and refreshed.
Eoin Keary, OWASP board member and Code Review Guide lead is going to
discuss the guide; What is SCR (Security Code review)? Measurements,
Metrics, SDLC, Automated and Manual Assessment, Risk based code
review, Transactional
Analysis,Examples & Code review Guide chapters.

The aim of the presentation is also to encourage individuals to contribute
to the guide, understand where it fits in and introduce people to source
code review.
