The next OWASP Dublin event will be in Workday on the 8th March and will
feature session from the speakers below.

   - *Adrian Bravo* from Workday will be discussing teh Java Virtual
   Machine (JVM), sandbxoin untrusted code and sandbox escapes;
   - *Subodh Sawant* from Paddy Power Betfair speaking about how risk can
   be measured in relation to application changes
