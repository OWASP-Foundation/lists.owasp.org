Hi Folks,

I'd like to let you know about the employment opportunity below:


 ==Software Developer/Engineer==



edgescan is a cloud based SaaS with its corporate HQ in Ireland. We deliver
continuous vulnerability management and security integration services to
our clients and assess thousands of endpoints for security vulnerabilities
every day.



This position is for an entry level developer with the ability to develop
and maintain security solutions in the cloud. The candidate shall hold a
degree in Computer Science, Software Engineering or equivalent, but we
shall consider developers without such credentials if they can demonstrate
the required skill and passion in relation to software development.


An EU Work permit is required to apply for this position.



The candidate should be proficient in at least one of the following
programming languages: C, C++, C#, Java, Ruby, Javascript or Python.

Knowledge of multiple languages is a bonus.



Candidates will most likely be expected to learn Ruby and/or Javascript as
part of the position if they are not already familiar with the language.
Candidates must demonstrate their ability to solve real world problems.



The following skills are given as an example of what this position job may
require. Preexisting knowledge of these skills is a bonus, but is not
required.

- Ruby, Javascript, HTML, CSS

- Knowledge of Angular and HTML5

- Use of Linux OS, and familiarity with the command line

- Use of AWS cloud computing technologies, especially EC2

- DevOps tools, especially using Chef for automating configuration



This is an opportunity for a talented software developer to join a rapidly
expanding company and enjoy the benefits that brings!!



Interest in Web application security, Risk Analytics and Threat
intelligence is a bonus, as is an understanding of secure coding techniques
(we are a security company after all!). Training shall be included!


Benefits include:

-              Joining a rapidly growing global team

-              Competitive pay

-              Health contribution

-              Bonus

-              Training

-              Beer

-              Sponsored staff ‘away days’

-              Flexible working environment

Send Ur CV's to *info at edgescan.com <info at edgescan.com>* if keen..
