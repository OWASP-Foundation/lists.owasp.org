<div dir="ltr">Hey Everyone, <div><br></div><div>I'm happy to be announcing the new version of the <a href="http://bit.ly/owaspSecurityShepherd">OWASP Security Shepherd</a> project has been released! </div><div><br></div>

<div>For those unfamiliar with Security Shepherd, it enables users to learn or to improve upon existing manual penetration testing skills. The new version is loaded with nearly 50 levels implemented with the aim of fostering and improving web and mobile application security awareness among a varied skill-set demographic. </div>

<div><br></div><div>Many of these levels include insufficient mitigations and protections to these risks, such as blacklist filters, atrocious encoding schemes, barbaric security mechanisms and poor security configuration. The modules have been crafted to provide not only a challenge for a security novice, but security professionals as well.</div>

<div><br></div><div>So check out the <a href="http://bit.ly/owaspSecurityShepherd">wiki</a> and try out the new suite of challenges. If anybody is going to AppSec EU stop by the <a href="http://bit.ly/shepherdAtAppSecEu">Security Shepherd talk</a> where we'll be going through the new version in detail! If anybody is interested in contributing to the project, mail me off list! </div>
