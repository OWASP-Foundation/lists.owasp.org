Now that I&#39;ve finally de-lurked, I thought I&#39;d share my frustration at the recent Twitter OAuth announcement.  <div><br></div><div>When I first heard about it, I was thinking it was fantastic, and that finally the great unwashed public would start to be taught what proper web security looks like.  </div>
