<div><span class="Apple-style-span" style="font-family: arial, sans-serif; font-size: 13px; background-color: rgb(255, 255, 255); "><br></span></div><b><font class="Apple-style-span" color="#ff9966">A practical introduction to pentesting using the OWASP Zed Attack Proxy (ZAP)<br>
</font></b><br>The OWASP Zed Attack Proxy (ZAP) is an easy to use integrated penetration testing tool for finding vulnerabilities in web applications.<br>It is designed to be used by people with a wide range of security experience and as such is ideal for developers and functional testers who are new to penetration testing as well as experienced pentesters.<br>
<br>This 2 hour course will include:<br>* An introduction to the OWASP Zed Attack Proxy (ZAP)<br>* The principals behind ZAP<br>* An overview of its functionality<br>* A hands-on session using ZAP to pentest an example vulnerable application<br>
* A demonstration of using ZAP to perform security regression tests<br>* An indication of where ZAP is heading<br><br>The course will be suitable for experienced pentesters who would like to learn about ZAP and developers or functional testers who are interested in using ZAP to learn more about application security.<br>
<br>Attendees who wish to take part in the hands-on session will need to bring along a laptop.<br><br></span><div><span class="Apple-style-span" style="font-family: arial, sans-serif; font-size: 13px; background-color: rgb(255, 255, 255); ">A list of software to be installed prior to the course will be distributed closer to the event.<br>
