conform to a specific syntax itself and there are standard tools for 
examining&nbsp;its infoset(ASN1 anyone?). For example the description of a Web

that service. It is a straightforward and important check to make before 
allowing it to reach the&nbsp;application itself . As Eoin said adding a regular 
expression to&nbsp;the schema is a pretty standard and easy patch. As XML 
processing becomes more the norm be prepared for more alerts like PHP XML Remote
