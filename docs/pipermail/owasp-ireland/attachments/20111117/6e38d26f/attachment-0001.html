<br>The CHISEL (Computer Human Interaction for Software Engineering) group, based in the University Of Victoria, Canada, has openings for Masters and PhD level researchers in the area of software security analysis tools research. <br>
<br>Working on a project for Defense Research and Development Canada, the successful candidates will help to develop better tool and visualization support for security researchers tasked with preventative assessment of software for unknown vulnerabilities.<br>
<br>Applicants should have a strong interest in: software security, security analysis tools, software visualization and software comprehension. Knowledge of assembly and software trace analysis tools would be a plus.<br><br>
Applicants should email a curriculum vitae, a list of three referees and short summary of your research interests to Dr. Brendan Cleary <a href="mailto:bcleary@uvic.ca">bcleary@uvic.ca</a><br>
