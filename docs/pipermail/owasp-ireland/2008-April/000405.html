Hello OWASP'ers

OWASP are having their annual "OWASP Day" during April. This time it shall
span the Month :)
We were hoping to have a gathering on the 22nd to reflect such a special
week.

We have a large box of goodies to give away such as OWASP Books and OWASP
Polo shirts & presentations on.......

*1. The OWASP Enterprise Security API:*
A presentation on the OWASP ESAPI. The OWASP open source Java API which
attempts to cover all the bases of the OWASP Top 10.
This can be used to assist in preventing vulnerabilities of the OWASP Top
10. This discussion shall cover the API and uses for it when developing
secure applications.

*2. PCI DSS:
I'm a developer, get me out of here...Application Security for PCI DSS
*
David Rook, Security Analyst for Realex Payments shall present on PCI DSS
(Payment Card Industry Data Security Standard) from an application security
perspective.
Covering strengths/weaknesses, the controversial compliant vs security
discussion and sections relating to application security in the PCI
standard.
