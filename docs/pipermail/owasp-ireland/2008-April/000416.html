 Hi,
A final reminder on the chapter meeting tomorrow evening.
(For those of you who have not already, drop me a mail if u intend to come
so I an organise refershments.)

This meeting is open to anyone, u dont have to be an OWASP member to attend.
CPE credits can be claimed for CISSP's who attend.

If you have an interest in Secure application development this meeting may
be beneficial so spread the word.

Also there are many OWASP books to give away such as the OWASP Testing
guide, development guide, CLASP,  Top 10 2007, Top 10 Ruby on rails.

The complete list of "freebies" can be found here:
http://www.lulu.com/browse/search.php?search_forum=-1&search_cat=2&show_results=topics&return_chars=200&search_keywords=&keys=&fSearch=OWASP&fSearchFamily=0

 Presentations shall be on the following topics:

*1. The OWASP Enterprise Security API:*
A presentation on the OWASP ESAPI. The OWASP open source Java API which
attempts to cover all the bases of the OWASP Top 10.
This can be used to assist in preventing vulnerabilities of the OWASP Top
10. This discussion shall cover the API and uses for it when developing
secure applications.
Short presentation on CRSF and Javascript Malware.

*2. PCI DSS: *
I'm a developer, get me out of here...Application Security for PCI DSS

David Rook, Security Analyst for Realex Payments shall present on PCI DSS
(Payment Card Industry Data Security Standard) from an application security
perspective.
Covering strengths/weaknesses, the controversial compliant vs security
discussion and sections relating to application security in the PCI
standard.
