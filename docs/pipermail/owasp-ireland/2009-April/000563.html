Reminder:

 Next Meeting 15th April

Venue: Ernst & Young, Harcourt Street

Time: 18:30



Speakers:

Mark Hillick (AIB)

Irish CERT Project

The Irish Reporting & Information Security Service (IRISS) is an independent
not for profit company limited by guarantee founded in 2008 to provide a
range of free services to Irish businesses and consumers in relation to
information security issues to help counter the security threats posed to
the Irish businesses and the Irish Internet space

Mairtin O'Sullivan (Espion);

Manual Web App Testing Methodology/Procedure.

 - Objective is to provide a basic testing methodology to ensure consistent
and comprehensive testing of applications.

- Will outline what information is required prior to testing, how to
identify all the functionality within the application that require testing,
how to ensure that all types of vulnerabilities are tested and how to keep
track of the functionality tested.

 - Will focus on free tools and manual testing but can be adopted for use
with any tools or automated testing.

 Mairtin is a senior security consultant with Espion, providing consultancy
on all areas of information security. He leads Espion's web application
security testing, has been performing web application security reviews for
over three years and is responsible for developing Espion's training
offerings in web application security.
