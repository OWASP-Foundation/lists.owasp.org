Hi,
Just thought id let you know about a new tool freely available called code
crawler.

it is based on the OWASP code review guide V1.1 which is to be officially
released very soon.

OWASP CC [binaries]:
http://www.cyphersec.com/software_archive/OWASP_Code_Crawler.zip

OWASP CC [source]:
http://www.cyphersec.com/software_archive/OWASP_Code_Crawler_Source.zip

OWAS CC [PowerPoint]:
https://www.owasp.org/images/6/61/OWASP_CodeCrawler_Presentation.ppt

The search strings are defined in XML so it is easy to add a new
technology/language to the tool.

Please send some feedback on this, what you like, dont like et and any
suggestions fro future improvement.
