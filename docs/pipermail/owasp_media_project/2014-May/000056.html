Hey all,

I've been playing around a bit with the new Ning 3.0 platform.  One of the
cool new features that I found is the ability to import and view RSS feeds
as part of the application.  The cool thing is that both YouTube and Vimeo
offer RSS feeds so this may be something that provides the ability to
combine all of these different feeds into a single location.  Check it out:

http://myowasp2.ning.com/videos

The "Videos" link at the second to last link is what I'm talking about.  I
told it to just display the titles, but it can do images and text as well.
Pretty neat and it automatically updates when new stuff is posted to the
channel.
