Hi, all!  This came across the OWASP Leaders list, I thought some of you
might be interested in it:

On Mar 20, 2017 03:03, "Liran Tal" <liran.tal at gmail.com> wrote:

Hi Everyone,



I wanted to share with you exciting news of my newly published book: Essential
Node.js Security <https://leanpub.com/nodejssecurity>.



Its audience is Node.js developers, and equipping them with secure code
methodologies, and common best practices in the Node.js ecosystem.



The book features many of OWASP security risks among the top 10
vulnerabilities and is fluent with practical source code examples, as well
as covering quite a few OWASP resources through-out.



I invite you to get a free copy of the book at
http://leanpub.com/nodejssecurity/c/OWASPIL.

I'll appreciate any feedback, and comments you may have.




On a personal note, I'm actively contributing to OWASP NodeGoat project,
and generally active around open source and Node.js security including my
talk on this topic in our previous OWASP AppSec 2016 conference.



So if anyone would like to connect and share more ideas I'll be happy to
meet you :)
