Hey, all - A recruiter I've worked with previously reached out to me with
this opening in the KC Area, as she's having a hard time finding
low-/mid-level AppSec folks.  I thought I would share the opening if anyone
here was looking (I promise I'm not going to spam the list with job
postings all the time):

Mid-level Security Analyst that has at least 1-2 years’ experience
preforming web application and network vulnerability assessments (both
manual/automated testing) as well as experience with different security
systems (SIEM, IDS/IPS, Anti-malware). You’ll be in charge of finding
vulnerabilities, remediating vulnerabilities, as well as documenting the
results.

Information Security Analyst Responsibilities:
•    Work with security team to perform tests and uncover vulnerabilities.
•    Security incident response management
•    Develop company-wide best practices for IT security.
•    Perform penetration testing (this can be trained!)
•    Research security enhancements and make recommendations to management.
•    Stay up-to-date on information technology trends and security
standards.

Information Security Analyst Requirements:
•    Bachelor's degree IS required
•    Certifications are preferred but not mandatory
•    At least 3 years of experience in IT Security as a whole, 1-2 years of
Web App Security
•    General PowerShell and Python scripting knowledge is a MUST

If you're interested, contact Katie Esklund -
katiee at alliedglobalservices.com
