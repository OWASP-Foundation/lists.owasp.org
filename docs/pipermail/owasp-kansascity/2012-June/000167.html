-       attacks over HTTP and WebSocket protocols against HTTP,
WebSocket , proxy,
cache,
-       breaking media streaming servers using new version ofslowhttptest
-       mitigation recommendations (how to fix it) and defensive techniques.

The details:

12345 College Boulevard
Overland Park, Kansas 66210
