<li style="margin-bottom:0.1em">Protection Against OWASP Top Ten!</li><li style="margin-bottom:0.1em">Very Few False Positives (i.e., should NEVER disallow an authorized request)</li><li style="margin-bottom:0.1em">Strength of Default (Out of the Box) Defenses</li>

<li style="margin-bottom:0.1em">Power and Ease of Learn Mode</li><li style="margin-bottom:0.1em">Types of Vulnerabilities it can prevent.</li><li style="margin-bottom:0.1em">Detects disclosure and unauthorized content in outbound reply messages, such as credit-card and Social Security numbers.</li>

<li style="margin-bottom:0.1em">Both Positive and Negative Security model support.</li><li style="margin-bottom:0.1em">Simplified and Intuitive User Interface.</li><li style="margin-bottom:0.1em">Cluster mode support.</li>

<li style="margin-bottom:0.1em">High Performance (milliseconds latency).</li><li style="margin-bottom:0.1em">Complete Alerting, Forensics, Reporting capabilities.</li><li style="margin-bottom:0.1em">Web Services\XML support.</li>

<li style="margin-bottom:0.1em">Brute Force protection.</li><li style="margin-bottom:0.1em">Ability to Active (block and log), Passive (log only) and bypass the web trafic.</li><li style="margin-bottom:0.1em">Ability to keep individual users constrained to exactly what they have seen in the current session</li>

<li style="margin-bottom:0.1em">Ability to be configured to prevent ANY specific problem (i.e., Emergency Patches)</li><li style="margin-bottom:0.1em">Form Factor: Software vs. Hardware (Hardware generally preferred)</li>
