>> *Simon Bennetts - "An introduction to OWASP ZAP"*
>>
>> The Zed Attack Proxy (ZAP) is now one of the most popular OWASP projects.
>> It is designed to be used by people with a wide range of security
>> experience and as such is ideal for developers and functional testers who
>> are new to penetration testing as well as being a useful addition to an
>> experienced pen tester's toolbox.
>> In this talk Simon will give an overview of the functionality ZAP
>> provides as well as an insight into some of the enhancements that will be
>> available in the forthcoming 2.3 release
>>

>>
>> Ce sunt shellcode-urile, care e diferenta dintre un executabil si un
