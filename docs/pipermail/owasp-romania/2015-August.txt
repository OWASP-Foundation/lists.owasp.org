Buna tuturor,

Pe *9 octombrie 2015* vom avea un eveniment parte a *OWASP EEE (Eastern
European Event).*

*OWASP EEE* este de fapt o saptamana de evenimente sincronizate organizata
de capitole din Europa de Est (5-10 octombrie). Evenimentele vor avea live
streaming pe youtube, astfel incat in fiecare zi oamenii din Europa si nu
numai vor putea urmari un eveniment live dintr-o tara participanta.
Capitolele implicate pana acum sunt: Romania (Bucuresti si Cluj), Polonia,
Ungaria, Rusia si Austria.

Acesta este "Call for Speakers
<https://docs.google.com/a/owasp.org/forms/d/1HnHihbW2FK8J9s1EaF4RKMphPcZnaS8Dim39h1bfttM/viewform>"
pentru evenimentul nostru.
Asteptam inscrieri pentru prezentari sau workshop-uri din urmatoarele arii
si nu numai:

? Security aspects of new / emerging web technologies / paradigms /
languages / frameworks
? Secure development: frameworks, best practices, secure coding, methods,
processes, SDLC, etc.
? Security of web frameworks (Struts, Spring, ASP.Net MVC, RoR, etc)
? Vulnerability analysis (code review, pentest, static analysis etc)
? Threat modelling of applications
? Mobile security and security for the mobile web
? Cloud security
? Browser security and local storage
? Countermeasures for application vulnerabilities
? New technologies, paradigms, tools
? Application security awareness and education
? Security in web services, REST, and service oriented architectures
? Privacy in web apps, Web services and data storage

Vor fi selectate prezentarile si temele cele mai interesante.

Important:

   - termenul limita pentru inscrierea prezentarilor este 17 septembrie
   - conferinta va avea loc pe 9 octombrie
   - prezentarile vor avea durata de 45 de minute fiecare
   - va exista un speaker agreement
   <https://www.owasp.org/index.php/Speaker_Agreement>


O seara frumoasa!

Buna tuturor,

In proces de fund raising pentru evenimentul din octombrie, va trimit
atasata oferta de sponsorizare si cateva clarificari pentru cei care ar
putea da informatia mai departe sau care ar putea ajuta.

Evenimentul din Bucuresti, parte a *OWASP EEE* va avea loc pe *9 octombrie*.
OWASP EEE (Eastern European Event) este o saptamana de evenimente
sincronizate organizata de capitole din Europa de Est (5-10 octombrie).

In fiecare zi oamenii din Europa si nu numai vor putea urmari un eveniment
live, pe Youtube, dintr-o tara participanta. Capitolele implicate pana acum
sunt: Romania (Bucuresti si Cluj), Polonia, Ungaria, Rusia si Austria.

Primele doua optiuni, Platinum si Gold, sunt pentru toata regiunea si
sponsorizarile se vor imparti intre capitolele participante iar varianta
Silver/Local este pentru capitolul nostru local.

*De ce sa sponsorizez?*

   - Veti fi alaturi de leaderi, consultanti de securitate, arhitecti de
   securitate si programatori din 6 tari (7 capitole locale OWASP) veniti sa
   schimbe idei despre initiative si noutatile tehnologice.
   - Evenimentele OWASP atrag o audienta interesata de "What's next?" in
   securitatea IT - Sponsorul va fi promovat ca raspuns la aceasta intrebare.
   - Cresterea gradului de constientizare si recunoasterea in comunitatea
   de securitate IT din Romania
   - Sustineti si jucati un rol activ in lumea pasionatilor de securitate a
   informatiei. Numarul de participanti cumulat pentru evenimentele din toate
   capitolele este de peste 1000.


Evenimentele trecute ale capitolului nostru:
OWASP Romania InfoSec Conference 2014
<https://www.owasp.org/index.php/OwaspRomaniaConference2014>
OWASP Romania InfoSec Conference 2013
<https://www.owasp.org/index.php/OWASP_Romania_InfoSec_Conference_2013>
OWASP EU Tour 2013 - Bucharest
<https://www.owasp.org/index.php/EUTour2013#tab=Bucharest>

Multumesc!
