Salut Gigi, all,

Incep a raspunde prin a face cunoscuta misiunea globala a OWASP disponibila
la www.owasp.org .

*The Open Web Application Security Project (OWASP) is a 501(c)(3) worldwide
not-for-profit charitable organization focused on improving the security of
software. Our mission is to make software security visible, so that
individuals and organizations worldwide can make informed decisions about true
software security risks.*
*
*
Continuand, vreau sa cred ca nimeni nu este atat de naiv incat sa se dea in
vileag ca doreste sa obtina avantaje banesti, sau de alta factura, in urma
exploatarii unei vulnerabilitati gasite.

In alta ordine de idei, activitatea unui Application / Infrastructure
Security Analyst, Security Auditor, IT Security Engineer, Vulnerability
Security Research Engineer, etc consta *si* in intelegerea principiului de
functionare a unei aplicatii / sistem / device, patrunderea in maruntaiele
ei/lui, in vederea hard-enizarii acesteia/acestuia si protejare lui de
persoane rau intentionate.

Multumesc Teodor pentru raspunsul prompt si totodata incurajam ca, daca nu
toti, cat mai multi dintre voi sa va spuneti parerea si sa contribuiti cu
idei, cunostinte sau sa deschideti noi teme de discutii.

Multumesc,

Ionel.
