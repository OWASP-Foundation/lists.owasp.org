Buna,

Iata niste update-uri cu ce s-a mai intamplat in ultimul timp, de ce puteti beneficia si modalitati de implicare:


	* A fost lansata prima parte a "Secure Application Design Project"  

https://www.owasp.org/index.php/OWASP_Secure_Application_Design_Project
Cei ce doresc, pot trimite feedback si pot contribui. Urmatoarele release-uri vor contine:
 1. Secure Design Techniques for:
    a. Web applications
    b. Thick/smart client applications
 2. Secure guidelines for commonly used design frameworks in web applications
	* Mobile Security Project - Mobile Tools a fost reactualizat. 
https://www.owasp.org/index.php/Projects/OWASP_Mobile_Security_Project_-_Mobile_Tools
Puteti afla mai multe despre iMAS (un proiect de cercetare initiat de MITRE focusat pe controale de securitate pentru iOS), GoatDroid, iGoat si MobiSec.
	* A fost dat "Mobile Top 10 Refresh- Call For Data" - pentru a stabili topul riscurilor de securitate ce afecteaza aplicatiile mobile. Este nevoie de un efort dedicat si continuu pentru a sustine aceasta initiativa. Daca ati fost implicati in proiecte ce au legatura cu securitatea aplicatiilor mobile si puteti oferi date/statistici anonimizate despre vulnerabilitati si incidente va incurajez sa contribuiti la acest proiect.  
Mai multe detalii gasiti aici: http://owasp.blogspot.com/ 2013/05/2013-owasp-mobile-top- 10-call-for-data.html
	* OWASP lanseaza "Summer 2013 Membership Drive" - Aici puteti afla de ce merita sa deveniti membrii si ce puteti castiga.
	* Un alt eveniment ce merita urmarit este WASPY Awards 2013 - Web Application Security People of the Year Awards 2013 - in functie de activitatea din comunitatea Owasp, din ultimul an, oricine poate fi nominalizat sau poate nominaliza pe cineva; insa doar paid members pot vota. 

	* OWASP Europe Tour este in desfasurare: https://www.owasp.org/index.php/EUTour2013
	* Agenda evenimentului nostru: https://www.owasp.org/index.php/EUTour2013_Bucharest_Agenda
	* Oferte de sponsorizare: http://owasp.com/images/8/82/European_Tour_2013_With_Some_Dates_V.1.1.pdf
	* Bluekaizen (cea mai mare comunitate de security din middle east) vor avea un curs tinut de Justin Searle in Istanbul, luna viitoare: "Assessing and Exploiting Web applications with samurai-WTF". Justin Searle tine acelasi curs inainte de conferinta BlackHat din Las Vegas, la un pret dublu. Pentru membrii Owasp, am primit si am atasat o oferta, iar mai multe detalii despre curs gasiti la: www.bluekaizen.org/samurai
Acestea sunt noutatile.
Nu ratati evenimentul nostru de miercuri! Va asteptam in numar cat mai mare!

O seara frumoasa!
