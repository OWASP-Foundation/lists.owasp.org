Event Details:
------------------------------

Owasp Romania organizes a chapter meeting, part of the Owasp Europe Tour.The Open Web Application Security Project is an open-source application security project. The OWASP community includes corporations, educational organizations, and individuals from around the world.Owasp meetings intend to simulate discussions on paalication security issues, raise awareness of security risks, motivate and involve members in Owasp open source projects.OWASP Europe TOUR, is an event across the European region that promotes awareness about application security, so that people and organizations can make informed decisions about true application security risks. Everyone is free to participate in OWASP and all of our materials are available under a free and open software license.For the meeting agenda and more details, please visit the Owasp EU Tour webpage.
