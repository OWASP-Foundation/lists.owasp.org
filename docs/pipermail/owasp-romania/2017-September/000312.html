Buna tuturor,

Echipa OWASP Bucharest are placerea de a va invita la *OWASP AppSec
Bucharest 2017*! Evenimentul va avea loc intre *11 si 13 octombrie
2017* la Hotel
Caro <http://www.carohotel.ro/>.

https://www.owasp.org/index.php/OWASP_Bucharest_AppSec_Conference_2017

Agenda conferintei cuprinde doua track-uri de prezentari:
[image: Inline image 1]
In zilele de 11 si 12 octombrie vom avea training-uri:

de o zi
<https://www.owasp.org/index.php/OWASP_Bucharest_AppSec_Conference_2017#tab=1_day_training>
:
pe 11 octombrie:

   - OWASP Top 10 vulnerabilities – discover, exploit, remediate
   - Introduction to Metasploit Framework

pe 12 octombrie:

   - Time critical DFIR: Key playbooks, techniques and tools for
   time-pressured investigations of security incidents

de doua zile:
<https://www.owasp.org/index.php/OWASP_Bucharest_AppSec_Conference_2017#tab=2_days_training>
pe 11 si 12 octombrie:

   - Assessing and securing applications using the OWASP ASVS (Application
   Security Verification Standard)
   - Advanced Malware Analysis

programul complet:
