Am placerea de a va invita la evenimentul anual organizat de capitolul
OWASP din Bucuresti.

*OWASP Bucharest AppSec Conference 2018 *va avea loc pe *25 si 26 octombrie
2018*. Ca si anul trecut vom avea training-uri/workshop-uri si o competitie
capture the flag.
https://www.owasp.org/index.php/OWASP_Bucharest_AppSec_Conference_2018

Inregistrarea prezentarilor se realizeaza aici.
<https://docs.google.com/forms/d/e/1FAIpQLScjfXpflQpE6ytoh5ALPjFeuw3g54Pt5g4rUqt0kZSOEvhovw/viewform>

<https://docs.google.com/forms/d/e/1FAIpQLSe8RXs_k6LeBVMxST4HxCNa5f9w7-ZSTUxXBaFWkXxxH1slag/viewform>
Oportunitatile de sponsorizare sunt in acest document
<https://www.owasp.org/images/d/d3/OWASPAppSecBucharest2018-sponsorshipopportunities.pdf>
.

Va puteti inscrie cu prezentari sau workshop-uri din urmatoarele arii si nu
numai:

   - Security aspects of new / emerging web technologies / paradigms /
   languages / frameworks
   - Secure development: frameworks, best practices, secure coding,
   methods, processes, SDLC, etc.
   - Security of web frameworks (Struts, Spring, ASP.Net MVC, RoR, etc)
   - Vulnerability analysis (code review, pentest, static analysis etc)
   - Threat modelling of applications
   - Mobile security and security for the mobile web
   - Cloud security
   - Browser security and local storage
   - Countermeasures for application vulnerabilities
   - New technologies, paradigms, tools
   - Application security awareness and education
   - Security in web services, REST, and service oriented architectures
   - Privacy in web apps, Web services and data storage

Important:

   - termenul limita pentru inscrierea prezentarilor este *24 septembrie*
   - lista speakerilor confirmati va fi anuntata pe *1 octombrie*
   - finala CTF-ului si training-urile vor avea loc pe *25 octombrie*
   - prezentarile vor avea loc pe *26 octombrie*
   - prezentarile vor avea durata de *40 de minute* fiecare
   - va exista un speaker agreement
   <https://www.owasp.org/index.php/Speaker_Agreement>si un trainer
   agreement
   <https://www.owasp.org/index.php/File:Training_Instructor_Agreement.doc>
