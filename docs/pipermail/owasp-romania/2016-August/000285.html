Buna tuturor,

Dupa cum stiti, pe *6 octombrie* vom avea urmatorul eveniment OWASP, o
conferinta de o zi focusata pe securitatea aplicatiilor.

Detaliile despre *OWASP Bucharest AppSec Conference 2016* sunt publicate
aici: https://www.owasp.org/index.php/OWASP_Bucharest_AppSec_Conference_2016

Vom avea 4 workshop-uri
<https://www.owasp.org/index.php/OWASP_Bucharest_AppSec_Conference_2016#tab=Workshops>
:

   - OWASP Top 10 vulnerabilities – discover, exploit, remediate
   - Secure Web Applications in Java
   - Shellcode Development and Exploiting
   - Practical Cryptography on the Internet

si un CTF
<https://www.owasp.org/index.php/OWASP_Bucharest_AppSec_Conference_2016#tab=CTF>
.

Link-uri utile:
