 Merhaba, Her gün Modsecurity loglarını incelemeniz gerekiyorsa ve modconsole kullanmıyorsanız (kullanamıyorsanız) Jarvinen tam size göre bir uygulama. Unix sistemlerinde Modsecurity web uygulama güvenlik duvarı kullananlar için web tabanlı log analiz uygulaması olan Jarvinen iki kısımdan oluşuyor. İlk kısmı “Serial” olarak üretilen logları MySQL veritabanına aktaran bir shell script. İkinci kısım bu veritabanından log bilgilerini okuyan bir PHP web uygulaması. NOT: Şu an sadece ilk parçasını (audit log to db dumper) yayınladığımız Jarvinen’in ikinci parçasını (web uygulaması) çok yakın bir zamanda (gelecek hafta) www.webguvenligi.org adresinde yayınlayacağız. Projenin ilk kısım Gökhan Alkan tarafından yazılmıştır, ikinci kısım Yusuf Çeri tarafından hazırlanmaktadır. bedirhan
