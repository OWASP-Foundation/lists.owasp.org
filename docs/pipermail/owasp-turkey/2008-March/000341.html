"OWASP Summer Of Code 2008", OWASP projelerine finansal olarak destek amacini tasimaktadir. Bu organizasyona, kisiler kendi sectikleri veya OWASP'in belirledigi web/uygulama guvenligi yazilim projeleri ile basvurabilmektedirler. Onerilen projeler OWASP kurulu tarafindan kabul edilirse, katilimcilar projelerini gercekleme asamasina gecmektedirler. Uretilen proje karsiliginda katilimcilar maddi olarak desteklenmektedir. Bu seneki butce 100.000$. Detaylar asagidaki linkte bulunabilir.  https://www.owasp.org/index.php/OWASP_Summer_of_Code_2008_Press_Release
