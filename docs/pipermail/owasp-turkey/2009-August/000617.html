Merhabalar,

Uzun zamandır OWASP Turkey <http://www.owasp.org/index.php/Turkey> ve 
Web Güvenlik Topluluğu <http://www.webguvenligi.org> olarak, web 
güvenliği eksenli bir e-dergi projesi 
<http://www.webguvenligi.org/projeler/web-guvenligi-e-dergi-zamani.html> 
üzerinde çalışıyorduk.

Periyodik olarak yayınlayacağımız e-dergi 
<http://dergi.webguvenligi.org/>`ye ücretsiz olarak internet üzerinden 
erişebileceksiniz. Stabil yazar kadrosunun yanında, e-dergi`ye katılım 
yapmak isteyen okurlarımızın ise web güvenliği eksenli yazılarını kabul 
edeceğiz.

E-dergi de;

- Web güvenliği eksenli teknik yazıların yanı sıra teorik yazılar ve 
farklı yaklaşımları,
- Web uygulamalarına yönelik saldırı vektörleri ve önlemlerini,
- Otomatik web açıklık tarayıcılarıyla alakalı bilgileri,
- Güvenli kod tekniklerini,
- Web güvenlik duvarlarını ve
- Yazılım güvenliği olgunluk modellerini

içeren konuları bulabileceksiniz.

Özgün bir proje olan, yazar kadrosu profesyonellerden oluşan ve web 
güvenliği alanında branşlaşmış ilk Türkçe yayın olan Web Güvenlik 
Topluluğu E-Dergi`yi yayımlamanın gururunu ve sevincini sizlerle 
paylaşmaktan kıvanç duyarız.

Basın Bülteni: 
http://www.webguvenligi.org/docs/webguvenligi_e_dergi_basin_bulteni.pdf
