ORM (object relational mapping) tarzi bi teknoloji kullanirsaniz ve/veya
Model-Controller-View tarzi bi yapi kullanirsaniz sql injection tipi
ataklardan basitce kurtulmus olursunuz. her ikisinde de temel mekanizma,
veritabani tablolarinin  model class'lara map edilmesi ve bu class'larin
bu tur saldirilara karsi zaten onlem almis olmasi.

ilgilenenler icin Ruby on Rails'i tavsiye ederim. ORM kullanan bir MVC
web framework. .net ciler icin (parali) code smith (ORM), java'cilar
icin spring ve pyhton'cular icin django birer alternatif.
