Merhaba,
JSecureImage <http://code.google.com/p/jsecureimage/>, java ortaminda
guvenli resim upload'i icin kullanilabilecek bir kutuphanedir. Upload edilen
resimlerin bazi ozelliklerini kontrol eden
JSecureImage<http://code.google.com/p/jsecureimage/> hatali
veya guvensiz resimleri reddeder. Standard Java ImageIO API'ini kullanan
JSecureImage, kucuk bozukluklari olan bazi resimlerde hatali sonuc
vermekteydi. Bu hatayi kutuphanenin kullanildigi isletim sisteminde JDK
uzerine JAI ImageIO kutuphanesini kurarak cozebilirsiniz.

https://jai-imageio.dev.java.net/binary-builds.html#Release_builds

Ayrica yeni JSecureImage
kutuphanesini<http://jsecureimage.googlecode.com/files/jsecureimage.jar>indirip
resim dosya upload ozelligi bulunan kendi J2EE web uygulamanizi da
guvenli hale getirebilirsiniz.
