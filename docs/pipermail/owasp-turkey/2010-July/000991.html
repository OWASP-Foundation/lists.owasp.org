Merhabalar,

ADEO Security Labs ekibi tarafından geliştirilen, Zaman Tabanlı SQL Injection zafiyetlerini otomatize olarak exploit etmeyi amaçlayan ColdEffect'i bugün yayınladık. 

.NET platformunda geliştirilen bu uygulamamız ile zaman tabanlı sql injection zafiyetlerini daha efektif olarak exploit etmeyi amaçladık. Programda, daha önce yayınladığımız 'Zaman Tabanlı SQL Injection Saldırıları <http://security.adeo.com.tr/Makale/13-zaman-tabanli-sql-injection-saldirilari.adeo> '  makalesindeki kullandığımızı klasik injection tekniğini kullanmakla beraber, 11 Ağustos tarihinde yayınlamayı planladığımız Private Edition'da ise daha farklı teknikler barındırmayı ve diğer veritabanı yönetim sistemlerine destek vermeyi düşünüyoruz.

Projede aktif rol oynayan ADEO Security Labs ekibindeki Onur Yılmaz ve Canberk Bolat'a buradan teşekkür ederim.

 

Detaylı Bilgi:  http://coldeffect.adeo.com.tr <http://coldeffect.adeo.com.tr>
