Merhabalar,
 
Ben bu arada NSecureImage uzerinde yaklasik olarak 1 haftadir calisiyorum ( vakit buldukca )
Orjinal versiyona gore eklediklerim:
 
1) C# Png Parser
2) C# Gif Parser
3) C# Bmp Parser
 
Bunlar GDI+ kullanmadan direkt olarak fileformat kontrolu yapiyor. 
Png Parser png dosyalari icin ozellikle kontrol bazlı duzenlenmiş Png Dosyalari uzerinde denendi ve ciddi anlamda libpng'de buffer overflow olusturabilecek yapilari simdiden kendisi tespit edebiliyor.
Gif Parser su anda baslangic asamasinda, ama uzun vadede planım ayni librarylerin Java'ya port edilmesi ( Aslinda Java'da avantajımız yuksek cunku IBM'in Eclipse SWT Paketi icinde bununla ilgili parser classları zaten bulunuyor.) 
Toplantıda NSecureImage hakkındaki gelistirmeler ve digerlerine nasil uyarlanabilecegi konusunuda tartışmak isterim.
 
Ayrıca http://getid3.sourceforge.net/source/ altındaki Php Projesini .Net Framework'e port edip,
bir webguvenligi.org projesi olarak yayinlamak istiyorum. Bu otomatik olarak mime type bazında detection ve file format kontrol yapan bir proje ve maalesef .Net Platformunda bunu yapabileceginiz halihazırda bir proje yok. 
 
Toplantıda detayları konusmak ümidiyle :) 
 
Saygılar , Sevgiler :)
 
NSecureImage http://www.codeplex.com/owaspturkey altından ulasilabilir durumda :)
