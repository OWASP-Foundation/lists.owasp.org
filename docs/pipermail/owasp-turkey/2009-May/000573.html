kullanılabilecek ultra modüler bir fuzzer olan Freaking Simple Fuzzer (aka
FSF) .NET uygulamasına ve detaylarına

*
Uygulama *Mono *ile OSX ve Linux altinda, .NET Framework 3.5 ile Windows
altinda calismaktadir. Kaynak kodlari google code sayfalarindan download
edilebilir - http://code.google.com/p/fm-fsf/.
