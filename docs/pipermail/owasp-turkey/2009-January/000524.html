Bir onceki mail ile alakali olarak; Microsoft'un Aralik ayinda duyurdugu
statik kod analizi aracini, .NET projenizdeki belli basli guvenlik
problemlerini gelistirmeniz devam ederken bulabilirsiniz. Stand-alone veya
Visual Studio IDE'ye entegre sekilde calistirilabiliyor. Ben IDE'ye entegre
ederek denedim, bedava bir urun icin guzel sonuclar uretiyor.

SQL Injection
Process Command Injection
File Canonicalization
Exception Information
LDAP Injection
XPATH Injection
Redirection to User Controlled Site
