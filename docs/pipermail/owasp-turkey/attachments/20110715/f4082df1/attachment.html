<div>aynen. yani jquery&#39;nin text metodu htmlencoded stringi alip decode ediyor (<a href="http://api.jquery.com/text/" target="_blank">http://api.jquery.com/text/</a>), cunku calisma prensibi xml element icindeki text alanlari okuyup string olarak vermek (dolayisiyla &amp;lt; gibi html kodlanmis stringi &lt; seklinde okuyor). index.php&#39;nin PHP kodunu veriyorum (<a href="http://pastebin.com/xPPx3p7W" target="_blank">http://pastebin.com/xPPx3p7W</a>), guvenlik prensiplerine output encoding acisindan uyulmus.</div>


<div> </div>
<div>Simdi gelelim developer&#39;lari ilgilendiren ilginc noktaya. Yukaridaki konuyu gercek hayata uyarlayalim. Guvenlik konusuna ilgili ve farkinda bir developer, uygulamasinin cikti noktalarinda HTMLEncode kullanmasi gerektigini biliyor. Her klasik html output ettigi yere framework&#39;un kendine sagladigi otomatik veya manuel HTMLEncode metodunu kullanir. </div>


<div> </div>
<div>Sonra arayuzu afilli olsun diye de guzel gorunen <strong>html select</strong>ler yapmak icin jquery plugin&#39;i <a href="http://www.filamentgroup.com/lab/jquery_ui_selectmenu_an_aria_accessible_plugin_for_styling_a_html_select/" target="_blank">selectmenu</a>&#39;yu kullanmak ister. Ama selectmenu kendi icinde jquery text&#39;i kullanir ve js rendering islemi sirasinda inanilmaz bir sekilde XSS olusur. :) Sonra demezler mi, &quot;bize HTMLEncode deyip duruyorsun, hala XSS olusuyor&quot; diye.</div>


<div> </div>
<div><a href="http://www.webguvenligi.org/jquerytext/selectmenu.php" target="_blank">http://www.webguvenligi.org/jquerytext/selectmenu.php</a></div>
<div><br>Bence JQuery&#39;nin bu text metodu sakat. Tek basina olasilik dusuk olabilir ama icinde kullanildigi pluginlerle beraber risk gayet iyi. Cunku innerHTML veya appendChild gibi DOM&#39;un kaba metotlari ile hem IE hem de FF&#39;de XSS calistirmak kolay degil. Ama JQuery append diyor, tak calisiyor.</div>


<div> </div>
<div>Sonuc olarak yapacak sey ya plugin&#39;i kullanmamak ya da double encode etmek. Double encode... kotu seyler cagristiriyor degil mi?</div>
