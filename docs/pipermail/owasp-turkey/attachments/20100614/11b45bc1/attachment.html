<div><em>SELECT * FROM `dvwa`.`users` where user_id=1 AND 1=(CASE LOCATE(0x61,(SELECT substr(&quot;baca&quot;,3)),1)<br>WHEN 1 THEN SLEEP(1)<br>WHEN 2 THEN SLEEP(2)<br>WHEN 3 THEN SLEEP(3)<br>WHEN 4 THEN SLEEP(4)<br>ELSE SLEEP(0) END)</em></div>

<div> </div>
<div>case&#39;lerin hic birine dokunmayiz, zarari yok kalmalarinin... ;)</div>

<div> </div>
<div><em> This function is multi-byte safe, and is case-sensitive only if at least one argument is a binary string. </em></div>
<div> </div>
