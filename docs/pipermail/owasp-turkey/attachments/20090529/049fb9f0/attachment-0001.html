Hocam elinize sağlık. Makalede eksik gördüğüm veya bana göre eksik olan yerler var. basit ama yinede olsa iyi olur diye düşünüyorum<div>Logon için</div><div><a href="http://1.si">1.si</a> yerel kullanıcılara ihtiyaç duyması <a href="http://2.si">2.si</a> ise etki alanı kullanıcıları </div>
<div><br></div><div>Yerel kullanıcılar için aşağıdaki komutu vermişsiniz. </div><div><br></div><div>C:\program files\openssh\bin \mkpasswd ‐l ‐u username &gt;&gt; ..\etc\passwd <br><br></div><div>domain kullanıcıları eklemeye çalıştığımda böyle bir user yok diyor. Readme ye baktığımda -d switchini kullanmam gerektiğini gördüm. </div>
<div>bir diğer konu Openssh ı xp üzerinde kullanmak isteyenler de olabilir. ince ayar kısmında belirttiğiniz security default olarak xp de gelmiyor. security görmeleri için klasör seçeneklerinden &quot;use simple file sharing&quot; i kaldırmaları gerekiyor. </div>
