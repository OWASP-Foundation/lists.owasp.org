Web Application Pentest eğitimi günümüz bilişim güvenliğinin en zayıf
halkalarından biri olan web uygulamalarına
hacker gözüyle bakıp açıklıkların nerelerde nasıl oluşabileceğini önceden
keşfetmeyi sağlamaya yönelik bir eğitimdir.

Eğitim boyunca katılımcılar sorumluluklarında olan web uygulamalarına
yönelik  güvenlik testlerinin hangi adımlardan
 oluştuğunu ve nasıl yapıldığını pratik olarak görme şansını
yakalayacaklardır.
