Merhabalar,

Bu hafta GüvenlikTV'de OWASP-Türkiye Lideri Bünyamin Demir'i konuk olarak ağırlayacağız. Bünyamin'in kısa bir biyografisini aşağıda bulabilirsiniz. Bünyamin'e sormamızı istediğiniz sorularınız  varsa benimle özelden paylaşabilirseniz yayın sırasında kendisine sorularınızı yöneltebilirim. Sorularınız güvenliğin herhangi bir alanıyla alakalı olabileceği gibi OWASP ile ilgili de sorabilirsiniz. Sorularınızı Twitter üzerinden de sorabilirsiniz. Güvenlik TV'nin Twitter adresi @guvenliktv. Şimdiden çok teşekkürler.

Bünyamin Demir Kimdir?

1982 Erzincan doğumlu olan Bünyamin Demir, lisans ve lisansüstü eğitimini Kocaeli Üniversitesi Matematik Bölümünde yapmıştır. 2006 yılında web developer olarak başladığı iş hayatına, daha sonra veritabanı güvenliği ve web güvenliği uzmanı olarak devam etmiştir. Şu an ağırlıklı olarak sızma testleri üzerine güvenlik danışmanlığı yapmaktadir. 2007 yılından itibaren OWASP-Türkiye ve Web Güvenliği Topluluğu (www.webguvenligi.org <http://www.webguvenligi.org> ) üyesi olup, web güvenliği alanında yazılım ve dokümantasyon projelerine katılmış, halen Web Güvenliği Topluluğu ile birlikte uygulama güvenliği üzerine çalışmalar yapmaktadır.

Teşekkürler, iyi çalışmalar.
