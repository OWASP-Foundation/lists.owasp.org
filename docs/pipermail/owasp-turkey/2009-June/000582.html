Damn Vulnerable Web App (DVWA), PHP ile open source olarak gelistirilmis,
temel web uygulamalari zafiyetlerini gorup tespit ederek exploit
edebileceginiz ve sonuclarini gorebileceginiz bir test ortami olarak
yayinlandi.

 

DVWA ile ozellikle PHP ile web uygulamalari gelistirenler uygulamalarda ne
gibi acikliklar olabilecegi konusunda fikir edinebilir, kendi uygulamalarini
yaptiklari testler sonucunda redakte edebilirler.

 

Yahut web uygulamalarinda zayiflik tespitinin nasil yapildigi konusunda
kendisini gelistirmek isteyenler, DVWA`yi sistemlerine kurarak burada
testler yapabilmektedir.
