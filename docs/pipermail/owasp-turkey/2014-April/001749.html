We are looking for a Security Engineer for our client which is a Turkey
leader in telecom sector.

Qualifications

·         BS / MS University degree in Computer, Electronics, Math
Engineering or similar engineering  disciplines

·         Good understanding of Web  Application Security, OWASP tools and
methodologies

·         Experience with  code security scanner tools such as Fortify,
Appscan and others.

·         Preferably software development experience in a programming
language such as C++, Java,.NET

·         Good understanding  and/or experience with security solutions such
as  Firewalls, VPNs, DOS, Two-factor Authencation Systems, DLP, PKI,
Certificates, Encryption, Hashing and Web/Database Firewalls.

·         Solution oriented with outstanding analytical, interpersonal,
communication and team working  skills

·         Eager to investigate and improve overall security and quality of
systems, applications and services

·         IT Security Certifications are desired including GIAC
certifications, CISSP, CISA, CEH, MCSE or others.

·         Project management experience is desired.

Job Description

·         As a security analyst identify security risks and develop/suggest
solutions for ongoing  IT projects actively taking part in the analysis and
design phase

·         Improve the SDLC  development phase by leading the application
security improvement efforts

·         Perform application  code review across a variety of programming
languages

·         Contribute to the  improvement of IT Security Requirements
standards and development  guidelines.

·         Working closely with system analysts, developers and business
teams act as a security liason to improve security awareness and overall
product quality

·         Coordinate and lead  the Vulnerability Management efforts

·         Develop reports to  show Security Risks, Vulnerabilities and
Improvements from a network, system and application domain perspective

·         Evaluate, recommend and implement security tools and solutions
based on identified risks and business needs

·         Maintain 7X24  operational availability of specific security
solutions related with application security and vulnerability management
such as Certificate Server, Vulnerability Scanner, Application Firewall
systems.
