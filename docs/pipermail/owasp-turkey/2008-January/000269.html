Eğer SQL injection gibi bir açık var ise sql sorgularına NOW() ifadesi ekleyerek öğrenebilirsiniz. Bunun dışından TCP timestamp yolu ilede öğrenmek mümkün olabilir. Yine icmp tstamp açıksa bu değerler elde edilebilir.sing gibi bir aracı bu iş için kullanmak mümkün olur. nmap sistem uptime'ini tcp timestamp değeri ile ölçebiliyor. Uptime 6.971 days (since Tue Aug 16 08:48:392005 Buradan sistem tarihini aşağı yukarı kestirmek mümkün olabilir. Bunun dışında bazı web uygulamaları sistem tarih ve saat bilgisini ekleyebiliyor. Sitedeki arama fonksiyonlarını bu iş için kullanabilirsiniz.

merhaba 

herhangibir serverside dille (php, .net, python, ... vs) basit şekilde sunucunuzun saatini öğrenebilirsiniz.

php ile örnek :
<?php
echo date("d.m.Y H:i:s");
?>

ayrıca kullandığınız diğer servislerinde satlerini öğrenebilirsiniz, mesela basit bir query ile mysql sunucunuzun kullandığı saati öğrenebilirsiniz.

mysql ile örnek :

SELECT NOW()

iyi çalışmalar
