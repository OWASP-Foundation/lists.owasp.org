Hping komutu ile icmp-timestamp request yollamissiniz. Eger sunucu
ICMP isteklerine kapali ise buna cevap alamazsiniz. Diger bir yontem
TCP Timestamp degerine bakmak olabilir. nmap'i bu is icin
kullanabilirsiniz. Tabi buda kesin sunuc vermeyebilir.

# nmap -sT -P0 www.google.com -p 80 -O

Ama en iyi yontem Halil beyin belirttiği http response larina bakarak
zaman bilgisini ogrenmek gibi duruyor. Tabi arada bir cache veya
reverse proxy yoksa.
