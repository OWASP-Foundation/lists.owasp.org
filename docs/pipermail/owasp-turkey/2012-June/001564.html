HTTP Flood saldırılarının Web uygulaması seviyesinde tespit edilerek, etkilerinin azaltılmasına dair hazırladığım makaleyi aşagıdaki linkte bulabilirsiniz.

IOSEC v.1.6 kodu Wordpress ekibin onayından geçerek plugin olarak da yayına girmiştir.
