>> *OWASP SUMMER CODE SPRINT 2015 *
>>
>> *OWASP Summer Code Sprint 2015 is a program that aims to provide
>> opportunities and incentives for students to gain experience with
>> Application Security coding as 'interns' contributing to OWASP projects.
>> We call this our Summer Sprint since it is a very short term initiative.
>> Students will gain valuable experience over the July-August timeframe.*
>>
>> *All students who successfully complete the program will receive in total
>> $1500.*
>>
>> *OWASP: Call for Students for Code/Tool Projects:*
>>
>>    - *Submit your proposal no later than July 3**rd**, 2015*
>>    - *We are looking for candidates who are committed to supporting the
>>    OWASP Mission to make application security visible.*
>>    - *Students will have the following opportunities:*
>>
>>
>>    - *Receive mentoring and dedicated time from OWASP Project Leaders*
>>    - *Get real-life experience while contributing to an open source
>>    project by using your creativity and knowledge.*
>>
>>
>>
>> *You do NOT need to be a proven developer--in fact, the program is meant
>> to facilitate relationships within the OWASP and Open Source communities
>> regardless of level of experience. Of course, experience in coding and
>> application development are welcome, just not required.*
>>
>>
>>
>> *Full details and requirements for participating:*
>>
