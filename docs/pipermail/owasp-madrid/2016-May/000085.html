Buenas a todos!

Tras un tiempo de inactividad, volvemos a la carga con el que esperamos sea el relanzamiento de OWASP Madrid, en la que trataremos de hacer iniciativas de forma más habitual.

Por lo pronto, el 28 de Junio (Martes), nos estrenamos con un meetup en el Auditorio de Flagship Store Telefónica (Gran Via, 28 28004, Madrid). Toda la información en: http://www.meetup.com/es-ES/OWASP-Madrid/events/231391701/ 

Con las siguientes charlas (pendientes de una tercera):

Seguridad en NodeJS con OWASP

Siempre hemos visto la seguridad como algo oscuro que solo tenia que ver con los hackers. Pero eso no es así. La seguridad tiene que partir de nosotros los desarrolladores. En esta charla veremos algunas de las vulnerabilidad enumeradas en el OWASP Top 10 y como evitarlas para aplicaciones en NodeJS utilizando el proyecto Node Goat.

Seguridad & Python: El desarrollo seguro no sólo es cosa de Java

Hace mucho tiempo que Python dejó de ser un lenguaje de scripting para convertirse en una potente herramienta mucho más genérica: Google, DropBox, Yahoo o NASA, entre otras, puedan dan buena cuenta de ello. 

En la charla repasaremos de forma práctica las principales vulnerabilidades, inyecciones y fallos de seguridad causados por la propia implementación y uso frecuente del lenguaje. ¿O pensabas que estas cosas solo eran cosa de Java o PHP? :-P

— 

Las charlas serán cortas. de unos 30m. La idea es que sea algo ligero, pero interesante :)

Os esperamos a todos!

Chau!
