<p><span><b>OWASP SUMMER CODE SPRINT 2015 </b></span></p>
<p><span><b>OWASP Summer Code Sprint 2015 is a program that aims to provide opportunities and incentives for students to gain experience with Application Security coding as 'interns' contributing to OWASP projects.  We call this our Summer Sprint since it is a very short term initiative.  Students will gain valuable experience over the July-August timeframe.</b></span></p>
<p><span><b>All students who successfully complete the program will receive in total $1500.</b></span></p>
<p><span><b>OWASP: Call for Students for Code/Tool Projects:</b></span></p>
<ul>
<li><b></b><span><b>Submit your proposal no later than July 3</b></span><span><b><sup>rd</sup></b></span><span><b>, 2015</b></span></li>
<li><b></b><span><b>We are looking for candidates who are committed to supporting the OWASP Mission to make application security visible.</b></span></li>
<li><b></b><span><b>Students will have the following opportunities:</b></span></li>
</ul>
<ul>
<li><b></b><span><b>Receive mentoring and dedicated time from OWASP Project Leaders</b></span></li>
<li><b></b><span><b>Get real-life experience while contributing to an open source project by using your creativity and knowledge.</b></span></li>
</ul>
<p><span> </span></p>
<p><span><b>You do NOT need to be a proven developer--in fact, the program is meant to facilitate relationships within the OWASP and Open Source communities regardless of level of experience. Of course, experience in coding and application development are welcome, just not required.</b></span></p>
