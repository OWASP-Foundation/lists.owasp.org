Security Committee,



Attached is a document/catalog that we’ve prepared to include all of the
security standards that are relevant and that we should be monitoring
including:



·         ISO 270001

·         ISO 270002

·         ISO 29115/ITU-T

·         NIST-SP 800-53

·         NIST-SP 800-63

·         GPG 43

·         GPG 44

·         GPG 45

·         ISO 20000 (Part 1&2)

·         NIST-SP 800-57

·         OWASP Application Security Verification Standard

·         PCI-DSS

·         BS 25999/ISO 22301

·         ISO 15489

·         ISO 29003 (Draft)

·         NASPO IDPV (Draft)

·         Oauth

·         NISTir R1 7621 (Draft)



Please review the list and provide any additional standards that you’d like
included in this email thread. If necessary I will distribute a comment
matrix.  We will discuss which set of standards should be nominated first
at our next meeting.







Adam
