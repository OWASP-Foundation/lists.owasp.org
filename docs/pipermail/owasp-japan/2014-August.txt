3. CISOサーベイそろそろ始まります

日本のCISOサーベイ翻訳チーム（有志ボランティア）により
企業・団体のセキュリティ担当者（CISOという役職か
どうかに関わらず、責任をもってやっている方）向けの調査
アンケートを開催しようとしています。

これは幅広く集めることができると、自社の立ち位置の相対化
につながりますし、今後のOWASPの活動の中で、より、社会に
貢献できるプロジェクト推進につながります。

そこで、ぜひ、
・皆さんのご所属の会社
・皆さんのお客様、ご友人の会社
・皆さんのご所属の企業団体の会員

への告知協力をお願いしたいです。

またもし、プロモーションを手伝えるよ、という方がおられたら、
ぜひお声がけください。


4. OWASP APAC TOUR

OWASPは、世界中の地域ごとに、近隣各国でコンファレンスを
立て続けに行う、「TOUR」というものの開催を試みています。

この秋から冬にかけて、APAC TOURの実現をしようという話が
舞い込んできました。１日の、入場無料の、カンファレンスです。

今のところ、１１月２９日にできないかな、などと話していると
ころですが、この開催の実行スタッフ、スポンサーシップ、など
などに関心のある方を募りたいと思っています。

OWASPは毎年、WASPY Awardという表彰を行っていますが、今年の WASPY (Web Application Security
People of the Year) アワードの投票〆切が迫っています。

今年のWASPY Awardに少なからぬ日本メンバーがノミネートさていますので、投票のメールが届きましたOWASPメンバの皆様は奮ってご投票ください。
なお、投票依頼のメールは「Open Web Application Security Project
<owasp ＠ simplyvoting.com>」から送信されていますので、ご注意ください。

Awardのカテゴリは4つに分かれ、カテゴリ毎に1票投票します。
各カテゴリでのノミネート情報を抜粋します。
1. Best Chapter Leader:
Japan Chapter Leaders(Riotaro OKADA & Sen UENO ) と他3候補がノミネートされています。

2. Best Project Leader:
OWASP XSecurity Project(Tokuji Akamine)、OWASP Zed Attack Proxy Project
(ZAP)と他5候補がノミネートされています。

3. Best Mission Outreach: 4候補がノミネートされています。

4. Best New Community Supporter:
AppSec APAC 2014 Teamと他1候補がノミネートされています。
＃AppSec APAC 2014 Teamは、3月に開催されたAppSec APACの実行委員会＆ボランティアスタッフからなる集合体です。

詳しくは、以下のリンクでご確認ください。
https://www.owasp.org/index.php/WASPY_Awards_2014

よろしくお願いします。
