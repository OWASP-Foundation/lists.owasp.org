各位：

2018年4月3日から6日まで、ハワイ カイルア・コナ(直行便あり)にて
アプリケーションセキュリティにフォーカスしたカンファレンスが
開催されます。

"The Loco Moco Security Conference is a builder and defenders conference
for people interested in making impactful and long lasting changes for the
state of web security. "

LOCO MOCO Security Conference
Application security training and conference
April 3rd-6th, 2018

https://locomocosec.com/

4/3-4: トレーニングデイ
4/5-6: コンファレンスデイ

このカンファレンスには、多くのOWASPに関係しているエキスパートや
企業がこのカンファレンスを支えており、スピーカーとしても参加して
います。
セキュア開発のGuruであり、日本のAppSecにも参加した Jim Manico,
元WhiteHat CTOの Jeremiah Grossman, そしてOWASPの基盤担当の
Matt Tesauroなどご存知の方は少なくないかと思います。
トレーニング、コンファレンスのほうのスピーカーもすべて決まった
模様で、Dropbox, Ping identity, Synopsys, Oracle, HPE Securityなどから
のスピーチがラインアップされています。
キーワード：DevOps, Website Hacking, Automation, Encryption

さて、この実行委員会から日本のアプリケーションセキュリティ関係者向けに
招待コードが届いていますので、よろしければお使いください。
クーポンコード：CYBERSECURITYINJAPAN

こちらは、トレーニング、コンファレンスチケットをオンラインで買うとき
に適用すれば、15%オフになるとのことです。遠慮なくお使いください。

また、スポンサーシップにご関心のある方がおられましたら、このサイトから
お申し出くだされば良いかと思います。
https://locomocosec.com/wp-content/uploads/2018/01/sponsor-doc.pdf

以上、情報共有まで。
