OWASP Japanメーリングリスト参加者各位：
岡田です。

多くのみなさまの意欲とご支援により、本年9月30日土曜に、OWASP
Foundation主催のトレーニングデイを開催する計画です。

場所は、現在、東京工業大学での開催を進めているところです。（この
開催地決定にあたっては、東京大学もオファーをくださり、大変ありが
たく思っています）

さて、このトレーニングデイは、OWASP Foundationの戦略的目標として
OWASPのドキュメントやツールをより有効に活用していただくため、
世界の４拠点が選ばれ、そこで数千人規模のトレーニングを展開するも
ので、日本はそのひとつとなっています。
このイベントはOWASP Foundationが主催し、ローカルチャプタは運営
をします。

そのため、今後以下のものを募集していくことになります。
1. インストラクター
これはCall for trainnerを日本含め全世界に対して募集し、OWASP
Foundationより選出・委任される形をとります。日本での開催のため、
日本人からの応募も期待されています。交通費や謝金などはOWASP
からの支給となります。募集はまだ開始されていません。

2. ボランティア
これは日本の運営組織としてOWASP Japanから募集することになると
思います。500名規模の参加者が期待されていますので、会場のあらゆる
支援が必要になることと思います。また、通訳、記録、ステージ、
ビデオ配信、受付など、日本語・英語での仕事があると思います。
この募集もまだ開始されていませんが、追々お願いしたいと思います。
ボランティアチームは、運営のどの仕事につくとしても無報酬を前提
としています。

3. サテライト会場
このイベントは、オンライン回線を用いて全国のサテライト会場に配信し
それによってより多くの参加者がメリットを得られるようにしたいと考え
ています。そこで、本日より、サテライト会場の検討を全国のチャプター
を中心に検討していただきたいと思っています。
これは各チャプターリーダーを中心に相談し、個別に岡田までご連絡いた
だけますでしょうか。
特に、仙台、名古屋、関西、福岡、沖縄はぜひ実施をお願いします。
また近隣にチャプターがない地域で、大小を問わずサテライトを組みたい
と思われる場合でも、お知らせいただけますとうれしいです。

4. プロモーション活動
プロモーション活動は、OWASP Japanプロモーションチームを中心に
計画的に行なっていきます。メディアの関係の方々、広報のプロフェッショナル
の方でお力を貸していただける方がおられましたら、ぜひお知らせ下さい。

最後に、

このイベントは、地域にアプリケーションセキュリティのコミュニティを
推進していくための拍車をかけることになります。また、OWASPの
オープンソースの成果をより活用していただくための優れた広告機会に
なります。

今年は、OWASP top 10、OWASP ZAP、OWASP SAMM、その他のドキュ
メントにもアップデートの多い年となっています。ぜひがんばって実現
しましょう。

引き続き、宜しくお願い致します。

岡田
