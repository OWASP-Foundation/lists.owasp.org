OWASP Japan MLのみなさんこんにちは：

年度末できっととてもお忙しくしていらっしゃることと思います。

先日、上野さんと、塩尻市のイベントでOWASPについて知っていただく
機会を持ちました。長野県塩尻市は、良いワイナリーで有名なところです。
コミュニティ発のイベントが行政とともに、また企業の関心を集める
動きに進展しており、今後もいろんな企画をなさっていかれるようです。

よろしければ機会を見つけて塩尻に赴いてみてください。歓迎される
と思いますよ。

さて、次回のOWASP Nightですが、3月11日に飯田橋IIJさんのセミナー
ルームをご提供いただいて、開催したいと考えています。

実は、Kubernetesのセキュリティのお話をお聞きできる予定
だったのですが、あいにくご家族の予定で延期になりました。

まあそれでも、昨今のアプリケーション環境にまつわるセキュリティコンセプト
や方法論などのニーズはとどまるところを知りません。

そこで、3/11 OWASP Nightそのものは開催しようということになりました。
つきましては、何名かのスピーカーを募集します。

OWASPミーティングでは、さまざまな技術や方法論のチャレンジやリサー
チ状況の情報共有をお願いしています。
・Web application security
・Mobile security
・Cloud Security
・Serverless security(FaaS)
・Machine Learning fro security
・Blockchain for security use
・Internet of Things
・Vulnerability analysis
・Threat modeling
・Penetration testing
・Application and solution architecture security
・Application level attacks
・Security for DevOps engineers
・Privacy controls

現在リサーチをなさっている方々、チャレンジした結果をお持ちの方、
アウトプットを継続したい方、OWASP Nightで共有しませんか。

パネルディスカッションの形でも、スピーチの形でも、共有しやすい方法
でご提供いただけるといいなあと思っています。次回のみならず、今後の
話ということでも結構ですので、自薦他薦問わず、ぜひお教えください。
ぜひ、このメールの返信にて、わたくしあるいは最寄りのアドバイザリ
ボードまでお声がけください。facebookのOWASPメンターズグループで
投稿していただいても結構です。

共有された情報は、たとえばCheat Sheet化してまとめると広く多くの人
の役にもたちますし、逆に情報も集まりやすくなります。広がりが出ますね！
（作業にあたるため、もくもく会や翻訳ソン、ハッカソンもやれるのではと
　思っています）

では、3月11日、よろしければスケジュールしておいてください。
いつもどおり、ライブ中継もやろうと思っていますので。

どういうトピックで開催できるか決まり次第、公開します。

よろしくお願いします！
