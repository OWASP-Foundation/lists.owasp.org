                            <h3><font face="Helvetica, Arial, sans-serif" size="4" color="#D85D33">Featured OWASP Project</font></h3>
                            <p><font face="Helvetica, Arial, sans-serif" size="3"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/cef5ada534/c38511f33d/2f5060634b" target="_blank">OWASP Reverse Engineering and Code Modification Project</a></font></p>


                            <p><font face="Helvetica, Arial, sans-serif">This project educates security professionals about the risks of reverse engineering and how to ensure that code cannot be reverse engineered or modified. If you are placing sensitive code in an environment in which an attacker can get physical access to that environment (read: mobile, desktops, cloud, particular geographies), you should be concerned with the risks of reverse engineering or unauthorized code modification. This umbrella project will help you understand the risks and how to mitigate them.
                                </font></p><p><font face="Helvetica, Arial, sans-serif"><font face="Helvetica, Arial, sans-serif">For more information, please contact the Project Leader, <a href="mailto:jonathan.carter.org" target="_blank">Jonathan Carter</a></font></font></p>

<font face="Helvetica, Arial, sans-serif">

                            <h3><font face="Helvetica, Arial, sans-serif" size="4" color="#D85D33">New OWASP Projects</font></h3>

                            <p><font face="Helvetica, Arial, sans-serif" size="3">OWASP Pyttacker Project</font></p>
                            <p><font face="Helvetica, Arial, sans-serif">The OWASP Pyttacker Project is a portable Web Server that includes the features needed for every Pentester when creating reports, helping to create PoCs that show a more descriptive way to create awareness to the businesses by demonstrating realistic but in-offensive "attacks" included as part of the tool.
                            </font></p><p><font face="Helvetica, Arial, sans-serif"><font face="Helvetica, Arial, sans-serif">For more information, please contact the Project Leader, <a href="mailto:Mario.Robles@owasp.org" target="_blank">Mario Robles.</a></font></font></p>

<font face="Helvetica, Arial, sans-serif">

                            <p><font face="Helvetica, Arial, sans-serif" size="3">OWASP XSecurity Project</font></p>
                            <p><font face="Helvetica, Arial, sans-serif">The OWASP XSecurity Project aims to provide the best free security tool integrated with the IDE to assist iOS developers to develop secure iOS apps. We now provide a security plugin for Xcode plus clang static analyzer checkers for iOS application development. This plugin aims to reduce the vulnerability made during development by detecting the vulnerability as it is being created.
                            </font></p><p><font face="Helvetica, Arial, sans-serif"><font face="Helvetica, Arial, sans-serif">For more information, please contact the Project Leaders, <a href="mailto:Tokuji.Akamine@owasp.org" target="_blank">Tokuji Akamine.</a> and Ramund Pedraita</font></font></p>

<font face="Helvetica, Arial, sans-serif">

                            <p><font face="Helvetica, Arial, sans-serif" size="3">OWASP Incident Response Project</font></p>
                            <p><font face="Helvetica, Arial, sans-serif">The OWASP Incident Response Project will provide users with a current set of tools and best practices for dealing with a hacked web application.</font></p>


                            <p><font face="Helvetica, Arial, sans-serif">For more information, please contact the Project Leader, <a href="mailto:Tomb@owasp.org" target="_blank">Tom Brennan.</a></font></p>
