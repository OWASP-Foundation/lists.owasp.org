<td> <h3><font face="Helvetica, Arial, sans-serif" size="4" color="#D85D33">Featured OWASP Project</font></h3> <p><font face="Helvetica, Arial, sans-serif" size="3"><a href="http://cts.vresp.com/c/?TheOWASPFoundation/02946169fb/1c1fc130d1/af7b7ee260">OWASP Click Me Project</a></font></p> <p><font face="Helvetica, Arial, sans-serif" size="2">The OWASP Click Me Project aimed at having a simple GUI which helps to create a test page for Clickjacking attacks.This is an attack which targets the clickable content on a website. OWASP Click Me tool will help you to test whether your site is vulnerable to this attack by creating a html page that will try to load your web site from a frame.</font></p> <p><font face="Helvetica, Arial, sans-serif" size="2">For more information, please contact the Project Leader, <a href="mailto:arun.kumar@owasp.org">Arun Kumar</a>.</font></p> <h3><font face="Helvetica, Arial, sans-serif" size="4" color="#D85D33">New OWASP Projects</font></h3> <p><font face="Helvetica, Arial, sans-serif" size="3">OWASP JSEC CVE Details Project</font></p> <p><font face="Helvetica, Arial, sans-serif" size="2">OWASP JSEC CVE Details is is an opensource application developed in Java that uses the api provided by cvedetails.com to receive latest CVE updates. The Common Vulnerabilities and Exposures (CVE) system provides a reference-method for publicly known information-security vulnerabilities and exposures. For more information, please contact the Project Leader, <a href="mailto:dibyendu.coder@gmail.com">Dibyendu Sikdar.</a></font></p> <p><font face="Helvetica, Arial, sans-serif" size="3">OWASP Persian Translation Project</font></p> <p><font face="Helvetica, Arial, sans-serif" size="2">The OWASP Persian Translation Project is a documentation project with the goal of translating fundamental OWASP projects, such as the Top 10 and Testing Guide. Several translations are already available <a href="http://cts.vresp.com/c/?TheOWASPFoundation/02946169fb/1c1fc130d1/e99d7c961
