OWASP Japan ML参加者各位：
（またBccにてOWASPの日本のチャプターリーダーに）

かねてよりお知らせしておりましたが、今年の真夏のWASNightは、
8/21 SuperDeluxeでやります。今しがた、来場者募集を開始しました
のでご案内いたします。

・また、発表者も募集しています。自薦他薦は問いません。
・どうぞ積極的に情報共有にご参画ください。
・オープンマイクもたぶんあります。

全国のチャプターのリーダーのみなさま：

チャプターの紹介や近況をお話になりませんか。短いビデオレター
をお送りください。ご来場になるのももちろん歓迎です。
https://owasp.doorkeeper.jp/events/63852

OWT2017 来場者登録開始 間近

また、現在OWASP FoundationとOWASP Japan Teamは、OWASP World
Tour 2017として、全世界3都市で無償の1日トレーニングを展開する計画を
進めており、来週半ばにCFT(インストラクター募集)と来場者登録を発表する
予定です。

日時は9月30日、東京会場は東京工業大学（大岡山）です。
各地域チャプターなどで、サテライトができるよう、ライブ配信をする予定
ですので、9月30日のサテライト会場の計画を検討していただければと思い
ます。

サテライトが可能で、かつ来場者募集ができそうだとの目処が立ちましたら
OWASP World Tour 2017 実行委員会 owt2017 [at] owaspjapan [dot] org まで
お知らせください。
これに関する質問があれば、それもこちらにお寄せください。

宜しくお願いします。
