OWASP Japanメーリングリスト参加者の皆様：

OWASP World Tour 2017 in Tokyoのオフィシャルページ、トレーナー募集、
そして来場者登録を開始したことをお知らせいたします。

このイベントは9月30日フルデイで、セキュアなソフトウェア開発に関する
トレーニングをOWASP Foundation主催で行うというものです。
2017年は、テルアビブ、ボストン、東京の３箇所で行われます。

official page:
https://www.owasp.org/index.php/2017_OWASP_World_Tour_Tokyo

CFT page:
https://owasp.submittable.com/submit/91359/owasp-world-tour-tokyo-japan

registration:
https://owasp.doorkeeper.jp/events/63959

詳細はウェブサイトをご覧ください。

これから、続々とプロモーションをしていかなければなりません。
皆様の社内、ローカルコミュニティ、関係する皆様にぜひお知らせください。

また当日のボランティア募集も行なっていきます。こちらはもう少しお待ち
ください。

特に日本国内のチャプターリーダーならびにボードの皆様は、当日同時中継の
サテライトを実施なさるようお勧めします。また、支援や相談が必要なら
ぜひ遠慮なくお知らせ下さい。

他の技術関連のコミュニティの皆様で、ローカルにチャプターがあるかどうか
わからないなどのところもあろうかと思います。オープンな募集が可能で、
少なくとも20人以上集まってこの日に一緒に学ぶ、サテライト会場を開設する
ことを希望なさる方はお知らせ下さい。

OWT 2017に関する　取材、ご質問、サテライトの設置に関するお問い合
わせなどはこちらまで：
OWASP World Training Tour 2017 in Tokyo <owt2017 at owaspjapan.org>

どうぞ、宜しくお願いします！

OWASP Japan
岡田

Hi Leaders and OWASPers:

We are happy to announce OWT 2017 in Tokyo as part of the OWASP World
Tour being held in accordance with the OWASP 2017 Strategic Goal. The
CFT and registration site for attendees are now open.

https://www.owasp.org/index.php/OWASP_World_Tour
it reads,
"OWASP is committed to improving the quality of the worlds software
security. We advocate approaching application security as a people,
process, and technology problem because the most effective approaches
to application security include improvements in all of these areas. As
part of this Philosophy, OWASP is offering basic AppSec training for
Developers in hopes of including security throughout the entire SDLC.
The 2018 cities for this training are Boston, Tel Aviv, and Tokyo."

official page:
https://www.owasp.org/index.php/2017_OWASP_World_Tour_Tokyo

CFT page:
https://owasp.submittable.com/submit/91359/owasp-world-tour-tokyo-japan

registration:

event in your local area should let us know.
