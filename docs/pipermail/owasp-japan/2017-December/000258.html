一部見て気になった言い回しです。

https://github.com/okdt/Top10/blob/master/2017/ja/0x06-release-notes.md
モバイルアプリケーションによって使いまくられているAPI

使いまくられている → 多用されている

同じ訳語ニュアンスなら上記が適当かと思います。

流石に丸暗記はしていませんが、こういうニュアンスだったかな？と思い原文をあたってみると、原文は

Old code never expected to be accessible from the Internet is now sitting
behind an API or RESTful web service to be consumed by Single Page
Applications (SPAs) and mobile applications.

be consumed byなので、単に「利用されている」が適当だと思います。
