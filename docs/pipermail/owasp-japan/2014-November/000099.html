明日、神保町EDITORYにて災害被災・支援経験者と、情報支援と情報技術
に関するブリーフィング＋アンカンファレンスを開催します。

http://gensaikaigi.doorkeeper.jp/events/15809
・岩手県総合防災室元室長 小山雄士様
（現在、岩手県オイルターミナル株式会社 代表取締役専務）
・慶應義塾大学看護医療学部准教授 宮川祥子様
・災害IT支援ネットワーク( http://saigaiit.net/ ) 代表 柴田哲史様
・山田町役場総務課課長補佐 船越海平様
のスピーチ（情報共有）をお願いしていますので、情報支援を考える上で
新たな視点も得られるのではないかと思います。

セキュリティそのものの話題ではありません。が、社会的に非常に意義のある、
技術の発揮しどころの分野ですので、もしかしたら関心をお持ちの方がおら
れるかもしれないと思い、告知する次第です。

都合でお越しになれなくなった方がおられるため、若干名空きがありまして、
受付を再オープンしています。

そのため直前の告知で恐縮ですが、もしよろしければお越しください。

将来にわたってこのボランティア活動（プロボノ）のトピックにご関心のある
方は、上記の参加にかかわらずお知らせください。
facebookグループにご招待します。
