OWASP Japan関係者の皆様：
岡田です。

11月8, 9日沖縄県にて開催されるWASForum Hardening Project
のイベントの詳細情報が公開されています。
これはOWASP Japanのリーダー２名やアドバイザリボードも
参画しています。これにご参加になりませんか、というご案内です。

http://wasforum.jp/2014/09/hardening-10-evolutions/

このプロジェクトは脆弱なECサイトを８時間で堅牢化する
という競技として、これまで４回開催されています。

今回は、さらに幅広くウェブの堅牢化にかかわる方々のご
参画が肝要となっています。
もっと汎用化、昇華させ、ドキュメントやチートシート、
またさまざまなツールなどの形で残して行きたいとの意志が
あり、アンカンファレンス、ハッカソン、アイデアソンの形
で開催されます。

OWASP Top 10、レビューガイド、OWASP ZAPなどのドキュ
メントやツールが、インシデントレスポンスにどのように実践
的に役立てられるか、OWASP The Broken Web Project や、
OWASP Cornucopiaなどの教育・ゲームツールの人材育成面で
の適用などもやれると思っています。

ぜひ、ご自身あるいは周辺の方々とご一緒にご参加いただけます
よう検討ください。早期申込だと、審査もほとんどなしで参加が
可能です。

関心のありそうな方に転送していただけますと幸いです。
不明点あればなんなりとお知らせください。

よろしくお願いいたします。

岡田
