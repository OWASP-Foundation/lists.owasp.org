mailing list here is the process:<br><ol><li>User cannot remember password</li><li>User clicks &#39;Forgot Password&#39; link.</li><li>User submits their registered email address (Not their username).</li><li>Applciation responds with screen message.</li>

</ul><li>User receives email and clicks link.</li><li>Application compares unique token in POST data to internally created and stored token.</li><ul><li>If a match occurs then user is presented with password reset function.</li>
</ul><ul><li>If a match does not occur then user is presented with initial logon page.</li></ul><li>Password reset function then goes on to ask one or more previously created &#39;secret&#39; questions.</li></ol><br>Any thoughts?&nbsp; Anyone created a process like this before and have any feedback?<br>
