I also support the idea, but I also like the idea of creating a [turnkey 
linux](http://www.turnkeylinux.org) distribution than one can download 
and deploy to their local environment to start running a CISO program. 
It could and should contain a complete copy of the wiki (and the wiki 
should point to other OWASP supporting resources when more information 
is required. It should allow automated emailing stakeholders the url 
with the questionnaire at set intervals to measure maturity, it should 
keep a history of every answer so you can monitor improvements or worse 
regressions - it should have a dashboard so you can see at a glance the 
current maturity levels.

We have a great Model; **we should make it as easy to use as possible**.

It should be a 'no-brainer' compared to the competition.
OWASP SAMM should be download, and deploy!

This is my 2¢…
