Hey Everyone.

I have been talking about it for a while, but I finally sat down and
wrestled through the process of generating an XML file with all the SAMM
content. Well, almost all the content... I left out the case study at the
end.

Lots of people have been asking for it since it really streamlines
translation work as well as making big opportunities for incorporating the
model into automation tools.

If you have interested and a few minutes to kill, take a look and proofread
the xml layout to make sure I didn't make any errors in tagging the content
(though the content itself is identical to the PDF, so no need to proof-read
that part unless you want to).

If it all looks good in a few days, I'll post it up on the site. I also want
to add a "resources" section to the site, but more on that under separate
cover.
