Nice work, Rohit! I definitely like the auditor "flavor" to the language you
used and I think it resonates well with the few folks I've showed it to.

As I was reading through it, I couldn't help but think that we should add a
something to help an auditor map it back to a SAMM scorecard. Have you
thought about that at all?

It's certainly not a requirement for any audit framework, but since an
organization can strive to achieve a level less than 3, they should be able
to easily tailor the audit framework to check for only the controls that are
appropriate. That being said, sometimes the SAMM activities don't directly
correspond to the traditional notion of a "control" so that might be tricky.
Just curious about your thoughts.

Thanks!

p.

>
