> When I interviewed Gary McGraw for the Cigital Podcast, I asked him a question that I knew the answer to but felt was important. If we look at the new world, we will see that there are now more CIOs who haven’t written a single line of code or even project managed an IT project than that have. So, much of the “success” of the build-it-in crowd, is neither portable across organizations nor likely to be “quantified” since it was done using intuition.
