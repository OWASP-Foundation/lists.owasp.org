With all due respect Christian,

Gary has said: "I hold this truth to be self evident - security is an
emergent property of the entire system, just like
quality."<https://docs.google.com/viewer?a=v&q=cache:4lYjcfmzcYIJ:www.cigital.com/presentations/bsimm10_McGraw.pdf+&hl=en&gl=uk&pid=bl&srcid=ADGEESibvC0Rx1ThseSaJ6diAP8J2u2lACHi2yYdzpe7RKmhlfxpKZL2Bhy0qH2UqA6KujDSM8iVYjVB-6oOBxjUeIRfFFL5LxWVOa2uKV_ONgVcfOTe94xvCJSHqxO5unk2Xlajt1WE&sig=AHIEtbQsIFe5AQm6GyHuXw6pQk0AIerYXw>

I hate to controdict; however, the primary assumption is that security
exists!
Security does not exist! Security is no more an emergent property than a
pot of gold at the end of a rainbow!

Humans for example are fantasticly complex systems with a number of systems
dedicated to their security. However, numerous pathogens still compromise
the system - for example viri like aids; to name a single example.
Survivability aka homeostasis in a hostile envoirnment is the **very best**
we can aim for.

Security is a unicorn, it is completely imaginary.

You should know this more than anybody Christian, since nobody can keep you
out of anything if you put your mind to it. Why? Because their is only one
way to be secure and that is control of the entire chain of trust; which is
impossible, and there are gazillions of was to be insecure. Second, even if
somebody did manage to be secure at some moment in time, the laws of
entropy would decay the trust/securue and it would returen to insecurity.

Again, I maintain *security* is as imaginary as
Equestria<http://www.equestriadaily.com/>.




Dennis
