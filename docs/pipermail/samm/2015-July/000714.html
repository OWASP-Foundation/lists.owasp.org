Hi,

we've recently developed and released a Javascript visualization framework for SAMM which can be found at https://github.com/qudosoft-labs/SAMMCharts.
As the CC licences are not suited for software projects as also stated by creativecommons.org<https://wiki.creativecommons.org/wiki/Frequently_Asked_Questions#Can_I_apply_a_Creative_Commons_license_to_software.3F> we would like to release this software under a different Free Open Source licence and hereby ask for permission for this.
My favorite licence would be Apache 2.0 but we could go for a different licence with stronger copyleft e.g. LGPL.

What are your thoughts about this?
