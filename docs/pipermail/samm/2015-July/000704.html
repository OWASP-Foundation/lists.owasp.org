 Dear all,

Not as fast as we would like it, but we are progressing towards the v1.1
release (for public review).

The main deliverables, status and links are below.

1) SAMM Core Model - status alpha v1.1
Pravir is working on the draft and we hope to have a review version next
week.
I will update the online wiki towards v1.1, once I have the updated
artwork/logos reflecting the name chances.

Some of the questions have been updated after v1.0, Brian has created an
overview of updated questions here:
http://lists.owasp.org/pipermail/samm/2015-June/000695.html
If you have some free cycles, please review the updated questions on typos
and possible improvements to leave less room for interpretation and add
possible clarifications.
You can do this as reply to me or Brian.

2) SAMM Quick Start Guide (QSG) - status beta v1.1
Bart has incorporated all your remarks/comments in a version here:
https://docs.google.com/document/d/1L0EwKANPZrgiZUZyEJTGkhtPeX2e04A6EfVuj3wcYfc/edit
Hugo just created the PDF version here:
https://github.com/OWASP/opensamm/blob/master/v1.1/OpenSAMM_quickstart_02.pdf

3) SAMM How To Guide (HTG)- status v1.1
I have incorporated your remarks/comments and added extra texts to explain
the new v1.1 release components and list acknowledgements to individuals
and sponsors. Latest version here:
https://github.com/OWASP/opensamm/blob/master/v1.1/OpenSAMM_HowTo_05.pdf

4) SAMM Toolbox - status v1.1
Brian has done some final updates. Latest version here:
https://github.com/OWASP/opensamm/blob/master/v1.1/tools/SAMM%20Assessment%20Toolbox%20v1.1.xlsx
The best way to test this is to take it for a spin in a a real life
assessment.

5) SAMM mapped resources - status v1.1
Latest list of mappings here:
https://www.owasp.org/index.php/Category:SAMM-Resources

Any comments, reviews and remarks are welcome!

Once all components of the v1.1 release are finished we will release these
for public review in the coming weeks.

Enjoy the summer holidays,

Kind regards

Seba
