Hi guys,

Now that the model is quite stable and people are starting using it  
for real, we definitely need to put together a tool to support  
assessements, roadmap creation and other SAMM related activities.

Ideally, this tool could aim initially "just" supporting enterprises  
in understanding the model, doing self-assessment, establishing a  
roadmap, creating nice reports and charts.

After this first stage, I'd like to give to the tool the ability to  
assist enterprises in actually implementing the security practices.
An already existing tool that goes in that direction is the Microsoft  
Threat Modeling Tool but it's closed sourced, not extensible and  
definitely not built around SAMM!

I'd like to brainstorm a bit with the SAMM community and of course  
find contributors to help ;)
Anyone here interested?
