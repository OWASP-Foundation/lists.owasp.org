All,

We have published the draft agenda online:
https://www.owasp.org/index.php/OWASP_SAMM_Summit_2015#tab=Project_Day

Topics that will be covered during the day are:

   - Analysis templates / tooling
   - SAMM model improvements
   - What to put in next release / roadmap
   - Finalizing / publishing OpenSAMM v1.1
   - Project 71 follow-up (benchmark repository)
   - opensamm.org website - next steps

We will end each session with defined outputs including action plans,
responsible and timing.

For each of the sessions we will have one topic leader and one note taker
to capture the proceedings, decisions and next steps (with owners & timing)

If you have input for additional topics or ideas to be covered during the
summit: please let us know?

Looking forward to see you all in Dublin next week!
