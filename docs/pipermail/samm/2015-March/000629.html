I am setting up categories to link OWASP pages / resources to SAMM
Practices.

The setup is to create a category for each maturity level in each practice
and use this throughout the OWASP wiki to link the pages/resources directly.

First try out is here:
https://www.owasp.org/index.php/Category:SAMM-CR-1

I have also created a category hierarchy to link them all together:
https://www.owasp.org/index.php/Category:SAMM-Resources

if you think this is useful - feel free to add categories (following the
same structure) and start adding OWASP resources?

any feedback  - ranging from "this might work" to "bad idea" or "how about
doing it like this: ..." - are welcome :-)
