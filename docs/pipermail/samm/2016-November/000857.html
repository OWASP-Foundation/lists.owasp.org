I took Adam's idea and incorporated it into the current draft version of
the v1.5 toolbox spreadsheet.

https://github.com/OWASP/samm/tree/master/v1.5/Tools

There is a new "Roadmap" Tab.
It contains a linked set of questions/answers from the Interview tab.

​

There are four "phases" available for indicating when work is projected to
be completed to the point that an answer would change.
There are not "dates", as each organization has different planning cycle
durations.
If the answer of a phase is better than a previous phase it is highlighted
as green.
If the answer of a phase is worse than a previous phase it is highlighted
as red.
This should help identify when answers are changing for better or worse.

​​

The spreadsheet will also calculate the improvements in the score for each
phase.
The results are also feed into the "Roadmap Chart" tab for charts/graphs.

​

​
I need to write up some notes and do a walkthough to make sure everything
is working as expected.
Feedback is welcome!
