Proposing an addition to the excellent Toolbox spreadsheet.

The intent of the proposed change is to allow the spreadsheet to be
used as a tool for forward planning.

Change: Add additional rows from J17-N17

Like: Target  | Q + 1  |  Q + 2  |  Q + 3  |  Q + 4 (perhaps up to Q
+8) where these boxes represent the current quarter plus 1,2,3
forward.

After completing the current state audit, the user determines the
ultimate target state for each category, filling the appropriate box.

For example, J18 would contain the target state for "Is there a
software security assurance program in place?"

The Guidance boxes can be filled in to note where work should begin
and where it will be completed up to the target completion level.  If
some target levels are + levels like 2+, you could record which of the
additional guidances will be followed and when, as well.

I've utilized the spreadsheet this way several times, it was much
better than figuring it out externally to the document. We could pull
the numbers over into the roadmap chart, too, which is fancier than
I've gotten with it. .
