Title
Mapping SAMM 1.0 to NIST SP 800-53 Rev 3

-------------------

Description
Develop a companion document that maps SAMM 1.0 security practices,
objectives and activities to NIST
SP 800-53 Rev 3 (Recommended Security Controls for Federal Information
Systems and Organizations).  The latter is in final draft and is due
for finalisation on 1 July 2009.  Will need to review how other
organisations have mapped standards (e.g. ISACA/COBIT) to determine
how to express partial relationships.

SAMM 1.0
http://www.opensamm.org/downloads/SAMM-1.0.pdf

NIST SP 800-53 Rev 3 Final Public Draft
http://csrc.nist.gov/publications/drafts/800-53/800-53-rev3-FPD-clean.pdf

COBIT Mapping Documents
http://www.isaca.org/Template.cfm?Section=COBIT_Mapping1&Template=/ContentManagement/ContentDisplay.cfm&ContentID=30523
