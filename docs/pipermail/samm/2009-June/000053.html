Hi Folks,

I thougt it might help some people get started with SAMM in some
organizations if they had an MS Project file with all the SAMM tasks,
rolled up to the various sections, so I made one. I added a calculated
column that shows a maturity score for each individual task and each
summary task based on the maturity level of a task and the %complete of
that task.  Sometimes it's easier to get efforts started, especially in
large organizations, if you present the effort in formats that project
management people are used to. The .mpp project file and notes are in
the OWASP file upload archive at
http://www.owasp.org/index.php/Special:ImageList; SAMMProject.zip.  Let
me know if you have any questions, comments, suggestions etc.
