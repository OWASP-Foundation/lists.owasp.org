Is it possible for us to create an accompanying primer such that we
provide clarity on the following?

SAMM is not about code review, Top X lists, etc

SAMM is a strategic tool. It provides no guidance on what to address
first, but tactically speaking, we should provide some guidance to help
folks figure it out?

ISACA and the auditor crowd would like to explore notions of
"compensating controls" so can we add a section that includes secondary
suggestions?

Some guidance on how to deploy SAMM depending on what type of
organization you are? For example, us Financial Services heavily
regulated types have a different take than say a regional grocery store
chain.

Metrics, to some we feel to qual, and won't satisfy the quant types.
Maybe a survey or two is in order?
