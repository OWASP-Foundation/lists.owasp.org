*_OWASP VaultDB Project
<http://www.cvent.com/events/mproc.aspx?m=5468e318-5e41-4ba0-8f34-0b5fe485e08f&u=https%3a%2f%2fwww.owasp.org%2findex.php%2fOWASP_VaultDB_Project&l=OWASP+VaultDB+Project>_*
*
Project Leader: Â Maxime Labelle

*
VaultDB is a secure NoSQL database management system (DBMS) for modern
applications. Â It supports multi-recipient encryption, table-level
encryption, group encryption and comes loaded with a strong cryptosystem.

VaultDB adds automatic transparent encryption to your application's data
at the table/document level. Â Instead of using it's own internal
storage engine, VaultDB stores the encrypted data inside your preferred
DBMS for storage.Â 

*OWASP WS-Amplification DoS Project
<http://www.cvent.com/events/mproc.aspx?m=5468e318-5e41-4ba0-8f34-0b5fe485e08f&u=https%3a%2f%2fwww.owasp.org%2findex.php%2fOWASP_WS_Amplification_DoS_Project&l=OWASP+WS-Amplification+DoS+Project>*
*
Project Leader: Â Thomas Vissers <mailto:Thomas.Vissers en owasp.org>
*
This project aims to explore the threat of an Amplification DoS attack
that utilizes web services. Â Currently, DNS servers are widely misused
to amplify DoS traffic. Â This is called a DNS Amplification or
Reflective attack. Â It appears that SOAP web services that implement
WS-Addressing might be vulnerable to similar abuse, as stated in this
paper. Â The aim of the project is to develop tools to test this
vulnerability and determine the threat magnitude on a global scale.Â

<http://www.cvent.com/events/mproc.aspx?m=5468e318-5e41-4ba0-8f34-0b5fe485e08f&u=https%3a%2f%2fwww.owasp.org%2findex.php%2fOWASP_Mutillidae_2_Project&l=OWASP+Mutillidae+2+Project>*
*
Project Leader: Â Jeremy Druin <mailto:Jeremy.Druin en owasp.org>
*
NOWASP (Mutillidae) is a free, open source, deliberately vulnerable
web-application providing a target for web-security enthusiast. Â NOWASP
(Mutillidae) can be installed on Linux and Windows using LAMP, WAMP, and
XAMMP for users who do not want to administrate a web server. Â
