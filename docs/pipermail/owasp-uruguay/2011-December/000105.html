> >                                 
> >                                       * First, review the How to
> >                                         Host a conference
> >                                         page (http://www.owasp.org/index.php/How_to_Host_a_Conference).  This page hosts a variety of information about planning an OWASP event, as well as various requirements and policies that have been put in place by the foundation to govern Global AppSec Conferences.

> >                                         planning team to acquire
> >                                         some basic information

> >                                         event.

> >                                         plan using the OWASP Budget
> >                                         Planning Tool
