Hola,

Yo usé Fortify varios años, es una buena opción y sin dudas líder de
mercado comercial.. El año pasado también evalué Checkmarx y dió buenos
resultados tanto para JEE como para ambientes mobile.
Ambos tienen demos/trials en la nube que pueden servirte para pruebas y
evaluación.

Dentro de las herramientas opensource a incorporar, les puedo recomendar:

   - YASCA
   - VisualCodeGrepper
   - LAPSE

Igual como recomendación, antes de implementar o ejecutar herramientas de
revisión de código es muy recomendable definir como será la metodología,
trabajar en los alcances de las revisiones y tener bien definidos los
puntos de función críticos y cuál es el modelo de madurez y/o hoja de ruta
de seguridad a seguir.

Espero sea de utilidad.
Saludos,,

Mateo
