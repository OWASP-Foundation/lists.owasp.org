Listas,

    Para la organizacion del OWASP Camp estaba pensando en utilizar
algun software que ayude con la tarea de coordinar los esfuerzos.
Saben si OWASP provee algún software de este estilo? Sino estuve
viendo que pyconference [0][1] está buenisimo!

    Mi objetivo es que los organizadores podamos coordinar:
        * TODO list
        * Fechas del evento, fechas limite que se acerquen
        * Temas propuestos
        * Mostrar a la comunidad informacion sobre el evento
        * Tener una plataforma abierta, para que todos puedan ver que
se esta haciendo en la organizacion y sumarse
