Title: OWASP Webcast en Español: Cómo ganar siempre al Poker usando OWASP  
ZAP
Estimados,

A continuación les envio un nuevo evento en linea de OWASP en Español a  
realizarse el proximo 6 de Octubre a las 13 horas GMT-3 (Argentina time).

Descripción: WebSocket es parte de la iniciativa de HTML5 que define una  
API que permite a las páginas web, la comunicación full-duplex y  
bidereccional a través de un solo conector TCP/HTTP para proporcionar una  
enorme la reducción del tráfico de red. Se analizará este nuevo protocolo y  
la foma de analizar el tráfico a través del proxy web OWASP ZAP.

*Charla: *El Proyecto GoLISMERO: Como auditar aplicativos web de manera
facil

GoLISMERO es un spider web capaz de detectar vulnerabilidades y formatear
los resultados de forma muy útil cuando se afronta una auditoría web. Cada
vez que nos enfrentamos a una nueva URL, ¿no sería genial poder disponer de
forma sencilla y rápida de todos los enlaces, formularios con sus
parámetros, detectar posibles URL vulnerables y que además de que se
presentasen de manera que nos permita hacernos una idea de la todos los
puntos de entrada donde podríamos lanzar ataques? GoLISMERO nos permite
hacer todo esto.

*Ponente: *Daniel Garcia A.K.A. cr0hn (twitter @ggdaniel)

Auditor de seguridad senior, pentester e investigador. Programador Python y
sysadmin de FreeBSD por afición. Amante del software y el conocimiento
libre. Tiene publicadas numerosas herramientas de hacking y seguridad en
general. Ponente en diversos foros y conferencias. Creador y organizador
del evento de seguridad Navaja Negra. Ha trabajado para una gran cantidad
de empresas: sector de telecomunicaciones, entidades financieras, abogados,
aseguradoras y otras empresas del IBEX 35 Español.
