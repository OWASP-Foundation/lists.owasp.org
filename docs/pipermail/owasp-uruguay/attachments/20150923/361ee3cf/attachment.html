          Topics<br>
    <br>
    <br>
    OWASP AppSec conferences are true security conferences, with
    expected<br>
    talks and presentations all around (web) application security.<br>
    Non-technical talks (see below) are welcome too. Please refrain from<br>
    submitting marketing talks or having sales pitches in your talk.<br>
    <br>
    We are interested in all topics related to Web Application Security
    and<br>
    OWASP, in particular:<br>
    <br>
      * Secure development: frameworks, best practices, secure coding,<br>
    methods, processes, SDLC<br>
      * Vulnerability analysis: code review, pentest, static analysis<br>
      * Threat modelling<br>
      * Mobile security<br>
      * Cloud security<br>
      * Browser security<br>
      * HTML5 security<br>
      * OWASP tools or projects in practice<br>
      * New technologies, paradigms, tools<br>
      * Privacy in web apps, Web services (REST, XML) and data storage<br>
      * Operations and software security<br>
      * Management topics in Application Security: Business Risks,<br>
    Outsourcing/Offshoring, Awareness Programs, Project Management,
    Managing<br>
    SDLC<br>
