      absolutamente vulnerable!<br>
      <br>

      para que no sea práctico hacer la calesita. Si hay que
      implementarlo para 2 puede que sea fácil agregar un campo
      "passanterior" en la misma tabla de usuarios, pero para tener 4,
      ya hay que hacer algo "bien" (otra tabla independiente, loops
      recorriendola, etc), y quien lo hace para 4, lo hace para 100.<br>
