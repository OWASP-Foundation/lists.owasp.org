Hi,
in our template we have defined it:
http://www.owasp.org/index.php/Template_Paragraph_Testing_AoC

The template is a guideline: maybe in some paragraphs not all the
items are applicable. It depends on the context.
In some case "Results expected" means that the tester should be expect
a particular information as answer from the application/server. For
example:
http://www.owasp.org/index.php/Analisys_of_error_code_AoC#Black_Box_testing_and_example
In other cases we explain it by describing an example with a real case
or just by words. For Example:
http://www.owasp.org/index.php/Cookie_and_Session_Token_Manipulation_AoC#Black_Box_Testing_and_Examples

Thanks,
Mat
