Hello List,
 
I wanted to update everyone with the status of the LiveCD and also start a
thread to document everyone's ideas on what should be on the CD. I started
this LiveCD around June this year with the following goals;
 

*	Portable LiveCD with all tools needed to perform a app
audit/pen-test
*	A single media image containing all the OWASP tools and documents
*	Including tools and research relating to;
*	        RFID
*	        VOIP
*	        Wireless
*	        BlueTooth
*	        Network/Infrastructure

As you know it was selected for AOC, so it's now at the 50-60% mark. That
is.. it's stable and it has a handful of tools/documents.
 
I have decided to follow the OWASP website structure for tools and documents
on the CD. This is as follows;
 
Releases
    Tools
    Docs
Alpha
    Tools
    Docs
Beta
    Tools 
    Docs
