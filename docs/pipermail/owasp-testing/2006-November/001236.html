Hi how are we all on this fine winter morning?
I added the Fuzz vectors some time ago to assist the Webscarab fuzzer.
One can cut and paste the vectors defined on that page and use in the
webscarab fuzzer.

I personally dont see any point in defining each vector, and saying what it
is or does. There are other resources in the guide to define responses such
as error codes etc.

I would prefer if the fuzz vector section is touched that a new page be
added so one can still "cut and Paste" the list into a test file to use with
webscarab et al.
