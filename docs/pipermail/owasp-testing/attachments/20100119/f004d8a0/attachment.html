<p class=MsoNormal style='margin-bottom:12.0pt'>Not to take away from the good
work, just an FYI. It appears that SSLScan is getting overlooked. <br>
<br>
SSLScan: <a href="http://sourceforge.net/projects/sslscan/">http://sourceforge.net/projects/sslscan/</a><br>
SSLScan Parser: <a href="http://search.cpan.org/~jabra/Sslscan-Parser-0.02/">http://search.cpan.org/~jabra/Sslscan-Parser-0.02/</a><br>

&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
_<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; ___ ___|

<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
Version 1.7.1<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; <a

&nbsp;&nbsp;&nbsp;&nbsp; Copyright (C) 2007-2008 Ian Ventura-Whiting<br>
<br>
SSLScan is a fast SSL port scanner. SSLScan connects to SSL<br>
ports and determines what&nbsp; ciphers are supported, which are<br>
the servers&nbsp; prefered&nbsp; ciphers,&nbsp; which&nbsp; SSL&nbsp;
protocols&nbsp; are<br>
supported&nbsp; and&nbsp;&nbsp; returns&nbsp; the&nbsp;&nbsp; SSL&nbsp;&nbsp;
certificate.&nbsp;&nbsp; Client<br>
certificates /&nbsp; private key can be configured and output is<br>
to text / XML.<br>
<br>
Command:<br>
&nbsp; sslscan [Options] [host:port | host]<br>
<br>
Options:<br>
&nbsp; --targets=&lt;file&gt;&nbsp;&nbsp;&nbsp;&nbsp; A file containing a list
of hosts to<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
check.&nbsp; Hosts can&nbsp; be supplied&nbsp; with<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
ports (i.e. host:port).<br>
&nbsp; --no-failed&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; List
only accepted ciphers&nbsp; (default<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
is to listing all ciphers).<br>
&nbsp;
--ssl2&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
Only check SSLv2 ciphers.<br>
&nbsp;
--ssl3&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
Only check SSLv3 ciphers.<br>
&nbsp;
--tls1&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
Only check TLSv1 ciphers.<br>
&nbsp; --pk=&lt;file&gt;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
A file containing the private key or<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
a PKCS#12&nbsp; file containing a private<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
key/certificate pair (as produced by<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
MSIE and Netscape).<br>
&nbsp; --pkpass=&lt;password&gt;&nbsp; The password for the private&nbsp; key
or<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
PKCS#12 file.<br>
&nbsp; --certs=&lt;file&gt;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; A file
containing PEM/ASN1 formatted<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
client certificates.<br>
&nbsp; --starttls&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
If a STARTTLS is required to kick an<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
SMTP service into action.<br>
&nbsp; --xml=&lt;file&gt;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
Output results to an XML file.<br>
&nbsp;
--version&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
Display the program version.<br>
&nbsp;
--help&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
Display the&nbsp; help text&nbsp; you are&nbsp; now<br>
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
reading.<br>
Example:<br>
&nbsp; sslscan 127.0.0.1<br>

<p class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto'><span
lang=EN-US>Hello,</span><o:p></o:p></p>

<p class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto'><span
lang=EN-US>&nbsp;</span><o:p></o:p></p>

<p class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto'><span
lang=EN-US>Last weekend I hacked together a piece of software that checks what
SSL protocols and ciphers a web server supports, which is available for
download at </span><a href="http://code.google.com/p/sslaudit/" target="_blank"><span
lang=EN-US>http://code.google.com&nbsp;&nbsp;&nbsp;&nbsp; /p/sslaudit/</span></a><span
lang=EN-US>.</span><o:p></o:p></p>

<p class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto'><span
lang=EN-US>&nbsp;</span><o:p></o:p></p>

<p class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto'><span
lang=EN-US>From the above mentioned website:</span><o:p></o:p></p>

<p class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto'><span
lang=EN-US>&nbsp;</span><o:p></o:p></p>

<p class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto'><span
lang=EN-US>--8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;-</span><o:p></o:p></p>

<p class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto'><span
lang=EN-US style='font-size:10.0pt;color:black'>SSLAudit is a tool that
verifies SSL certificate and supported protocols/ciphers of a SSL-enabled
webserver. The result is graded according to&nbsp;</span><span
style='font-size:10.0pt;color:black'><a
href="https://www.ssllabs.com/projects/rating-guide/index.html" target="_blank"><span
lang=EN-US>SSLLabs SSL Server Rating Guide</span></a></span><span lang=EN-US
style='font-size:10.0pt;color:black'>.</span><o:p></o:p></p>

<p><span lang=EN-US style='font-size:10.0pt;color:black'>The tool is similar in
function to&nbsp;</span><span style='font-size:10.0pt;color:black'><a
href="http://www.foundstone.com/us/resources/proddesc/ssldigger.htm"
target="_blank"><span lang=EN-US>SSLDigger from Foundstone</span></a></span><span
lang=EN-US style='font-size:10.0pt;color:black'>&nbsp;and&nbsp;</span><span
style='font-size:10.0pt;color:black'><a
href="http://freeworld.thc.org/root/tools/" target="_blank"><span lang=EN-US>THCSSLCheck
from The Hacker Choice</span></a></span><span lang=EN-US style='font-size:10.0pt;
color:black'>&nbsp;but is different that it is open source and is easily
modified to support new protocols and ciphers as they become available and the
result is graded.</span><o:p></o:p></p>

<p style='max-width: 65em'><span lang=EN-US style='font-size:10.0pt;color:black'>This
project is sponsored by&nbsp;</span><span style='font-size:10.0pt;color:black'><a
href="http://www.omegapoint.se" target="_blank"><span lang=EN-US>Omegapoint AB</span></a></span><span
lang=EN-US style='font-size:10.0pt;color:black'>&nbsp;and was created to assist
security assessments done according to&nbsp;</span><span style='font-size:10.0pt;
color:black'><a
href="http://www.owasp.org/index.php/Category:OWASP_Testing_Project"
target="_blank"><span lang=EN-US>OWASP Testing Guide</span></a></span><span
lang=EN-US style='font-size:10.0pt;color:black'>.</span><o:p></o:p></p>

<p><span lang=EN-US style='font-size:11.0pt'>--8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;-</span><o:p></o:p></p>

<p class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto'><span
lang=EN-US>It currently performs the following tests:</span><o:p></o:p></p>

<p><span lang=EN-US style='font-family:Symbol'>·</span><span lang=EN-US
style='font-size:7.0pt'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span><span
lang=EN-US>SSL Protocol support detection</span><o:p></o:p></p>

<p><span lang=EN-US style='font-family:Symbol'>·</span><span lang=EN-US
style='font-size:7.0pt'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span><span
lang=EN-US>SSL Cipher support detection</span><o:p></o:p></p>

<p><span lang=EN-US style='font-family:Symbol'>·</span><span lang=EN-US
style='font-size:7.0pt'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span><span
lang=EN-US>Public cert PEM extraction</span><o:p></o:p></p>

<p><span lang=EN-US style='font-family:Symbol'>·</span><span lang=EN-US
style='font-size:7.0pt'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span><span
lang=EN-US>Certificate timeframe validation (and warns if it is 30 days or less
until the certificate expires)</span><o:p></o:p></p>

<p><span lang=EN-US style='font-family:Symbol'>·</span><span lang=EN-US
style='font-size:7.0pt'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp; </span><span
lang=EN-US>Grading of result according to SSLLabs SSL Server Rating Guide</span><o:p></o:p></p>

<p class=MsoNormal style='mso-margin-top-alt:auto;mso-margin-bottom-alt:auto'><span
lang=EN-US>&nbsp;</span><o:p></o:p></p>
