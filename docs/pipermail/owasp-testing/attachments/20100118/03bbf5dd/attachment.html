software that checks what SSL protocols and ciphers a web server supports,

<p class="MsoNormal"><span lang="EN-US">--8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;-</span></p>

<p class="MsoNormal"><span style="font-size: 10pt; color: black;" lang="EN-US">SSLAudit is a tool that verifies SSL certificate and supported
protocols/ciphers of a SSL-enabled webserver. The result is graded according to<span> </span></span><span style="font-size: 10pt; color: black;"><a href="https://www.ssllabs.com/projects/rating-guide/index.html" target="_blank"><span lang="EN-US">SSLLabs SSL Server Rating Guide</span></a></span><span style="font-size: 10pt; color: black;" lang="EN-US">.</span><span lang="EN-US"></span></p>





<p><span style="font-size: 10pt; color: black;" lang="EN-US">The tool is similar in function to</span><span><span style="font-size: 10pt; color: black;" lang="EN-US"> </span></span><span style="font-size: 10pt; color: black;"><a href="http://www.foundstone.com/us/resources/proddesc/ssldigger.htm" target="_blank"><span lang="EN-US">SSLDigger from Foundstone</span></a></span><span><span style="font-size: 10pt; color: black;" lang="EN-US"> </span></span><span style="font-size: 10pt; color: black;" lang="EN-US">and</span><span><span style="font-size: 10pt; color: black;" lang="EN-US"> </span></span><span style="font-size: 10pt; color: black;"><a href="http://freeworld.thc.org/root/tools/" target="_blank"><span lang="EN-US">THCSSLCheck from
The Hacker Choice</span></a></span><span><span style="font-size: 10pt; color: black;" lang="EN-US"> </span></span><span style="font-size: 10pt; color: black;" lang="EN-US">but

protocols and ciphers as they become available and the result is graded.</span></p>
