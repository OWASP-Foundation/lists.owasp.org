<p class="MsoNormal"><span lang="EN-US">Last weekend I hacked together a piece of
software that checks what SSL protocols and ciphers a web server supports,
which is available for download at </span><a href="http://code.google.com/p/sslaudit/" target="_blank"><span lang="EN-US">http://code.google.com/p/sslaudit/</span></a><span lang="EN-US">.</span></p>

<p class="MsoNormal"><span lang="EN-US"> </span></p>

<p class="MsoNormal"><span lang="EN-US">From the above mentioned website:</span></p>

<p class="MsoNormal"><span lang="EN-US"> </span></p>

<p class="MsoNormal"><span lang="EN-US">--8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;---8&lt;-</span></p>

<p class="MsoNormal"><span lang="EN-US" style="font-size:10.0pt;color:black">SSLAudit is a tool that verifies SSL certificate and supported
protocols/ciphers of a SSL-enabled webserver. The result is graded according to<span> </span></span><span style="font-size:10.0pt;color:black"><a href="https://www.ssllabs.com/projects/rating-guide/index.html" target="_blank"><span lang="EN-US">SSLLabs SSL Server Rating Guide</span></a></span><span lang="EN-US" style="font-size:10.0pt;color:black">.</span><span lang="EN-US"></span></p>




<p><span lang="EN-US" style="font-size:10.0pt;color:black">The tool is similar in function to</span><span><span lang="EN-US" style="font-size:10.0pt;color:black"> </span></span><span style="font-size:10.0pt;color:black"><a href="http://www.foundstone.com/us/resources/proddesc/ssldigger.htm" target="_blank"><span lang="EN-US">SSLDigger from Foundstone</span></a></span><span><span lang="EN-US" style="font-size:10.0pt;color:black"> </span></span><span lang="EN-US" style="font-size:10.0pt;color:black">and</span><span><span lang="EN-US" style="font-size:10.0pt;color:black"> </span></span><span style="font-size:10.0pt;color:black"><a href="http://freeworld.thc.org/root/tools/" target="_blank"><span lang="EN-US">THCSSLCheck from
The Hacker Choice</span></a></span><span><span lang="EN-US" style="font-size:10.0pt;color:black"> </span></span><span lang="EN-US" style="font-size:10.0pt;color:black">but
is different that it is open source and is easily modified to support new
protocols and ciphers as they become available and the result is graded.</span></p>
