<div>Alessio has put a new version of the OWASP code review tool, Code crawler, available for download.</div>
<div>It has a totally new look and feel and also runs SQL Server express for the definitions.</div>
<div>The source is available also. Suggestions on new&nbsp;functionality are welcome.</div>
<div>&nbsp;</div>
<div>Code crawler is a complementary tool associated with definitions defined in the code review guide. It works best using both the guide and tool together.</div>
<div>Remember this is a beta tool, stable, but much functionality to implement.</div>
<div>It can be downloaded from the OWASP code review project site:</div>
