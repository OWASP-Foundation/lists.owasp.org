<div clear="all"><strong>OWASP will hold its annual Ireland Application Security conference in Trinity College, Dublin, September 2010. </strong></div>
<div><strong>The &quot;Who&#39;s Who&quot; in application security for Ireland.</strong></div>
<div>  </div>
<div>The Conference will consist of one day of training sessions, followed by a one-day conference with 2 tracks. </div>
<div>  </div>
<div><strong>In 2009, we attracted a large number of local and international speakers and attendees. We hope to achieve the same again in 2010. Delegates from across Europe, Middle East and the USA attended in 2009 event.</strong></div>

<div>  </div>
<div>We are seeking presentations on any of the following topics: </div>
<div>  </div>
<ul>
<li>   Web Services and Application Security </li>
<li>   Common Application related Threats and Risks</li>
<li>   Business Risks with Application Security </li>
<li>   Vulnerability Research in Application Security </li>
<li>   Web Application Penetration Testing </li>
<li>   OWASP Tools and Projects </li>
<li>   Secure Coding Practices </li>
<li>   Technology specific presentations on security such as AJAX, XML, etc. </li>
<li>   Anything else relating to OWASP and Application Security. </li></ul>
<div>  </div>
<div>Other suggestions welcome.</div>
<div>  </div>
<div><strong>The call for papers/presentations is out.!!</strong></div>
<div>  </div>

<div>  </div>
