The new OWASP Content Management system is getting rolled out this week. It
integrates Magnolia (www.magnolia.info) into oPortal so that users can
updates and publish content directly from their browsers. It rocks and rocks
hard !

Every project leader will be responsible for maintaining an up to date and
accurate project page
Every Chapter chair will be responsible for maintaining an accurate and up
to date Chapter home page
The OWASP Leaders collectively will take care of the rest of the site
Mads you can create your resources section with pages for links, whitepapers
and tools using this

It is everyone's responsibility to check spelling and grammar before
publishing. 

We will be creating templates for as much possible so that you just need to
complete sections of the template and hit publish. 

In order to use the system you will need an account on the CMS. To obtain an
account you must send your name, the reason you need an account (I run the
Ireland Chapter) and a STATIC IP address. If you do not have an static IP
address you can not use the system at present and will need to hand over
your responsibilities to someone else. 

To obtain an account you should email larry.casey at aspectsecurity.com (Do No
Mail or CC Me!)

We will setup an IM on Friday for basic Magnolia training. Watch for the
invite.

If you do not request an account by Friday we will auto-assign your sections
to someone else. If you do not create your pages by next Monday we will also
be forced to do so. This should take no one more that 30 mins. Sorry for the
harshness but we need to do some spring cleaning ;-)
