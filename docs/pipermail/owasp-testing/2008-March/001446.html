You are right Brad, "Infrastructure Configuration Management Testing"
is not included in the report chart because the following:
- Testing Web Application Fingerprint
- Application Discovery
- Spidering and Googling
- Analysis of Error Codes
- SSL/TLS Testing
- DB Listener Testing
- Testing for File Extensions Handling
- Old, backup and unreferenced files
are a subset of "Information Gathering" and "Infrastructure
Configuration Management Testing".

Yes, we are planning to rewrite the Testing Guide, so we can explain
better this issue in the new one.
So keep in touch for the new project.
