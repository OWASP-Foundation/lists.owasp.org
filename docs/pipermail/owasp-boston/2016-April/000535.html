Hi Folks,
We are planning to provide 2 days of web application penetration testing
Monday and Tuesday July 18th, 19th from 8:00AM ET to 5PM ET, in Waltham MA
for a very low price by well known pentest trainer Tim Tomes. We're looking
for sponsors to help us lower the price even more and help defray the cost.
Sponsorship includes some free employee admission, logo placement on the
event and chapter web pages, logo placement and short presentation at the
training venue. For the complete sponsorship document please email
boston at owasp.org.

The course title is Practical Web Application Penetration Testing.
This course provides customized training on the latest open source tools
and manual techniques for performing end-to-end web application penetration
testing engagements. A major goal of this course is teaching students the
glue that brings the tools and techniques together to successfully perform
a web application penetration test from beginning to end, an oversight in
most web application penetration testing courses.
The majority of the course will be spent performing an instructor led,
hands-on web application penetration test.  This experience will help
students gain the confidence and knowledge necessary to perform web
application penetration tests as an application security professional.

The instructor is Tim Tomes, who is the Director of Application security at
nVisium, and has taught for SANS and Blackhat.

Thanks, Jim
