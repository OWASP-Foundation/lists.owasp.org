Topic: Identity Federation : Prospects and Challenges - 

Speaker: Prateek Mishra, Ph.D. - Director, Security Standards and Strategy,
Oracle Corp
Identity federation is a collection of technologies that reduces the number
of
passwords that users must posses and administrators must maintain. Instead
of maintaining one password per-site, identity federation allows users to
"re-use" their logins from one site to securely access applications at a
second site. 
This talk will provide an overview of this area. It will discuss standards
such as SAML, Liberty Alliance ID-FF and ID-WSF, WS-Federation and
Shibboleth. It will also survey the progress made in actual deployments in
this space, as well as some of the unresolved challenges.
 
 
Topic: Application Security Gateways
 
Speaker: Ryan Shorter - System Engineer - Netcontinuum
What's involved in stopping application level attacks such as cross site
scripting and sql injection by inspecting the TCP/IP packets as they head
toward your web applications?
