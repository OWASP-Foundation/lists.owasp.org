Hi Folks,

In order to try to keep track of people we have contacted to let them
know about attending BASC 2010, if you have or intend to contact anybody
in academia, please check with our Academic Outreach coordinator,
Himanshu Vakil, at info at ebusinessmantra.com. We'd like to make sure
people are only contacted once and what their response was, if any. So
please send Himanshu the name, email and institution/college/department
of who you contacted and what their reply was, if any.  

 

Feel free to use the text below for any invites - 

 

The Boston chapter of OWASP (Open Web Application Security Project) is
holding a free, one day informal conference on web application security
on Saturday Nov. 20, 2010 at Microsoft New England Research and
Development Center, Cambridge, MA. The conference is intended for both
people new to web application security and those experienced in web
application security. You can find out more at
http://www.owasp.org/index.php/Boston_Application_Security_Conference.
You can register at http://basc2010.eventbrite.com/  If you would like
to submit a paper, see the instructions at
http://www.owasp.org/index.php/Boston_Application_Security_Conference_-_
Call_For_Papers.

 

 

And registration is now open at http://basc2010.eventbrite.com/  . We
need people to register so we can judge the food order as well as make
sure we do not exceed the allowable room capacity.
