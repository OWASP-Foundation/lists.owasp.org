In order for OWASP Boston to better serve the metrowest Boston application
security community, we'll be having some meetings at Constant Contact,
Reservoir Place, 1601 Trapelo Road   Waltham, MA 02451

The first meeting will be Thursday May 21, 7 to 8:30 pm, in the Constant
Contact Innovation Center. This will be an informal meet and greet meeting,
no formal presentations, just talking about application security topics,
what OWASP has to offer, and how future meetings can be useful.  Constant
Contact will provide beer, snacks and soda. If you have any topic
suggestions let us/me know and we'll try to schedule something in the
future or talk about them on Thursday.

One topic I'll use to start conversations is what tools are people using
and not using - source code scan, web application firewall, web application
vulnerability scanner, threat intelligence service etc.

Directions -
Right off of Rt 128 in Waltham Ma, you should exit onto Trapelo Road and
take a right into Reservoir place.  Continue around to the front of the
building.

Park in the front of the building and enter in the main building lobby,
continue down the hallway and you will see the innovation center, enter in
the large glass doors
