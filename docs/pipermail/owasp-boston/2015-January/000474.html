7 pm     *Incident Response and Web Application Security - Stories from the
Trenches*



Mandiant has conducted numerous incident response engagements throughout
its history, including a record number of 200 incidents in 2014. Mandiant
consultants Fred House and Joe Ceirante will discuss case studies and
trends Mandiant has observed in relation to web application security.


Topics will include the types of threat actors that target web
applications, the techniques they use to compromise web applications, and
the activities they perform once inside the network. Techniques for
detecting and mitigating web compromises will also be reviewed. All content
will be derived from the real-world compromises that Mandiant has
investigated.
