Hi Folks,
One of the things we'll be doing going forward is to try to have 3 or 4
meetings a year devoted to training on OWASP security topics, not just a
technology presentation.

So we're starting this schedule by having the Feb. 5 meeting as a training
session on

1. SQL Injection - intro (what is it, what can you do with it, how to
prevent); intermediate - real world example of attacks (how does malware
get onto 'drive-by' web sites that serve malware?); advanced - how do web
application vulnerabity scanners check for SQL Injection and false
positives.

2. Cross Site Request Forgery

We may have some more training topics for this meeting.

Would anyone like to do a short training session on some topic?

We'll also have the usual news review, including the Target breach,
announcements etc.
