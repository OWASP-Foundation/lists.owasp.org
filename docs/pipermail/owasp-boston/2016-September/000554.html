Boston Application Security Conference - BASC 2016
Saturday, October 1, 2016 from 8:30 AM to 6:30 PM (EDT)
Burlington, MA
https://www.owasp.org/index.php/2016_BASC_Homepage


The OWASP Boston Chapter's Boston Application Security Conference (BASC) is
a free, one day, informal conference, aimed at increasing awareness and
knowledge of application security in the greater Boston area.

While many of the presentations will cover state-of-the-art application
security concepts, BASC is intended to appeal to a wide-array of attendees.
Application security professionals, professional software developers,
software quality engineers, computer science students, and security
software vendors should be able to come to the BASC, learn, and hopefully
enjoy themselves at the same time.

To obtain more information on BASC and sign up, please visit:

https://www.owasp.org/index.php/2016_BASC_Homepage

and to stay in touch, please consider signing up at our Meetup:
https://www.meetup.com/owaspboston/

OWASP Boston Board
