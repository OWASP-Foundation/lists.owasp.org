Hi Folks,
OWASP BASC 2014 will have one room devoted to informal demonstrations and
explaining OWASP tools and software and other open source application
tools.  If you would like to demonstrate and explain any OWASP or open
source tools during BASC, please contact Mark Arnold at lotusebhat at gmail.com.
This is not for product demos, even if it is a free version of the
product.  You don't have to be an expert, but if you have used any OWASP or
open source software, you have experience that others could benefit from.
This was a big hit at AppSec USA in Denver last month. This is an informal
way to share with the Boston AppSec comunity and have fun. We call it the
OWASP   Petting Zoo because you get to interact with ( or ask the presenter
to try something with)  the software.

Thanks, Jim
