<p class=MsoNormal><span style='font-size:9.0pt;font-family:"Helvetica","sans-serif"'>In
this installment of the Safelight lightning talks series, Rob will present the

methods of XSS attack, reflected and persistent, as well as provide detailed
demonstrations that show how an attacker would use these attacks in the real

&quot;weaponized&quot; JavaScript that could be used to steal another user's
session information.<o:p></o:p></span></p>

scripting flaws in your own applications.<o:p></o:p></span></p>
