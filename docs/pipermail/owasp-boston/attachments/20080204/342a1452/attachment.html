<DIV><BR>Description - <BR>This presentation will discuss penetration testing 
techniques for<BR>analyzing unknown data in web applications and demonstrate how 
encrypted<BR>data can be compromised through pattern recognition and only 
a<BR>high-level understanding of cryptography concepts.&nbsp; Techniques will 
be<BR>illustrated through a series of detailed, step-by-step case 
studies<BR>drawn from the presenter's penetration testing experience.</DIV>
<DIV>This is not a talk on brute forcing encryption keys, nor is it 
a<BR>discussion of weaknesses in cryptographic algorithms.&nbsp; Rather, the 
case<BR>studies will demonstrate how encryption mechanisms in web 
applications<BR>were compromised without ever identifying the keys or even 
the<BR>underlying ciphers.</DIV>
<DIV>Bio - <BR>Prior to joining Veracode, Mr. Eng was a Technical Manager for
