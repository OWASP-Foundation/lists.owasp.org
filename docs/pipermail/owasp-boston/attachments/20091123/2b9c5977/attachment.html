<p>The statistic is a bit odd, to be sure, but I think the intent was to convey that 10-20% of TYPES of vulnerabilities can be found, and that automated scanners are not a complete solution. Remember, 70% of statistics are completely made up. ;)</p>


<p>Using multiple scanners can be useful since the weaknesses of one can be complemented by the strengths of the others, but imagine that you have an oddly named database backup file (complete with plaintext admin passwords) sitting in a web-accessible, indexable directory. An automated scanner will likely never find this, but with a good web spider and a keen eye, a pen tester certainly will, and this is DEFINITELY a vulnerability.</p>
<p>I suggest using an intercepting proxy like OWASP&#39;s WebScarab or Burp Suite or Paros when doing manual web app pen testing to complement your automated scans.</p><p><br></p>
