Hi all,

On Thursday August 30th from 6pm to 8pm Cory Fowler, Windows Azure Technical Evangelist from Microsoft, will be flying in from Redmond to demonstrate developing PHP Applications from Command-line to the Azure Cloud. He will outline the different deployment patterns in Windows Azure, as well as how to leverage several building block services offered by the platform for use in your PHP applications. This will all be done from his Mac :)

For anyone who hasn't seen the new Open Source features of Azure it will be really interesting to see the possibilities the platform now opens up. The event is free to attend and will be held at Madlab (with drinks afterwards in Common) and the sign up page is here:
http://www.eventbrite.com/event/4135647834

Many Thanks,
