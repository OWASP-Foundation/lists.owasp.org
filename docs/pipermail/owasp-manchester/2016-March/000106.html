Hello everyone,

We have a new speaker confirmed for the OWASP meeting on 17th March. The
details follow:

Name: Julian Horoszkiewicz



Topic: Blind detection of path traversal-vulnerable file uploads


Description: Presentation of an experimental web penetration testing
technique, aiming at detection of path traversal issues in file upload
implementations, with zero knowledge about the remote directory structure.


Bio: Julian Horoszkiewicz, IT Security Consultant at Pentest Ltd, OSCP,
open source and security enthusiast, recently focused on methodology


If you still have not booked your place on the event you can do it on
EventBrite:
https://www.eventbrite.co.uk/e/owasp-manchester-meeting-17th-march-tickets-22527547501

The event will be held in SpacePortX in Manchester city center and it will
kick off at 18:00.

Hope to see you on the event next week.
