Hello, and welcome to the OWASP Manchester mailing list!

We are planning 2 chapter meetings in 2011 on August 24th and November 16th.

I've already got some talks lined up for both dates, but there are still
more slots available, so please get in touch if you or anyone you know would
like to speak at one of these meetings.

And if you have any requests for talks then please email this list - we'll
do our best to find someone who can talk about any security topic you are
interested in.

For more details see https://www.owasp.org/index.php/Manchester, and I will
(of course) keep you updated via this list.
