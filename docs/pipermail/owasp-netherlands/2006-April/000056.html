We are pleased to announce the release of draft 0.2 of the Information
Systems Security Assessment Framework (ISSAF).

ISSAF is the product of the collaborative effort of several security
professionals that aims to provide a comprehensive framework for penetration
testers and systems auditors. It includes detailed techniques, procedures
and tool usage examples to perform a wide range of tests for a several
platforms and devices.

The document is available for download from: http://www.oissg.org/issaf

ISSAF is maintained by the Open Information Systems Security Group (OISSG):
www.oissg.org. All feedback and new volunteers for the project are always
welcomed  ;-) .

Changes from draft 0.1 include a huge amount of new and updated information.
Most of the sections in draft 0.2 are in a fairly advance stage and the
document in general is ready to be used as support for most phases in real
pentest engagements.

Regards,
