<h3><span>Applying static analysis:</span><br></h3>Where architectural analysis finds flaws (design) in the application, 
static analysis is an excellent way of weeding out the bugs 
(implementation). However, static analysis is neither trivial from a 
theoretical nor from a practical perspective. In this session, we will 
discuss the theory behind static analysis as well as how to tune static 
analysis in such way that the solution becomes an excellent way to weed 
out security bugs in your organization.
<br></div><div><br></div><div><font size="4"><b>Speakers:</b></font><br><h3><span>Daan Raman</span></h3>
<p>Daan Raman works at NVISO, where he is responsible for R&D. His
