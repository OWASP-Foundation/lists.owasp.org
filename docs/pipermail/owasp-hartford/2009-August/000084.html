Web Developer - Philadelphia, PA - Contract - Hire

 

A client of ours in Philadelphia is looking to bring on a Web Developer to
help them create a Web Portal for an existing software application.  This
position will be the lead on the Web-portal aspect of this new project.
This person will be responsible for standing up a web-based solution for
creating the web portal associated with a Collateral Management Software and
a new Messaging platform.  This portal will be hosted in-house and will
interact with core collateral management product as the back-end.  This
portal will be constructed using Ruby on Rails and may involve some Java
programming.  Essentially, the web developer will look at the existing User
Interface of the core product and replicate the "look and feel" in a web
version.  This web portal will need to be High Availability, High Visibility
and be able to support high volume in a 24x7 transaction based environment.

 

Requirements:

-              5+ years of experience building large-scale High Availability
websites from scratch.  

-              Must have experience of Web Design/Development best practices
in an Agile Environment

-              Must have strong Ruby on Rails development experience along
with other Open Source technologies.

-              Experience in Restful development

-              Must be able to hand-code web languages such as HTML, DHTML,
Javascript, CSS, PHP.

-              Table-based UI layout development 

-              Table-less (CSS) UI layout development preferred

-              Use of function/script/content includes 

-              Functionality developed for re-usability 

-              Web-data capture and search functionality development 

-              Cross-browser, cross-platform compatibility for layout (to
the pixel) and functionality

-              Script/code/markup development from scratch (not from WYSIWYG
editors) 

-              Must have some experience with User Interface design and
principles of User Experience
