<P><FONT COLOR="#0000FF" SIZE=2 FACE="Arial">The Open Web Application Security Project will be holding its next meeting on Wednesday September 24th at The Hartford, Tower Building, Atrium Conference Room. We will have two great speakers:<B> Nick Selby</B>, Industry Analyst from the<B> 451 Group</B> along with<B> Andy Stone</B> of<B> Accenture</B>. </FONT></P>

<P><FONT COLOR="#0000FF" SIZE=2 FACE="Arial">Andy will be speaking on the topic of:</FONT><B><FONT COLOR="#0000FF" SIZE=2 FACE="Arial"></FONT> <FONT COLOR="#0000FF" SIZE=2 FACE="Arial CE">Improve Security by Removing It from Applications</FONT></B>
</P>

<P><FONT COLOR="#000080" SIZE=2 FACE="Arial CE">The next evolutionary advancement in securing custom applications will</FONT><FONT COLOR="#000080" SIZE=2 FACE="Arial"></FONT> <FONT COLOR="#000080" SIZE=2 FACE="Arial CE">involve removing security code from the application package.</FONT><FONT COLOR="#000080" SIZE=2 FACE="Arial"></FONT> <FONT COLOR="#000080" SIZE=2 FACE="Arial CE">Application security has evolved from almost nonexistent to include</FONT><FONT COLOR="#000080" SIZE=2 FACE="Arial"></FONT> <FONT COLOR="#000080" SIZE=2 FACE="Arial CE">security focused coding practices and functions.&nbsp; Although applications</FONT><FONT COLOR="#000080" SIZE=2 FACE="Arial"></FONT> <FONT COLOR="#000080" SIZE=2 FACE="Arial CE">will continue to need secure coding practices such as failing to a</FONT></P>

<P><FONT COLOR="#000080" SIZE=2 FACE="Arial CE">secure mode and secure defaults, security functions such as</FONT><FONT COLOR="#000080" SIZE=2 FACE="Arial"></FONT> <FONT COLOR="#000080" SIZE=2 FACE="Arial CE">authentication and authorization have no place in the application</FONT><FONT COLOR="#000080" SIZE=2 FACE="Arial"></FONT> <FONT COLOR="#000080" SIZE=2 FACE="Arial CE">package.&nbsp; Abstraction of these functions can result in improved security</FONT><FONT COLOR="#000080" SIZE=2 FACE="Arial"></FONT> <FONT COLOR="#000080" SIZE=2 FACE="Arial CE">and reduced cost of application development.</FONT></P>

<P><FONT COLOR="#800080" SIZE=2 FACE="Arial">The Open Web Application Security Project (OWASP) is a worldwide free and open community focused on improving the security of application software. Our mission is to make application security &quot;visible,&quot; so that people and organizations can make informed decisions about application security risks. Everyone is free to participate in OWASP and all of our materials are available under a free and open software license.</FONT> </P>
