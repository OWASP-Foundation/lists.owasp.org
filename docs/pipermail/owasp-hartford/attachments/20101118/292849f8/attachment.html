<p class=MsoNormal>The OWASP community at large has been very busy on a variety
of open source security related projects. Wanted to highlight a few of my
personal favorites:<o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoListParagraph style='text-indent:-.25in;mso-list:l0 level1 lfo1'><![if !supportLists]><span
style='mso-list:Ignore'>1.<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span><![endif]>The OWASP Secure Coding Practices Quick Reference
Guide: <a
href="http://www.owasp.org/index.php/OWASP_Secure_Coding_Practices_-_Quick_Reference_Guide">http://www.owasp.org/index.php/OWASP_Secure_Coding_Practices_-_Quick_Reference_Guide</a><o:p></o:p></p>

<p class=MsoListParagraph style='text-indent:-.25in;mso-list:l0 level1 lfo1'><![if !supportLists]><span
style='mso-list:Ignore'>2.<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span><![endif]>The OWASP AppSensor Project: <a
href="http://www.owasp.org/index.php/Category:OWASP_AppSensor_Project">http://www.owasp.org/index.php/Category:OWASP_AppSensor_Project</a>
This project is useful if you want to understand how to make your apps defend
against automated attacks.<o:p></o:p></p>

<p class=MsoListParagraph style='text-indent:-.25in;mso-list:l0 level1 lfo1'><![if !supportLists]><span
style='mso-list:Ignore'>3.<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span><![endif]>The OWASP Alchemist Project: <a
href="http://www.owasp.org/index.php/OWASP_Alchemist_Project">http://www.owasp.org/index.php/OWASP_Alchemist_Project</a>
This project enables a software development team in the realization of highly
secure and defensible application with built-in defense/controls against
security-related design, coding and implementation flaws.<o:p></o:p></p>

<p class=MsoListParagraph style='text-indent:-.25in;mso-list:l0 level1 lfo1'><![if !supportLists]><span
style='mso-list:Ignore'>4.<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span><![endif]>The OWASP Secure Web Application Framework Manifesto: <a
href="http://www.owasp.org/index.php/OWASP_Secure_Web_Application_Framework_Manifesto">http://www.owasp.org/index.php/OWASP_Secure_Web_Application_Framework_Manifesto</a>
This project is a document detailing a specific set of security requirements
for developers of web application frameworks to adhere to .Some of the things
in it may feel familiar <span style='font-family:Wingdings'>J</span><o:p></o:p></p>

<p class=MsoListParagraph style='text-indent:-.25in;mso-list:l0 level1 lfo1'><![if !supportLists]><span
style='mso-list:Ignore'>5.<span style='font:7.0pt "Times New Roman"'>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
</span></span><![endif]>The OWASP Application Security Skills Assessment: <a
href="http://www.owasp.org/index.php/OWASP_Application_Security_Skills_Assessment">http://www.owasp.org/index.php/OWASP_Application_Security_Skills_Assessment</a>
This is an online multiple-choice quiz built to help individuals understand
their strengths and weaknesses in specific application security skills. My
personal interest in this effort is to encourage PCI/DSS auditors to find a
simple way of telling when enterprises who need to comply to PCI/DSS aren&#8217;t
fudging the truth and have people who know what they are doing <span
style='font-family:Wingdings'>J</span><o:p></o:p></p>

<p class=MsoNormal><o:p>&nbsp;</o:p></p>

<p class=MsoNormal>I encourage everyone to check out these projects and see if
they add value to your organization. If your organization finds value in OWASP,
maybe it will also consider becoming a member (<a
href="http://www.owasp.org/index.php/Membership">http://www.owasp.org/index.php/Membership</a>).
Please also consider forwarding the above information to those who could
benefit. <o:p></o:p></p>
