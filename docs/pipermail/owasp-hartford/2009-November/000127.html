Job description : (  Key words : SQL server, SSIS, SSAS, SSRS, OLAP, MDX
) 

The BI developer is responsible for the overall execution of the data
mart build, aggregations and OLAP cube development.This includes
analyzing, planning and developing the architecture to ensure that it is
in line with business requirements. The success of this role will be
dependent upon the ability to comprehend the technical implementation
environment using SQL Server 2005, 2008 SSIS, SSAS aligned with that of
the technical design and with business requirements to successfully
implement this solution. This incumbent deal directly with the
technology and will require hands on experience with the Microsoft
toolset and general data warehouse, data mart and aggregation
constructs. The candidate must be able to provide technical expertise,
knowledge transfer and operational documentation for the solution
implemented.

 

DUTIES & RESPONSIBILITIES: 

 

 

* Build SSIS Master and child packages for initial load of data and
incremental (monthly) loads of data for small data mart consisting of 10
standard type1 dimensions and 4 aggregated fact tables.

 

* Manages the ETL function and use of Staging tables.

 

* Proactively diagnoses, researches, monitors for and resolves SQL and
database-related faults and performance problems

 

* Develops, maintains, monitors, and does performance tuning of the data
mart databases

 

* Develops, maintains, monitors, and does performance tuning of SSAS
OLAP cube(s)

 

* Acts as a high-level technical data expert, and mentors less
experienced colleagues in addressing data-related problems of systems
integration and migration 

 

* Develops and maintains appropriate repositories of data model and
dataflow diagrams relating to the data warehouse

 

* Develops and communicates project plans, work assignments, target
dates and other aspects of assigned projects. Ensures that data projects
follow prescribed agile development life cycle and meet quality
standards

 

* Functions as a technical expert in the design, development,
modification and debugging of data OLAP cubes and databases, etc. 

 

* Advises technology architects on database subjects appropriate for
supporting ongoing development efforts

 

* Develops ETL specifications, source to target mapping and other
documentation required for ETL development. Ensures appropriate
documentation is completed and stored in the repository database.

 

* Develops test cases and plans, and assists with system testing 

 

* Facilitates communication between developers and business users to
ensure that customer requirements are correctly translated to
appropriate data design specifications

 

* Works with business users to optimize information usage 

Required Skills: 5+ years experience working with data warehouses, data
marts, and OLAP technologies

 

* Strong knowledge of data warehousing concepts, terminology, and
architecture 

 

*Ability to write reconciliation and validation scripts

 

* Experience designing, maintaining and implementing aggregate data
marts 

 

* Ability to mentor and lead team members in the areas ETL and OLAP cube
development

 

* Understanding of data analysis, modeling (PowerDesigner preferred)

 

* Ability to hit the ground running and begin to build SSIS packages for
already designed solution 

 

* Keen attention to detail

 

* Solid multitasking, analytical, and problem solving skills

 

* Strong SQL Server 2005, 2008 administration skills

 

* Strong SQL. T-SQL skills

 

* Excellent communication, negotiation, and collaboration skills

 

* Excellent documentation, organization, and presentation skills

 

* Experience training end-users is a plus

 

* Experience with agile development methodologies is a plus

 

Critical Skills: Undergraduate degree or equivalent experience in a data
warehousing environment with at least 3 years SSIS, SSAS, SSRS(a plus)
on either SQL Server 2005 or SQL Server 2008. In-depth knowledge of
dimensional databases, dimensional modeling and data warehousing
required. Must demonstrate capabilities in building attribute
relationships, hierarchies, named sets, MDX within SSAS. Must
demonstrate capabilities in building security models for drill level
access to cubes as well as within the physical dimensional schema.
