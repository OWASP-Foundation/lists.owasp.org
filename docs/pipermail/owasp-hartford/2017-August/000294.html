Title: Chapter Planning Meeting (Virtual)
Calling all CT OWASP members:

OWASP CT is looking to formalize its volunteer committee to spearhead  
several important chapter initiatives as well as get input on other  
community activities for the chapter.

OWASP CT NEEDS YOU!

Come attend this session to get a preview of activities being planned for  
CT, and discuss initiatives you have an interest in. If you think something  
is missing from the list below, come to the session to discuss what we can  
do as an OWASP CT community!


Existing initiatives:

• Penetration testing Lab updates

• Event Marketing

Static and Dynamic Analysis Security testing tools

Digital Ethics

IOT Security

Hacking voting machines



Initiatives needing leads:

• Mentorship

• Academic Outreach

• Media Outreach

• Government and Legislative Outreach

• [Your idea here!]
