Hola Samuel, si estas intentando ejecutar el analisis desde una red que
tiene un ISA hacia un Site externo, no vas a poder ejecutar el analisis
con la herramienta (por ahora).
Porque la opcion de utilizar proxy de w3af no se autentica a traves de
NTLM ni NTLMv2.

Si queres ejecutar un analisis bajo este escenario, te conviene
instalarte un proxy local como el paros proxy (o el que mas te guste). 
Configurar el Paros para que salga por el ISA y a W3af para  que salga
por el Paros...  asi te va a funcionar.

Saludos,

Claudio.
