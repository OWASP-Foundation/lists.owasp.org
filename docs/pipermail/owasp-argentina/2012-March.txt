   Hola a todos,

   Aca les paso un link a una herramienta desarrollada por Matias Fontanini
   y equipo que sirve para la explotación automatica de SQL Injections.

   Entre sus features pueden encontrar:
   * Multiplataforma (hecho en py3k) con ejecutable para windows (32bits)
   * Explota SQLi en modo union y blind.
   * Soporta inyecciones en parametros GET/POST/Cookie.
   * Explota inyecciones en MySQL, Oracle, SQL Server y Postgres.
   * Interfaz de linea de comandos con autocompletion para facilitar su uso.
   * Soporta filtros para evadir IPS o modificar las consultas en runtime.

   En la página (http://themole.nasel.com.ar) van a encontrar los links
   de descarga así como un par de tutoriales de como usarlo (
   http://themole.nasel.com.ar/?q=tutorial). Al final del tutorial hay
   un video por si quieren verlo en acción. Próximamente va a estar en
   los repositorios de Debian(y derivados).

   Si alguien lo prueba y tiene feedback será más que agradecido por Matias.

   Saludos,
   Fabio Cerullo

Estimados colegas,

OWASP fue invitado a participar en el Google Summer of Code 2012. Esto
quiere decir que estudiantes universitarios que quieran colaborar en
mejorar alguno de los proyectos de OWASP listados abajo podran participar
de esta iniciativa global durante tres meses y al mismo tiempo ser
remunerados por Google con sumas de hasta U$S 5000 por el trabajo realizado.

Los proyectos OWASP en los que pudieran participar son:

- OWASP Enterprise Security API (ESAPI) Project
- OWASP ESAPI Swingset Project
- OWASP AntiSamy .NET Project
- OWASP Zed Attack Proxy Project
- OWASP AppSensor Project
- OWASP Hackademic Project
- OWASP GoatDroid Project
- OWASP O2 Project

En la siguiente pagina puedes ver algunas de las ideas propuestas por
los líderes de cada uno de los proyectos:

https://www.owasp.org/index.php/GSoC2012_Ideas.

Hay tiempo hasta el 6 de abril para hacer una propuesta formal a
Google+OWASP, la cual será evaluada por un comité.

Todos los estudiantes que deseen participar deben suscribirse a la
siguiente lista de correo:

https://lists.owasp.org/mailman/listinfo/gsoc

Asimismo adjunto material informativo en ingles para ser distribuido en
universidades.

https://docs.google.com/open?id=0Bx8Vln0JUGI1Yk13QWVxVkpRNy11RmVZemVldGNQdw

Cualquier duda, por favor no duden en contactarse conmigo.

Saludos y gracias.

Fabio Cerullo
OWASP GSoC Admin
