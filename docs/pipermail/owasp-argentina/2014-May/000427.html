>> Para los que no estuvieron en alguna de las charlas donde intenté
>> lavarles la cabeza sobre esta idea, lo que quiero hacer es organizar
>> un OWASP Camp / Hackaton al estilo de las organizadas por la comunidad
>> de Python Argentina [0].
>>
>> La organizacion basica del evento es:
>>     * Nos reunimos en un lugar durante 3 dias a trabajar en grupos
>> sobre distintos proyectos relacionados con la seguridad en
>> aplicaciones
>>
>>     * Trabajamos en proyectos de codigo abierto, investigamos sobre
>> nuevas técnicas de hacking, y luego compartimos los resultados con el
>> resto de los grupos
>>
>>     * Hacemos unos asados para calmar el hambre y compartir en conjunto
>>
>>     * Se duerme (muy posiblemente) en carpa/bolsa de dormir. Veremos
>> luego como evoluciona este tema, posiblemente se pueda tambien hacer
>> en algun camping que ofrezca camas en habitaciones compartidas.
>>
>>     * El evento en si mismo es GRATIS, pero habrá que pagarse
>> comidas/alojamiento/internet
>>
>> Los objetivos principales del evento son:
>>     * Aprender en conjunto con otros profesionales
>>
>>     * Mentoring de los menos experimentados para integrarlos y permitirles
>>
>>     * Pasar un buen rato e integrar más las comunidades de seguridad
>> de la región (no hace falta ser miembro de OWASP para asistir)
>>
>> Gracias a Mateo Martinez (OWASP Uruguay) tenemos un lugar para hacerlo
>> en las cercanias de Montevideo. Dependiendo de la cantidad de gente
>> quizás debamos cambiar el lugar; pero eso lo veremos luego.
>>
>> Idealmente deberia ser en un mes donde no haga mucho frio/calor, por
>> lo que Octubre suena bien. Adicionalmente deberiamos encontrar un fin
>> de semana largo (3 dias) en Sep/Oct/Nov para hacerlo en esos dias.
>>
>> Este email es para empezar a hablar de este tema, hacer algunas
>> definiciones generales y ver si realmente podemos llevar a la realidad
>> esta locura divertida y nerd in extremis :)
>>
>> Que les parece? Avanzamos?
