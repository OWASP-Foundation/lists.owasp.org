- Solo funciona para Python.
- Es intrusivo.

único set TAINTED.

Pero bue... quería programar algo como para empezar a probar. Es
código de una hora.

- ej2.py : la versión corregida del primer programa. Ejecuta normalmente.

objetivo empezar a pensar en el asunto.
