Hola tod en s:

Estoy desarrollando un script de nmap que sirve para probar virtual hosts contra un host determinado. La idea es que analizando las respuestas http poder determinar cuales estan hosteados aun si no estan registrados. Esto permite hallar entornos de desarrollo. Este tipo de exploración no es novedoso, metasploit tiene un módulo similar, según me enteré cuando ya lo habia hecho.

Si alguien lo quiere probar, que vea las referencias. Cualquier dificultad, me contacta. 



Mi pregunta es, qué políticas se están usando para una exploración de este tipo. Usar un default virtual host para dar 200 a los nombres inexistentes? Redirigir? No hacer nada?
