Try this one instead -

SecRule TX:'/PHPIDS-30-(.*)-ARGS_NAMES:smaction[save][1]/' "@contains ][" 
"chain,phase:2,t:none,nolog,pass"
       SecRule MATCHED_VAR_NAME "TX\:(.*)" "capture,t:none,setvar:!tx.
%{tx.1},setvar:tx.anomaly_score=-4"
