On 21. okt. 2016 10:50, Ilyass Kaouam wrote:
> 
> How can I configure modsecurity to send an e-mail when match some rule?
> I want something "global", for all rules.

We used AuditConsole[1] at some point. It works with a model that is
similar to Sentry in the python world.

You have mlogc, a collector that gathers all modsec alerts, queues them
up and then sends them in batches to the central audit console, where
you can search and lookup all alert details, and send emails too
if that's what you *really* want :-)

Sadly AuditConsole is not updated very frequently and
it's a bit rough around the edges. If you know of any
alternatives, please do share.
