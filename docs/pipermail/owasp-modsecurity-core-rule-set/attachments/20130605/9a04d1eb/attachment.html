<p class="MsoNormal"><span style="font-size:10.5pt; color:black">Right…. The issue here seems to be that, only for "/", we have a sub-string match.  Here is a quick fix.  Essentially you need to make that rule into a chained rule.  The first like is used to
 simply create a new custom variable that uses # chars to surround the request filename data.  The 2nd rule in the chain then runs the same -</span></p>
</div>
<div>
<p class="MsoNormal"><span style="font-size:10.5pt; color:black"> </span></p>
</div>
<div>
<div>
<p class="MsoNormal"><span style="font-size:10.5pt; color:black">SecRule REQUEST_FILENAME ".*" "chain,phase:5,id:'981039',t:none,nolog,pass,setvar:'tx.filename=#%{request_filename}#',skipAfter:END_BRUTE_FORCE_PROTECTION_CHECKS"</span></p>
</div>
<div>
<p class="MsoNormal"><span style="font-size:10.5pt; color:black"> SecRule TX:FILENAME "<a href="mailto:!@within%20%25%7btx.brute_force_protected_urls%7d">!@within %{tx.brute_force_protected_urls}</a>"</span></p>
</div>
</div>
<div>
<p class="MsoNormal"><span style="font-size:10.5pt; color:black"> </span></p>
</div>
<div>
<p class="MsoNormal"><span style="font-size:10.5pt; color:black">For this to work, you would then need to modify your URL list in the 10 file like this -</span></p>
</div>
<div>
<p class="MsoNormal"><span style="font-size:10.5pt; color:black"> </span></p>
</div>
<div>
<p class="MsoNoSpacing"><span style="font-size:8pt; font-family:'Courier New'; color:black">SecAction \</span><span style="color:black"></span></p>
<p class="MsoNoSpacing"><span style="font-size:8pt; font-family:'Courier New'; color:black">  "id:'900014', \</span><span style="color:black"></span></p>
<p class="MsoNoSpacing"><span style="font-size:8pt; font-family:'Courier New'; color:black">  phase:1, \</span><span style="color:black"></span></p>
<p class="MsoNoSpacing"><span style="font-size:8pt; font-family:'Courier New'; color:black">  t:none, \</span><span style="color:black"></span></p>
<p class="MsoNoSpacing"><span style="font-size:8pt; font-family:'Courier New'; color:black">  setvar:'tx.brute_force_protected_urls=#/login.jsp# #/partner_login.php#', \</span><span style="color:black"></span></p>
<p class="MsoNoSpacing"><span style="font-size:8pt; font-family:'Courier New'; color:black">  setvar:'tx.brute_force_burst_time_slice=60', \</span><span style="color:black"></span></p>
<p class="MsoNoSpacing"><span style="font-size:8pt; font-family:'Courier New'; color:black">  setvar:'tx.brute_force_counter_threshold=10', \</span><span style="color:black"></span></p>
<p class="MsoNoSpacing"><span style="font-size:8pt; font-family:'Courier New'; color:black">  setvar:'tx.brute_force_block_timeout=300', \</span><span style="color:black"></span></p>
<p class="MsoNoSpacing"><span style="font-size:8pt; font-family:'Courier New'; color:black">  nolog, \</span><span style="color:black"></span></p>
