<p style="-qt-paragraph-type:empty; margin-top:0px; margin-bottom:0px; margin-left:0px; margin-right:0px; -qt-block-indent:0; text-indent:0px; -qt-user-state:0;"><br></p>My understanding of HPP (http://www.owasp.org/images/b/ba/AppsecEU09_CarettoniDiPaola_v0.8.pdf) is that it includes adding in/injecting new parameters.  This can include adding in the data before the "=" into the parameter name section or after the "=" into the parameter payload section.  Either way, the attacker is adding in new parameters.<br>
