    <DIV>First you need to define where the GeoDB is. This can be done in the 
    CRS 10 file. </DIV>
    <DIV> </DIV>
    <DIV>SecGeoLookupDb /opt/modsecurity/bin/GeoLiteCity.dat<BR></DIV>
    <DIV> </DIV>
    <DIV>Next you need to pass the client's IP address (REMOTE_ADDR) to the 
    @geoLookup operator via a rule like:</DIV>
    <DIV> </DIV>
    <DIV>SecRule REMOTE_ADDR "@geoLookup" 
    "id:1,phase:1,t:none,pass,nolog"<BR></DIV>
    <DIV> </DIV>
    <DIV>Finally you limit access to the upload script (e.g. /UploadFiles.aspx)

    <DIV> </DIV>
    <DIV>
    <DIV>SecRule GEO:COUNTRY_CODE3 "!@streq JPN" 
    "id:2,phase:1,t:none,log,deny,msg:'Client IP not from 
    Japan',chain"     </DIV>

    /UploadFiles.aspx"</DIV></DIV>
