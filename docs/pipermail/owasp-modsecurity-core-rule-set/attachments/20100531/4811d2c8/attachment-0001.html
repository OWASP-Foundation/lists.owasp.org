Any script will work that just looks at the given filename (first and only argument to the script).&nbsp; It just needs to output a &quot;0&quot; (zero) as the first character of the output on failure.&nbsp; Any other output is a success.&nbsp; The exit code is ignored.&nbsp; For example:<br>
<br>
#!/bin/sh<br>
<br>
FILE=$1<br>
<br>
# Fail if any of the words are found in the file if grep &quot;list|of|words&quot; $FILE &gt; /dev/null; then<br>
&nbsp; echo &quot;0 $0: FAILED&quot;<br>
&nbsp; exit<br>
fi<br>
<br>
echo &quot;1 $0: OK&quot;<br>
