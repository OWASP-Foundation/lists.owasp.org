All:

Everyone please register for the upcoming talk to OWASP San Antonio on *Friday
May 20* about Runtime Application Self-Protection (RASP) Tools. MeetUp link
is here:
*http://www.meetup.com/OWASP-San-Antonio/events/231044053/
<http://www.meetup.com/OWASP-San-Antonio/events/231044053/>*

This will be a great presentation discussing Runtime Application
Self-Protection (RASP) and a new high-performance methodology called
language theoretic security (LANGSEC). Kunal will also break down how
lexers, tokenizers and parsers work, and construct an open source toolchain
to analyze data and explore interactive data visualizations -- covering the
challenges of modern AppSec along the way.

Also FREE LUNCH! (Everyone loves free lunch.)

And for all you folks on Twitter, we’ve created an OWASP San Antonio
Twitter account @owaspsanantonio:
https://twitter.com/owaspsanantonio

Hope to see you there!
