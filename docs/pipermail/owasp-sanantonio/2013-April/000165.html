OWASP San Antonio is going to be working with the BSides San Antonio folks
to offer three free mini-courses for developers. BSides San Antonio is
current SOLD OUT but we have a special entry code for developers
interested in this training.

When: Saturday May 4th, 2013
Where: Geekdom (downtown)

See the OWASP San Antonio page here for more info:
https://www.owasp.org/index.php/San_Antonio#Local_News


The three mini-courses are:

Threat Modeling for Developers
Developing Secure Web Applications
Developing Secure Mobile Applications

Each will be around 1.5 to 2 hours and the specific schedule will follow.
Attendees get full access to the Bsides event and it is all FREE. Please
send this along to developers you know who might be interested.
