Title: Vision of the SWAMP

Abstract:
This Presentation is an overview of the Software Assurance Marketplace
(SWAMP) project. The SWAMP is a 23 Million dollar DHS funded initiative to
build a physical and virtual facility to help improve the quality of
software that is being designed, both open-source and proprietary. This
platform democratizes the SwA environment by providing a free, virtual
market where the user community can come to share tools, techniques,
resources and experiences with the goal of improving software assurance
(reducing software vulnerabilities)
