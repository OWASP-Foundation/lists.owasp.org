There is no change password function in AppSensor, you might be
thinking of ESAPI. In that case, ESAPI's default is to use a file
based authentication scheme where the user "data store" is a file on
disk. However, most people who actually use ESAPI build their own
authenticator for a given database or ldap or something, and then use
that as their backing data store.
