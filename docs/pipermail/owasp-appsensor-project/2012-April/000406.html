>>> Ahmad,
>>> This is a Java-based project. It does not require linux, just a
>>> platform that runs java. You can use it on windows just fine. It is
>>> not a GUI application, but rather is a component that gets deployed in
>>> java web applications. The developer guide should show you enough to
>>> get started once your environment is setup. Essentially, the basic
>>> steps are:
>>> 1. setup environment (you'll probably want to use eclipse on some
>>> platform - maybe windows if you're comfortable with that)
>>> 2. download code into eclipse
>>> 3. create a web application that uses the project as a 3rd party jar
>>> library
>>> 4. create detection points in your code
>>> 5. create appsensor.properties file and put in your configuration options
>>> 6. run application to test that intrusion detection is working properly.
>>>
>>> Thanks,
>>> John
