Samantha has kindly produced a summary of the telephone conference call:

   http://lists.owasp.org/pipermail/owasp-leaders/2013-May/009287.html

Seba and Samantha have asked if projects want to participate in this
bid for funding.  It seems it is "match funding" and voluntary work
doesn't count. So ideas have to cost money, and be paid equally by
OWASP and the EU.  Some examples that came up during the call were:

- Pay a designer to rework the existing code review guide into a more
accessible and stylish document
- Fund travel and expenses for an individual project summit where
volunteers (in their own time) create extensions to ZAP for example

Part of the cost of these examples might also be contributions to
OWASP staff salaries.  OWASP will not pay volunteers of course, so
anything we have to do to make any of these ideas happen has to be
done in our own time.

I hope I have summarised the issues correctly.

So, do we have any ideas of things we think this type of funding could
be practically used for in the AppSensor project? I am not sure of the
funding timescales, but think it might be from next year. However we
need to come up with ideas ...today.

Colin
