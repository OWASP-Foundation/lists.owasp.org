Rauf,
Thanks very much for the interest in appsensor for the GSOC program.

1. Just a clarifying point. There are a couple of features the SOAP
services will expose - the engine (which you listed) and the reporting
features.
2. To your question about using Axis2: I'm the dev lead, and I
honestly haven't decided. I was leaning towards using the web services
built into Spring since I plan on using that for the REST
implementation as well. I'm not necessarily opposed to using something
else, but I'd have to think about it a bit. The only real requirement
is that the web services library has support for the WS-I Basic
Security Profile (BSP), which I think Axis2 does.

Hopefully I answered your question well, but if you have any further
questions, please feel free to contact me anytime.
Thanks,
John
