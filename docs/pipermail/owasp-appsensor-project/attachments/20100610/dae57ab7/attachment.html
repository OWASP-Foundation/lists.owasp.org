I could have sworn that unexpected headers was on the list. I remember
testing it in my old demo app.&nbsp; Oh well, its not at that link so we
should add it.<br>
<br>
I would divide this into two detection points. One for missing an
expected header and one for receiving an unexpected header.&nbsp; The later
is actually very tricky as I found because all sorts of proxies will
attach weird x-something headers.&nbsp; We should mention that in the
comments for that detection point.<br>
