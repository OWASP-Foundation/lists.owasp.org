Description<br>
-----------------------------------<br>
The user is identified as using an IP address associated with a<br>
blacklist (e.g. internal blacklist, list of Tor nodes e.g.<br>
<a href="https://torstat.xenobite.eu/" target="_blank">https://torstat.xenobite.eu/</a> and HTTP blacklist e.g.<br>
<a href="http://www.projecthoneypot.org/httpbl.php" target="_blank">http://www.projecthoneypot.org/httpbl.php</a> and Dshield<br>
<a href="http://www.dshield.org" target="_blank">http://www.dshield.org</a> and spammers e.g. Spamhaus<br>
<a href="http://www.spamhaus.org/" target="_blank">http://www.spamhaus.org/</a> and known botnets e.g.<br>
<a href="http://www.shadowserver.org/wiki/" target="_blank">http://www.shadowserver.org/wiki/</a>).  &quot;Suspicious&quot; may also depend upon<br>
the type of user e.g. users in the &quot;CMS manager&quot; role should be using<br>
an internal network IP address, public users could be from anywhere,<br>
customers should only be accessing the application from a particular<br>
geographical region, search engine robots  should be from a limited<br>
range of IP addresses.<br>
