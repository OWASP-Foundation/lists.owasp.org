Not sure I&#39;m on board with this one ... someone else can correct me if I&#39;m wrong, but this actually doesn&#39;t fit in the &quot;application&quot; doing detection.  By definition, something outside the app is doing the detection and is feeding that info to the app.  I think these are worthwhile sensors that can produce data that an application could use to make decisions, but as for it being considered app detection, I don&#39;t generally see these as falling into that category.  I may be convinced otherwise however :&gt;.<br>
