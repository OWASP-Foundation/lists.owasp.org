>> >>>>>>>>><br>
>> >>>>>>>>> Approach<br>
>> >>>>>>>>> 1. Collectively agree on the skeleton / table of contents<br>
>> >>>>>>>>> (pretty<br>

>> >>>>>>>>> 4. Determine a larger number of contributing writers<br>

>> >>>>>>>>><br>
>> >>>>>>>>><br>
>> >>>>>>>>> Core Author<br>

>> >>>>>>>>> - Drives the main ideas covered in the sections<br>

>> >>>>>>>>> section,<br>
>> >>>>>>>>> organization of ideas<br>
>> >>>>>>>>> - Works with contributing writers to arrange material, flow<br>

>> >>>>>>>>> section they feel comfortable<br>
