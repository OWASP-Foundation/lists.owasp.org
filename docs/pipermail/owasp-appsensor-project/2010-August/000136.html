 AppSensor Crew,

I hope to see many of you at the upcoming OWASP AppSec conference in
Irvine, CA. 

I will be presenting on day 2 about AppSensor. The talk will cover
AppSensor's current status and its integration with ESAPI. In addition I
will also be talking about how AppSensor ideas are being incorporated
into Mozilla web applications to actively identify application attackers. 

Finally, I will also be debuting a new demo application called
AppSensor-Tutorial.  This demo code base is a fully working jsp app with
ESAPI and AppSensor integration. The purpose is to provide lesson based
examples of how to use AppSensor. The code is intended to be as clear
and concise as possible so that it can be referenced as an example of
how to easily use AppSensor with ESAPI. 

You can get a preview of the code here:

http://code.google.com/p/appsensor/source/browse/#svn/trunk/AppSensor-Tutorial

Detailed instructions for setup will be coming soon.
