All,

We've now created a specific mailing list to focus on AppSensor 
development.  This current mailing list owasp-appsensor-project will 
remain live.

Here is a basic breakdown of the list usage. This is just a guide to 
help you decide which list(s) you want to be on.

owasp-appsensor-project
- Ideas & Project Updates
- New Detection Points
- AppSensor concepts
- Upcoming AppSensor presentations or related articles
- Any other discussions on the topic or related topics

owasp-appsensor-dev
- Code & Techie Stuff
- Discuss AppSensor code
- ESAPI integration with appsensor (code and implementation)


Thanks and please sign up for the dev list if that info sounds 
interesting to you.

https://lists.owasp.org/mailman/listinfo/owasp-appsensor-dev
