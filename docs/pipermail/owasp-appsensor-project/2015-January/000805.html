We just released 2.0.0-RC3. This is the third (and planned final) release
candidate for 2.0.0 final.  More info is here:
https://github.com/jtmelton/appsensor/releases/tag/v2.0.0-RC3

This was an update to address issues in the SOAP web service. Many thanks
to Juan Calderon for finding the issue and helping me resolve it.

Unless there are significant bugs that come up, we plan on a v2 final going
out in mid-late January. This is a delay from the planned date at the end
of 2014, but this issue was important enough to resolve before 2.0.

As always, happy to hear any feedback.
