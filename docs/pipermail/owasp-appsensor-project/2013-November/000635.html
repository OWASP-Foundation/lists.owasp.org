Dear all

Additional extensive content from Dennis Groves and reviewer feedback
from Graig Munson have been incorporated into the draft new AppSensor
Guide (now currently at v1.36).

The plan is to finalise all content during December and issue it for
more final review late December. Some of you have already offered to
help with that, but if anyone else can contribute, please let me know.
But if you have time before then to review the existing draft content,
please do so anyway.

We have also received some funding from the HOST program (see separate
emails to this list) to design the figures and book layout, as well as
a flyer. The timing of that, and reviewer feedback, may affect when we
release the v2, but we are now aiming for early 2014. I have some more
content to write myself but will do that mid-December.

Note Dennis will seeking review and feedback on the draft book at
Appsec USA next week:

    Writing and Documentation Review Session for documentation
projects including AppSensor, Michael Hidalgo and Samantha Groves,
Wednesday 20th November at 9.00am - 1.00pm (Sky Lounge, 16th Floor)

I'll be around too!

Colin
