Colin has uploaded the zip file containing my .Net port to
https://www.owasp.org/index.php/File:AppSensor_Core_-_dotNet.zip

I basically took the Java version and did my best to port it to .Net. It
does Build but I very quickly realised that the design didn't work for my
system since it is largely user-centric in that the actions mostly relate
to blocking/choking a user whereas in my system, the main threats are from
anonymous users. For this reason, although it did seem to work for my
limited testing, I cannot guarantee that everything will behave as expected
so if someone wants to use it and write some unit tests etc then you are
most welcome.

I really like the idea of AppSensor though. In my case, as in others, the
most informed intrusion detection can only take place at application level.

happy coding!

Luke
