My general opinion from experience is that people will write adapters if
they have to (and in this case I don't think it's too difficult), but
they're more likely to use your tool if it supports their environment
already. I'd love to have support for the different popular formats out of
the box. I'll try to put all this in the GSOC request in a cleaner form
hopefully this weekend. I'll also include feedback from Louis/others if
more comes in.
