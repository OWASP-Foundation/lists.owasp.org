All,

A minor snapshot was just released to address an issue where the same
detection point label could not be reused (
https://github.com/jtmelton/appsensor/issues/18 ) (Thanks Juan for bringing
this up).

After this update, you should be able to use the same detection point label
(eg. "IE3") multiple times with different configurations and that should be
respected. This gives you the capability in particular to trigger different
responses at different intervals, which adds a lot of flexibility.

If this is a feature that interests or is important to you, please test it
out. I'll give it a week or two for feedback, then will push it as a point
release if there are no objections.

Thanks,
John
