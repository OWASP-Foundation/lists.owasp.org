The SANS Institute report says the top impediment to attack detection
and response is lack of visibility into applications, underlying
systems and vulnerabilities (and that excludes lack of visibility into
cloud applications which is a separate category).

http://www.sans.org/reading-room/whitepapers/analyst/analytics-intelligence-survey-2014-35507

But then goes on to list all the networky defency/detectiony things,
but no mention of application intrusion detection. AppSensor-like
technology doesn't even appear in "[categories of] future investments
related to analytics/intelligence in order to obtain better visibility
and response" unless it is in "other".
