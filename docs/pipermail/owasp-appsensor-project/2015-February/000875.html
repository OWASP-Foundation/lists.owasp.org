I had some time on a train journey today. A new version (exported from
Illustrator) uploaded to the same place:

    https://www.owasp.org/index.php/File:Appsensor_intro_for_developers.pdf

2-sided. The dark side is meant to be eye catching, with all the
details on the reverse. I used colours and icons from www.apsensor.org
primarily but also the book cover image faintly in the dark background
as texture. I added the OWASP logo and some key words as suggested by
Louis. I moved the app sensor bullet points up into the first part
after reading Louis's feedback. I think the "why should I.." is still
relevant. I assume we are just trying to catch people's interest and
get them to read the website, so maybe things like "Tomcat" can be
mentioned more there?

Maybe the back/detailed size looks a little cramped. Do you think I
could use a smaller text size (the main text is 12pt, the about OWASP
text is 11pt).

I will need it check alignment and so on, once we agree on the copy
etc. I will also be able to export a press-ready high-resolution file.

Colin
