Colin,

Thanks so much. Looks amazing!

Feedback:
- I would remove the rest/soap/thrift/native from the front page. It's
technical detail and can be found on the website - it'll make more sense
there.
- I printed out page 2, and it could definitely have a smaller font and not
suffer. 11pt should be fine - I actually think it will look better with the
extra whitespace. I would leave the section header font as is. Feel free to
reword/shorten/remove content of mine to get some extra space.
- Under the "what is appsensor" section, we should bring up Louis' point
about a single instance of appsensor supporting multiple client
applications at once.
- If we have space (lowest priority), maybe we can also add a point that we
can correlate analysis for multiple applications for applications using
SSO.
- We should do a once over for spelling/grammar (I noticed a few mistakes I
made)

Thanks,
John
