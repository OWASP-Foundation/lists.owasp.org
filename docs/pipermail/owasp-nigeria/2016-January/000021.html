Hi Team,

We want to invite you to the very first OWASP meeting in Nigeria scheduled
for February 13, 2016. It's going to be a fun hacking experience.

We will provide some background on OWASP to set the stage and have already
lined up 3 talks:


Talk 1: Introduction to OWASP ZAP, the integrated web application
penetration testing tool

Talk 2: Exploiting a Vulnerable website to steal user credentials and gain
root

Talk 3: The OWASP Web Security Shepherd, a web and mobile application
security training platform
