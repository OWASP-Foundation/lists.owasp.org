Dear Members,


We have requested more space for OWASP Members and we are pleased to invite
you to attend NITDA Penetration Testing Checklists Standard Workshop on 24th
March, 2016 at NITDA Software Testing Centre, 21B Niyi Okunibi Street Near
Admiralty Road, Lekki Phase 1 Lagos.



The workshop will provide a forum for technical peers to exchange
experiences, best practices, ideas and developed an acceptable framework
for Penetration Testing Checklists Standard which can be adopted by NITDA
for public and private organizations as well as for educational purposes.



You can register here
<https://www.eventbrite.co.uk/e/nitda-penetration-testing-checklists-standard-workshop-tickets-23231183094>



*NB: No need to register again, if you have done so.*


We hope that you will find this workshop of interest and we look forward to
welcoming you.





Yours truly,
NITDA Software Testing Centre, Lekki
