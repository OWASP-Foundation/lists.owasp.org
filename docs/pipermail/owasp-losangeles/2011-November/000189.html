Topic: Cucumber and friends: tools for security that matters

Behavior-Driven Development (BDD) helps focus software development on
delivering prioritized, verifiable business value by providing a
common vocabulary that spans the divide between Business and
Technology. Cucumber is a widely used tool in Ruby community for
implementing BDD and it executes plain-text functional descriptions as
automated tests. In this talk, Mani and Tin will discuss how Cucumber
and related tools can be used to define and verify security features
that matter in software.

Topic: passw3rd: friends don't let friends store passwords in source code

Passw3rd is a encryption library intended to encrypt and store
passwords outside of source code. This is a problem that has been
solved in hundreds of half-baked ways, or it is a problem that is
often overlooked.
