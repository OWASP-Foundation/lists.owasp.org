The position is more Infrastructure (LAMP + Networking) with Application
security being a plus. Full-Time perm in Woodland Hills, CA.

*Job Summary:*

The Security Engineer role within the Security Operations team is directly
responsible for actively defending both the Internet-facing perimeter as
well as sensitive internal facilities from attack and compromise. In being
tasked with the defense of ReachLocal’s infrastructure and applications,
the Security Engineer operates closely with both the broader Technology
Operations teams and Software Engineering resources to identify, mitigate,
and remediate vulnerabilities.

*Competencies: *

   - Infrastructural Security

      - IPv4/v6 Stateful Firewall ACL Best Practices
      - Operating System (Linux) Security Best Practices
   - Application Security


   - Web Application Security Analysis (XSS, Input Sanitization, etc.)
      - Application Server Component (Apache, Varnish, et al) Configuration
      Validation


   - ITIL-Based Operational Policies and Procedures

      - Incident Response
      - Change Management

*

Job Skills & Responsibilities:
*

   - On-Call Response to Security-Related Alerts
   - Daily Review of Log Data for Security-Related Events
   - Review and Processing of Requests to Modify Firewall ACLs
   - Close Monitoring of Public (Community and Vendor) Security Advisories
   - Active Analysis/Penetration of ReachLocal’s Applications and
   Infrastructure
   - Close Cooperation with Operations and Engineering Personnel to Locate
   and Repair Security-Related Issues

* **

Qualifications:
** *

    - 3+ Years in Security-Focused Systems and Network Administration
   - Familiarity with Operation of Common Security Tools (Nessus, Snort, et
   al)
   - Familiarity with Design, Implementation, and Operation of Common
   Stateful Firewalls (ScreenOS, JunOS, et al)
   - Experience with Web Application Security Management (PHP and Java
   Preferred)


Thanks,
Anthony M. Trummer
Sr. Security Engineer
