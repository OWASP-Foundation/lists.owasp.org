Software is a crucial component of everyday life for much of the world. At
the same time, not a day goes by without news of a software breach affecting
thousands if not more. Application security is a serious problem impacting
organizations globally. In fact, over 10,000 information security
professionals ranked application vulnerabilities as the number one threat in
this year’s (ISC)2®  Global Information Security Workforce Study.

Join (ISC)2, along with OWASP and IEEE, and attend our September 29th
e-Symposium sponsored by Fortify, as we examine application security and
software assurance. Sessions will include looks at threat assessment, policy
development and educating users on the dangers of insecure software.
