Information Systems Security Association Los Angeles Chapter

February 17, 2010 Monthly Member Meeting

The ISSA LA Chapter cordially invites you to the February luncheon
where you will have the opportunity to hear from some of your
colleagues about state-of-the art security products they are using to
solve technical problems in their workplace.
The agenda includes:

1. Topic: Access Governance
    Product: Aveksa's Access Governance
    Speaker: Artyom Poghosyan, Advancive Technology Solutions Inc.
2. Topic:  Availability Monitoring
    Product: Nagios Exchange
    Speaker: Mikhael Felker, Security Engineer, Aerospace
3. Topic: Configuration Control and Audit
    Product: Enterprise Tripwire
    Speaker: Jill Williams, Security Operations Analyst, Disney
4. Topic: Endpoint Security
    Product: Safendprotector
    Speaker: Richard Greenberg, Information Security Officer
5. Topic: Web Application Security
    Product: QualysGuard
    Speaker: Clark Yogi, Sony Pictures

Wednesday, February 17th, 2010
11:30 A.M. to 1:45 P.M.
