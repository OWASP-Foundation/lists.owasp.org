<p>**OWASP APPSEC BRASIL 2010**<br>**CALL FOR PRESENTATIONS**</p>
<p>Colleagues,</p>
<p>OWASP is currently soliciting presentations for the OWASP AppSec<br>Brasil 2010 Conference that will take place at CPqD Foundation in<br>Campinas, SP, Brazil on November 16th through 19th, 2010.  There will<br>be training courses on November 16th and 17th followed by plenary<br>

sessions on the 18th and 19th with each day having one single track.</p>
<p>We are seeking people and organizations that want to present on any of<br>the following topics (in no particular order):<br> - Application Threat Modeling<br> - Business Risks with Application Security<br> - Hands-on Source Code Review<br>

 - Metrics for Application Security<br> - OWASP Tools and Projects<br> - Privacy Concerns with Applications and Data Storage<br> - Secure Coding Practices (J2EE/.NET)<br> - Starting and Managing Secure Development Lifecycle Programs<br>

 - Technology specific presentations on security such as AJAX, XML, etc<br> - Web Application Security countermeasures<br> - Web Application Security Testing<br> - Web Services-, XML- and Application Security<br> - Anything else relating to OWASP and Application Security</p>


<p>To make a submission you must fill out the form available at<br><a href="http://www.owasp.org/images/f/f7/OWASP_AppSec_Brasil_2010_CFP.rtf.zip">http://www.owasp.org/images/f/f7/OWASP_AppSec_Brasil_2010_CFP.rtf.zip</a><br>

and submit through the easychair conference interface at<br><a href="http://www.easychair.org/conferences/?conf=appsecbr2010">http://www.easychair.org/conferences/?conf=appsecbr2010</a></p>
<p>Each presenter will have 45 minutes for the presentation, followed by<br>10 minutes reserved for questions from the audience. The presentations<br>must respect the restrictions of the OWASP Speaker Agreement.</p>
<p>**Important Dates:**<br> Submission deadline is August 17, 2010 at 11:59 PM (UTC/GMT -3).<br> Notification of acceptance is September 8, 2010.<br> Presentation slides are due September 30, 2010.</p>
