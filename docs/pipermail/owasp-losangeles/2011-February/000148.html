*Redspin is hiring Web Application Penetration Testers*

Redspin (*www.redspin.com* <http://www.redspin.com/>) one of Outside
Magazine's 2010 “Best Places to Work" is hiring security engineers for web
application security assessments. We’re in Carpinteria. We like independent
thought, so we make time for independent research. We audit network
security. We like Earth, so we give to 1% For The Planet. We like life so we
have 20 days of paid time off annually.

We’re looking for someone who knows a little something about web-application
security assessments.

Send resumes and letters of interest to j <jmiller at redspin.com>
obs at redspin.com.

*
*

*Job Requirements:*

· At least one area of strong technical experience, in: web app testing,
source code review, secure web development.

· Must have operational knowledge of information technology infrastructure,
operating systems (Unix and Windows) and networking (TCP/IP, Ethernet)

· Commitment to continuous improvement of assessment processes and practices

· Those seeking contract work need not reply

*Work Environment:*

· This is a hands-on technical job that requires strong written and verbal
communication, problem solving and customer-handling skills

· Working in ambiguous and/or undefined situations

· Opportunities for independent research, tool development, and improving
our testing methodology

· Comfortable with compiling and running tools in a Unix environment

· Clients include financial services firms - a background security check and
credit checks are required*

· Office Location: Carpinteria, CA (telecommuting possible for the right
candidate)

· 10% travel

· Relaxed, start-up-like environment

*Relevant Security Experience:*

Web application testing, penetration testing, OWASP, prior scripting/coding
experience is a plus. This requires the ability to think holistically and
identify areas of risk within an IT environment.

Familiarity with HIPAA, PCI-DSS, FFIEC, and NERC compliance a plus.

*Background:*

4 Yr. technical degree preferred but not necessarily required for someone
with a strong technical background. CISSP and/or SANS certifications would
be beneficial. 1-3 years of security experience desired.

*Full-Time Benefits:*

· 20 Days Paid Time Off – Annually

· 401K / Profit Sharing / Stock Options

· Medical / Dental

· Competitive Salary

*Contact Information*

Send resumes and letters of interest to j <jmiller at redspin.com>
obs at redspin.com.

*Please note: A pre-employment background security check and credit check
will be conducted.

Redspin, Inc. is an equal opportunity employer.
