Greetings LADOTNET,

Below is the updated details of Bret's talk.

Session 1 – “Getting Started with Windows Azure”  - In this session we will
introduce the Windows Azure platform, and walk through building an
application that can be hosted on Windows Azure.  Well cover the
differences between web roles and worker roles, and introduce the various
types of storage available.



Session 2 – “Using the Windows Azure APIs”  - Developers writing
applications that run in Windows Azure have access to a great set of APIs
for accessing storage, inspecting their environment, and instrumentation.
In this session we will review the various APIs available.
