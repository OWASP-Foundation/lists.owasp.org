I recently created the Ghost Box broken web app that is included in the
OWASP bwa project. There was some problems with the app which have been
fixed. If anyone is still having problems with it, i will be adding a copy
of the updated tar to the support site at
http://ghost.webdevelopmentsolutions.org

or if you would rather, email me and ill send you the files that need to be
changed. The main problems were in the php.ini, and htaccess file. The SVN
should be up by now so you can try to update the app.
**
*Where I'm at now:*
 I am currently working on updating the content on the support site. I am
looking for some people to volunteer some time to create vids for
walkthroughs, usage of some of the tools on the list, and just plain old
good practices for penetration / application testing. If anyone is
interested please email me @ ghost at webdevelopmentsolutions.org or you can
find me at the OWASP meeting this month.
I am also interested in any suggestions for further releases of the app.
