Dear Members,

Aiming to better identify bugs that could lead to security issues,
Microsoft announced on Wednesday that it planned to release a tool to
help developers classify and assess program crashes.

The tool, known as !exploitable and pronounced "bang exploitable," is a
plugin for the Windows debugger that categorizes crash information using
two hashes, members of Microsoft's Security Science group told
SecurityFocus in a briefing. Using the hashes, which Microsoft called
major and minor hashes, the tool can group crashes that are caused by
the same bug, even if the program executed in different ways to produce
the crashes. The tool also estimates the exploitability of the
vulnerability, rating them as Exploitable, Probably Exploitable,
Possibly Exploitable, and Unknown.
