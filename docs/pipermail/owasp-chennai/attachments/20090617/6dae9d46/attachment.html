All<br><br>There are some ways to protect oneself from malicious links in websites (especially social networking sites). These are addons to Mozilla which can be used to preview links or understand more about them before actually clicking on the link. These are LinkAlert and CoolPreviews.  Link Alert is a neat little widget which provides info about the site which is behind the click, like whether it is a JS being accessed, or a regular site or a secure site. It provides info about things like a new window opening up. CoolPreviews is also very useful as it provides the preview of the site, before you can click on it. I use both of these in tandem, and I am quite sure using these 2 effectively in your daily browsing activities can prevent, to a large extent, a phishing issue, or an XSS issue, or a clickjacking issue also. <br>
