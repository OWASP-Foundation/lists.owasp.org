Here are the full details of the event.

Checkmarx (an OWASP Contributing Corporate Member), Radware, and Critical
Start are hosting an Application Security event at Joystix on August 15.
It will be a fun event with refreshments and retro pinball machines.

>From development to delivery, securing your applications is critical to
implementing a best-practice approach to cybersecurity.

Join Critical Start, Checkmarx and Radware on Tuesday, August 15 at 4:30
p.m. at JoyStix and learn how you can strengthen your security posture with
tools that:
Build secure applications during the development process
Streamline application security testing for developers
Keep web applications secure after deployment
Baseline application use and detect anomalies
Ensure optimal applications service levels across enterprise and cloud data
centers
