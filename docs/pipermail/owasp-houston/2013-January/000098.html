Greetings,

This is a reminder about our secure code review workshop that is scheduled
for 6PM this Thursday at Stag's Head. Space will be limited for this event,
so get there early. If you intend on participating, make sure your laptop
is charged.

We have decided to focus the code review session on three open source
applications. These applications are not known to have vulnerabilities.
Please download and install these before the event. If you have some
problems with install, we will have some time from 6:00PM-6:30PM to help.
Attendees are encourage to
RSVP<http://www.meetup.com/OWASP-Houston/events/95725772/>.
Applications being reviewed include...

§  phplist-2.10.19.tgz<http://sourceforge.net/projects/phplist/?source=directory>
*"phpList is the world's most popular opensource email campaign manager."*

§  xoops-2.5.5.zip <http://sourceforge.net/projects/xoops/?source=directory>
*"XOOPS is one of world's leading open source CMS and portal systems,
written in PHP for the MySQL database. Its object orientation makes it an
ideal tool for developing small or large community websites, intra company
and corporate portals, weblogs and much more."*

§  Open_Source_Point_of_Sale_2.0.zip<http://sourceforge.net/projects/opensourcepos/>
*"Open Source Point of Sale is a web based point of sale system written in
the PHP language. It uses MySQL as the data storage back-end and has a
simple user interface."*

These were the most recent versions of the application available when we
planned the session. Please download and install this version, so we are
all working with the same application. It's advised that, until we have
checked out the source code of these applications, the webserver on your VM
only listen on a loopback address.

Additionally, I would like to announce that registration for our February
Mini-Con is now open. Seating will be limited so you must sign up for a
ticket <https://owasp-feb-mini-con.eventbrite.com>!

Please join us in the Imperial Suite, located on the 24th floor of the
Westin Galleria, on Thursday, February 21st at 6PM for the first OWASP
Houston conference-style event of 2013.

The featured presentation will be delivered by Jason Chan, Cloud Security
Architect from Netflix. Jason will be joining us to discuss Netflix
approach to application security testing.

The reception starts at 6PM and presentations begin promptly at 6:30PM.
Food, beverage, and venue are provided by our sponsors, Imperva and
AlertLogic. This is a free event, but donations are accepted. Just type the
amount you'd like to donate next to your ticket on
eventbrite<https://owasp-feb-mini-con.eventbrite.com/>.
Donations will be used by the Houston to chapter to provide food, beverage,
speaker travel, and meeting space. You can also support OWASP Houston by
becoming an OWASP member. Check out the benefits of becoming an OWASP
Member to see if it is something that you're interested in.

*6PM CST Thursday February 21st at Westin Galleria, Imperial Suite, 24th
Floor
5060 W Alabama St  Houston, TX 77056
Call (713) 960-8100 if you need help with parking or directions.*
