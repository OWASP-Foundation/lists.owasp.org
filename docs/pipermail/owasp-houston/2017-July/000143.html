Greg Anderson did a great job talking about OWASP DefectDojo and how it has
drastically improved the value that he provides to his development and
management teams. There are many opportunities to contribute in various
roles helping make this project even better. Since documentation is key,
the project has placed a huge emphasis making it solid.

https://www.owasp.org/index.php/OWASP_DefectDojo_Project

Please make sure to check out his slides.

https://docs.google.com/presentation/d/1eXNnJTbIY5HCrMrYV3XbHiLVK4rzxQSJ_obHM0RnGAY/edit?usp=sharing

He also reminded us that all OWASP members should participate in the
upcoming elections. It is extremely important to actively participate in
this process. You need to be an OWASP member in order to vote. Full details
can be found below.

https://www.owasp.org/index.php/2017_Global_Board_of_Directors_Election#Election_Timeline

We also discussed possible future topics such as Python programming since
it is becoming crucial to various security tools like DefectDojo.
