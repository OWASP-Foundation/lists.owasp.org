OWASP Houston,

As we kickoff 2009, we would like to invite anyone in our OWASP  
Chapter to submit presentation topics for review.  If you are  
interested in presenting at an upcoming meeting, please submit your  
topic via email to:

	icrew_at_owasp.org

To provide an idea of what might be applicable...

   * Web Services and Application Security
   * Common Application related Threats and Risks
   * Business Risks with Application Security
   * "Cloud" Security
   * Vulnerability Research in Application Security
   * Web Application Penetration Testing
   * OWASP Tools and Projects
   * Secure Coding Practices
   * Technology specific presentations on security such as AJAX, XML,  
etc.
   * Anything else relating to OWASP and Application Security.


We are continuously seeking out fun and interesting topics (from  
vendors, consultants, etc), but we would like to encourage personal  
growth within our local organization as well.
