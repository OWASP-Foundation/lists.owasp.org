Moin Hamburg und Umgebung,

in 14 Tagen (s.o.) gibt's den nächsten OWASP-Stammtisch in Hamburg.
Diesen Monat mit Vortrag.

Es freut mich sehr, dass Tim Sattler seinen Standpunkt zum Thema
Cloud-Sicherheit erzählen wird. So wie ich rausgehört habe wird das
eine schöne bodenständige Mischung werden von Machbarkeit und Sicherheit
in ihren verschiedenen Schattierungen. Hier die Daten:

--snip

Titel: Sicherheit in der Cloud - Chancen und Risiken

Zusammenfassung: Was sind die Chancen und Risiken von Cloud Computing im
Hinblick auf die Informationssicherheit? Wie können Organisationen Cloud
Computing nutzen, ohne sich unnötigen Gefährdungen auszusetzen? Der Vortrag
setzt sich mit technischen, organisatorischen und rechtlichen Aspekten
auseinander.

Bio: Dr. Tim Sattler ist IT Security Officer bei einem Verlagshaus in
Hamburg und verfügt über eine mehr als 10-jährige Erfahrung im Bereich
Informationssicherheit.

Ort: Uni-Hamburg, RZ Schlüterstr. 70, R 304

Zeit: 19 Uhr, s.t.

--snap

Dieses Treffen findet gemeinschaftlich mit der GUUG Hamburg statt.

Bitte gerne diese Info an potentiell Interessierte weiterleiten. Thema und
hinreichendes Kriterium ist in erster Linie Web Application Security. Man
muss überhaupt nichts von OWASP wissen, ein vorhergehender Blick auf die
Website indes schadet sicher nicht. *)
