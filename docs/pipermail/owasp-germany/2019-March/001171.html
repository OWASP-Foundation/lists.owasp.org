Tut mir leid, dass Sie erneut in Ihre Mail-Liste aufgenommen werden müssen,
aber ich muss die Mitglieder der Owasp-Germany-Liste daran erinnern, dass
die OWASP Foundation alle von Mailman am 22. März 2019 auf mails.owasp.org
geführten Mail-Listen storniert zur Liste wird abgelehnt.

Hier ist ein Link mit einem Klick, um der neuen Google-Gruppe beizutreten,
die diese Liste ersetzt:
https://groups.google.com/a/owasp.org/forum/#!forum/germany-chapter/join

Sie können auch beitreten, indem Sie eine E-Mail an
germany-chapter+subscribe at owasp.org senden

Die neue Gruppe ist auch online verfügbar unter:
https://groups.google.com/a/owasp.org/forum/#!forum/germany-chapter

Ausführliche Informationen zum Beitritt zur Gruppe finden Sie unter
https://drive.google.com/open?id=1TBzgvB8Tb0aAZnEy2qYzLnrzJuRzK_T91Em09DVf5YA

HINWEIS: Sie müssen KEIN Google-Konto, eine E-Mail-Adresse von @ owasp.org
oder Ihre Telefonnummer angeben, um sich der neuen Gruppe unter
germany-chapter at owasp.org anzuschließen. Dies ist lediglich eine
E-Mail-Adresse.

Ausführliche Informationen zum Rückzug von Mailman, Links und häufig
gestellten Fragen finden Sie unter:
https://www.owasp.org/index.php/Staff-Projects/Mailman-EOL

Vielen Dank für Ihre bisherigen und zukünftigen Beiträge zu unserer
Community!
