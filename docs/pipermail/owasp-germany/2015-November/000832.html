Hallo zusammen,

dies ist ein freundlicher Reminder für den Münchner OWASP Stammtisch 
morgen (17.11.2015) um 19:00 Uhr in der Gaststätte Villa Flora.

Der letzte Münchner OWASP Stammtisch in diesem Jahr - daher sollte man 
sich diesen Vortrag nicht entgehen lassen und von Mirko Dziadzka etwas 
über den neuen HTTP Standard erfahren:

###Thema: HTTP/2 - was ist das und warum will man das [oder auch nicht]
Seit diesem Jahr ist der HTTP/2-Standard offiziell verabschiedet. Alle 
großen Browser unterstützen ihn inzwischen, die Server werden langsam 
ausgerollt.
Dieser Vortrag beleuchtet die wesentlichen Unterschiede zwischen 
HTTP/1.1 und HTTP/2, was hat man sich dabei gedacht, wo gibt es 
Vorteile, was kann schief gehen und was können wir in Zukunft erwarten.
Abgerundet wird er mit einer Liste von ein paar Tools (aus Admin Sicht, 
nicht aus Pen-Tester Sicht).
Wie immer freuen wir, der Vortragende und der Stammtisch, uns auf rege 
Diskussionen über Ecken mit potentiellen Problemen für die Security

Der Vortragende Mirko Dziadzka ist set über 20 Jahren in der IT 
unterwegs. Primär im Bereich Unix, Internet und Security; von Konzeption 
über
Implementation bis zum Betrieb; in Hochschule, Startup und 
Großunternehmen. Aktuell interessiert an Web-Application Security, 
Software Entwicklung
sowie Cloud Computing. Aktiv in OWASP und GUUG.
