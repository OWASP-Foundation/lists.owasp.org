Fadi Serhan spricht über Docker Security.

Abstract
--------
Für die Virtualisierung von Anwendungen benutzen viele Unternehmen heutzutage
Container-Technologien wie Docker. Mit diesem Tool werden Anwendungen auf schnelle und einfache
Weise verladen, verpackt und transportiert. Docker findet man heutzutage in fast allen CI/CD
Pipelines, die aufgesetzt oder betrieben werden. Aber wie sicher ist Docker? Können wir mit
unserem jetzigen Wissen und Set-Up Docker in Produktion einsetzen? Vielleicht kommen euch diese
oder ähnliche Fragen bekannt vor.

Fadi wird in diesem Vortrag auf die aktuellen Sicherheitsaspekte von Docker eingehen und ein
paar nützliche Tipps & Tricks aufzeigen, wie Benutzer Ihre Docker-Container härten können, um
Hacker Bedrohungen entgegenzuwirken.


Generelles zum OWASP-Stammtisch
-------------------------------
