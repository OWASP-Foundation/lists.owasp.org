Hallo OWASP-Freunde,
 
am Donnerstag, 18. November 2010 findet in München die WJAX statt.  Die WJAX ist 
eine große Entwickler Konferenz (normalerweise mehr als 1000 Teilnehmern), die 
auch Spezielle „Tage“ anbietet.  Letztes Jahr habe ich ehrenamtlich einen „Web 
Application Security Day“ organisiert.  Die Veranstaltung war ein richtiger 
Erfolg (Standing Room Only für viele Vorträge), und ich möchte dieses Jahr den  
„Web Application Security Day“ weiter ausbauen.  Daher…
 
Wenn Sie eine Präsentation halten möchten, dann bitte in der nächsten 10  Tage 
kurz melden, so dass einen Platz gesichert werden kann. Wir haben nur 5 Slots, 
so bitte ein kurzes Abstract liefern.  Das Abstract muss nicht detailliert 
sein.  


 
Mögliche Themen sind alles was Web App Sicherheit betrifft, z.B.:
 
·         Sichere Entwicklungs-Frameworks und Best Practices
·         Secure Development Lifecycle
·         Code Sicherheit
·         Sicheres Single Sign On
·         Sichere Architektur und Design von Programmen
·         Verteidigungsstrategien gegen Hackerangriffe
·         uvm…
 
Die WJAX ist eine sehr professionelle und angesehene Konferenz und daher eine 
gute Möglichkeit, Ideen einem breiten Zuschauerkreis zu präsentieren.
 
Hier die Daten: 
 
Konferenz:  Donnerstag, 18. November 2010 in München
Website: http://jax.de/
Ihre Vorschläge bis: 23.07.2010
 
PS: Wenn Sie jemanden kennen, der einen guten Vortrag halten kann, ist aber 
nicht an die OWASP-Liste, dann bitte diese Mail weiterleiten!
 
Viele Grüße,
 
Bruce Sams
