### Einladung zum 53. Münchner OWASP Stammtisch am 16.09.2014 um 19:00 Uhr im "Hackerhaus"

### Thema: "XSS und andere Sicherheitslücken aus der Perspektive des Programmcodes" von Thomas Schön

"Wie sieht eigentlich der Programmcode auf der Serverseite aus, wenn Security Schwachstellen auftreten? Und mit welchen Verfahren kann man solche Lücken im Code finden? Der Vortrag diskutiert diese Fragen anhand von XSS und weiteren Sicherheitslücken. Schließlich soll der Vortrag in eine allgemeine Diskussion münden, wie solche Lücken am besten im Code behoben werden."
