zum Start ins Wochenende ein kurzer Zwischenstand unsererseits.

Die 1. deutsche OWASP Konferenz ist tatsächlich auf dem besten Wege zum
Branchentreffpunkt für Web Application Security in Deutschland.

Inklusive Vortragenden und Sponsoren-Vertretern sind wir nun schon ca.
90 Teilnehmer.

die magische Schwelle von "100 Teilnehmern" packen ;-))
