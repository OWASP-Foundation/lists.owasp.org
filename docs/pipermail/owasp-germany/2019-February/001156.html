Hallo zusammen,

das Programm für den OAuth Security Workshop wurde heute veröffentlicht:
https://sec.uni-stuttgart.de/events/osw2019/agenda

Themen sind unter anderem die Abschaffung (Deprecation) des Implicit
Flows, neue Sicherheitsempfehlungen für OAuth (OAuth Security BCP),
OAuth in Single Page Applications und Ergebnisse aktueller
Sicherheitsanalysen von OAuth-Flows.

Eine vergünstigte Early-Bird-Anmeldung ist bis zum 5. März möglich.

Viele Grüße
Daniel

>
> The aim of the OAuth Security Workshop (OSW) is to improve the
> security of OAuth and related Internet protocols by a direct exchange
> of views between academic researchers, IETF OAuth Working Group
> members, and industry experts.
>
> Call for Sessions
>
> We seek proposals for sessions (talks, interactive discussions, …) and
> tutorials related to OAuth, OpenID Connect, and other technologies
> using OAuth under the hood. Contributions regarding technologies that
> are used in OAuth, such as JOSE, or impact the security of OAuth, such
> as web technology, are also welcome.

>
