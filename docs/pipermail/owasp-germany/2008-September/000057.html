Hallo Martin,

das ist eine funktionale Adresse, um Dinge, die man sonst an eine Person
schicken würde, statt an die Liste, zu 'virtualisieren'. Die Entkopplung von
einer persönlichen Emailadresse ist sinnvoll, da die personelle Zuordnung,
was Organistorisches betrifft, im Fluss ist. Die Adresse ist auf einen
Gmail-Account abgebildet (von der OWASP so eingerichtet), darüber können
Forwards eingerichtet werden. Z.Zt. stehe ich als Einziger dort drin. Sie
ist nicht zum Versenden von Emails gedacht.
