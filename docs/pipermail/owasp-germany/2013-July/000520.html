### Einladung zum 42. Münchner OWASP Stammtisches am 16.07.2013 um 19:00 Uhr im "Alten Hackerhaus"

### Thema "Taint Analyse - eine Einführung" von Thomas Schoen:
"Auch mit einer gezielten Analyse ihres Programmcodes kann man Security-Schwachstellen in einer Applikation auffinden. Das zur Zeit etablierte Verfahren, 'Taint Analyse', sucht im Programmcode nach möglichen Datenflüssen, mit denen 'Tainted Data' zu kritischen Stellen innerhalb der Applikation transportiert werden könnten. Dieser Vortrag liefert eine Einführung zu diesem Verfahren, schildert damit verbundene technische Herausforderungen und deren Grenzen und erläutert dies anhand von Beispielen."
