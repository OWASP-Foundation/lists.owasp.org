<p>The next chapter meeting will be hosted at CAST in Darmstadt.<br>
This time the focus is on technical presentations and discussion. <br>
<br>
<b>Agenda</b> <br>
Technical presentation slots will consist of 20-30 minute presentation
and 15 minute discussion. </p>
<ul>
  <li> 1. (11:00 - 11:15) Welcome, Introduction and Administrativia </li>
  <li> 2. (11:15 - 11:30) Vorstellung von CAST (Dr. Heinemann)
  </li>
  <li> 3. (11:30 - 11:45) Short OWASP organisation introduction and
update (Tobias Gondrom)
  </li>
  <li> 4. (11:45 - 12:30) First technical presentation "Best
Practices beim Einsatz einer Web Application Firewall 1.0" (Alexander
Meisel) </li>
  <li> 5. (12:30 - 13:15) Break
  </li>
  <li> 6. (13:15 - 14:00) Second technical presentation "Defending
against Web Application DoS Attacks" (Maximilian Dermann)
  </li>
  <li> 7. (14:00 - 14:45) 20-Minutes Talks (15 Min Presentation + 5-10
Min Discuss)
    <ul>
      <li> "Input validation in ASP.NET -- tips, tricks &amp; pitfalls"
(Boris Hemkemeier)
      </li>
      <li> "Managing of extremely large Web Application Firewall
Installations" (Alexander Meisel)
      </li>
    </ul>
  </li>
  <li> 8. (14:45 - 15:00) Coffee Break
  </li>
  <li> 9. (15:00 - 15:45) Fourth technical presentation "Secure Coding
and Development Guidelines (part of CLASP)" (Tobias)
  </li>
  <li> 10. (15:45 - 16:00) Wrap-up and outlook </li>
</ul>
