/<span><font face="Arial" size="2"><a href="http://www.owasp.org/index.php/Category:OWASP_Guide_Project" target="_blank">OWASP 
Development Guide Project</a></font></span>/ -- After many months of 
planning and preparation, the OWASP Development Guide project announced 
today that it is ready to begin work on the next revision of the Guide, and 
that that the project is looking for volunteers to do the work, both individuals 
and organizations. </font></span></div>
<div><span><font face="Arial" size="2"></font></span> </div>
<div><span><span><font size="2"><font face="Arial">The OWASP Development Guide is aimed at architects, 
developers, consultants and auditors and is a comprehensive manual for 
designing, developing and deploying secure web applications. The original OWASP 
Development Guide has become a staple diet for many web security professionals. 
Since 2002, the initial version was downloaded over 2 million times. Today, the 
Development Guide is referenced by many leading government, financial, and 
corporate standards and is the Gold standard for Web Application and Web Service 
security.</font></font></span></span></div>
<div><span><span><font face="Arial" size="2"></font></span></span> </div>
<div><span><span><font face="Arial" size="2">The next version of the OWASP Development Guide will be in effect the 
detailed design guide for the requirements of the OWASP Application Security 
Verification Standard (ASVS), which can be found here: </font><a href="http://www.owasp.org/index.php/ASVS" target="_blank"><font face="Arial" size="2">http://www.owasp.org/index.php/ASVS</font></a><font face="Arial"><font size="2">. Key features of the next Guide will include use of the new OWASP common 
numbering scheme. The new numbering scheme will be common across OWASP 
Guides and References, more information can be found here:<span> </span></font></font><a href="http://www.owasp.org/index.php/Common_OWASP_Numbering" target="_blank"><font face="Arial" size="2">http://www.owasp.org/index.php/Common_OWASP_Numbering</font></a><font face="Arial"><font size="2">. Additional key features will be the inclusion of 
worksheets and checklists<span>, such as the sample 
input validation worksheet which can be found here: <a href="http://code.google.com/p/owasp-development-guide/wiki/WebAppSecDesignGuide_D5_2_1_1" target="_blank">http://code.google.com/p/owasp-development-guide/wiki/WebAppSecDesignGuide_D5_2_1_1</a></span></font></font></span></span></div>
