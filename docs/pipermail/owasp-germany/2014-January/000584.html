*Heimvorteil – Software-Qualität aus Deutschland*

*Karlsruher Entwicklertag 2014*



*Reminder : Call for Contributions  *





Für Sie noch einmal ein Reminder zum CFC für den Karlsruher Entwicklertag
2014 (21.-23.05.2014). Sie haben noch bis *Sonntag, 12. Januar 2014* die
Gelegenheit einen Beitrag einzureichen.



Neben dem Themenschwerpunkt *Qualitäts-Engineering* freuen wir uns über

Einreichungen zu den Themen:

- Application Security

- Mobile/Apps/Cloud

- Usability

- Continuous Delivery



Außerdem sind Einreichungen zu Spezialthemen von lokalen Initiativen
herzlich

willkommen:

- KA-IT-Si/OWASP: Security

- VKSI SIG: Enterprise Agility

- Dot.net User Group: .NET

- Java User Group: Java

- Softwerkskammer: Craftsmanship

- Google Developer Group: Android, Mobile



Konzeptionelle Beiträge sind ebenso willkommen wie die Vorstellung
innovativer

Produktideen sowie insbesondere Erfahrungsberichte aus der Praxis.



Bitte reichen Sie folgende Formate ein:

- Vorträge (45 min)

- interaktive Sessions wie Dojos, Fishbowl, Live Coding Sessions etc.

- Pecha Kuchas

- Hands-On Sessions

- Tutorials (halbtägig- oder ganztägig)



Wir sind aber auch aufgeschlossen für weitere Formate, die hier nicht
aufgezählt sind.

Ihre Einreichung sollte beinhalten:

- Abstract (ca. ½ -1 DIN A4 Seite)

- Kurzbio des Referenten

- Referentenfoto



Außerdem benötigen wir für Ihren Vortrag eine Einstufung der
Teilnehmer-Kenntnisse in

„basic“ oder „advanced“.



Bitte geben Sie auch an, wenn Sie Ihr Thema in verschiedenen Zeitformaten
halten

können (z.B. als klassischer Vortrag oder als Pecha Kucha). Dies erhöht die
Chancen Ihrer

Einreichung.

Beachten Sie: Bitte reichen Sie keine „Marketingvorträge“ ein, da diese in
der

Vergangenheit von den Teilnehmer vergleichsweise negativ beurteilt wurden.





Das Programmkomitee besteht aus:



Dr. Susanne Kaufmann (Regionalgruppe Karlsruhe der Gesellschaft für
Informatik, GI)

Dr. Klaus Krogmann (Verein der Karlsruher Software Ingenieure, VKSI)

Kai Jendrian (OWASP; Secorvo Security Consulting GmbH)

Jan Baumann (andrena objects ag)

Jochen Winzen (andrena objects ag)



Bitte laden Sie Ihre Einreichung auf folgender Webseite hoch:

http://www.entwicklertag.de/karlsruhe/2014/

Dort finden Sie weitere Informationen und Hinweise für Referenten.



*Einreichungsschluss ist Sonntag, der 12. Januar 2014*



Ca. einen Monat später können Sie mit einer Zu- oder Absage Ihrer
Einreichung rechnen.
