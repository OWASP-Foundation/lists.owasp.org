Hallo zusammen,

dies ist ein freundlicher Reminder für den Münchner OWASP Stammtisch morgen (19.05.2015) um 19:00 Uhr in der Gaststätte Villa Flora.

Diesmal werden wir einen Vortrag über ein hochaktuelles Forschungsprojekt hören:
###*Thema:**  An Ontology for Secure Web Applications - Wie hängen Sicherheitseigenschaften, Schwachstellen, Bedrohungen, Tools und Methoden zusammen? (Marianne Busch  <http://www.pst.ifi.lmu.de/Personen/team/busch>, LMU München)*
Es ist bekannt, dass viele Anwendungen Sicherheitslücken haben, die früher oder später ausgenutzt werden. Ein Grund dafür ist, dass "Sicherheit" für Entwickler schwer greifbar ist. An der Ludwig-Maximilians-Universität München wurde ein Modell namens SecEval entwickelt, das unter anderem die Zusammenhänge von Sicherheitseigenschaften, Schwachstellen, Bedrohungen, Tools und Methoden (sogenannte Knowledge Objects) grafisch darstellt. Diese Struktur kann für die Evaluation von Tools oder als Grundlage einer Knowledge Base genutzt werden.
Darauf aufbauend wird SecWAO vorgestellt, eine Ontologie für die Entwicklung sicherer Webanwendungen. Eine Ontologie ist in dem Fall eine Art "Wissenslandkarte" und gibt einen Überblick über die Beziehungen zwischen Knowledge Objects, wie z.B. control flow integrity, data integrity, cross-site scripting oder injection prevention.
Der Vortrag basiert auf M. Busch and M. Wirsing: An Ontology for Secure Web Applications. In: International Journal of Software and Informatics, 2015 (accepted)
