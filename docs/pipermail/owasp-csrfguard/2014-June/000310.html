I submitted a pull request for the creation of EL functions to mimic the custom tags.  There are functions for the token, token name, token value, token with URI, and token value with URI.  I also made utility methods in the custom tags so these two features can share the logic.  Finally, I noted that the token name and value tags had a name change.  I duplicated these in the taglib with the old name so that the old name can still be used for backward compatibility.

https://github.com/mchyzer/OWASP-CSRFGuard/commit/9cf68563390ea11aa1bf694f34ea5d9c77b05270

This is useful because sometimes you want to use the token name or value in a custom tag, or maybe you just like EL.  Here is an example:

      <html:link page="/deleteGroup.do?${csrf:token()}">
          Delete
      </html:link>
