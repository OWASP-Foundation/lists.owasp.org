<p><b><span lang="EN">Title:</span></b><span lang="EN"> Man in the Middle Attacks:
SSL Spoofing</span></p>

<p><span lang="EN">Duane Peifer, UberGuard Information Security</span></p>

<p><span lang="EN">Ralph Durkee, Durkee Consulting </span></p>

<p><b><span lang="EN">Abstract:</span></b><span lang="EN"> </span></p>

<p><span lang="EN">Duane Peifer will talk about SSL spoofing, or SSL
man-in-the-middle attacks, which are a very effective for obtaining sensitive
information such as accounts and passwords for web applications. SSL spoofing
is so effective because it exploits services that people assume to be secure.
The discussion will start with an explanation of how SSL connections work to
provide authentication and privacy, then the steps to exploit an SSL connection
will be explained, along with with SSL spoofing counter measures. Duane will
then demonstrate a couple of popular tools and techniques for these attacks
including SSLStrip. Ralph Durkee will follow with a brief presentation and a
demonstration using dsniff for SSL man-in-the-middle attacks. After these
demonstrations, using you web email and on-line banking just won&#39;t be the same.
</span></p>
