The Summit Schedule is now on-line including this years OWASP track.   
https://www.rochestersecurity.org
Also, We have added 4 In-Depth Training sessions to this year’s October
7-8 event.
 
Dennis Allen and others from CERT will conduct a hands-on session titled
“Hands-on Threat Detection and Analysis”.This session will be offered twice.
Also, we have software security training with Jim Manico. Jim is an
author and educator of developer security awareness training and has a
17 year history building software as a developer and architect.
 
Jim will host 2 different sessions:
 
The first session covers: HTTP Basics and Introduction, Cross-site
Scripting and HTML Hacking, Cross Site Request Forgery, Clickjacking,
SQL and other Injection
 
The second covers: HTML5 Security Considerations, App Layer Intrusion
Detection, Mobile Security
Webservice Security, Applied SSL
 
Detailed descriptions of these sessions are available at:
https://www.rochestersecurity.org/schedule/in-depth-training/
 
Due to limited seating for these in-depth training sessions, session
registration is requires in addition to conference registration.
Registration is free for paid attendees of the summit. Register for
these sessions as well as the summit at:
https://www.rochestersecurity.org/schedule/in-depth-training/
