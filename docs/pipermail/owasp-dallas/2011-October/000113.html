If you're qualified at interested, feel free to email me directly or email careers at praetorianlabs.com

Principal Software Security Consultant


Overview: Praetorian is seeking highly motivated individuals to join its team of elite software security consultants. We are seeking the top 5% of the industry to join our ranks. As a Principal Software Security Consultant, you will be responsible for management and execution of client-facing engagements that include, but are not limited to: application penetration tests, source code reviews, threat models, software security requirement reviews, and Praetorian’s strategic consulting offerings.

Qualified applicants must embody the following traits: an aptitude and willingness to learn new technologies; an ability to work well with others as part of a whole; proficient communication skills when demonstrating  technical findings; and a willingness to share knowledge with the practice and the community.

Location: Austin, TX

Qualifications: 
Successful applicants should have:
3-5 years of information security consulting experience.
Strong understanding of software and application security.
Experience with languages such as C, C++, C# / ASP.NET, and Java.
Strong oral/presentation and written communication skills.
Attention to detail and strong organization skills.
The ability to self-start, self-motivate, and self-manage.


Preference will be given to candidates with the following:
Background in web application penetration testing, web services penetration testing, and automation development.
Background in mobile application security, mobile platform security, or mobile vulnerability research.
Background in embedded system security or distributed/cloud systems security.
Background in reverse engineering, binary analysis, and vulnerability research.
Involvement in software community via oraganizations such as OWASP, WASC, and/or open source development.
Track record speaking at major security conferences such as OWASP Appsec, SANS Appsec, and Blackhat.
Ability to travel 25 to 30% of the time.
Mimimum 4-Year Bachelor of Science Degree in Computer Science, Engineering, or equivalent from a "top ten" institution.  Masters degree preferred.
