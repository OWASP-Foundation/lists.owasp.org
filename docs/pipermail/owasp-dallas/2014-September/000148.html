You are invited to the OWASP Dallas Chapter meeting.

Enterprise software security has been a hot topic for over a decade, yet enterprises of all sizes continue to fail at explaining measurable gains, but why? The answer lies in what and how is measured. Success begins with realistic, business-aligned KPIs which can be effectively measured to demonstrate improvement, or lack thereof. Come learn what, and how to effectively measure software security.

When: Wednesday, October 1, 2014 from 11:30 AM to 1:00 PM
Where: Richland College (12800 Abrams Road, Dallas, TX 75243) Room: SH117 (Sabine Hall)
Presentation Title: Succeeding with Enterprise Software Security Key Performance Indicators
Presenter: Rafal Los
Cost: As always Free; Bring you own lunch;

Please RSVP so we can ensure we have enough seats for everyone.
