Señores

Se propone este nuevo objetivo. Esperamos su participación y trabajo ens
como equipo.

Durante LatamTour vimos esta oportunidad: "traducción al español" del
material que falta para completar la colección
de metodologías OWASP, ya traducidas en el proyecto
OWASP_Spanish<https://www.owasp.org/index.php/OWASP_Spanish>


*_**Application Security Verification Standard(**ASVS), [faltan app.15 ó 20
pags]  link al proyecto
owasp<https://www.owasp.org/index.php/Category:OWASP_Application_Security_Verification_Standard_Project>

_La guia de revisión de código [233 pags.]*   *link al proyecto
owasp<https://www.owasp.org/index.php/Category:OWASP_Code_Review_Project>
*


Les pido a los interesados anotarse vía mail, para armar el equipo de
traductores+revisores y establecer el metodo de trabajo.

**Nuestro incentivo... ;-)
las reuniones periodicas de coordinacion seran de happy hour para degustar
cerveza artesanal en algunos locales de tradición cervecera.   ( nada de
cafe y galletas... ya somos  rrrandes )

...la  expectativa es poder concluir la traducción antes de fin de año y
hacer un evento para celebrarlo.

              ....ya estamos buscando patrocinador oficial para esta
actividad  y para el happy-hour!!
