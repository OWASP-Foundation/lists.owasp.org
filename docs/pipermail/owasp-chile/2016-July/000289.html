*TEMARIO*

*ASVS: Application Security Verification Standard*

Un estándar de OWASP, que documenta y organiza un checklist de 179
verificaciones de seguridad.  “Requisitos a cumplir”, para medir y
certificar el nivel de seguridad de una aplicación.


*TOP10*

Un listado de los 10 riesgos de seguridad web más criticos
Informa su descripcion, ejemplos de  vulnerabilidades y de ataques, con las
indicaciones para evitarlos.
