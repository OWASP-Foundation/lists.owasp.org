Hi everyone,

$30,000 would equal 600 new memberships for OWASP.  Whatever project we did with that money would need to provide something that they felt had similar value to them as an organization.  If someone would like to volunteer to write up a proposal with a plan the committee officers can review, assuming we approve of the plan and proposal, that volunteer can run with it.  

In fact, if anyone would like to run a WIA project, such as planning an event locally at your chapter, please tell us about it so we can help you promote it and you can put WIA all over it.  

To try to help set future expectations: The WIA committee officers will not be offering to run events, projects or other initiatives for other groups, if you suggest something for WIA, it is highly likely we will help you find volunteers or promote your event, but it is highly unlikely that we will offer to run it for you.  

If someone would like to volunteer to lead this please let me know.

Tanya
