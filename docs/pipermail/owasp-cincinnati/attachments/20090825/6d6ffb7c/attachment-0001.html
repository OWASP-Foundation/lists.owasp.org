New vulnerability was discovered a couple of weeks ago that can allow a local user (from telnet, ssh, etc…) to execute code in user-space, and gain full root access to the system. Linux systems with kernels up to  2.6.30.2  are vulnerable<br>
<br>Patch your systems. Practically every distribution has released a fix by now.<br><br>Overview from SANS.org<br><a href="http://isc.sans.org/diary.html?storyid=6964">http://isc.sans.org/diary.html?storyid=6964</a><br><br>
