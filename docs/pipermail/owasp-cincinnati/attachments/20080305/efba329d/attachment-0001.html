<ul style='margin-top:0in' type=disc>
 <li class=MsoNormal style='mso-list:l4 level1 lfo1'><font size=2 face=Arial><span
     style='font-size:10.0pt;font-family:Arial'><a

     title="https://www.owasp.org/index.php/OWASP_Summer_of_Code_2008">The
     OWASP Summer of Code 2008 (SoC 2008)</a> aims to financially sponsor
     contributions to OWASP Projects. SoC 2008 follows the previous <a
     href="http://www.owasp.org/index.php/OWASP_Spring_Of_Code_2007"
     title="http://www.owasp.org/index.php/OWASP_Spring_Of_Code_2007">OWASP
     Spring of Code 2007</a>, in which 21 projects were sponsored with a budget
     of $117,500, and the <a
     href="https://www.owasp.org/index.php/Owasp_Autumn_Of_Code_2006"
     title="https://www.owasp.org/index.php/Owasp_Autumn_Of_Code_2006">OWASP
     Autumn of Code 2006</a>, in which 9 projects were sponsored with a budget
     of $20,000. <o:p></o:p></span></font></li>
</ul>

<ul style='margin-top:0in' type=disc>
 <li class=MsoNormal style='mso-list:l2 level1 lfo2'><font size=2 face=Arial><span
     style='font-size:10.0pt;font-family:Arial'>The SoC 2008 is an open
     sponsorship program were participants/developers are paid to work on OWASP
     (and web security) related projects. <o:p></o:p></span></font></li>
</ul>

<ul style='margin-top:0in' type=disc>
 <li class=MsoNormal style='mso-list:l3 level1 lfo3'><font size=2 face=Arial><span
     style='font-size:10.0pt;font-family:Arial'>The SoC 2008 is also an
     opportunity for external individual or company sponsors to challenge the
     participants/developers to work in areas in which they are willing to
     invest additional funding, provided that these areas are relevant and
     beneficial to the OWASP community. <o:p></o:p></span></font></li>
</ul>

<ul style='margin-top:0in' type=disc>
 <li class=MsoNormal style='mso-list:l7 level1 lfo4'><font size=2 face=Arial><span
     style='font-size:10.0pt;font-family:Arial'>The initial Budget for SoC 2008
     will be $100,000 and it is funded by OWASP using current membership fees
     and profits from past conferences. In parallel with the Request for
     Proposals, OWASP would like to invite individuals and companies that
     benefit from OWASP projects to join OWASP as a member. In addition to the
     current Membership benefits, the new OWASP members will be able to
     allocate membership fees to SoC 2008 projects they are interested in. <o:p></o:p></span></font></li>
</ul>
