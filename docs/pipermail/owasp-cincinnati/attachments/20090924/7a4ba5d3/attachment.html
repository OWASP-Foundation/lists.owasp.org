Event Abstract:<br>
The IMI Security Symposium is a multi-track conference focused on information
security in several technology sectors.  This year’s conference will cover
key findings from “The State of IT Security” study, completed by the College of
Informatics in 2009.  This nation-wide study provides a snapshot of IT
security practices today.  Attendees will have the opportunity to
participate in a variety of sessions on topics such as virtualization &amp;
security, security implications for networking, security in cloud computing,
open source security, and many more.  Tracks for this year’s symposium
include: Application Software Security, Infrastructure Protection and
Information Security Governance &amp; Compliance.</span></i><span style="font-size: 10pt; font-family: &quot;Arial&quot;,&quot;sans-serif&quot;; color: navy;"></span></p>
