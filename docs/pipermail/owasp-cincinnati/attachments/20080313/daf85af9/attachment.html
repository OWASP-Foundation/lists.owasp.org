<p><font size=2 face=Georgia><span lang=EN style='font-size:10.0pt;font-family:
Georgia'>Static analysis is the process of analyzing software for security
vulnerabilities. Static analysis can be a costly and time consuming process,
but is a link in the chain for producing secure software. Join us as we
explorer building a business case for static analysis and review the current
open source static analysis tools. <o:p></o:p></span></font></p>

<p><b><font size=2 face=Georgia><span lang=EN style='font-size:10.0pt;
font-family:Georgia;font-weight:bold'>An Introduction to Web Proxies</span></font></b><font
size=2 face=Georgia><span lang=EN style='font-size:10.0pt;font-family:Georgia'>
<o:p></o:p></span></font></p>

<p><b><font size=2 face=Georgia><span lang=EN style='font-size:10.0pt;
font-family:Georgia;font-weight:bold'>Presented by</span></font></b><font
size=2 face=Georgia><span lang=EN style='font-size:10.0pt;font-family:Georgia'>:<st1:City
w:st="on"><st1:place w:st="on">Blaine</st1:place></st1:City> Wilson, Technology
Information Security Officer, Citigroup <o:p></o:p></span></font></p>

<p><font size=2 face=Georgia><span lang=EN style='font-size:10.0pt;font-family:
Georgia'>Web proxies will be explained and the group will be shown how to
install and configure WebScarab. WebScarab operates as an intercepting proxy,
allowing the operator to review and modify requests created by the browser
before they are sent to the server, and to review and modify responses returned
from the server before they are received by the browser. The presentation will
include several examples of intercepting, reviewing and modifying HTTP requests
and responses. <o:p></o:p></span></font></p>
