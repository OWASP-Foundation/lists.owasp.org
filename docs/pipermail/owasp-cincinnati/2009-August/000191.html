New vulnerability was discovered a couple of weeks ago that can allow a
local user (from telnet, ssh, etc…) to execute code in user-space, and gain
full root access to the system. Linux systems with kernels up to  2.6.30.2
are vulnerable

Patch your systems. Practically every distribution has released a fix by
now.

Overview from SANS.org
http://isc.sans.org/diary.html?storyid=6964

Full Details
http://archives.neohapsis.com/archives/fulldisclosure/2009-08/0174.html

I am sure most people are already aware, but wanted to point out that this
vulnerability can affect web applications too in conjunction with a local
execution vulnerability.
