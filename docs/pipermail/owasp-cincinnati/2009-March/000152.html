I found the updated OWASP Live CD that was released in February to be a very
useful resource for testing.  It comes with all three major proxies (Burp,
Paros, and WebScarab), a variety of vulnerability scanners, and Firefox with
all of the web application security add-ons installed, including FoxyProxy
so you can switch which proxy application you're using with a single click.

It's great because it lets you use all of these applications without having
to install anything.  It runs fast under a virtual machine, so you don't
even need to reboot your computer to use the live CD.  It can be downloaded
from the following URL:

http://mtesauro.com/files/owasp-livecd-AustinTerrier-Feb2009.iso
