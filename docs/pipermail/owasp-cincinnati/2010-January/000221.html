Looks like a great year Marco, thanks for organizing it.

Suggestion:

On the owasp home page there is an embedded google calendar with the
Cincinnati events in it. The event times listed on this calendar are
different from the eventbright invite that was distributed.

It looks like this is a single calendar. If there was a Cincinnati
calendar then that could be shared? It would be much easier for us to
add a calendar than copy/past the events from the wiki plus it makes
it auto updated if dates/times change. The main owasp embedded
calendar can include events from multiple calendars so it would still
work the same way.

Volunteers:

I will be continuing to push the word out about the OWASP meetings but
if you want to get some volunteers together to organize a coordinated
strike I would love to be involved.

You mentioned local tech-groups. I think this is the most effective
avenue for getting the OWASP word to developers. Last year the group
touched on the idea of an "OWASP Outreach" program that might ensure
that the user groups both know about OWASP and can include security
bulletins and presentations into their meetings. Perhaps this is
another topic for your volunteer group to come up with a plan for.

One of the big things that happened last year was the creation of the
TechLife Cincinnati Meet-up group. OWASP should be involved with this
group since in attracts all the leaders of all the user groups in
town. Even advertising the events on the group site may attract
developers.

    http://www.meetup.com/TechLife-Cincinnati/
