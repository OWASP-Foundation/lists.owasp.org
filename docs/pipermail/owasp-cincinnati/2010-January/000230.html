May be most of you notice the news about the Chinese attack on Google.
Apparently this attack was taking advantage of an IE6 un-disclosed exploit
(but known to MS vulnerabilities) on desktop running XP. When the exploit
was released and the metasploit attack method, the risk was rated critical
since allowed for a desktop running XP to be compromised.
http://www.foxnews.com/scitech/2010/01/18/google-exploit-leaked-internet-sec
urity-experts-urge-vigilance/

 

The risk prompted both German and France government CERTS to issue an alert
not to use IE at all
https://www.bsi.bund.de/cln_183/ContentBSI/presse/Pressemitteilungen/Sicherh
eitsluecke_IE_150110.html

with translation here:
http://mashable.com/2010/01/15/german-government-stop-using-internet-explore
r/

 

MS released an Advisory first and a special out of band patch last Thursday.

You can look at Microsoft Security Bulletin MS10-002 - Critical also 

http://www.microsoft.com/technet/security/advisory/979352.mspxhttp://blogs.t
echnet.com/srd/

http://www.microsoft.com/technet/security/bulletin/ms10-002.mspx

 

The patch can be automatically be downloaded with Windows update

http://www.update.microsoft.com

 

In my opinion, the best practice for the future would probably  use Firefox
for everyday browsing or upgrade to IE8/Vista/WIN7

 

Continue to use IE 6/on XP-2000 only if you have to browse old internal web
sites that does not support new browsers.
